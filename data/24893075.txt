They do a git clean if needed followed a git fetch &#x2F; git pull equivalent.<p>GoCD for example maintains a single copy of the repo on the server for every pipeline that refers to it and the agents have the repos that they work on checked out. Any local changes or untracked files are by default cleaned. There are settings to force reclone etc, but it&#x27;s not the default.')
