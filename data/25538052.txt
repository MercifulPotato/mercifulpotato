Item(by='segfaultbuserr', descendants=None, kids=[25542429], score=None, time=1608920446, title=None, item_type='comment', url=None, parent=25534258, text='For day-to-day development, WSL is definitely better.<p>But I think MSYS2 [0] still deserves a mention in the age of Windows Subsystem for Linux. It&#x27;s a complete mingw-w64 distribution with a friendly package manager (ArchLinux&#x27;s pacman). If you suddenly need to use some CLI tools on a Windows machine, MSYS2 can be quicker - no need to enable virtualization or upgrade Windows 10, just download a 91 MB installer and you can start right away. I recently needed to use wget on Windows to solve a problem for someone, instead of MSYS2, I made the mistake of downloading a random Windows build at SourceForge, later I was hit by a bug...<p>[0] <a href="https:&#x2F;&#x2F;www.msys2.org&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.msys2.org&#x2F;</a>')
