Item(by='vbezhenar', descendants=None, kids=[24862672], score=None, time=1603389820, title=None, item_type='comment', url=None, parent=24859732, text='One reason to avoid bare metal is drivers. There is a lot of hardware around. New hardware is released every day. And you need to write and maintain drivers for that hardware. That&#x27;s a tremendous effort and hardware manufacturers usually write drivers for a very few operating systems and Go unikernel certainly won&#x27;t be one of them.<p>Using virtualization provides a simple common layer, so host operating system can deal with drivers and guest operating system can deal with more interesting things.')
