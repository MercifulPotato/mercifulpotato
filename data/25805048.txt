Item(by='vaduz', descendants=None, kids=None, score=None, time=1610827784, title=None, item_type='comment', url=None, parent=25804954, text='Dropbox DL and Preview urls take a form of <a href="https:&#x2F;&#x2F;uc[26" rel="nofollow">https:&#x2F;&#x2F;uc[26</a> character hex string].dl.dropboxusercontent.com&#x2F;... and <a href="https:&#x2F;&#x2F;uc[26" rel="nofollow">https:&#x2F;&#x2F;uc[26</a> character hex string].preview.dropboxusercontent.com&#x2F;... - it does not have to be a separate TLD to avoid being blocked, but it has to be differentiated.<p>This is the same reason why the block of the TFA company did not cause an outage of everyone using CloudFront - GSB does not block full TLDs if it can be shown content is distinct. Same for anyone using S3, Azure equivalents and so on.')
