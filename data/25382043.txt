Item(by='erulabs', descendants=None, kids=[25385324, 25385914, 25382114], score=None, time=1607658383, title=None, item_type='comment', url=None, parent=25381983, text='The M1 does one thing: run MacOS and MacOS apps. They can control the vast majority of the compiled code that will be run on the chip - unlike an x86 platform where the exact same architecture is used for desktops, servers and everything in between - including Linux, windows, and Mac.<p>Specifically there is a reference counting optimization on the M1 that dramatically helps performance of compiled Swift apps - something only worthwhile if you know the majority of what the chip will ever do is run swift apps.')
