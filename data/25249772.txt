Item(by='Jenda_', descendants=None, kids=None, score=None, time=1606698673, title=None, item_type='comment', url=None, parent=25243549, text='By the way there is a tool called &quot;progress&quot; which will scan all running processes of (supported) tools like gzip, cat, grep etc., and report progress of their file operations.<p>If you want to emulate this by hand, first get the fd number of the file of interest by `ls -l &#x2F;proc&#x2F;PID&#x2F;fd&#x2F;` and then `cat &#x2F;proc&#x2F;PID&#x2F;fdinfo&#x2F;NUMBER`. There is a line called &quot;pos&quot;, which is the position in the file.')
