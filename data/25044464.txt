Item(by='blub', descendants=None, kids=[25045439], score=None, time=1605004523, title=None, item_type='comment', url=None, parent=25044094, text='Documentation is written in a human language, can include other media like graphics and is addressed to humans. We write documentation for the same reason that we don&#x27;t produce maps at a 1:1 scale: documentation offers a higher-level description that can be grasped without having to understand all the code.<p>Not only is code not the best documentation, it&#x27;s not documentation at all, outside of special cases where the entire code describes e.g. an algorithm and is encapsulated into one function.')
