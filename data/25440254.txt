Item(by='Twisol', descendants=None, kids=[25440762], score=None, time=1608107038, title=None, item_type='comment', url=None, parent=25415896, text='I&#x27;m probably a bit of an outlier, but I do find it more readable to use type parameters instead of concrete types. If the type is opaque, I know the only ways to get or use values of that type are whatever explicit means the specifier of that type provides.<p>This is less valuable the more specialized the type you&#x27;re looking at -- generifying everything is just as much an antipattern as anything else -- but String is so widespread and abusable that it really is helpful to abstract over it in many circumstances.')
