Item(by='badsectoracula', descendants=None, kids=None, score=None, time=1603500939, title=None, item_type='comment', url=None, parent=24875622, text='You can try Lazarus[0], which is essentially an open source Delphi, itself being a more &quot;serious&quot; alternative to classic VB using a real native compiler and more robust language (Object Pascal).<p>It is more complicated than VB6, though nothing that can&#x27;t be figured out by playing around with it and the included examples (and IMO it has by far the best <i>visual</i> form designer - placing controls in a freeform way and then &quot;tying&quot; them together with anchors with padding, etc so that they resize properly feels much more intuitive than using layout managers like in other toolkits that emphasize more on making stuff with code and have WYSIWYG editors as an afterthought at best).<p>Sadly there isn&#x27;t any open source tool that i know that approaches classic VB&#x27;s simplicity (though when it comes to that i am a bit more of a VB4 fan - i think by the time VB5 was introduced, the UI got a bit too complicated for the &quot;form editor with a scripting language attached&quot; that classic VB was at its core).<p>[0] <a href="https:&#x2F;&#x2F;www.lazarus-ide.org&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.lazarus-ide.org&#x2F;</a>')
