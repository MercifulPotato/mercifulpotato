Item(by='1vuio0pswjnm7', descendants=None, kids=[25076713, 25076754], score=None, time=1605220751, title=None, item_type='comment', url=None, parent=25074053, text='Some obvious differences...<p>Bing sets cookies.\nQwant does not set cookies.<p>Qwant, i.e., lite.qwant.com, prefixes search result URLs to point to Qwant servers.  Qwant redirects www.qwant.com to lite.qwant.com when Javascript is disabled.\nBing does not prefix search results.<p>Bing requires sign-up in order to use their API.\nQwant&#x27;s undocumented API is freely accessible, no sign-up.<p>Qwant, i.e., lite.qwant.com, requires a User-Agent header.\nBing does not require a UA header.<p>Example of Qwant API<p><pre><code>   curl &quot;https:&#x2F;&#x2F;api.qwant.com&#x2F;api&#x2F;search&#x2F;videos?q=example&amp;count=150&amp;offset=0&amp;f=xyz&amp;t=xyz&amp;l=en_gb&amp;uiv=xyz&quot;</code></pre>')
