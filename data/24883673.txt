Item(by='roywiggins', descendants=None, kids=None, score=None, time=1603597146, title=None, item_type='comment', url=None, parent=24883478, text='There are basically three broad sets of handhelds: OpenDingux devices, ARM Linux devices, and Android ones. The RG350P is an OpenDingux device.<p>There&#x27;s a <i>lot</i> of OpenDingux devices- they have MIPS CPUs, so they don&#x27;t run the same variety of software as the ARM-based ones. There&#x27;s still a bunch of homebrew for OpenDingux, but it&#x27;s not exactly organized, compared to Android and Linux devices. It involves hunting down poorly-documented executables, etc. Here&#x27;s one list of software available:<p><a href="https:&#x2F;&#x2F;github.com&#x2F;retrogamehandheld&#x2F;OpenDingux" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;retrogamehandheld&#x2F;OpenDingux</a><p>There are two Pico-8 emulators that run on those things, tac08 and retro8. They both work on many games, but it&#x27;s a bit hit-and-miss.<p><a href="https:&#x2F;&#x2F;0xcafed00d.itch.io&#x2F;tac08-rg350" rel="nofollow">https:&#x2F;&#x2F;0xcafed00d.itch.io&#x2F;tac08-rg350</a> &#x2F;\n<a href="https:&#x2F;&#x2F;github.com&#x2F;Jakz&#x2F;retro8" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;Jakz&#x2F;retro8</a><p>However, it is apparently possible to get the <i>official</i> Raspberry Pi version of pico-8 running on the Odroid Go Advance, which is an ARM CPU running Linux, just like the RPi.<p><a href="https:&#x2F;&#x2F;forum.odroid.com&#x2F;viewtopic.php?f=193&amp;t=37256&amp;start=800#p295789" rel="nofollow">https:&#x2F;&#x2F;forum.odroid.com&#x2F;viewtopic.php?f=193&amp;t=37256&amp;start=8...</a><p>Additionally to that, I dug up a version of TIC-80 ( <a href="https:&#x2F;&#x2F;tic80.com&#x2F;" rel="nofollow">https:&#x2F;&#x2F;tic80.com&#x2F;</a> ), an open-source PICO-like environment, compiled for the OpenDingux which will run on the RG350:<p><a href="https:&#x2F;&#x2F;github.com&#x2F;retrogamehandheld&#x2F;OpenDingux&#x2F;raw&#x2F;master&#x2F;Emulators&#x2F;TIC-80_2020-03-21.opk" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;retrogamehandheld&#x2F;OpenDingux&#x2F;raw&#x2F;master&#x2F;E...</a>')
