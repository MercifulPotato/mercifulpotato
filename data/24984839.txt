Item(by='oalae5niMiel7qu', descendants=None, kids=None, score=None, time=1604443038, title=None, item_type='comment', url=None, parent=24976449, text='The compiled&#x2F;interpreted architecture is only important because Emacs is written in C and Elisp isn&#x27;t incrementally compiled to native code.<p>If Emacs was written in Lisp (like Zmacs), then it would have the same flexibility it currently has without having to contain an interpreter of any kind, and the user-written customizations would have the benefit of being compiled to optimized native code. Some things that you can&#x27;t do in Elisp for performance reasons would be perfectly doable in a Common Lisp version of Emacs.')
