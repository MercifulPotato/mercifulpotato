In case someone wants to build the it using Nix I have pasted a quick and dirty port below. Not sure if I am willing to act as maintainer in Nixpkgs given my lack of experience with Qt and the tool itself, but hopefully it can prove as a starting point for someone that is or perhaps even if someone wants to build it on macOS (it does not look that difficult to adapt it for Mac):<p><pre><code>  # vim: set ts=8 sw=2 sts=2 et:\n  \n  # Needs post-20.09 nixpkgs for `lensfun`, I have nixpkgs checked out and import it when building:\n  #\n  #   nix-build --no-out-link -I nixpkgs=$HOME&#x2F;repo&#x2F;nixpkgs -E &#x27;with import &lt;nixpkgs&gt; {}; libsForQt5.callPackage .&#x2F;default.nix {}&#x27;\n  \n  { pkgs ? import &lt;nixpkgs&gt; { }, cmake, curl, exiv2, libarchive, libjpeg, libraw\n  , libtiff, mkDerivation, pkg-config, qtbase, qtquickcontrols2 }:\n  let\n    librtprocess =\n      pkgs.librtprocess.overrideAttrs (oldAttrs: rec { version = &quot;0.10.0&quot;; });\n    # “[Use the] latest git version for Linux and MacOS.”\n    lensfun = pkgs.lensfun.overrideAttrs (oldAttrs: rec {\n      version = &quot;202012142&quot;;\n      src = pkgs.fetchFromGitHub {\n        owner = &quot;lensfun&quot;;\n        repo = &quot;lensfun&quot;;\n        rev = &quot;4672d765a17bfef7bc994ca7008cb717c61045d5&quot;;\n        sha256 = &quot;00x35xhpn55j7f8qzakb6wl1ccbljg1gqjb93jl9w3mha2bzsr41&quot;;\n      };\n    });\n  in mkDerivation rec {\n    version = &quot;0.11.0&quot;;\n    pname = &quot;filmulator-gui&quot;;\n    src = pkgs.fetchFromGitHub {\n      owner = &quot;CarVac&quot;;\n      repo = &quot;${pname}&quot;;\n      rev = &quot;v${version}&quot;;\n      sha256 = &quot;0gdw4mxawh8lh4yvq521h3skjc46fpb0wvycy4sl2cm560d26sy6&quot;;\n    };\n    sourceRoot = &quot;source&#x2F;filmulator-gui&quot;;\n  \n    nativeBuildInputs = [ cmake pkg-config ];\n  \n    cmakeFlags = [ &quot;-DCMAKE_BUILD_TYPE=RELEASE&quot; ];\n  \n    buildInputs = [\n      curl\n      exiv2\n      lensfun\n      libarchive\n      libjpeg\n      libraw\n      librtprocess\n      libtiff\n      qtbase\n      qtquickcontrols2\n    ];\n  \n    meta.broken = builtins.compareVersions qtbase.version &quot;5.15&quot; &lt; 0;\n  }</code></pre>')
