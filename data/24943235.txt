Item(by='oftenwrong', descendants=None, kids=None, score=None, time=1604071637, title=None, item_type='comment', url=None, parent=24942308, text='One time I made an authentication system for an application that had some types like PlainTextAccountPassword and HashedAccountPassword. The storage backend for accounts accepted a HashedAccountPassword only. This way it would be less likely that a maintainer would end up storing a plain text password.<p>It is possible to do something like this in nearly any language with a type system.')
