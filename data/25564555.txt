Here&#x27;s how I do it:<p><pre><code>    git clone --bare &lt;dotifles repo&gt; ~&#x2F;.dotfiles\n    git --git-dir=$HOME&#x2F;.dotfiles&#x2F; --work-tree=$HOME checkout\n    git --git-dir=$HOME&#x2F;.dotfiles&#x2F; --work-tree=$HOME config --local status.showUntrackedFiles no\n</code></pre>\nAnd then I add an alias:<p><pre><code>    alias cfg=&#x27;git --git-dir=$HOME&#x2F;.dotfiles --work-tree=$HOME&#x27;\n</code></pre>\nNow I can just manage my dotfiles as a git repo, and I don&#x27;t have to worry about accidentally adding local secrets to the repo.')
