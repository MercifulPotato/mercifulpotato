Item(by='jandrese', descendants=None, kids=None, score=None, time=1605294300, title=None, item_type='comment', url=None, parent=25081853, text='One thing I like about Perl is the module system.<p>A big helper is when the module interface uses named parameters and has sensible defaults for unspecified parameters.  This allows the module designer to add features without breaking existing code and makes it easier for someone to integrate the module in the first place.  Having the documentation built into the module itself is also a huge win.')
