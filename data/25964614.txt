Item(by='Blikkentrekker', descendants=None, kids=None, score=None, time=1611965653, title=None, item_type='comment', url=None, parent=25962939, text='Almost anything can be simulated in any other data type, often with poor performance and easy bugs.<p>If a programmer use lists in lisps where vectors or hash tables are more appropriate, he is doing something wrong.<p>Likewise, simulating arrays in hash tables, which is what <i>PHP</i> expects one to do, is a very bad idea.<p><i>Bash</i> lacks multidimensional arrays, but it does have hash tables, so some programmers resort to simulating multidimensional arrays by using keys such as `&quot;3,4&quot;` as a string in hash tables, this would be a very bad idea if <i>Bash</i> had true multidimensional arrays. In it&#x27;s lack thereof, it is only a bad idea where no better idea is possible.')
