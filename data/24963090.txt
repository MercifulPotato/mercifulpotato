Item(by='brucehoult', descendants=None, kids=None, score=None, time=1604269605, title=None, item_type='comment', url=None, parent=24962597, text='The dependency could be arbitrarily many millions of instructions away.<p>The only way to know there isn&#x27;t a dependency is if that register gets clobbered by something else very soon afterwards.<p>But this whole topic of &quot;checking for signed overflow is expensive&quot; is overblown. It&#x27;s simply not <i>that</i> important an operation, especially in the context of those languages that do it a lot also doing a lot of memory references, which are far more expensive.<p>Adding arbitrary completely unknown integers is pretty rare. If you know both numbers are greater than zero then a single compare-and-branch is all you need. If one of the numbers is a constant then a single compare-and-branch is all you need.')
