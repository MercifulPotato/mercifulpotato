Item(by='anuragsoni', descendants=None, kids=[25902721], score=None, time=1611579981, title=None, item_type='comment', url=None, parent=25902377, text='Lwt [1] based web frameworks can be used on MirageOS if they provide an interface that allows someone to swap out the unix dependent pieces. One such framework is Opium [2], that provides most of its features in a unix independent code library `opium_kernel`. Its fairly straightforward to run an opium app on mirage. This example needs to be updated to the newest apis in the released version of Opium but this can provide some hints about how to run an opium app on mirage: <a href="https:&#x2F;&#x2F;github.com&#x2F;anuragsoni&#x2F;ocaml-opium-unikernel" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;anuragsoni&#x2F;ocaml-opium-unikernel</a>.<p>If a lower level web toolkit is sufficient then ocaml&#x27;s cohttp [3] library has been available on mirage for quite some time now.<p>[1] <a href="https:&#x2F;&#x2F;github.com&#x2F;ocsigen&#x2F;lwt&#x2F;" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;ocsigen&#x2F;lwt&#x2F;</a><p>[2] <a href="https:&#x2F;&#x2F;github.com&#x2F;rgrinberg&#x2F;opium" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;rgrinberg&#x2F;opium</a><p>[3] <a href="https:&#x2F;&#x2F;github.com&#x2F;mirage&#x2F;ocaml-cohttp&#x2F;" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;mirage&#x2F;ocaml-cohttp&#x2F;</a>')
