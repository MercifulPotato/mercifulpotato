Item(by='OliverJones', descendants=None, kids=[25288858], score=None, time=1607001888, title=None, item_type='comment', url=None, parent=25287562, text='For what it&#x27;s worth the php implementation of password hashing offers a function to determine whether it&#x27;s time to rehash a password.  <a href="https:&#x2F;&#x2F;www.php.net&#x2F;manual&#x2F;en&#x2F;function.password-needs-rehash.php" rel="nofollow">https:&#x2F;&#x2F;www.php.net&#x2F;manual&#x2F;en&#x2F;function.password-needs-rehash...</a><p>The idea is to alert an application program to the need to regenerate the hash at the time it has the plaintext password in hand (when the user has just presented it for login).<p>This is a great idea; rehashing a long-standing bcrypt password with a larger work factor makes it safer.<p>And, an extension or new version of the runtime can add a new hashing scheme.<p>It would be sweet if other password-hashing APIs added the same kind of thing. User accounts can last far longer than LTS versions of software runtimes, and this can help future-proof them.')
