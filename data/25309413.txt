Item(by='trinovantes', descendants=None, kids=None, score=None, time=1607122420, title=None, item_type='comment', url=None, parent=25306071, text='Back in my undergrad software testing course, we were taught test coverage in the context of a control flow graph.<p>I think most coverage measurements (at least in the GitHub projects where I&#x27;ve seen a coverage badge) only refer to statement coverage where it checks if every node of the CFG is reached. It&#x27;s one of the weakest forms of testing and will rarely reveal the more obscure bugs that edge&#x2F;(simple, prime, complete) path coverage testing can reveal.<p>I personally think the industry should stop using (node) coverage as some kind of golden metric.')
