Item(by='z3t4', descendants=None, kids=None, score=None, time=1607587526, title=None, item_type='comment', url=None, parent=25370388, text='One problem is that we developers are lazy and use Access-Control-Allow-Origin: * (wild star) instead of actual hostname. (eg. allowing all origins to access the backend)<p>No modern browser allows access to localhost without that header.<p>But it&#x27;s still possible to forge a request using curl or whatever to bypass CORS.\nSo as the parent post suggest - use a token of some sort.<p>I also recommend using a strict Content-Security-Policy to stop X-site injection attacks. (eg someone adding an image to your page&#x2F;app with src=&quot;&#x2F;api&#x2F;cmd=rm -rf &#x2F;&quot;')
