Item(by='achairapart', descendants=None, kids=[25356938], score=None, time=1607474125, title=None, item_type='comment', url=None, parent=25352940, text='While the source looks like JSON, the page is still sent by the server as text&#x2F;html. Browsers are so resilient to malformed markup that actually parse the content inside the &lt;html&gt; tag at the end of the document (firing the JS script that render the actual page), ignoring all the (pseudo) JSON around.<p>The page is rendered in QUIRKSMODE because the source is missing the &lt;!DOCTYPE html&gt; at the beginning of the document, to resemble valid JSON.<p>Well, at least that&#x27;s clever!')
