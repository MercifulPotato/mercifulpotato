The &quot;full&quot; notation is x_H for a pattern variable x that needs to have head H. Dropping the H, it matches anything. Dropping the x, it just doesn&#x27;t set a variable when matching. Dropping both, it matches anything and doesn&#x27;t set a variable.<p>[1] <a href="https:&#x2F;&#x2F;reference.wolfram.com&#x2F;language&#x2F;guide&#x2F;Patterns.html" rel="nofollow">https:&#x2F;&#x2F;reference.wolfram.com&#x2F;language&#x2F;guide&#x2F;Patterns.html</a>')
