Item(by='jacobolus', descendants=None, kids=None, score=None, time=1606512465, title=None, item_type='comment', url=None, parent=25229533, text='&gt; <i>tan(π&#x2F;8) would be halfway between 0 and 1 ie .5?</i><p>tan(π&#x2F;8) = √2 - 1, not 1&#x2F;2, but the value is indeed halfway between 0 and 1 if you take the distance function to be<p>d(a, b) = |b - a| &#x2F; (√(1 + a²)√(1 + b²))  or<p>d(a, b) = |b - a| &#x2F; |1 + ab|<p>(These are chordal distance or stereographic distance, respectively, when the real number line is used to represent points on a circle under stereographic projection.)<p>The halfway point is 0.5 when you use the distance function d(a, b) = |b - a|.')
