Item(by='h3rald', descendants=None, kids=None, score=None, time=1606530971, title=None, item_type='comment', url=None, parent=25231558, text='Well yes it started off for prototyping and then I kept adding more features like JavaScript middleware and token validation... I have used it as a backend for quite a few private projects of mine.<p>Regarding indexing json... here&#x27;s how I do it:<p><pre><code>     proc createIndex*(store: Datastore, indexId, field: string) =\n       let query = sql(&quot;CREATE INDEX json_index_$1 ON documents(json_extract(data, ?) COLLATE NOCASE) WHERE json_valid(data)&quot; % [indexId])\n       store.begin()\n       store.db.exec(query, field)\n       store.commit()\n</code></pre>\nBasically, SQLite supports indexes in arbitrary expressions and so you can use it in conjunction with json_extract. In this case, <i>field</i> is an arbitrary json path.<p>I then added a REST API in LiteStore to create (JSON) indexes... in that way you can make your queries more performant based on the specific json documents that you store in the data store.')
