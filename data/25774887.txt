Item(by='GianFabien', descendants=None, kids=None, score=None, time=1610621709, title=None, item_type='comment', url=None, parent=25774445, text='Your IP address can be any 32 bit number you choose if you don&#x27;t have DHCP choose it for you.  But, the routers will only forward and respond to addresses that have been correctly configured. There are conventions, standards and protocols that define what addresses are valid and how they should be used.  In a lot of cases, invalid IP addresses simply get dropped.<p>If you&#x27;d like to learn more, this Wikipedia entry is a good starting point: <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Internet_protocol_suite" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Internet_protocol_suite</a>')
