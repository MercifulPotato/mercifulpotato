Item(by='hyperrail', descendants=None, kids=[25117824], score=None, time=1605559690, title=None, item_type='comment', url=None, parent=25116286, text='The new Edge&#x27;s installer does not uninstall the old Edge, it just hides it. On versions of Windows before 20H2 (build 19042), you can bring back the old Edge icons by following these instructions: <a href="https:&#x2F;&#x2F;docs.microsoft.com&#x2F;en-us&#x2F;deployedge&#x2F;microsoft-edge-sysupdate-access-old-edge" rel="nofollow">https:&#x2F;&#x2F;docs.microsoft.com&#x2F;en-us&#x2F;deployedge&#x2F;microsoft-edge-s...</a><p>In short:<p>1. Run this command as an administrator: reg add hklm\\SOFTWARE\\Policies\\Microsoft\\EdgeUpdate &#x2F;v Allowsxs &#x2F;t REG_DWORD &#x2F;d 1<p>2. Download and reinstall the new Edge.<p>Note that Microsoft has said that old Edge will stop getting security updates soon, though I don&#x27;t know what this actually amounts to as the old Edge&#x27;s engine remains in Windows for application compatibility even when old Edge is nominally removed.<p>I&#x27;d also suggest that you send feedback to Microsoft about any problems you have with new Edge, using the built-in &quot;Send feedback&quot; tool or Windows&#x27; Feedback Hub.')
