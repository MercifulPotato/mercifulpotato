Item(by='pjmlp', descendants=None, kids=None, score=None, time=1607770546, title=None, item_type='comment', url=None, parent=25388219, text='Only devs that don&#x27;t code for Android can ever consider it an open source Linux based OS.<p>The fact that it uses Linux kernel and related features is an implementation detail, only visible to Google and OEMs.<p>What regular developers see is a Java&#x2F;Kotlin based userland, with a native layer having a set of predefined stable APIs, which includes ISO C, ISO C++ standard libraries, a POSIX subset, GL&#x2F;Vulkan and a couple of Android specific ones.<p>Tomorrow Android can be another Fucshia subsystem, and again, only Google and OEMs will notice the change.')
