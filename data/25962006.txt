Item(by='Nextgrid', descendants=None, kids=[25962145], score=None, time=1611952139, title=None, item_type='comment', url=None, parent=25961420, text='Something that has dramatically reduced my reliance on outside resources is an editor that can lookup symbol definitions.<p>I’m using Django and I used to use Sublime Text without any plugins which meant (at the time) it couldn’t automatically show me the definition of a function or object from a library, forcing me to search around for its documentation.<p>Switching to PyCharm&#x2F;IntelliJ meant I could now just Command-Click (Ctrl-Click for Linux &amp; Windows I believe) and see the actual definition and logic of the underlying function without having to rely on the online documentation. This of course works recursively, so I can repeat this process again and again up until I eventually reach the core standard library which is in C and isn’t introspective (at this point I can rely on my knowledge or some trial and error to figure out its behavior).')
