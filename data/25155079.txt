Item(by='dbaupp', descendants=None, kids=None, score=None, time=1605824512, title=None, item_type='comment', url=None, parent=25151687, text='I think there&#x27;s another perspective on doctests, that seems more convincing to me: doctest functionality allows testing for the documentation, not a way of writing tests in documentation.<p>In particular, it&#x27;s great if the documentation includes clear and simple examples for learning from, and it&#x27;s even better if these are validated as working. This means that the focus of the code in documentation is typically different to a test (it doesn&#x27;t need to include such precise validation or look at all the edge cases or regressions), but it&#x27;s still really useful to have them automatically run.')
