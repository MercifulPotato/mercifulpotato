Item(by='ppeetteerr', descendants=None, kids=[25625775], score=None, time=1609711444, title=None, item_type='comment', url=None, parent=25623125, text='Pagination of an immutable collection is one thing and can be parallelized. Pagination of a mutable collection (e.g. a database table), on the other hand, is risky since two requests might return intersecting data if new data was added between the requests being executed.<p>True result sets require relative page tokens and a synchronization mechanism if the software demands it.')
