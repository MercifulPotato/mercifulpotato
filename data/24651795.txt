Item(by='derefr', descendants=None, kids=None, score=None, time=1601565078, title=None, item_type='comment', url=None, parent=24648582, text='Less of a mess if you use a YAML block literal (basically a heredoc):<p><pre><code>    jobs:\n      format:\n        runs-on: ubuntu-latest\n        if: |\n          ! contains(github.event.head_commit.message, ‘wip’)\n</code></pre>\nOther systems that output YAML (e.g. Kubernetes) always emit block literals for embedded user-supplied strings, reserving inline string literals for the cases where the string isn’t expected to need quoting — e.g. string-serialized enum option-values; symbol&#x2F;atom&#x2F;keyword-typed values; and sometimes URLs.')
