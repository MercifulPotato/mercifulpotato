Item(by='sritchie', descendants=None, kids=[25114500], score=None, time=1605546261, title=None, item_type='comment', url=None, parent=25112907, text='If anyone wants to play with these ideas in Clojure(script), take a look at the SICMUtils project:  <a href="https:&#x2F;&#x2F;github.com&#x2F;littleredcomputer&#x2F;sicmutils" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;littleredcomputer&#x2F;sicmutils</a><p>I picked up co-maintenance on this project a few months ago; it&#x27;s a Clojure port of the computer algebra system behind Sussmans&#x27; &quot;Structure and Interpretation of Classical Mechanics&quot; [0] and &quot;Functional Differential Geometry&quot; [1], and can run all the code from those books. (I&#x27;m currently working on integrating it into <a href="https:&#x2F;&#x2F;Maria.cloud" rel="nofollow">https:&#x2F;&#x2F;Maria.cloud</a> so all of this will be usable from the browser.)<p>`sicmutils.generic.simplify` plugs in to a rule-based term rewriting system like Mathematica&#x27;s, along with polynomial and rational function simplification.<p>The core idea of the library is that if you build everything on top of extensible generic arithmetic operators (multimethods in Clojure), then a lot of good stuff shows up for free:<p>- If I implement all the operations for symbols, I get a symbolic arithmetic system for free.\n- Extend the generics to Dual numbers[2]? Boom, forward-mode AD pops out!<p>I&#x27;m getting carried away. Check it out, get your term-rewriting fix.<p>[0] <a href="https:&#x2F;&#x2F;mitpress.mit.edu&#x2F;books&#x2F;structure-and-interpretation-classical-mechanics-second-edition" rel="nofollow">https:&#x2F;&#x2F;mitpress.mit.edu&#x2F;books&#x2F;structure-and-interpretation-...</a>\n[1] <a href="https:&#x2F;&#x2F;mitpress.mit.edu&#x2F;books&#x2F;functional-differential-geometry" rel="nofollow">https:&#x2F;&#x2F;mitpress.mit.edu&#x2F;books&#x2F;functional-differential-geome...</a>\n[2] <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Dual_number" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Dual_number</a>')
