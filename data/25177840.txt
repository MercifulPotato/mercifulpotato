Item(by='jhallenworld', descendants=None, kids=[25177977], score=None, time=1606056614, title=None, item_type='comment', url=None, parent=25176531, text='Use arrays of 1 to avoid using &#x27;.&#x27; and &#x27;&amp;&#x27;.  So if you have a structure, say:<p><pre><code>    struct foo { int a, b; };\n</code></pre>\nA function that works on the structure looks like this, you use &#x27;-&gt;&#x27;:<p><pre><code>    void pr(struct foo *n) { printf(&quot;%d %d\\n&quot;, n-&gt;a, n-&gt;b); }\n</code></pre>\nBut when you make an instance of this structure, you have to use this different syntax:<p><pre><code>    struct foo p;\n    p.a = 1;  &#x2F;&#x2F; Yuck!\n    pr(&amp;p); &#x2F;&#x2F; Bad!\n</code></pre>\nBut you can do this instead:<p><pre><code>    struct foo p[1];\n    p-&gt;a = 1; &#x2F;&#x2F; Very nice..\n    pr(p); &#x2F;&#x2F; Yes!</code></pre>')
