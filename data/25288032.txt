You can get the same core design in 4 core and 64 core part with ryzen for example - the difference is price and power.<p>Single core performance is critical and core count has rapidly diminishing returns for the majority of everyday tasks (Amdahl&#x27;s law and all that).<p>When you have independent tasks that need to run massively parallel you are better off using a different architecture because you can get better value from simpler compute units.')
