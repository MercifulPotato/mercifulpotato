What you want is precisely what templates are, but you also want them to not be &quot;templates&quot; for... some bizarre reason.<p>&gt; I can have an array of T, but I can&#x27;t specify that T is Numeric?<p>Sure you can. If you have C++20 concepts:<p><pre><code>  template&lt;class T&gt;\n  concept Numeric = std::integral&lt;T&gt; || std::floating_point&lt;T&gt;;\n  \n  template&lt;Numeric T&gt;\n  T twice(T x)\n  { return x + x; }\n</code></pre>\nOr if you&#x27;re on a C++11 compiler:<p><pre><code>  template&lt;class T&gt;\n  typename std::enable_if&lt;\n    std::is_arithmetic&lt;T&gt;::value,\n    T&gt;::type twice(T x)\n  { return x + x; }</code></pre>')
