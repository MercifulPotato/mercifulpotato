Item(by='bregma', descendants=None, kids=[24931866, 24929771, 24929499], score=None, time=1603969874, title=None, item_type='comment', url=None, parent=24926925, text='For those interested, the current <i>de facto</i> ELF standard is <a href="http:&#x2F;&#x2F;www.sco.com&#x2F;developers&#x2F;gabi&#x2F;latest&#x2F;contents.html" rel="nofollow">http:&#x2F;&#x2F;www.sco.com&#x2F;developers&#x2F;gabi&#x2F;latest&#x2F;contents.html</a> (yes, for historical reasons, that SCO) although there is a rumour it will move to a GitHub project at some unspecified time in the future.<p>There is, in fact, no <i>de jure</i> standard for the ELF binary format. It is, however, pretty much a universal standard format for executable, dynamic shared object, and partially-linked compiled file format used on every platform except Apple and Microsoft systems.')
