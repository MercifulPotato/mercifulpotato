Item(by='dmytton', descendants=None, kids=[25655424], score=None, time=1609917620, title=None, item_type='comment', url=None, parent=25654222, text='Moving large numbers of files - like the 50k of Google Photos the author mentions - can be done using <a href="https:&#x2F;&#x2F;rclone.org" rel="nofollow">https:&#x2F;&#x2F;rclone.org</a>. Set up Google Photos as a source and then use the copy command to move it to a different location[1]. There are a lot of different commands which work with many different cloud storage systems e.g. sync everything one way locally to a backup, mount a FUSE filesystem, do two-way sync, etc.<p>[1] <a href="https:&#x2F;&#x2F;rclone.org&#x2F;commands&#x2F;rclone_copy&#x2F;" rel="nofollow">https:&#x2F;&#x2F;rclone.org&#x2F;commands&#x2F;rclone_copy&#x2F;</a>')
