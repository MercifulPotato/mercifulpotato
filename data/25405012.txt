Item(by='jsancho', descendants=None, kids=[25411475], score=None, time=1607845685, title=None, item_type='comment', url=None, parent=25385296, text='It&#x27;d be nice if Datasette provided support for other file formats than SQLite. I made <a href="https:&#x2F;&#x2F;github.com&#x2F;pytables&#x2F;datasette-connectors" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;pytables&#x2F;datasette-connectors</a> that gives a simple API for making &quot;connectors&quot; for other formats by monkey patching Datasette, but it&#x27;d be more efficient if Datasette included that API (or something similar) directly.<p>For example, <a href="https:&#x2F;&#x2F;github.com&#x2F;pytables&#x2F;datasette-pytables" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;pytables&#x2F;datasette-pytables</a> implements a connector for HDF5 files. You can read how to make a connector for your favorite file format in <a href="https:&#x2F;&#x2F;github.com&#x2F;PyTables&#x2F;datasette-connectors&#x2F;blob&#x2F;master&#x2F;DEVELOPERS.md" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;PyTables&#x2F;datasette-connectors&#x2F;blob&#x2F;master...</a><p>Currently, datasette-connectors supports Datasette 0.51.1 but every few months an update is needed in order to be compatible with last versions.<p>I wonder if it&#x27;d be possible to merge datasette-connectors into Datasette. Then, &quot;connectors&quot; could be like other plugins and people could publish their data in a wider variety of ways.')
