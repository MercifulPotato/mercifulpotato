Item(by='tasty_freeze', descendants=None, kids=[25680660], score=None, time=1610068167, title=None, item_type='comment', url=None, parent=25678945, text='Imagine telling the students to use vim if they have never used it before. In theory, one only needs to know a few things to edit:<p>- type vi &lt;filename&gt; to start editing<p>- there are two modes: &#x27;i&#x27; to start insert mode, &lt;esc&gt; to leave it for normal mode<p>- in insert mode, what you type is entered as you type<p>- in normal mode, use the cursor keys to move around<p>- in normal mode, type &#x27;x&#x27; to delete the character under the cursor<p>- type :wq to save and quit<p>What are the odds a vim-virgin could complete a programming assignment without getting tripped up, despite the explicit directions to use just the features above?')
