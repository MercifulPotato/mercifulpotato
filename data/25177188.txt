Item(by='flohofwoe', descendants=None, kids=None, score=None, time=1606050787, title=None, item_type='comment', url=None, parent=25176531, text='Declaring, instancing and initializing a global state struct in one go:<p><pre><code>    static struct {\n        int a;\n        float b;\n        const char* c;\n        struct {\n            float x, y, z;\n        } nested;\n    } state = {\n        .a = 1,\n        .b = 2.0f,\n        .c = &quot;Hello World!&quot;,\n        .nested = {\n            .x = 1.0f, .z = 3.0f\n        },\n    };\n</code></pre>\nCareful if the the struct contains big arrays though, this will let the executable size explode because a copy of the struct content is placed in the executable.')
