Item(by='gamegoblin', descendants=None, kids=[25595120], score=None, time=1609433301, title=None, item_type='comment', url=None, parent=25593892, text='The methods on `String` should be a strict superset of the methods on `str` because `String` dereferences to `str`, thus `String` gets all of the `str` methods for free.<p>If you&#x27;re familiar with say, `Vec&lt;u8&gt;` and `[u8]`, `String` and `str` are basically the same except they are contractually  valid UTF bytes. So just like you can `push` and `extend` to a `Vec`, you can do the same to a `String`.<p>With regard to generic type parameters, if you want to code in Java or Go style, you can use dynamic dispatch trait objects to remove type parameters.')
