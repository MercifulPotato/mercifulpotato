Work with the electric utilities, so the smart meter is the same one they&#x27;re using for billing, and you can largely prevent fraud by rerouting the electricity.<p>Decentralized finance is one area where blockchains have been pretty successful, so I can see why they&#x27;re using one for a project that boils down to decentralized finance.')
