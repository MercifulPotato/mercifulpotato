Item(by='monkeynotes', descendants=None, kids=None, score=None, time=1607694429, title=None, item_type='comment', url=None, parent=25382826, text='There is a difference between integration testing (Cypress UI click through UI) and backend unit testing.<p>We run integration tests with Cypress and then rspec or some other Rails testing platform for the backend.<p>Cypress uses fixtures so ideally you would not let it hit a backend API unless you don&#x27;t care about reproducibility of the response. For example a PUT will augment the data on the backend so when you run the test again the backend data will probably give you a different response to assert and fail your expected.')
