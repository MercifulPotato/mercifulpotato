Item(by='timoth', descendants=None, kids=None, score=None, time=1605826737, title=None, item_type='comment', url=None, parent=25154705, text='I remember using Interix back in the late 90s to port our home-grown source control system that we used on various UNIX flavours over to NT.  The source control system was basically a bunch of shell scripts which wrapped either SCCS or RCS depending on which was available on the system in question.  Porting them to Interix on NT worked pretty well, though there were some wrinkles with file permissions.')
