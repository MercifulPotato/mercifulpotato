Item(by='jcubic', descendants=None, kids=None, score=None, time=1610800518, title=None, item_type='comment', url=None, parent=25783152, text='I&#x27;m working on my Scheme based lips interpreter in JavaScript. With great integration with JavaScript, literal regular expression, auto async&#x2F;await (with option to quote the promise), self documented like ELisp. I&#x27;m working on compatibility with R7RS, recently implemented rest of the functions from R5RS. Big part of the standard library is written in Scheme (small LIPS core) and I&#x27;m also working right now on Rollup plugin that will allow to created one bundle for all the Scheme code is I no longer need to load them with Ajax.<p>What&#x27;s missing are continuations and proper tail calls. And syntax-rules macros need to be heavy refactored (I need to add proper expansion time).<p>If you want to take a look make sure to check out 1.0 Beta version, that is completely different than what&#x27;s suppose to be stable version.<p>Website: <a href="https:&#x2F;&#x2F;lips.js.org&#x2F;" rel="nofollow">https:&#x2F;&#x2F;lips.js.org&#x2F;</a>\nGitHub: <a href="https:&#x2F;&#x2F;github.com&#x2F;jcubic&#x2F;lips" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;jcubic&#x2F;lips</a><p>The most up to date documentation is on the wiki: <a href="https:&#x2F;&#x2F;github.com&#x2F;jcubic&#x2F;lips&#x2F;wiki&#x2F;v1.0-draft" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;jcubic&#x2F;lips&#x2F;wiki&#x2F;v1.0-draft</a>')
