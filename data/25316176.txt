Item(by='kazinator', descendants=None, kids=[25320269], score=None, time=1607188355, title=None, item_type='comment', url=None, parent=25313222, text='Yes. We would want the descriptive names for identifiers which have broad scope, but short names for the local variables that work with them.<p>I would rather give a function the descriptive name in such a way that the parameters can then be given trivial, mnemonic names.<p>Strawmannish example:<p><pre><code>   incrementAccountBalance(a, n)\n   {\n      ...\n   }\n</code></pre>\nThe descriptive function name gives away what <i>a</i> and <i>n</i> are, so it is unhelpful to make this:<p><pre><code>   incrementAccountBalance(account, amountToIncrementBy)\n</code></pre>\nReasonable middle ground:<p><pre><code>   incrementAccountBalance(acct, delta)</code></pre>')
