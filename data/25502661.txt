Item(by='jrott', descendants=None, kids=[25502801], score=None, time=1608609102, title=None, item_type='comment', url=None, parent=25501594, text='&gt; In my experience, code with too many functions is more difficult to grok than spaghetti code. It&#x27;s like trying to read a book with each sentence reference a different page. So, I try to code like I would write, in digestible chunks.<p>This is so true. The worst code that I&#x27;ve dealt with is the code that requires jumping to a ton of different files to figure out what is going on. It&#x27;s usually easier to decompose a pile of spaghetti code than to figure out how to unwrap code that has been overly abstracted.')
