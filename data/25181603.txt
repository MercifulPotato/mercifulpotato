Item(by='jleahy', descendants=None, kids=[25182889], score=None, time=1606083130, title=None, item_type='comment', url=None, parent=25181584, text='A 1-bit error in a compressed file will cause every subsequent bit in the entire file to end up wrong (well, 50% of them), and probably the file length to end up wrong, and probably the decompression state machine to get stuck and confused. This is why basically all compressed formats carry a CRC (eg. zip, gz, xz).')
