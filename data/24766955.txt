Item(by='fzimmermann', descendants=None, kids=None, score=None, time=1602605759, title=None, item_type='comment', url=None, parent=24764605, text='I see some issues with the problem description and the solution: The author assumes the problem can be split in x and y independently. Firstly, this has not to be true for all projections and grids. Secondly, the solution minimizes Manhattan distance in lat&#x2F;long., not actual distance (you can construct points where the minima are different).\nThis will not matter if you can approximate both grids as regular (equidistant and rectangular grids). But in that case, you could easily calculate for each point in one grid the corrosponding point in the other grid by scaling and rounding...')
