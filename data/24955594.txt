Item(by='wolf550e', descendants=None, kids=[24955903], score=None, time=1604182011, title=None, item_type='comment', url=None, parent=24955447, text='To match the original zlib&#x2F;gzip stream, it needs to match the exact hash table and parsing algorithm used by the exact zlib implementation used originally, to choose the same matches. It would not do to use a stronger deflate compressor that produces optimal parsing to generate smaller deflate streams but then not be able to match the original file. The hash of the compressed data and even the length of the compressed data might be important to the application using the file.')
