Real-world UI design aims to be as easily understandable and usable as possible, even if it falls short. In movies, they want to convey that the hackers&#x2F;astronauts&#x2F;super heros have deep expertise. It would be frustrating if you saw them using a familiar Windows Start Menu because you&#x27;d feel like you yourself could just step in and do their job.<p>They could (and for hacker movies, sometimes do) just use a command line, but then it&#x27;s hard to convey what they&#x27;re doing to an audience. For a similar reason, important text in a fantasy UI is impractically large, so it&#x27;s visible when the UI itself is only a fraction of the movie screen.')
