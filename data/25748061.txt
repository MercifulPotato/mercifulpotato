Item(by='kortex', descendants=None, kids=None, score=None, time=1610465258, title=None, item_type='comment', url=None, parent=25747453, text='Without the right framework to manage sidecar files, this system rots quickly. Tools like DVC are good but there are still rough edges.<p>The issue partly stems from the lack of coupling between data and metadata on file systems. I think the next evolution of file systems involves the ability to link files together in an organic way. But that is tricky since it breaks the abstraction of files as stream of bytes. Might need to go up a level of abstraction and create new primitives for cp, mv, etc.')
