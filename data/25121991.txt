Item(by='profclems', descendants=0, kids=None, score=1, time=1605607932, title='Gitlab Command Line', item_type='story', url=None, parent=None, text='Do you can create a GitLab merge request with your latest commit on the current branch with `glab mr create --fill` ??<p>You can also create a merge request for an issue with `glab mr for &lt;issue-id&gt;` ; this will create the source branch using the issue title and mark it as draft just as it is done on the UI.<p>You can then checkout to the merge request branch with `glab mr checkout &lt;id&gt;`.<p>To mark it as ready for review use `glab mr update --ready`. (ID is not required when already checked out to that branch).<p>Approve with `glab mr approve` and merge with `glab mr merge`<p>Check it out at https:&#x2F;&#x2F;github.com&#x2F;profclems&#x2F;glab. \nMirror at https:&#x2F;&#x2F;gitlab.com&#x2F;profclems&#x2F;glab')
