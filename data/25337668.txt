Item(by='mbirth', descendants=None, kids=[25340615, 25338933], score=None, time=1607375860, title=None, item_type='comment', url=None, parent=25335248, text='XMPP is anything but dead yet. There were a lot of XEPs added and updated in recent years and it turned into a modern protocol being able to provide all features you could expect from a modern messaging service.<p>It&#x27;s just that you need a client and server that supports those features. Daniel Gultsch has written an Android client[1], that&#x27;s as good as it can be. He also offers a maintained XMPP server [2] providing the full-featured counterpart. But if you want to self-host, ejabberd supports all the features as does Prosody. For iOS users, there&#x27;s Monal which is slowly getting there.<p>[1] <a href="https:&#x2F;&#x2F;conversations.im" rel="nofollow">https:&#x2F;&#x2F;conversations.im</a><p>[2] <a href="https:&#x2F;&#x2F;account.conversations.im" rel="nofollow">https:&#x2F;&#x2F;account.conversations.im</a>')
