You can install the needed Google app and see if it works. You can use Termux Widget [1] to add enable&#x2F;disable links for apps so you can keep them around in a disabled state, to be enabled and disabled on demand. Create a directory named <i>.shortcuts</i> in your <i>Termux</i> home directory, any script placed there will be available through the Termux widget. Take Google Maps, an often demanded but extremely intrusive app. If you occasionally want to use it you can add the following scripts to enable and disable it:<p>googlemaps-enable:<p><pre><code>   #!&#x2F;data&#x2F;data&#x2F;com.termux&#x2F;files&#x2F;usr&#x2F;bin&#x2F;sh\n   \n   exec sudo pm enable com.google.android.apps.maps\n   exit 0\n</code></pre>\ngooglemaps-disable:<p><pre><code>   #!&#x2F;data&#x2F;data&#x2F;com.termux&#x2F;files&#x2F;usr&#x2F;bin&#x2F;sh\n   \n   exec sudo pm disable com.google.android.apps.maps\n   exit 0\n</code></pre>\nYes, you need root (and sudo) to do this.<p>[1] <a href="https:&#x2F;&#x2F;wiki.termux.com&#x2F;wiki&#x2F;Termux:Widget" rel="nofollow">https:&#x2F;&#x2F;wiki.termux.com&#x2F;wiki&#x2F;Termux:Widget</a>')
