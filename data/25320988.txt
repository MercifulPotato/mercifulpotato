Item(by='ncmncm', descendants=None, kids=[25321147], score=None, time=1607228187, title=None, item_type='comment', url=None, parent=25316713, text='That a library implementation of pattern-matching would be unsatisfactory exposes a weakness in the core language. Usually, it is better to strengthen the language to the point where a library component can do the job. Sometimes, though, that is too hard, and you give up and do it directly in the core language. It should always be noted as a failing.<p>Something similar happened with coroutines, although what we did get has hooks that a library should be able to patch into, to achieve wonders not yet seen.')
