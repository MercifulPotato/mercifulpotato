Item(by='gls2ro', descendants=None, kids=None, score=None, time=1610889636, title=None, item_type='comment', url=None, parent=25798186, text='I recommend (subjectively of course) Ruby on Rails.<p>It is a full-stack offering together with Ruby Gems ecosystem solutions to a wide range of things that I want to achieve with a webapp. I like a lot how people in this community tries to simplify a lot of the complexities of modern tools.<p>In the Ruby (and Rails ecosystem) you should take a look at <a href="https:&#x2F;&#x2F;hotwire.dev" rel="nofollow">https:&#x2F;&#x2F;hotwire.dev</a> - helping create amazing web apps with less Javascript code. There are a lot of things to like there, but you can start small with Stimulus JS, which for me offers good enough JS code to do small UI things.<p>Then there is <a href="https:&#x2F;&#x2F;docs.stimulusreflex.com" rel="nofollow">https:&#x2F;&#x2F;docs.stimulusreflex.com</a> - for creating reactive web apps without going full SPA mode.<p>And here it is the latest addition to this eco-system (that I know of): <a href="https:&#x2F;&#x2F;github.com&#x2F;rails&#x2F;tailwindcss-rails" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;rails&#x2F;tailwindcss-rails</a>. This is integrating Tailwind directly in assets pipeline of Rails. No need to configure webpack and postcss and whatever other configs. You can directly use Tailwind by adding this gem.')
