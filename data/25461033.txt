E.g.:<p><pre><code>  let file = file |? raise Some_error\n</code></pre>\nor<p><pre><code>  let parse_result = parse &lt;$&gt; file\n</code></pre>\nFor the last two examples respectively.<p>EDIT: For completeness, the operators are<p><pre><code>  let (|?) opt y =\n    match opt with \n    | Some x -&gt; x\n    | None -&gt; y\n\n  let (&lt;$&gt;) f opt =\n    match opt with\n    | Some x -&gt; Some (f x)\n    | None -&gt; None</code></pre>')
