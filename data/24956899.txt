Item(by='lowdanie', descendants=None, kids=[24956941, 24961120], score=None, time=1604199321, title=None, item_type='comment', url=None, parent=24956761, text='Indeed, the statement is that for any list of axioms there exists a countable set of objects satisfying them.<p>For example, you could write down axioms for the real numbers by specifying that there should be relations called + and x with the standard properties such as commutativity, as well as an ordering relation &lt; such that for all elements x and y there is an element z for which: x &lt; z &lt; y.<p>Clearly the real numbers are a model for these axioms. But as it turns out the countable set of rational numbers is a model as well.')
