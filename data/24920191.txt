Item(by='kbumsik', descendants=None, kids=None, score=None, time=1603900246, title=None, item_type='comment', url=None, parent=24919526, text='I think learning a language is not the point, but learning underlying system (OS) is.<p>What you want to learn (especially in embedded world) is how various Linux&#x2F;POSIX APIs work, not language features.<p>So either choice is good as long as you learn to use them, but I personally recommend C because C is the first class language for Linux APIs e.g. they are exposed directly via #include &lt;sys&#x2F;<i>&gt; or #include &lt;linux&#x2F;</i>&gt;.<p>As others recommend, you can learn C and Rust at the same time though.')
