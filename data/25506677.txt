Item(by='garethrowlands', descendants=None, kids=None, score=None, time=1608648226, title=None, item_type='comment', url=None, parent=25505138, text='Passing a big structure to a function doesn&#x27;t make that function impure. But passing a structure containing objects with impure methods does make the called function impure.<p>Passing a big structure to a function is a bad idea in any paradigm because it&#x27;s a big dependency. Instead, you should write the function to operate on the data that it needs and pass just that when you call it.')
