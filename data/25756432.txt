Item(by='stncls', descendants=None, kids=None, score=None, time=1610499201, title=None, item_type='comment', url=None, parent=25756270, text='Many refactoring tools need the AST (and more) from a frontend to do type-aware refactoring, among other things.<p>If we have, for example,<p><pre><code>  struct a { int x; } sa;\n  struct b { float x; } sb;\n</code></pre>\nand we want to rename sa.x into sa.y, then the refactoring tool needs to understand types in order to leave sb.x alone. And if<p><pre><code>  sa.x - sa.x\n</code></pre>\nis not in the AST, it will not be changed into<p><pre><code>  sa.y - sa.y\n</code></pre>\nwhich will yield errors.')
