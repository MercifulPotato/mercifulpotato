(Source code for 10.14 might be at <a href="https:&#x2F;&#x2F;opensource.apple.com&#x2F;source&#x2F;text_cmds&#x2F;text_cmds-99&#x2F;grep&#x2F;grep.c.auto.html" rel="nofollow">https:&#x2F;&#x2F;opensource.apple.com&#x2F;source&#x2F;text_cmds&#x2F;text_cmds-99&#x2F;g...</a>.) Not sure if it was ever GNU grep. Maybe before the big purge in 2011?<p>IME, most of the basic shell commands are either directly from FreeBSD or a project in the BSD universe. For example tar is from libarchive, a distinct project (see <a href="https:&#x2F;&#x2F;github.com&#x2F;libarchive&#x2F;libarchive" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;libarchive&#x2F;libarchive</a>) that DragonflyBSD, FreeBSD, NetBSD, and macOS import and track. (But oddly not OpenBSD, which has evolved the 4.4BSD-Lite implementation.)<p>According to a 2012 accounting at <a href="http:&#x2F;&#x2F;meta.ath0.com&#x2F;2012&#x2F;02&#x2F;05&#x2F;apples-great-gpl-purge&#x2F;" rel="nofollow">http:&#x2F;&#x2F;meta.ath0.com&#x2F;2012&#x2F;02&#x2F;05&#x2F;apples-great-gpl-purge&#x2F;</a> (via <a href="https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=3559990" rel="nofollow">https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=3559990</a>), macOS was down to 16 GPL packages in macOS 10.12 from 47 in macOS 10.5.')
