Item(by='the_duke', descendants=None, kids=[25596679], score=None, time=1609433136, title=None, item_type='comment', url=None, parent=25594010, text='Rust has the best tooling for wasm and is easiest to get started with, in my opinion.<p>`rustup target add wasm32-unknown-unknown`<p>`cargo build --target=wasm32-unknown-unknown`<p>Is all you need to compile a Rust library to a `.wasm` file. (almost, you also need a setting in Cargo.toml:  [lib] crate-type = [&quot;cdylib&quot;, &quot;rlib&quot;] )<p>In the browser you can get access to DOM apis via web-sys.<p>The documentation has quite a lot of examples and info: <a href="https:&#x2F;&#x2F;rustwasm.github.io&#x2F;wasm-bindgen&#x2F;web-sys&#x2F;index.html" rel="nofollow">https:&#x2F;&#x2F;rustwasm.github.io&#x2F;wasm-bindgen&#x2F;web-sys&#x2F;index.html</a><p>wasm-pack (<a href="https:&#x2F;&#x2F;github.com&#x2F;rustwasm&#x2F;wasm-pack" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;rustwasm&#x2F;wasm-pack</a>) makes building code for the browser easy.')
