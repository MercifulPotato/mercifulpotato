Item(by='Kednicma', descendants=None, kids=None, score=None, time=1602424383, title=None, item_type='comment', url=None, parent=24729396, text='A TCB should be dozens of lines, not thousands. More code means more places for more bugs to hide.<p>My experience in Safe Haskell was that, if you have to ask each module individually whether it has a safety property, then you&#x27;ve already created too much work for yourself. Instead, require every module to structurally encode the desired invariant.<p>Or, in fewer words: If you want memory safety, don&#x27;t have `unsafe` blocks.')
