Item(by='wenbin', descendants=None, kids=[25130558], score=None, time=1605650087, title=None, item_type='comment', url=None, parent=25127371, text='It&#x27;s almost impossible to build a decent web search engine from scratch today (i.e., build your own index, fight SEO spams, tweak search result relevance...). The web is already so big and so complex. Otherwise Google won&#x27;t need to hire so many people to work on search alone.<p>If you didn&#x27;t start at the very early stage of tiny web (e.g., Google in 1996 as a research project) and grew with the web over the past 20+ years, or you don&#x27;t have super deep pocket (e.g., Microsoft Bing in mid 2000s), then it&#x27;s almost impossible to build a decent web search engine within a few years.<p>It&#x27;s possible to build vertical search engines on far smaller scale, far less complex, far less lucrative things that Google&#x2F;Microsoft has little interest today (e.g., recipes [2], podcasts [3], gifs [4]...)<p>It&#x27;s also possible to come up with a different discovery mechanism for web (or a small portion of web), other than a traditional complete web search engine. Essentially you don&#x27;t cross moat to attack a huge castle (e.g., Google).
