Item(by='dragontamer', descendants=None, kids=[24675187, 24676601], score=None, time=1601763789, title=None, item_type='comment', url=None, parent=24675039, text='I think you underestimate the domination of large numbers.<p>Let&#x27;s say you have code that is 50% main memory and 50% L1 cache. Let&#x27;s say there are 1000 operations.<p>You have (500 x 100ns) + (500 x 1ns) or 50500ns total time.<p>Now let&#x27;s say you optimize the code so that they are all L3 operations: 20ns x 1000 operations is 20000ns, or over twice as fast.')
