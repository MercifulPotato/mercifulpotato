Item(by='hardwaresofton', descendants=None, kids=None, score=None, time=1609131924, title=None, item_type='comment', url=None, parent=25551950, text='It looks like everyone has blissfully forgot that API access is often one of the first things to go&#x2F;get significantly limited once companies reach monopoly velocity (i.e. competition exit velocity).<p>APIs are awesome, but we need APIs that don&#x27;t break. I think that happens one of two ways:<p>- APIs built on scraping (EULAs make this untenable -- companies will just login-wall everything that isn&#x27;t already)<p>- Standards-compliant APIs that lock companies into complying to be able to even compete with the others that are standards-compliant (ex. if your API doesn&#x27;t comply with the SimpleAccountingV1 standard, no one would dare use it)')
