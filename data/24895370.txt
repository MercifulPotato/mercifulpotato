Item(by='_ph_', descendants=None, kids=None, score=None, time=1603715220, title=None, item_type='comment', url=None, parent=24888960, text='While sometimes it would be nice to have a source-code confirmation, that struct Foo implements the interfaces Bar and Baz, there is one huge benefit of not requiring to declare the interfaces a struct implements: a struct can implement an interface the struct creator doesn&#x27;t know about. So you can design new interfaces, which for example structs from the standard library or any third party library happen to implement. You don&#x27;t have to change the original libraries for that. This is quite a benefit for modularity.')
