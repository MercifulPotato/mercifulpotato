Item(by='jlokier', descendants=None, kids=None, score=None, time=1604667028, title=None, item_type='comment', url=None, parent=25007138, text='It sees the filesystem the way Cygwin maps it, so what&#x27;s transferred isn&#x27;t an exact match for Windows files and attributes.<p>For example, Windows symlinks and Cygwin symlinks are quite different.  I don&#x27;t think Cygwin rsync can replicate a directory containing Windows symlinks properly, so that the replica behaves the same as seen by Windows programs.<p>I&#x27;m not sure about the other Windows attributes such as ACLs but I would be surprised if Cygwin rsync can replicate those.')
