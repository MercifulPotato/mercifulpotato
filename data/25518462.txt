Item(by='dmitriid', descendants=None, kids=None, score=None, time=1608736946, title=None, item_type='comment', url=None, parent=25517500, text='YMMV but, almost every single GUI framework outside the web has:<p>- virtual lists<p>- customizable controls (including selects)<p>- easy solutions for layouts of any complexity (CSS got flexbox in 2015, Qt had it in early 2000s)<p>- a plethora of complex components out of the box (and most complex controls are rather trivial to implement yourself): date pickers, modals and dialog boxes, tree views, splitters, you name it.<p>Yes, building GUIs is hard. But building GUIs on the web is approaching <i>impossible</i>. There&#x27;s a reason why most GUI frameworks on the web only manage to implement the same paltry set of components: buttons, badges, inputs. Very rarely there&#x27;s a date picker and a table. And... that&#x27;s about it.')
