Item(by='ing33k', descendants=None, kids=None, score=None, time=1603014229, title=None, item_type='comment', url=None, parent=24813795, text='I run a replicated ClickHouse server setup, Clickhouse uses zookeeper to enable replication.\nThe zookeeper instance was not replicated.it was a single node.\nThe server on which zookeeper was running ran out of hard disk and Clickhouse went into read only mode.\nLuckily,no data was lost while this happened because we use RabbitMq to store the messages before it gets written to the db.\nThanks to RabbitMq&#x27;s ACK mechanism.')
