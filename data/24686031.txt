Item(by='londons_explore', descendants=None, kids=[24686094, 24686387, 24686059, 24690940, 24686376, 24690393], score=None, time=1601890749, title=None, item_type='comment', url=None, parent=24685998, text='People use excel sheets as databases all the time because it doesn&#x27;t require any specialist knowledge to build or use.<p>Sure, we techy types prefer a proper database with a web frontend and an API, but that requires significantly more skill to build than an excel file.<p>No database has the flexibility and flat learning curve that excel has.<p>Until a database manages to meet these goals, it remains a good tool for certain usecases.<p>We as engineers should be working towards making a real database thats just as easy to use as excel, and <i>only then</i> can we complain that people are using excel sheets for this kind of stuff.')
