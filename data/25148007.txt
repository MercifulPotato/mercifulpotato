Item(by='acqq', descendants=None, kids=None, score=None, time=1605781341, title=None, item_type='comment', url=None, parent=25144434, text='It‘s really a nice example of what can be achieved when one starts solving a specific problem and solves it by generalizing it to create something like an ad hoc language for that kind of reports. Additional bonus for designing it exactly so that it fits to the concepts which one gets already by design of awk.<p>The author explains the idea: “ The program keeps a running sum that it clears when it sees a blank line (NF==0). Other calculations are performed in place. The first occurrence of a variable name defines it as that sum. Subsequent occurrences become the stored value.”<p>The regexp that defines what the variable name is is simply: \n    &#x2F;^[A-Z]+[A-Z0-9]*$&#x2F; and numbers to sum and variables are always only the first word in the line.')
