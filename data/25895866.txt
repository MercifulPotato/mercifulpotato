Item(by='mikem170', descendants=None, kids=[25896892, 25896209], score=None, time=1611521434, title=None, item_type='comment', url=None, parent=25895165, text='A Thinkpad that runs OpenBSD!<p>OpenBSD developers mostly use Thinkpads, so everything works - suspend, hibernation, wifi, camera, etc. Packages for firefox, chrome, and all the usual other open source software are just a pkg_add command away. Two commands and a reboot to upgrade the system and all installed apps a couple times a year.<p>It might not be the right OS if you are a gamer, or you gotta have the latest phone-app-thingy, but if you&#x27;re a unix person and want a rock solid well organized and documented system that you can trust it&#x27;s bliss.')
