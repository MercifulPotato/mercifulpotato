Item(by='gpderetta', descendants=None, kids=[24921335], score=None, time=1603903842, title=None, item_type='comment', url=None, parent=24919607, text='This is quite unfair.<p>Boost.container::vector implements the full standard vector interface (that&#x27;s already a large interface), plus quite a few non-standard extensions (significantly it has built-in support for default initialization).<p>Boost.container also has support for stateful allocators and custom pointer types so that it can be used, for example, on shared memory (in fact boost container was originally a sub-library of boost.interprocess). That alone will increase the complexity significantly.<p>It also I believe still support pre-c++11 (and substandard post c++11) compilers, so there is a lot of compiler workarounds and emulation of c++11 features.<p>Also, being a template, the whole implementation is just in the _header_.')
