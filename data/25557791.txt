Item(by='rav', descendants=None, kids=[25557837], score=None, time=1609146266, title=None, item_type='comment', url=None, parent=25556286, text='I have Wine installed on my Linux system, which uses &quot;binfmt_misc&quot; to automatically run Windows-looking executable files through Wine. This causes an issue with Cosmopolitan, since it uses an output format that triggers binfmt_misc. I had to run this to disable binfmt_misc on my system:<p><pre><code>    sudo systemctl mask proc-sys-fs-binfmt_misc.automount\n</code></pre>\n...and this to disable it for just the current boot:<p><pre><code>    sudo bash -c &#x27;echo 0 &gt; &#x2F;proc&#x2F;sys&#x2F;fs&#x2F;binfmt_misc&#x2F;status&#x27;</code></pre>')
