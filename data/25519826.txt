Item(by='schmooser', descendants=None, kids=None, score=None, time=1608744195, title=None, item_type='comment', url=None, parent=25498364, text='I&#x27;ve been using jq for several years but still can&#x27;t write anything more complex than simple XPath without going to manual and googling.<p>Babashka[1] with Cheshire[2] aliased as json solves this to me, the code is longer but no need for googling anymore:<p><pre><code>  USERX=&#x27;{&quot;name&quot;:&quot;duchess&quot;,&quot;city&quot;:&quot;Toronto&quot;,&quot;orders&quot;:[{&quot;id&quot;:&quot;x&quot;,&quot;qty&quot;:10},{&quot;id&quot;:&quot;y&quot;,&quot;qty&quot;:15}]}&#x27;\n\n  echo $USERX | jq &#x27;.orders[]|select(.qty&gt;10)&#x27;\n  {\n    &quot;id&quot;: &quot;y&quot;,\n    &quot;qty&quot;: 15\n  }\n\n  echo $USERX | bb -i -o &#x27;(-&gt; *input* first (json&#x2F;decode true) (-&gt;&gt; :orders (filter #(&gt; (:qty %) 10))))&#x27;\n  {:id y, :qty 15}\n\n</code></pre>\n[1]: <a href="https:&#x2F;&#x2F;github.com&#x2F;borkdude&#x2F;babashka" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;borkdude&#x2F;babashka</a>\n[2]: <a href="https:&#x2F;&#x2F;github.com&#x2F;dakrone&#x2F;cheshire" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;dakrone&#x2F;cheshire</a>')
