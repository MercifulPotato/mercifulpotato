Item(by='shortercode', descendants=None, kids=None, score=None, time=1609608825, title=None, item_type='comment', url=None, parent=25613035, text='In an ideal world it probably would!<p>WebAssembly only provides very minimal capabilities, anything more complicated than basic arithmetic and linear memory access require the features to be built. malloc, closures, arrays, error throwing, GC etc. all must be implemented. Some language features like console logs and visual output physically aren&#x27;t possible, and must call out to the host ( normally JS ). Requiring the compiler to emit 2 languages.<p>In comparison JS has analogues to many common language features, making it easy to transpile to. There are other advantages to JS as well, but probably not worth listing.<p>It&#x27;s complicated, and emitting JS is easier.')
