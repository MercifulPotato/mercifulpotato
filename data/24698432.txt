Item(by='Jtsummers', descendants=None, kids=[24699799], score=None, time=1601996860, title=None, item_type='comment', url=None, parent=24697105, text='In SPARK&#x2F;Ada you&#x27;d have to constrain the input for something like that or it wouldn&#x27;t be able to discharge the proof. Suppose you wanted a simple function (which is too simple to write in reality, but for brevity):<p><pre><code>  function Increment(A : Integer) return Integer\n    with Post =&gt; Increment&#x27;Result = A + 1;\n</code></pre>\nIt would find the counterexample for the situation where A is equal to Integer&#x27;Last. So you&#x27;d need to add a precondition like:<p><pre><code>  function Increment(A : Integer) return Integer\n    with Pre =&gt; A &lt; Integer&#x27;Last,\n         Post =&gt; Increment&#x27;Result = A + 1;\n</code></pre>\n(doing this from memory and not fluent, so syntax may be off but the idea isn&#x27;t)')
