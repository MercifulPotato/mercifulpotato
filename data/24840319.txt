Item(by='optymizer', descendants=None, kids=None, score=None, time=1603216152, title=None, item_type='comment', url=None, parent=24839817, text='You can use packages that have been converted to not use the GC and all of the libraries you would be using in C or C++.<p>You still get the following language features, which is more than what C, C++ or Rust have to offer (though Rust is hot on D&#x27;s trail).<p><pre><code>   Unrestricted use of compile-time features\n   Full metaprogramming facilities\n   Nested functions, nested structs, delegates and lambdas\n   Member functions, constructors, destructors, operating overloading, etc.\n   The full module system\n   Array slicing, and array bounds checking\n   RAII (yes, it can work without exceptions)\n   scope(exit)\n   Memory safety protections\n   Interfacing with C++\n   COM classes and C++ classes\n   assert failures are directed to the C runtime library\n   switch with strings\n   final switch\n   unittest\n   printf format validation\n</code></pre>\n<a href="https:&#x2F;&#x2F;dlang.org&#x2F;spec&#x2F;betterc.html" rel="nofollow">https:&#x2F;&#x2F;dlang.org&#x2F;spec&#x2F;betterc.html</a>')
