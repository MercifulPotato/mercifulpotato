For normal, Web Mercator maps we just use the free download of the whole world, available at [1].  Running the OpenMapTiles tools to generate a current tileset for a particular region is straightforward, but if you need all of Europe you will need some decent compute capacity. Unless everything can be done on a single computer, you&#x27;ll also need to implement some missing pieces, e.g. a message system to distribute the rendering across your computers.<p>[1] <a href="https:&#x2F;&#x2F;data.maptiler.com&#x2F;downloads&#x2F;dataset&#x2F;osm&#x2F;" rel="nofollow">https:&#x2F;&#x2F;data.maptiler.com&#x2F;downloads&#x2F;dataset&#x2F;osm&#x2F;</a>')
