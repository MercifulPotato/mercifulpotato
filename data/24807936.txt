Item(by='black3r', descendants=None, kids=None, score=None, time=1602917443, title=None, item_type='comment', url=None, parent=24805717, text='for me the problem with C++ is not about the verbosity itself... it&#x27;s more about how generally incomprehensible maybe even hostile it is towards you if you lack experience compared to other languages I&#x27;ve worked with (JS, Python, Java, Go, ..):<p>- navigating through template compilation error messages is a pain.\n- standard library is getting better, but still feels lacking.\n- different commonly used libraries have different code styles and reimplement the same things in different ways.\n- combine the previous two and you get small-ish projects where you have 4 or more different types of strings (char<i>, const char</i>, std::string, QString, ...)\n- no standard build&#x2F;publish&#x2F;compile&#x2F;library distribution system makes using less-known external libraries a giant pain in the ass.')
