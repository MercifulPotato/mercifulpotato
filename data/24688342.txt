Forget about EMMC, it&#x27;s gone and not supported by anyone.<p>2.) Boot is only possible from microSD card. USB is not supported.<p>3.) Build BareBox with <a href="https:&#x2F;&#x2F;www.barebox.org&#x2F;doc&#x2F;latest&#x2F;boards&#x2F;rockchip.html" rel="nofollow">https:&#x2F;&#x2F;www.barebox.org&#x2F;doc&#x2F;latest&#x2F;boards&#x2F;rockchip.html</a><p>Use `rk-splitboot` to split FlashData from the original boot loader, other option doesn&#x27;t work. Build it, it will create barebox-radxa-rock.img. It&#x27;s <i>.img, not the </i>.bin.<p>Write an SD card as in the instruction.<p>4.) Follow these instructions, but use the generic image.<p><a href="https:&#x2F;&#x2F;archlinuxarm.org&#x2F;platforms&#x2F;armv7&#x2F;amlogic&#x2F;odroid-c1" rel="nofollow">https:&#x2F;&#x2F;archlinuxarm.org&#x2F;platforms&#x2F;armv7&#x2F;amlogic&#x2F;odroid-c1</a><p><a href="http:&#x2F;&#x2F;os.archlinuxarm.org&#x2F;os&#x2F;ArchLinuxARM-armv7-latest.tar.gz" rel="nofollow">http:&#x2F;&#x2F;os.archlinuxarm.org&#x2F;os&#x2F;ArchLinuxARM-armv7-latest.tar....</a><p>5.) Build kernel as in this tutorial, ignore the `dts` stuff. The `dts` comes with the kernel sources.<p><a href="https:&#x2F;&#x2F;wiki.radxa.com&#x2F;Rock&#x2F;Linux_Mainline" rel="nofollow">https:&#x2F;&#x2F;wiki.radxa.com&#x2F;Rock&#x2F;Linux_Mainline</a><p><a href="https:&#x2F;&#x2F;github.com&#x2F;torvalds&#x2F;linux&#x2F;blob&#x2F;master&#x2F;arch&#x2F;arm&#x2F;boot&#x2F;dts&#x2F;rk3188-radxarock.dts" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;torvalds&#x2F;linux&#x2F;blob&#x2F;master&#x2F;arch&#x2F;arm&#x2F;boot&#x2F;...</a><p>6.) Copy the zImage, rk3188-radxarock.dts to the &#x2F;boot and plug in the UART.<p>7.) sudo picocom &#x2F;dev&#x2F;ttyUSB1 -b 115200 -e w<p>8.) Boot in the bootloader and execute the following commands:<p>global.bootm.appendroot=true<p>global.bootm.image=&#x2F;mnt&#x2F;mshc1.0&#x2F;boot&#x2F;zImage<p>global.bootm.oftree=&#x2F;mnt&#x2F;mshc1.0&#x2F;boot&#x2F;dtbs&#x2F;rk3188-radxarock.dtb<p>bootm<p>9.) If it boots, you can make it permanent by following the BareBox docs.')
