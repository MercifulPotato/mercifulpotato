Item(by='finderdotcool', descendants=None, kids=None, score=None, time=1610207750, title=None, item_type='comment', url=None, parent=25701313, text='The stack is pure browser side JavaScript (not even jQuery) + Linode Java servers (no Spring&#x2F;JBoss&#x2F;Hibernate&#x2F;Play&#x2F;nothing) which use just the NoQMS microservice framework found at <a href="https:&#x2F;&#x2F;github.com&#x2F;noqms&#x2F;noqms" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;noqms&#x2F;noqms</a> + Amazon S3 for site pages and images. Amazon hosted postgreSQL database. It turned out to be quite a bit simpler, organized, and maintainable than I ever imagined. The microservice framework gives scalability. Failover and availablility is provided via simple client side failover to alternate servers, accommodated with completely stateless services.')
