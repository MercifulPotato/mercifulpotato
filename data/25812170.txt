There is no universally valid way to measure understandability and any note can be subdivided so many different ways.<p>&gt; Each note should contain a single idea and should be understandable when reading in isolation.<p>It&#x27;s really a guiding principle. Unlike index cards in filing drawers, markdown with wikilinks allows for infinitely long notes and a low cost for context switching. you can choose whatever balance of note size&#x2F;quantity you want.<p>In my markdown-based second brain built originally in Obsidian and now in Dendron, I have around 350 notes with what is probably a Poisson distribution of sizes. The important thing is that jumping from note to note costs very little.<p>further, if a note warrants splitting, now you can do it as fast as you can think it.
