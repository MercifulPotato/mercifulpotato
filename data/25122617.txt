Item(by='account42', descendants=None, kids=None, score=None, time=1605613508, title=None, item_type='comment', url=None, parent=25108609, text='You can still have thrashing without a swap partition&#x2F;file: as memory pressure increases, the kernel will flush disk cache and buffers, slowing down operations that need them again. In the extreme case this can mean the program code you are executing being evicted from ram only to be re-read page by page for many times until the kernel finally decides to kill whatever was eating all the ram.')
