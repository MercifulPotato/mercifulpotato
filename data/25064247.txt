Item(by='stevesimmons', descendants=None, kids=None, score=None, time=1605137129, title=None, item_type='comment', url=None, parent=25061918, text='many-requests is the name the package was registered with on pypi.org.  So you install with: pip install many-requests.<p>That installation creates a folder named many_requests inside the site-packages directory on your PYTHON_PATH. It uses underscore rather than hyphen because modules&#x2F;packages must be legal Python names. A hyphen would be parsed as subtraction.')
