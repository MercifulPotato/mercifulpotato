Item(by='temuze', descendants=None, kids=[24725417], score=None, time=1602199277, title=None, item_type='comment', url=None, parent=24719722, text='I agree that AWS Parameter store isn&#x27;t great... that&#x27;s why we use Chamber:\n<a href="https:&#x2F;&#x2F;github.com&#x2F;segmentio&#x2F;chamber" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;segmentio&#x2F;chamber</a><p>Chamber is an open source wrapper for AWS Parameter Store. You add a secret by doing:<p>`chamber write ENV_NAME SECRET_NAME SECRET_VALUE`<p>And when you want to execute a command with those values in your environment...<p>`chamber exec ENV_NAME -- yarn start:prod`<p>Parameter Store comes with auditing tools and versioning. Chamber uses KMS for encryption. Also, you can control permissions (including namespacing) with IAM.<p>My question is - what does Doppler provide that would make me want to switch? I&#x27;m weary about letting someone else own our secrets.')
