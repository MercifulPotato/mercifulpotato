Item(by='Spivak', descendants=None, kids=[24708402, 24710115, 24708283], score=None, time=1602078378, title=None, item_type='comment', url=None, parent=24708015, text='I don&#x27;t think it would be infringement. The main difference between WINE and Android is where the interoperability integration points are.<p>WINE is using the Windows names for the purposes of allowing Windows programs to run. This is a case where a judge would easily grant fair-use because you have to use the API surface area to provide compatibility for existing programs.<p>Android is using the Java names for the purposes of allowing existing Java <i>developers</i> to be productive, not so that existing Java programs can run.')
