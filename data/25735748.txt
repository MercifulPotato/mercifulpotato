Item(by='vel0city', descendants=None, kids=None, score=None, time=1610395821, title=None, item_type='comment', url=None, parent=25721374, text='There&#x27;s been some changes and refinements since Windows 3.1. The current default file chooser dialogue shipped with Windows allows you to put in a full path into both the top file path or in the Filename dialog.<p>Reference photo: <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;File_dialog#&#x2F;media&#x2F;File:An_%22open%22_file_dialog_from_a_web_browser_on_Windows_10.jpg" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;File_dialog#&#x2F;media&#x2F;File:An_%22...</a><p>In this, a full path can be pasted in to both the top bar where it says &quot;This PC  &gt; WINDOWS (C:)&quot; and also into the blank &quot;File name: |        |&quot; box.<p>I&#x27;m pretty sure I remember this dialogue being available since NT days. It seems fairly discoverable to me, clicking in the empty space of the current working directory path changes it to show the text path of the directory. The &quot;File name&quot; box offers autocomplete if you did something like &quot;C:&quot; as a file name, or started typing a folder name of the current working directory.')
