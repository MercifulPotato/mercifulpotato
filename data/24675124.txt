You just define a message for your configuration schema:<p><pre><code>  message Config {\n    repeated Server server = 1;\n  }\n\n  message Server {\n    string address = 1;\n    int32 port = 2;\n    bool standby = 3;\n  }\n</code></pre>\nAnd then you use the text representation in a config file:<p><pre><code>  # main instance\n  server { address: &quot;127.0.0.1&quot; port: 4567 }\n  # backup instance\n  server { address: &quot;127.0.0.1&quot; port: 9876 standby: true }\n</code></pre>\nAnd load it into a message instance:<p><pre><code>  Config config;\n  google::protobuf::TextFormat::ParseFromString(input, &amp;config);</code></pre>')
