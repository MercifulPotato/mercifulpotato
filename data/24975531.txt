Item(by='klodolph', descendants=None, kids=None, score=None, time=1604367987, title=None, item_type='comment', url=None, parent=24975430, text='It&#x27;s deterministic, you&#x27;ve just changed the inputs. The &quot;committer date&quot; is usually excluded from logs so you don&#x27;t see it, but it&#x27;s the only missing input. Here is a demo:<p><pre><code>    $ git rev-parse HEAD \n    78d66a4f169fec9cb9b3252f7a22bb020e967cd2\n\n    $ git log HEAD      \n    commit 78d66a4f169fec9cb9b3252f7a22bb020e967cd2 (HEAD -&gt; master)\n    Author: XXXX\n    Date:   Mon Nov 2 20:44:04 2020 -0500\n\n        temp2\n\n    $ git reset --soft HEAD\\^\n\n    $ GIT_COMMITTER_DATE=&#x27;Mon Nov 2 20:44:04 2020 -0500&#x27; GIT_AUTHOR_DATE=&#x27;Mon Nov 2 20:44:04 2020 -0500&#x27; git commit -m temp2\n    [master 78d66a4] temp2\n     1 file changed, 0 insertions(+), 0 deletions(-)\n     create mode 100644 b\n\n    $ git rev-parse HEAD\n    78d66a4f169fec9cb9b3252f7a22bb020e967cd2</code></pre>')
