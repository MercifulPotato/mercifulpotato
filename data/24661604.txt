Item(by='aesh2Xa1', descendants=None, kids=[24662320], score=None, time=1601639056, title=None, item_type='comment', url=None, parent=24661498, text='PPA (personal package archive) is the Ubuntu name, but other distributions make it possible to do the same. The system is a way to distribute packages through the package manager, and repository metadata is hosted centrally.<p>Fedora has such repositories (RPM Fusion, Copr). Arch does, too (AUR). Other distributions, including Debian, can use them as well, so long as there exists a community.')
