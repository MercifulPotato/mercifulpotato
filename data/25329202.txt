Item(by='mtVessel', descendants=None, kids=None, score=None, time=1607313470, title=None, item_type='comment', url=None, parent=25327385, text='My favorite feature of logical names was that they didn&#x27;t just point to directories -- they could be a search path!<p>To test new code, all I had to do was create a directory to hold my changes, and copy in just the files I wanted to update.  Set the logical name aliasing the code location to search first in my &quot;branch&quot; directory, then in the main directory.  Any code in found the top-level search location takes precedence over the main directory.<p>Boom! Instant branches, but supported at the OS level.')
