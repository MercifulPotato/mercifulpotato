Item(by='ubercow13', descendants=None, kids=None, score=None, time=1605612414, title=None, item_type='comment', url=None, parent=25121647, text='It&#x27;s like shorthand for git checkout &lt;current branch&gt; -- &lt;files&gt;. You&#x27;re only omitting the current branch which is the default. I&#x27;m not sure if it&#x27;s intuitive but it seems pretty coherent.<p>edit: on second thought no it&#x27;s really not very coherent. If you exclude &lt;files&gt; it means checkout <i>all</i> files if you write a &lt;branch&gt;, but no files if you write no &lt;branch&gt;. So excluding &lt;branch&gt; doesn&#x27;t consistently do the same as using &lt;current branch&gt;. I guess git does kind of suck.')
