Item(by='hut0', descendants=None, kids=None, score=None, time=1607371073, title=None, item_type='comment', url=None, parent=25334087, text='I use transform feedback, which allows vertex shaders to send data back to the application.  You could do this with textures as well, as described by pjmlp.  I did this in an earlier version (<a href="https:&#x2F;&#x2F;github.com&#x2F;hut&#x2F;cellmade#" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;hut&#x2F;cellmade#</a>) but it just felt SO WRONG. I was very happy that I found out about transform feedback to be able to do it more cleanly.<p><a href="https:&#x2F;&#x2F;developer.mozilla.org&#x2F;en-US&#x2F;docs&#x2F;Web&#x2F;API&#x2F;WebGLTransformFeedback" rel="nofollow">https:&#x2F;&#x2F;developer.mozilla.org&#x2F;en-US&#x2F;docs&#x2F;Web&#x2F;API&#x2F;WebGLTransf...</a><p>&gt; The WebGLTransformFeedback interface is part of the WebGL 2 API and enables transform feedback, which is the process of capturing primitives generated by vertex processing. It allows to preserve the post-transform rendering state of an object and resubmit this data multiple times.')
