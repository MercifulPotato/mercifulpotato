Item(by='btschaegg', descendants=None, kids=None, score=None, time=1601760725, title=None, item_type='comment', url=None, parent=24674683, text='I just did some experiments, and I think this should help identify such merge commits from the commit IDs of two copies:<p>Run this in bash (if you&#x27;re using Windows, use &quot;Git Bash&quot;). Make sure you replace the `&lt;hash&gt;` tokens with the appropriate commit IDs. The hash it prints should be the merge commit tying the two copies together.<p><pre><code>  commit_1=&lt;hash&gt;\n  commit_2=&lt;hash&gt;\n  \n  awk &#x27;\n          ARGIND == 1 { h[$1]++ }\n          ARGIND == 2 &amp;&amp; h[$1] { last = $1 }\n          END { print last }\n      &#x27; \\\n      &lt;(git log --ancestry-path --format=%H $commit_1..HEAD) \\\n      &lt;(git log --ancestry-path --format=%H $commit_2..HEAD)</code></pre>')
