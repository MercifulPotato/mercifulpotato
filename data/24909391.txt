Here is a directory I made with 1000+ pages, prerendered and hosted for free on netlify: <a href="https:&#x2F;&#x2F;www.givepuppylove.com&#x2F;organizations&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.givepuppylove.com&#x2F;organizations&#x2F;</a> . I&#x27;ve found that single digit thousands is about the maximum you can currently have and keep build times reasonable.<p>If you want to recognize the user&#x27;s location and display the appropriate currency or language, you would probably want to wrap content dependent on location in a &quot;ClientOnly&quot; type component. So it would not be prerendered in that case. Example code:<p><pre><code>  export default function ClientOnly({ children, ...delegated }) {\n    const [hasMounted, setHasMounted] = useState(false)\n    useEffect(() =&gt; { setHasMounted(true) }, [])\n\n    return hasMounted ? &lt;div {...delegated}&gt;{children}&lt;&#x2F;div&gt; : &lt;div&gt;&lt;&#x2F;div&gt;\n  }\n</code></pre>\nThis allows you to grab things off the &quot;window&quot; and change behavior based on that info, which you can&#x27;t do with prerendering.')
