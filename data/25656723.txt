Item(by='michaelt', descendants=None, kids=[25658206], score=None, time=1609932228, title=None, item_type='comment', url=None, parent=25656113, text='Naturally DHCP is handled by the original network when the original network can be seen, yes!<p>But if the original network is unavailable - perhaps the SSID has changed for example - you need to be able to get to the repeater&#x27;s configuration page somehow. And the user can&#x27;t google their problems or download any configuration tools while their connection is down, so you&#x27;d better make it simple. So many devices like this will be transparent under normal operation, but if there&#x27;s a network problem will start their own DHCP server.')
