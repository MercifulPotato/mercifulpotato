Item(by='mcaruso', descendants=None, kids=None, score=None, time=1602167089, title=None, item_type='comment', url=None, parent=24718827, text='Let&#x27;s take a look at how we might write a function like `map` the (pure) functional way in a language like JS:<p><pre><code>    const head = ([x, ...xs]) =&gt; x;\n    const tail = ([x, ...xs]) =&gt; xs;\n    \n    const map = (list, fn) =&gt; {\n      if (list.length === 0) {\n        return [];\n      } else {\n        return [fn(head(list)), ...map(tail(list), fn)];\n      }\n    };\n    \n    map([1, 2, 3], x =&gt; x + 1); &#x2F;&#x2F; [2, 3, 4]\n</code></pre>\nWe&#x27;re not keeping track of any state here. Using recursion you don&#x27;t need to keep track of the current element in the list for example (when you run this on a physical machine it <i>will</i> of course, but not at the conceptual level).')
