Item(by='alexfrydl', descendants=None, kids=[25993619], score=None, time=1612209828, title=None, item_type='comment', url=None, parent=25991563, text='I hate the logic that because people have successfully used something, it is good. Having used MongoDB extensively, it is absolutely true that MongoDB is powerful and useful. However, it is also true that I would never choose it over something else for a new project. It has likely improved since 2013, but so has everything else.<p>Certainly I could endeavor to build amazing modern software in C, but unless for some reason C is an absolute must, I would rather try any other language first. I doubt this is a controversial statement, and yet someone will always be there to defend the opposite stance.')
