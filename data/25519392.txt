Item(by='oarsinsync', descendants=None, kids=None, score=None, time=1608741609, title=None, item_type='comment', url=None, parent=25519135, text='To emphasise, using ECC RAM is another step that can be used to increase the integrity of your data.<p>It’s like security - it’s done in layers.<p>Using a checksumming filesystem without ECC RAM is an improvement on not using a checksumming filesystem at all. Do not consider ECC RAM to be a requirement for ZFS &#x2F; BTRFS.<p>It does add another layer of improvement though, so if you can, you should!<p>And even if you can’t, and even if you can’t provide the redundancy to enable bitflips to be resolved, still use a checksumming filesystem if you can!')
