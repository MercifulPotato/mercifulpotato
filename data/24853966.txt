I fix it by running this script once a month (it deletes the temporary containers from the containers.json file):<p><pre><code>    import json\n\n    with open(&#x27;containers.json.bac&#x27;) as f:\n        s = f.read()\n        j = json.loads(s)\n        ids = [i for i in j[&#x27;identities&#x27;] if not i[&#x27;name&#x27;].startswith(&#x27;tmp&#x27;)]\n        print(len(j[&#x27;identities&#x27;]), len(ids))\n        j[&#x27;identities&#x27;] = ids\n        with open(&#x27;containers.json&#x27;, &#x27;w&#x27;) as wf:\n            wf.write(json.dumps(j))</code></pre>')
