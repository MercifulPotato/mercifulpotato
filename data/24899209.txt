Item(by='rodw', descendants=None, kids=None, score=None, time=1603736038, title=None, item_type='comment', url=None, parent=24898372, text='Self-plug warning but FWIW if you&#x27;re using cheerio _just_ for the selector syntax a related tool is Stew [1] which is a dependency-free [2] node module that allows one to extract content from web pages (DOM trees) using CSS selectors, like:<p>var links = stew.select(dom,&#x27;a[href]&#x27;);<p>extended with support for embeded regular expressions (for tags, classes, IDs, attributes or attribute values). E.g.:<p>var metadata = stew.select(dom,&#x27;head meta[name=&#x2F;^dc\\.|:&#x2F;i]&#x27;);<p>It&#x27;s on npm as `stew-select`<p>[1] <a href="https:&#x2F;&#x2F;github.com&#x2F;rodw&#x2F;stew&#x2F;" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;rodw&#x2F;stew&#x2F;</a><p>[2] there&#x27;s an optional peer-dependency-ish relationship to htmlparser or htmlparser2 or similar to generate a DOM tree from raw HTML but anything that creates a basic DOM tree (`{type:, name:, children:[] }`) will suffice')
