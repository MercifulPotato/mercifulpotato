Item(by='marcan_42', descendants=None, kids=None, score=None, time=1609884066, title=None, item_type='comment', url=None, parent=25650744, text='Worth noting that that OSS release does not include M1 support (and Apple does not open source most of their drivers, just the kernel core), although it does include (partially redacted, not buildable, because Apple) support for other mobile chips using the same CPU cores. Perhaps the next OSS dump will finally have the M1 bits. But we&#x27;re definitely on our own for all of the fun stuff beyond &quot;Linux boots with a serial tty&quot;.<p>At least some really basic parts are useful as a hardware reference, e.g. things related to the interrupt controller, UARTs, and CPU quirks&#x2F;errata workarounds. That said, their license is incompatible with the GPL, so we cannot take any code directly.
