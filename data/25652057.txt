Item(by='claytonjy', descendants=None, kids=None, score=None, time=1609885604, title=None, item_type='comment', url=None, parent=25650140, text='I do very similarly, except I avoid installing jupyter lab for each project, instead installing `ipykernel` as a dev dependency. I install jupyter lab system-wide with pipx [1], and for each project issue a command like<p><pre><code>    pipenv run python -m ipykernel install --user --name=this_directory\n</code></pre>\nThen if I open Jupyter Lab I see &quot;this_directory&quot; as a listed kernel to create a notebook from.<p>This allows me to manage Jupyter settings and plugins in one place rather than in each env, have multiple project&#x27;s notebooks open in the same Jupyter Lab instance, etc.<p>[1]<a href="https:&#x2F;&#x2F;pipxproject.github.io&#x2F;pipx&#x2F;" rel="nofollow">https:&#x2F;&#x2F;pipxproject.github.io&#x2F;pipx&#x2F;</a>')
