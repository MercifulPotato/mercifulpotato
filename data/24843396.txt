Item(by='burntsushi', descendants=None, kids=[24843480, 24843435], score=None, time=1603237007, title=None, item_type='comment', url=None, parent=24843200, text='That&#x27;s not quite true... My blog post there says mmap is slower in some cases (where you&#x27;re searching a lot of files), but if you&#x27;re just searching a single file, mmap does give a bit of a speed boost:<p><pre><code>    $ time rg &#x27;Sherlock Holmes&#x27; OpenSubtitles2018.raw.en -c --mmap\n    7673\n\n    real    1.754\n    user    1.235\n    sys     0.516\n    maxmem  12510 MB\n    faults  0\n\n    $ time rg &#x27;Sherlock Holmes&#x27; OpenSubtitles2018.raw.en -c --no-mmap\n    7673\n\n    real    2.663\n    user    1.130\n    sys     1.530\n    maxmem  6 MB\n    faults  0\n</code></pre>\nSee the sibling comment for a link to a bit more discussion and some interesting observations! Namely, it looks like using mmap can be slower when searching a lot of files because of CPU cache effects.')
