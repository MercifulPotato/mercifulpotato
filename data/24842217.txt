Item(by='shepik', descendants=None, kids=[24842443], score=None, time=1603226823, title=None, item_type='comment', url=None, parent=24841811, text='The deal with multiple dispatch, as I understand it, is that instead of obj-&gt;somemethod, you define function named somemethod, which is defined for obj.\nMaybe some other code calls it.<p>You can then define somemethod for other data types, without touching the original definition, and that makes your new definition available to the other code that used the original somemethod.\nIt is a unique feature of Julia (at least unique in a sense that it is absent from mainstream languages)')
