Item(by='JamesBarney', descendants=None, kids=None, score=None, time=1601966556, title=None, item_type='comment', url=None, parent=24694235, text='Runtimes -&gt; This runs the IL code, includes Mono, CLR, and CoreCLR.  Mono was written by Miguel Icaza to support cross platform mobile development.  CoreCLR is a new version of the CLR that supports .NET core code.<p>Libaries&#x2F;Frameworks -&gt; .NET core, .NET framework.  Two different versions of the libraries, .NET Core is newer and cross platform but older technologies such as webforms can&#x27;t run on it, .NET framework supports older technologies but isn&#x27;t cross platform.  Plenty of libraries are usable inside either framework.<p>Ahead of Time Compilation Technologies -&gt; .Net Native, CoreRT, AOT<p>Most new devs don&#x27;t have to know any of this to be productive.  Just pick .NET core unless you need to use .NET Framework for backwards compatibility reasons.')
