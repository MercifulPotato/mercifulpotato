Item(by='PaulHoule', descendants=None, kids=[25883650, 25882845, 25883844, 25882679, 25882610], score=None, time=1611412815, title=None, item_type='comment', url=None, parent=25881911, text='I like mmap and I don&#x27;t.<p>It is incompatible with non-blocking I&#x2F;O since your process will be stopped if it tries to access part of the file that is not mapped -- this isnt a syscall blocking (which you might work around) but rather any attempt to access mapped memory.<p>I like mmap for tasks like seeking into ZIP files,  where you can look at the back 1% of the file,  then locate and extract one of the subfiles;  the trouble there is that the really fun case is to do this over the network with http (say to solve Python dependencies,  to extract the metadata from wheel files) in which case this method doesnt work.')
