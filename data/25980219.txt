This thanks to an almost complete classpath library (based on Apache Harmony) and a macro system [1] that allows to produce POJO to JSON conversion code at compile time.<p>[1] <a href="http:&#x2F;&#x2F;teavm.org&#x2F;docs&#x2F;runtime&#x2F;metaprogramming.html" rel="nofollow">http:&#x2F;&#x2F;teavm.org&#x2F;docs&#x2F;runtime&#x2F;metaprogramming.html</a>')
