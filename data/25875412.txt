Item(by='majormajor', descendants=None, kids=[25875680], score=None, time=1611344538, title=None, item_type='comment', url=None, parent=25875289, text='If you&#x27;re porting between two languages&#x2F;frameworks with radically different approaches, preserving the original project&#x27;s approach in the new code is going to give you something that&#x27;s probably <i>worse</i> than the original!<p>Now when you hire someone new you&#x27;re going to be looking for React experience but then having to re-train them to think in .NET, or looking for .NET experience but training them to write React... and any time you have dependency upgrades, you&#x27;re gonna be fighting more impedance mismatches.<p>Most tech debt that slows down feature development has more to do with the code structure than with the language or framework, so if you&#x27;re intent on preserving the structure and style I don&#x27;t know that you should be doing a rewrite at all!')
