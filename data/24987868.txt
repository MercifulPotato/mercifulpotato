Item(by='dmitriid', descendants=None, kids=[24988513], score=None, time=1604485959, title=None, item_type='comment', url=None, parent=24987549, text='Libraries and library-like code. Any code that fetches data from generic storage&#x2F;protocols.<p>Prime examples:<p>- http requests. For an API it&#x27;s almost always a generic request parametrized by some type.<p>For example, you API always returns `{result: ...some data..., nextPageToken: ...}`. Well, that&#x27;s a `PagedResponse&lt;...the type of data...&gt;`<p>- cache<p>Caches store objects. In go any `.get` from a cache will return an `interface{}` that you have to type-assert because you can&#x27;t do a `Cache&lt;T&gt;`.<p>and so on.')
