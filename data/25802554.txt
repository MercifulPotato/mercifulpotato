Item(by='marcinzm', descendants=None, kids=None, score=None, time=1610808936, title=None, item_type='comment', url=None, parent=25801260, text='Let&#x27;s say you&#x27;re loading data from Postgres into Snowflake. Pretty simple integration ETL. You need to:<p>* Make sure you&#x27;re only loading incremental updates<p>* Make sure you don&#x27;t miss any data while doing incremental updates. This includes deleted rows.<p>* Update the Snowflake table schemas as Postgres schemas change. If this is impossible you need to alert someone.<p>* Keep historical metrics so you know if the job is slow or too much data or whatever.<p>Now do this for twenty different data sources every hour.')
