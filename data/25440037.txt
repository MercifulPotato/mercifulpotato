I set up a local test and <a href="http:&#x2F;&#x2F;mytest.localhost" rel="nofollow">http:&#x2F;&#x2F;mytest.localhost</a> works without having to add an entry to &#x2F;etc&#x2F;hosts, but <a href="https:&#x2F;&#x2F;mytest.localhost" rel="nofollow">https:&#x2F;&#x2F;mytest.localhost</a> with a self-signed TLS certificate will still raise a SEC_ERROR_UNKNOWN_ISSUER error. You still need to ignore those or install your own root CA with something like <a href="https:&#x2F;&#x2F;github.com&#x2F;FiloSottile&#x2F;mkcert" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;FiloSottile&#x2F;mkcert</a> for that to work seamlessly.')
