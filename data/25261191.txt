Item(by='grovesNL', descendants=None, kids=None, score=None, time=1606785948, title=None, item_type='comment', url=None, parent=25260441, text='There are lots of options for graphics in Rust, so it&#x27;s really just a choice of which graphics API you&#x27;d like to use or learn more about.<p>(full disclosure: I contribute to some of these crates)<p>For example, there is an implementation of WebGPU (<a href="https:&#x2F;&#x2F;github.com&#x2F;gfx-rs&#x2F;wgpu-rs" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;gfx-rs&#x2F;wgpu-rs</a>) that can run both inside and outside of the browser that has been growing in popularity lately.<p>There are also bindings for Vulkan (ash, erupt), D3D12 (d3d12-rs), Metal (metal-rs), cross-platform abstractions at various levels (gfx-hal, wgpu), OpenGL (glow, glium), etc.<p>It&#x27;s also possible to simply render graphics to an image (a pretty common approach for Raytracing in One Weekend) or blit pixels onto a window surface.')
