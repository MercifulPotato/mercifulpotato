Item(by='ryukafalz', descendants=None, kids=None, score=None, time=1606679546, title=None, item_type='comment', url=None, parent=25243659, text='I run Guix System on my laptop. To change my system configuration (think installed packages, installed services, users, filesystem mounts, etc) I would run:<p>&#x27;guix system reconfigure ~&#x2F;guix&#x2F;system.scm&#x27;<p>...after editing that system config file. To build an ISO image from the same configuration, I would run:<p>&#x27;guix system disk-image ~&#x2F;guix&#x2F;system.scm&#x27;<p>...which would build a bootable ISO image of the same configured system. (Obviously I&#x27;d need to change things like the filesystem mounts to do this as they&#x27;re specific to my laptop; it&#x27;s possible to make more generic configs and inherit them per-machine but I only have one so haven&#x27;t bothered yet.)
