Item(by='flohofwoe', descendants=None, kids=[24941569, 24941342], score=None, time=1604058358, title=None, item_type='comment', url=None, parent=24940650, text='I&#x27;m a big fan of WASM to run my cross-platform code in browsers, but everytime I see it pitched as wholesale replacement for Javascript frameworks which sit on top of the DOM I flinch a bit.<p>This is essentially the reverse of &quot;with Electron you can now create native applications entirely in Javascript&quot;. Yes it works, but only with a lot of overhead that wouldn&#x27;t exist by doing it the &quot;straightforward&quot; way.<p>A much smarter approach is to create hybrid JS&#x2F;WASM applications, manage the UI in Javascript, move isolated standalone features (cross-platform code or performance-sensitive code) into WASM modules, and call into those WASM modules from JS.')
