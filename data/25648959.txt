Item(by='grey-area', descendants=None, kids=[25649923], score=None, time=1609871257, title=None, item_type='comment', url=None, parent=25647733, text='The Go authors have (ab)used comments to extend the language in experimental ways for code generation, build pragmas and recently file embedding.<p>They’ve done this to avoid breaking the language stability guarantee, not sure if there were other reasons.  I think it’s a mistake personally to create this sort of metalanguage in comments (now you have two problems!) but it’s easy to avoid using this most of the time as the uses are pretty esoteric.')
