Item(by='DaiPlusPlus', descendants=None, kids=[25893713], score=None, time=1611504854, title=None, item_type='comment', url=None, parent=25890787, text='Some things aren’t expressable in winmd though - like Code Contracts’ preconditions, invariants, and postconditions: those require directly executable (but pure, stateless, and side-effect-free) methods to represent and encode those contracts. As Winmd binaries can’t contain executable code it means you’re limited to only using “common”&#x2F;“known” contract functions and&#x2F;or resorting to re-encoding the contract to a string containing the expression that’s stored in an attribute. Good luck with that.')
