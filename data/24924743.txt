Item(by='delsarto', descendants=None, kids=[24926005], score=None, time=1603923454, title=None, item_type='comment', url=None, parent=24919826, text='The git-review [1] tool makes it trivial to interface with Gerrit (it&#x27;s likely packaged for your distro, see [2]).  I&#x27;ve found many people struggling with Gerrit don&#x27;t know about it and it has made their life significantly easier.  It handles all the magic of pushing to refs so that you never need to know about it.  You drop a .gitreview in your project and then your work-flow is literally<p><pre><code>    $ git checkout -b my-feature-branch\n    $ emacs ... &lt;edit edit edit&gt;\n    $ git add -i ...\n    $ git commit\n    $ git review\n     read reviews, edit\n    $ git add -i ...\n    $ git commit --amend\n    $ git review # push new change revisions\n</code></pre>\nYou can download an upstream change to use locally with &quot;git review -d 123456&quot;<p>[1] <a href="https:&#x2F;&#x2F;docs.openstack.org&#x2F;infra&#x2F;git-review&#x2F;" rel="nofollow">https:&#x2F;&#x2F;docs.openstack.org&#x2F;infra&#x2F;git-review&#x2F;</a>\n[2] <a href="https:&#x2F;&#x2F;www.mediawiki.org&#x2F;wiki&#x2F;Gerrit&#x2F;git-review" rel="nofollow">https:&#x2F;&#x2F;www.mediawiki.org&#x2F;wiki&#x2F;Gerrit&#x2F;git-review</a>')
