Item(by='piaste', descendants=None, kids=None, score=None, time=1605195876, title=None, item_type='comment', url=None, parent=25070480, text='&gt; There are pretty much no tutorials, no recent articles and no performant native F# libraries out there.<p>Giraffe [1] is the de facto standard F# web framework. It&#x27;s a plugin on top of ASP.NET (for all the reasons described elsewhere - performance, security, ecosystem) that lets you write functional-style &#x27;route &gt;=&gt; function&#x27; code without controllers.<p>If you want a more batteries-included approach, Saturn [2] builds on top of Giraffe to provide, well, more backend batteries (database layer, etc.)<p>Finally, the SAFE stack [3] are a set of project templates that combine the above with a F# frontend (via the Fable JS compiler) and some Azure deployment helpers.<p>[1] <a href="https:&#x2F;&#x2F;giraffe.wiki&#x2F;docs" rel="nofollow">https:&#x2F;&#x2F;giraffe.wiki&#x2F;docs</a><p>[2] <a href="https:&#x2F;&#x2F;saturnframework.org" rel="nofollow">https:&#x2F;&#x2F;saturnframework.org</a><p>[3] <a href="https:&#x2F;&#x2F;safe-stack.github.io&#x2F;" rel="nofollow">https:&#x2F;&#x2F;safe-stack.github.io&#x2F;</a>')
