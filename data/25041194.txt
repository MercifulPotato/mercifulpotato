Item(by='tenebrisalietum', descendants=None, kids=None, score=None, time=1604964182, title=None, item_type='comment', url=None, parent=25039981, text='- You run into copyright issues, because unless you duplicate a commercial UEFI (SMBIOS and all that good stuff) within the VM you&#x27;ll be detected.<p>- You run into performance issues - virtualization&#x27;s speed depends on paravirtualized drivers, which are readily identifiable (e.g. &quot;virtio&quot; disk controller).  These can be emulated instead of paravirtualized but will cost performance.  Windows drivers are signed so you can&#x27;t just change names of things.<p>- Chipsets - some are often emulated by VMs.  For example, if you are running a system in 2020 and are using an Intel 440BX chipset, it&#x27;s probably a VM.')
