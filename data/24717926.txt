Item(by='flohofwoe', descendants=None, kids=[24718024, 24718035], score=None, time=1602154796, title=None, item_type='comment', url=None, parent=24717727, text='The problem is that there are dozens (hundreds?) of extensions adding very tiny vendor-specific features all over the place, so your code feathers out into vendor-, GPU- and driver-version specific code paths, the &quot;hell&quot; there is maintaining and testing such code.<p>So instead of one &quot;Vulkan API&quot; you have a mess of sub-APIs which offer different features on each device, that&#x27;s the opposite of a &quot;cross-platform API&quot; that Vulkan apparently wants to be.<p>Other APIs group capabilities into a very small matrix of API versions and feature levels which keeps the amount of optional code paths very small or in many cases eliminates the need for them completely.')
