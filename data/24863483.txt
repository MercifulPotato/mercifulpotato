Item(by='nicwolff', descendants=None, kids=None, score=None, time=1603402049, title=None, item_type='comment', url=None, parent=24849933, text='In an enterprise production environment, it&#x27;s much easier to update a JSON schema, which is stored as data, than to alter the DB schema. The point of having a JSON field isn&#x27;t to let everyone with write access to the API stick whatever they want in there; it&#x27;s to let the administrators of the service change what it can store without engineers deploying code changes and DB migrations.')
