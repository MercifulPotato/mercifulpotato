Item(by='benrbray', descendants=None, kids=[24880966], score=None, time=1603562807, title=None, item_type='comment', url=None, parent=24880749, text='Functional programming languages, more often than not, distinguish themselves with some combination of the following features:<p>* First-order functions<p>* Partial function application &#x2F; currying<p>* Type inference<p>* Algebraic data types<p>* Typeclasses<p>* An emphasis on pure functions (side effects must be explicitly annotated), opening up more possibilities for static analysis<p>* Lazy evaluation<p>This is not to say that the above features cannot be emulated by other languages.  For instance, Rust has traits, which serve more or less the same practical purpose as Haskell typeclasses.<p>However, FP is designed to prioritize the above features, whereas other languages may not be.')
