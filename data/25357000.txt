Item(by='qball', descendants=None, kids=None, score=None, time=1607507934, title=None, item_type='comment', url=None, parent=25351578, text='Word <i>has</i> compiler errors, and it even has <i>bugs</i> in its compiler sometimes (Word 2007 has one where it needed a character to show a table cell when that cell contained nothing but a certain kind of image; this was fixed in Word 2010+ but not there).<p>This is only really relevant if you&#x27;re generating Word documents programatically from other data sources (it&#x27;s very easy to do this with the MS-provided tools which can auto-generate code-behind for programs that need to make documents like this), but it&#x27;s entirely possible to end up with invalid OOXML that means the document doesn&#x27;t load.<p>Some are recoverable, some aren&#x27;t.')
