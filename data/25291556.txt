Item(by='capableweb', descendants=None, kids=None, score=None, time=1607016370, title=None, item_type='comment', url=None, parent=25291444, text='&gt; Generally when releasing new code you want to do a gradual release so that a bad release is mitigated<p>Make each stored procedure not depend on any other procedure. If you change one procedure, make sure that every existing procedure can work with both the current and the future version. Once compatible, start upgrading procedures one by one. Add in monitoring and automatic checking of the values during deployment, and you have yourself a system that can safely roll out changes slowly.<p>I&#x27;m not advising anyone to do this, it sounds horribly inefficient to me, but it&#x27;s probably possible with the right tooling.')
