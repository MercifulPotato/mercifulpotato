It doesn&#x27;t support running as TrustedInstaller (AFAIK), but when I run into a command that requires elevated permissions, I just prepend `sudo` at the beginning, and it elevates the current shell (without a new window) and lets me run the command.<p><pre><code>    PS&gt; choco upgrade deno\n    Chocolatey detected you are not running from an elevated command shell\n    \n    ^C\n    \n    PS&gt; sudo choco upgrade deno\n    Chocolatey v0.10.15\n    Upgrading the following packages:\n    deno\n</code></pre>\n[0]: <a href="https:&#x2F;&#x2F;github.com&#x2F;gerardog&#x2F;gsudo" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;gerardog&#x2F;gsudo</a>')
