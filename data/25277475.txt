Item(by='pheelicks', descendants=None, kids=[25281127], score=None, time=1606924368, title=None, item_type='comment', url=None, parent=25277221, text='Gladly:<p>- Procedural GL (<a href="https:&#x2F;&#x2F;github.com&#x2F;felixpalmer&#x2F;procedural-gl-js" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;felixpalmer&#x2F;procedural-gl-js</a>) is a JavaScript library for rendering 3D terrains. It is built on top of THREE.js but you don&#x27;t need to know anything about 3D to use it - a high-level API is provided. The software is open source, but of course you need to supply it with source data, for it to be useful. In particular you need elevation data to give you the shape of the terrain, and aerial imagery to give the color on the terrain.<p>- NASADEM is an elevation dataset published by NASA and integrated into this library via the <a href="https:&#x2F;&#x2F;www.nasadem.xyz&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.nasadem.xyz&#x2F;</a> service. You don&#x27;t need to understand how this works, all you need is an API key. A free tier is available.<p>- LINZ is a (incredible and free) basemap provided by the New Zealand government: <a href="https:&#x2F;&#x2F;basemaps.linz.govt.nz&#x2F;" rel="nofollow">https:&#x2F;&#x2F;basemaps.linz.govt.nz&#x2F;</a>. Integrating with Procedural GL was just a matter of configuing the URL for the tiles and adding an API key. Note it is New Zealand only, by default Procedural GL uses www.maptiler.com&#x2F; (again a free tier is available)<p>As for what one would need to do make this for another part of the world:<p>1) Fork <a href="https:&#x2F;&#x2F;github.com&#x2F;felixpalmer&#x2F;new-zealand-3d" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;felixpalmer&#x2F;new-zealand-3d</a>\n2) Update the locations of interest in <a href="https:&#x2F;&#x2F;github.com&#x2F;felixpalmer&#x2F;new-zealand-3d&#x2F;blob&#x2F;main&#x2F;parks.geojson" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;felixpalmer&#x2F;new-zealand-3d&#x2F;blob&#x2F;main&#x2F;park...</a>\n3) Optionally change the imagery datasource (Maptiler, Mapbox etc) [don&#x27;t forget attribution]\n4) Profit')
