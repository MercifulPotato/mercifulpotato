Item(by='Spivak', descendants=None, kids=None, score=None, time=1606932185, title=None, item_type='comment', url=None, parent=25277095, text='It&#x27;s an easy-to-state largely foolproof test to see if data really is anonymized.<p>The thing that you&#x27;re worried about with poorly-anonymized datasets is that if you have another non-anonymized dataset you can combine them to deduce the original information. &quot;Your data set must not be able to be combined with any others that would allow them to infer the original data&quot; is hard. How could you possibly test them all?<p>Well it turns out that there is one such non-anonymized dataset with the property that if you can&#x27;t connect your anonymized data with it at all then you can be pretty sure that you couldn&#x27;t connect them with any others -- the original data!')
