The workflow at a high level:<p>1) Make code changes to a specific microservice.\n   (note: The Open API Spec also lives in code.)<p>2) CI&#x2F;CD pipelines get triggered.<p>3) New microservice is built.<p>4) Call every REST endpoint defined in the microservice&#x27;s Open API Spec and validate it using the example requests&#x2F;responses.<p>5) If successful, regenerate the API documentation from the Open API Spec.<p>This ensures that our documentation and services will stay inline with each other.')
