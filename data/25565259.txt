Item(by='TeMPOraL', descendants=None, kids=[25565537, 25565433], score=None, time=1609202818, title=None, item_type='comment', url=None, parent=25564597, text='&gt; <i>One benefit (with respect to the CPU) of pure functions is that they can be cached</i><p>Only if the input space is very small. If your function takes as input two booleans and a string you know is an ID from a relatively static and small set of IDs, memoization is the way to go. If your function takes in two arbitrary floats, trying to memoize anything will only make your program run slower and leak memory.')
