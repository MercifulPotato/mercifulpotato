Having to find and install custom extensions adds a regular frictional cost which encourages you to just do more work in a full programming language since you know you&#x27;ll be able to satisfy any requirement that way.<p>I saw so many developers sour on XML after hitting the “This would be easy if we used XPath 2 but instead it&#x27;s hard” wall that I wonder if anyone on the relevant standards committees ever thought about how much libxml2 would make their work relevant.')
