Item(by='jjav', descendants=None, kids=[25217790, 25217813, 25218291, 25217845], score=None, time=1606375858, title=None, item_type='comment', url=None, parent=25217422, text='You should not have any full GC (pause) events, ever, in properly written production Java code.<p>Source: I worked on several extremely high performance Java systems at Sun and also worked optimizing various Java servers at subsequent jobs based on my Sun experience.<p>At one company (after Sun) I took on a backend service which was doing GC pauses every 3-5 minutes, the code was a dumpster fire mess. After cleaning things up we didn&#x27;t hit a single full GC in two months of production use (they redeployed every two months so that&#x27;s as long as the process lasted).')
