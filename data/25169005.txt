Item(by='spijdar', descendants=None, kids=None, score=None, time=1605956631, title=None, item_type='comment', url=None, parent=25168908, text='For anyone interested, the problem with homebrew is it relies on &quot;&#x2F;usr&#x2F;local&quot; being owned by the current user to allow installing software without needing a password every time you invoke &quot;brew&quot;. Naturally this requires you only use that user to invoke homebrew.<p>Macports does things in a more conventional way, requiring sudo to elevate privileges to root before installing software, but should also work properly on a multi-user system, whereas homebrew sort of assumes a single-user install.<p>(I think homebrew has some capacity for being used multi-user -- years back my CS professor had a lab of macs and managed to get this working I believe)')
