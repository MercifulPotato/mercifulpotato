Item(by='Closi', descendants=None, kids=None, score=None, time=1607290290, title=None, item_type='comment', url=None, parent=25326613, text='The differences seem to be:<p>* You can write it in one language (excel formula language)<p>* The language is simpler and known by almost all users, while Javascript and VBA are only used by a tiny proportion of users.<p>* The language is more secure (i.e. you can&#x27;t execute arbitrary code, access files, call DLLs etc)<p>* Because of the above, users don&#x27;t need any security permissions &#x2F; get warnings when running it.<p>* Because they are standard excel formulas, you get OOTB support for other excel features such as dynamic array formulas and access to the full catalogue of worksheet functions (even in VBA, Application.Worksheet only had access to a few basic excel workbook functions, so if you wanted to do a Xlookup for example you are implementing it yourself with arrays and loops)')
