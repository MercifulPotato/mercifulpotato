Item(by='kazinator', descendants=None, kids=None, score=None, time=1604204240, title=None, item_type='comment', url=None, parent=24955976, text='Try TXR Lisp.<p>Though not available in a Ubuntu repository, there are binaries for Ubuntu 18 (32 or 64 bit) in tarball format: you can unpack these relative to &#x2F; to get it into &#x2F;usr&#x2F;local. Or anwyhere in your tree: TXR is self-contained and relocatable: it will run from wherever you unpack it, finding its own files relative to the executable location.<p>It&#x27;s easy to build from sources: just clone the repo; .&#x2F;configure; make; make install.  In the most recent release, I dropped the build-time dependencies on several tools, to make things even easier. The only special library you need is libffi (so at build time you need the libffi-devel package on Ubuntu). It will configure and build without it, but then you can&#x27;t use FFI.<p>Git: <a href="http:&#x2F;&#x2F;www.kylheku.com&#x2F;cgit&#x2F;txr&#x2F;" rel="nofollow">http:&#x2F;&#x2F;www.kylheku.com&#x2F;cgit&#x2F;txr&#x2F;</a><p>The documentation is all in one big man page, which is very convenient in the GNU&#x2F;Linux and Unix environment. There is a hyperlinked HTML form:<p><a href="https:&#x2F;&#x2F;www.nongnu.org&#x2F;txr&#x2F;txr-manpage.html" rel="nofollow">https:&#x2F;&#x2F;www.nongnu.org&#x2F;txr&#x2F;txr-manpage.html</a>')
