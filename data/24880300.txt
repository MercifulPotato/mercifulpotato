Item(by='gnulinux', descendants=None, kids=[24880707], score=None, time=1603557908, title=None, item_type='comment', url=None, parent=24880264, text='Lisp runtime implicitly runs each top-level expression as statement. When you run (def x y) it&#x27;s not only true that this expression returns &#x27;x, it&#x27;s also true that it alters the static state of the program by adding &#x27;x to the scope.<p>To make a purely statementless language, I think you need to make everything happen in a single monadic computation. E.g. in pseudo-Haskell-ish:<p><pre><code>    main : Scope -&gt; StaticIO Scope\n    main scope0 = do\n      scope1, x&#x27; &lt;- def scope0 &quot;x&quot; Int\n      scope2, xv &lt;- dec scope1 &quot;x&quot; 3\n</code></pre>\nBut the main part is still statement.')
