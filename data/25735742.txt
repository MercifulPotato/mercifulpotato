Item(by='uryga', descendants=None, kids=None, score=None, time=1610395813, title=None, item_type='comment', url=None, parent=25733848, text='pedantically, talking to the database will involve sending a request or some kind of IPC, all of which are usually considered side effects.<p>[handwavy analogy alert]<p>people often call it &quot;side-effects&quot; as a shorthand. in reality an expression being of type `IO Foo` mostly just tells the compiler that order of execution matters (which isn&#x27;t the case with pure functions):<p><pre><code>  do print &quot;a&quot;\n     print &quot;b&quot;\n \n  -- obviously not the same as\n  \n  do print &quot;b&quot;\n     print &quot;a&quot;\n</code></pre>\nand also that it can&#x27;t eliminate common expressions:<p><pre><code>  do a &lt;- readBytes file 100\n     b &lt;- readBytes file 100\n     doStuff a b\n\n  -- obviously not the same as\n\n  do x &lt;- readBytes file 100\n     doStuff x x\n</code></pre>\nit&#x27;s a way of enforcing ordering in a lazy language where evaluation order isn&#x27;t really defined.')
