Item(by='amzans', descendants=None, kids=[25872074], score=None, time=1611326652, title=None, item_type='comment', url=None, parent=25871981, text='Thanks for the comment!<p>Yes, I&#x27;d like to clarify Timeflake (and many alternatives suggested) is NOT meant to be used for security.<p>There&#x27;s an important note on the readme about this:<p>&gt; Since the timestamp part is predictable, the search space within any given millisecond is 2^80 random numbers, which is meant to avoid collisions, not to secure or hide information. You should not be using timeflakes for password-reset tokens, API keys or for anything which is security sensitive. There are better libraries which are meant for this use case (for example, the standard library&#x27;s secrets module).')
