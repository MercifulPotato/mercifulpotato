Item(by='ptx', descendants=None, kids=[24750016], score=None, time=1602455702, title=None, item_type='comment', url=None, parent=24749412, text='If the generated code is a representation of certain unchangeable data about the hardware, you might still want to<p>1) represent it more compactly;<p>2) represent it in a form that can more easily be read and transformed to handle future use-cases for the data;<p>3) after some future restructuring of the driver, represent the data in a form that better fits with that structure.<p>If you have to regenerate the code using the proprietary tool in order to restructure the driver, the generated code is not &quot;the preferred form of the work for making modifications&quot;.')
