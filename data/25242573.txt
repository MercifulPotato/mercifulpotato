Item(by='xyzzy123', descendants=None, kids=None, score=None, time=1606618283, title=None, item_type='comment', url=None, parent=25242070, text='IMHO the ideal situation is that both distribution mechanisms are available; single file static binary and also packages for different OS&#x27;s.<p>Which one you want to use in a given situation is contextual.<p>Murphy&#x27;s law seems to dictate that the version of the build tool lovingly cared for by the OS&#x27;s package manager is never the version you actually need.<p>Build tools in particular have a slightly different lifecycle to many things running on a host, which are &quot;owned&quot;, cared for and configured by a system administrator.<p>Usually the project you&#x27;re building &quot;owns&quot; the version of the build tool it needs and there&#x27;s no guarantee that it will build with a different version. As a consequence, genuine build tool security issues must first be handled by the team making the tool and then by the projects depending on it when they verify compatibility.')
