Item(by='lmm', descendants=None, kids=[24988472], score=None, time=1604475135, title=None, item_type='comment', url=None, parent=24986288, text='&gt; The problem with &quot;Make it an error to not initialise a struct field&quot; is you lose source compatibility when adding new struct fields.<p>If you allow the struct definition to specify a default value then the struct author can set default in the cases where there is a sensible default, and leave it compile-time incompatible in the cases where there really is no good default and the person doing the upgrade needs to make a decision.')
