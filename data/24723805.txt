Item(by='pyb', descendants=None, kids=None, score=None, time=1602190206, title=None, item_type='comment', url=None, parent=24719878, text='It would be initially problematic, but I do not believe that an Oracle win would be that disruptive in the long run. What would happen is that API providers would from now on have to provide standardized legal assurances that users are allowed to implement their API (in part or in whole). APIs that do not provide these assurances will go mostly unused.<p>As an imperfect analogy, think about the fact that many companies no longer want to use GPL code because of the legal ramifications. Open source authors would want their code used at those places use other licenses such as MIT.')
