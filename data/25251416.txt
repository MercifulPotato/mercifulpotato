Item(by='neurostimulant', descendants=None, kids=None, score=None, time=1606720443, title=None, item_type='comment', url=None, parent=25251361, text='The next.js method is completely static. You can run your wordpress instance locally in your laptop&#x2F;pc, run next.js static site generator, then upload the resulting static site to aws or github pages. No need to run the wordpress instance 24&#x2F;7 as it&#x27;s only required as a data source during static site generation and can be turned off when you&#x27;re done with it.<p>There is also an option in cloudflare to pre-cache everything on their edge servers so visitors will never hit the origin server, but it&#x27;s not available in free plan (I think it&#x27;s enterprise only plan).')
