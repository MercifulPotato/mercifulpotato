Item(by='ajxs', descendants=None, kids=[25354330], score=None, time=1607476737, title=None, item_type='comment', url=None, parent=25352385, text='It&#x27;s a neat trick using the Browser&#x27;s Quirks Mode to load JSON that can bootstrap a framework, I&#x27;m not sure I really understand the benefit though. In order to actually get to the point of bootstrapping, the server has already transferred all of the content, plus the bootstrapping Javascript in a single payload. It seems like this implementation has the weaknesses of using a Javascript framework, without the benefits. Together with all of the weaknesses of static content, without the benefits.
