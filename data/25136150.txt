Item(by='micksabox', descendants=None, kids=[25138109, 25137077], score=None, time=1605703716, title=None, item_type='comment', url=None, parent=25124867, text='&gt; If you don’t touch code, it doesn’t intrinsically degrade.<p>This is one misunderstanding of code quality which takes an absolute approach to measuring code health. Code health is relative to the changing requirements of it’s use. If you suddenly add a requirement that doesn’t fit existing design assumptions, the code hasn’t changed but the relative to the ideal has deviated...without touching the code.<p>The tech debt metaphor is not perfect. The pandemic has given us all a better metaphor: the code health epidemic.
