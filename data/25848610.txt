Item(by='Androider', descendants=None, kids=None, score=None, time=1611161234, title=None, item_type='comment', url=None, parent=25846504, text='In Chrome, make sure you set your less frequently used extensions to run &quot;On click&quot; instead of &quot;On all sites&quot;. Extensions -&gt; extension details -&gt; Site access.<p>For dev tools and such, I set a whitelist of the sites they&#x27;re allowed to run on, using that same extension details page. There&#x27;s no need for your JSON formatter etc. to run on every single page you visit. Also speeds up browsing.')
