In that sense, it was &quot;discovered&quot;, not invented.<p>Depending on the language, the syntax of &quot;if then else&quot; is not needed and is syntactic sugar.<p>In the Lambda Calculus you can implement &quot;if else&quot; using &quot;first&quot; and &quot;second&quot;.<p>Here is something equivalent in JavaScript.<p><pre><code>  const trueFn  = (first, second) =&gt; first\n  const falseFn = (first, second) =&gt; second\n\n  const ifFn = (boolFn, first, second) =&gt; boolFn(first, second)\n  \n  ifFn(trueFn,  &#x27;yes&#x27;, &#x27;no&#x27;)  &#x2F;&#x2F; returns &#x27;yes&#x27;\n  ifFn(falseFn, &#x27;yes&#x27;, &#x27;no&#x27;)  &#x2F;&#x2F; returns &#x27;no&#x27;</code></pre>')
