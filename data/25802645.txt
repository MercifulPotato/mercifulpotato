Item(by='mumblemumble', descendants=None, kids=[25802846], score=None, time=1610809743, title=None, item_type='comment', url=None, parent=25802590, text='I draw a big distinction between user-facing error messages and dev- and ops-facing error messages.<p>Back when I was doing desktop GUI apps full time, my standard pattern was to display a terse and readable error message, and provide a link the user could click to bring up a much more detailed error report. The more detailed report was purely for the benefit of tech support, and very, very few users would click it unless specifically told to.<p>I&#x27;m no Web developer, but I imagine the equivalent for a Web app would be to display a terse error in the browser and save a verbose error in the logs.')
