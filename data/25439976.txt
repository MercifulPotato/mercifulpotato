Item(by='valenterry', descendants=None, kids=[25445783], score=None, time=1608103628, title=None, item_type='comment', url=None, parent=25439871, text='The GraphQL equivalent to creating a new, specialized REST route would be to create a new, specialized query.<p>E.g. the following REST<p><pre><code>    &#x2F;articles-with-comments?number=20\n</code></pre>\nwhich returns title, contents etc. would map to a completely new query in graphql<p><pre><code>    articlesWithComments(number: Integer) {\n      title\n      contents\n      ...\n    }\n</code></pre>\nso it is exactly as easy to optimize. Of course this is not very composable, but that is equally true for both solutions.')
