Item(by='mbirth', descendants=None, kids=None, score=None, time=1607221185, title=None, item_type='comment', url=None, parent=25319464, text='I&#x27;ve been using floccus (<a href="https:&#x2F;&#x2F;floccus.org" rel="nofollow">https:&#x2F;&#x2F;floccus.org</a>) for quite some time. It&#x27;s a browser extension available for Chrome and FF that syncs bookmarks to a WebDAV server. I had the occasional sync conflict where I needed to re-initialise the sync, but it mostly just worked fine.<p>I&#x27;ve recently switched to using Safari more and to sync that, you&#x27;ll sadly need a macOS running e.g. Synkmark (<a href="http:&#x2F;&#x2F;www.sheepsystems.com&#x2F;products&#x2F;" rel="nofollow">http:&#x2F;&#x2F;www.sheepsystems.com&#x2F;products&#x2F;</a>).<p>Edit: I think the iCloud tools for Windows also can sync bookmarks between Safari and e.g. Firefox or Chrome.')
