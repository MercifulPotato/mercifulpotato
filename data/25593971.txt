Item(by='dom96', descendants=None, kids=[25594830], score=None, time=1609431356, title=None, item_type='comment', url=None, parent=25593844, text='&gt;  The Nim forum has recurring questions about deserializing JSON that only make sense if you assume Nim is runtime dynamic like Python.<p>Nim is a statically typed Python, and actually the way you can deserialise JSON is very Python-like, so I&#x27;m not sure where you got this from. Here is an example:<p>Python:<p><pre><code>    &gt;&gt;&gt; import json\n    &gt;&gt;&gt; j = json.loads(&#x27;{&quot;foo&quot;: 42}&#x27;)\n    &gt;&gt;&gt; j[&quot;foo&quot;]\n    42\n</code></pre>\nNim:<p><pre><code>    import json\n    let j = parseJson(&quot;{\\&quot;foo\\&quot;: 42}&quot;)\n    echo(j[&quot;foo&quot;]) # 42\n</code></pre>\n<a href="https:&#x2F;&#x2F;play.nim-lang.org&#x2F;#ix=2KpU" rel="nofollow">https:&#x2F;&#x2F;play.nim-lang.org&#x2F;#ix=2KpU</a>')
