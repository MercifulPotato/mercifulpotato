Item(by='indymike', descendants=None, kids=[24979318], score=None, time=1604405830, title=None, item_type='comment', url=None, parent=24976224, text='Open source is a trade-off. In many cases, it&#x27;s rocket fuel for early and mid-stage growth in exchange for reduced barriers to entry for competitors. It also points out a real weakness in infrastructure: why is provisioning, configuring and deploying code so difficult and expensive? Shouldn&#x27;t we have OS level tools that make this trivial at this point? It&#x27;s pretty amazing that we&#x27;re taking patterns like load balancer + Shared Cache + Database + Libraries + App Code + config files and assets and making it so difficult to configure them and deploy them. Reducing deployment complexity is still a huge opportunity...')
