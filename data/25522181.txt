Item(by='sargun', descendants=None, kids=None, score=None, time=1608757265, title=None, item_type='comment', url=None, parent=25521909, text='The trouble is that Docker does not enable user namespaces by default, and thus resulting in these CVEs. A lot of integrations (like the examples of secrets, and sidecars) do not work properly when used in conjunction with user namespaces, and tend to require modification. We did the work to make this work, and created a model (injected processes into the container) in order to create this clear boundary layer.<p>Many people use Docker with Kubernetes. Unfortunately, the Kubernetes Kubelet does not work with Docker and user namespaces (<a href="https:&#x2F;&#x2F;github.com&#x2F;kubernetes&#x2F;enhancements&#x2F;issues&#x2F;127" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;kubernetes&#x2F;enhancements&#x2F;issues&#x2F;127</a>). There is still work being done in this area by folks like Kinvolk.')
