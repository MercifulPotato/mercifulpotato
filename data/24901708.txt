I have some abstractions for making interfacing easier, but I don&#x27;t have any auto-generated structures.<p><a href="https:&#x2F;&#x2F;github.com&#x2F;fwsGonzo&#x2F;rvscript" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;fwsGonzo&#x2F;rvscript</a><p>I have done that here. Example:<p><pre><code>    APICALL(api_math_smoothstep)\n    {\n        auto [edge0, edge1, x] = machine.sysargs &lt;float, float, float&gt; ();\n        x = std::clamp((x - edge0) &#x2F; (edge1 - edge0), 0.0f, 1.0f);\n        machine.cpu.registers().getfl(10).set_float(x * x * (3 - 2 * x));\n        return 0;\n    }\n</code></pre>\n&quot;machine&quot; is an argument to each system call handler, so you can have multiple machines for many purposes.
