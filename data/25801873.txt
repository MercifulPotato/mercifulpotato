Item(by='beagle3', descendants=None, kids=None, score=None, time=1610801018, title=None, item_type='comment', url=None, parent=25801376, text='It isn’t so black and white.<p>Nim, if you let it use C as a backend, will let you use C macros, despite being generally safe and not remotely a C superset.<p>Yes, the use of C macros (and any kind of ffi) is unsafe, that cannot be avoided. But it doesn’t need to be a c superset (and indeed, when you use JS as a backend, or the native LLVM backend, you get no access to C macros)')
