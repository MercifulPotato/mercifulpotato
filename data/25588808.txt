Item(by='kennu', descendants=None, kids=[25588893], score=None, time=1609380656, title=None, item_type='comment', url=None, parent=25588216, text='But when you share some HTML in practice, e.g. send a HTML game exported from Godot to a friend, they will need to unzip it and install a local web server to actually play it with appropriate permissions.<p>Whereas .swf files could be opened directly in the browser without any preparation steps.<p>It was also possible to read  metadata from .swf files (background color, canvas size, etc), which made it very simple to automatically embed Flash games on web pages with the correct layout.<p>So there are a lot of small practical details that made .swf a very convenient format for sharing games and multimedia content.')
