Item(by='onion2k', descendants=None, kids=None, score=None, time=1611485986, title=None, item_type='comment', url=None, parent=25890472, text='<i>The plugin issue is not specific to WordPress.</i><p>The fact that other platforms and applications are insecure isn&#x27;t relevant; we&#x27;re comparing static sites to WordPress.<p>However, to answer the point, static sites are <i>significantly</i> more secure than every single dynamic platform that supports a plugin architecture because plugins can be, and often are, written without security in mind.<p>Unless you <i>really</i> need a dynamic website you should be deploying static assets to the enduser. Practically every business website would be better off being delivered as a static site, even if the admin still use WordPress to edit the content.')
