Item(by='dragonwriter', descendants=None, kids=[24724304, 24724511, 24724602], score=None, time=1602193144, title=None, item_type='comment', url=None, parent=24724164, text='&gt; An API is literally just a table of contents.<p>An API is more like a set of mechanical gears that just happen to be shaped a little like words. If I change the names of things in a table of contents to synonyms, I can still use it to find what I am looking for. If I change the names of things my implementation of an API, it no longer functions as an interface for the things that worked with it previously.<p>An API, despite looking like works, is not a literary expression of a process, it is a machine for executing the process itself, and its shape, is essential to its function.')
