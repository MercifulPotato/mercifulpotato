Item(by='kkoncevicius', descendants=None, kids=None, score=None, time=1607194253, title=None, item_type='comment', url=None, parent=25317027, text='There is a big difference between what is allowed from within a package and what is possible in native implementation. As an example - currently this pipe operator seems to be implemented at the parser level. The parser simply takes the pipe expression and translates it into standard nested function call f(g(x)). Which means - there will be almost no cost in speed (%&gt;% was notoriously slow). In addition the user will get the usual error stack in case something within this pipe fails.')
