Item(by='garethrowlands', descendants=None, kids=None, score=None, time=1608649408, title=None, item_type='comment', url=None, parent=25502226, text='A higher order function is a function that takes another function as a parameter or returns a function as its result. Famous higher order functions include &quot;map&quot; and &quot;filter&quot;, for example. Javascript, for example, uses higher order functions all the time. See [1].<p>The term &quot;higher order logic&quot; typically means program logic that uses higher order functions.  An object-oriented programming style is inherently higher order because objects typically contain functions and are passed to methods.<p>The term &quot;higher order logic&quot; can also mean a system of logic that allows statements about logical statements.
