Item(by='Rochus', descendants=None, kids=None, score=None, time=1602178733, title=None, item_type='comment', url=None, parent=24720777, text='The origin of the graphical user interface was actually elswhere. The <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Semi-Automatic_Ground_Environment" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Semi-Automatic_Ground_Environm...</a> system had a display system with a &quot;graphical user interface&quot; and a &quot;ligth gun&quot; (i.e. pointing device) by 1958; also e.g. <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Sketchpad" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Sketchpad</a> had similar features in 1963; but these were CRT displays with vector drawings. The Alto in 1973 in contrast had a bitmaped screen and the first GUI following a desktop metapher (see e.g. <a href="https:&#x2F;&#x2F;www.microsoft.com&#x2F;en-us&#x2F;research&#x2F;publication&#x2F;alto-personal-computer&#x2F;?from=http%3A%2F%2Fresearch.microsoft.com%2Fen-us%2Fum%2Fpeople%2Fblampson%2F25-alto%2F25-alto.pdf" rel="nofollow">https:&#x2F;&#x2F;www.microsoft.com&#x2F;en-us&#x2F;research&#x2F;publication&#x2F;alto-pe...</a> or <a href="http:&#x2F;&#x2F;bwlampson.site&#x2F;38-AltoSoftware&#x2F;Abstract.html" rel="nofollow">http:&#x2F;&#x2F;bwlampson.site&#x2F;38-AltoSoftware&#x2F;Abstract.html</a>).')
