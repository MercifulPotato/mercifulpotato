Item(by='danielheath', descendants=None, kids=None, score=None, time=1608591692, title=None, item_type='comment', url=None, parent=25500549, text='I should clarify; that technique doesn’t work if there are calls between systems on different versions unless they are tested on both.<p>I wouldn’t run a system where the two servers call one another.<p>This is a technique for reducing risk via toggles at the load balancer. The reduction is achieved by making it possible to deploy smaller changes to users, and by reducing the latency of rollback.<p>These are only useful if you have the ability to introspect the system state in production and confirm whether a new version is working correctly.')
