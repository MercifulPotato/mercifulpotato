Item(by='matthiaswh', descendants=None, kids=[25593016, 25592948], score=None, time=1609422716, title=None, item_type='comment', url=None, parent=25592476, text='I&#x27;ve written a large-ish amount of Nim code on personal projects while using Python day to day at work, and almost every time I&#x27;m working with Nim my brain is screaming at me at how unlike Python it is.<p>Nim took some inspiration from Python&#x27;s syntax, but the similarity is only skin deep. The construct of the language is very, very different resulting in code that usually looks and is structured differently.<p>IMO going into Nim thinking it will be like Python but compiled! and faster! with types! is not a good way to approach it. It is an entirely different language altogether.')
