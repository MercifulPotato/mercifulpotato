Item(by='pdw', descendants=None, kids=[25793181], score=None, time=1610670808, title=None, item_type='comment', url=None, parent=25782567, text='I&#x27;ve recently spent some time experimenting with Tcl, and I&#x27;ve developed some affection for it, but what tripped me up is that &quot;everything is a string&quot; means that Tcl is one of the few truly weakly typed languages. This Tcl notation:<p><pre><code>    {a b}\n</code></pre>\nmight represent any of the following data structures in JSON notation:<p><pre><code>    &quot;a b&quot;\n    [&quot;a&quot;, &quot;b&quot;]\n    [[&quot;a&quot;], [&quot;b&quot;]]\n    {&quot;a&quot;: &quot;b&quot;}\n    {&quot;a&quot;: [&quot;b&quot;]}\n</code></pre>\nThis can make reading some Tcl code quite a puzzle.')
