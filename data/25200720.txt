Item(by='the_af', descendants=None, kids=[25213589], score=None, time=1606239218, title=None, item_type='comment', url=None, parent=25199573, text='I think the parent comment meant that typescript is a static checker, which means it&#x27;s not exercised at run time but in a prior step. Therefore &quot;performant typescript&quot; means to shorten the time it takes to perform these static checks, not the time it takes to run your code. In contrast, when people talk about optimizing C code they most often (but not always, of course) mean to write code that runs fast.')
