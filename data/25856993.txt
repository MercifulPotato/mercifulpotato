Item(by='ronenlh', descendants=None, kids=None, score=None, time=1611220466, title=None, item_type='comment', url=None, parent=25845925, text='My interest in ReScript is that it doesn&#x27;t aim to add more features to JavaScript, it&#x27;s a completely different language which has a JavaScript-like interface and compiles to JavaScript.<p>It&#x27;s a statically-typed functional language from the ML family, which is a huge deal. It implements a completely different approach to code, even if it looks similar.<p>It&#x27;s currently academical, &quot;ivory tower,&quot; but in my opinion it will trickle into industry. Someone described ReScript as ECMAScript 2030.')
