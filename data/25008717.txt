Transfers don&#x27;t work if the scp program is missing on the remote.<p>I use tinyssh in a few contexts where I don&#x27;t want file transfers of any kind.<p>___<p>TinySSH doesn’t have SCP?<p>No, ‘rsync -e ssh’ makes same job. If you really need scp, use scp for example from OpenSSH. TinySSH doesn’t have problem with scp protocol, only doesn’t have scp program.<p>Can I use sftp using TinySSH?<p>Yes. TinySSH doesn’t have sftp program, but can run e.g. OpenSSH &#x2F;usr&#x2F;libexec&#x2F;openssh&#x2F;sftp-server.\nSftp support can be enabled using switch ‘-x’<p><pre><code>    ... tinysshd -x sftp=&#x2F;usr&#x2F;libexec&#x2F;openssh&#x2F;sftp-server  &#x2F;etc&#x2F;tinyssh&#x2F;sshkeydir\n</code></pre>\n<a href="https:&#x2F;&#x2F;tinyssh.org&#x2F;faq.html" rel="nofollow">https:&#x2F;&#x2F;tinyssh.org&#x2F;faq.html</a>')
