Item(by='jlokier', descendants=None, kids=None, score=None, time=1610634184, title=None, item_type='comment', url=None, parent=25767777, text='You don&#x27;t need to delete the extra info.<p>After reading the output, you run the entire computation backwards (it&#x27;s reversible after all) until the only state in the system is the original input and initial state.<p>Then you can change the input to do another calculation.<p>If implemented &quot;perfectly&quot;, heat is produced when changing the input and to read the output.<p>Reading the output will disturb it so the computation won&#x27;t perfectly reverse in practice, however if the disturbance is small, most of the energy temporarily held in circuit is supposed to bounce back to the initial state, which can then be reset properly at an appropriately small cost.<p>Quantum computation is <i>very</i> similar to all of this.  It too is reversible, and costs energy to set inputs, read outputs and clear accumulated noise&#x2F;errors.  The connection between reversible computation and quantum computation is quite deep.')
