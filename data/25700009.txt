Item(by='kdkeyser', descendants=None, kids=None, score=None, time=1610197184, title=None, item_type='comment', url=None, parent=25699885, text='Haxl is actually an example of &quot;the more impressive and more useful&quot; approach: it is an abstraction embedded within a Haskell library, that allows you to express your &quot;intent&quot; in simple, standard Haskell code, decoupled from the way this intent will be reached (i.e. the execution is determined&#x2F;optimized by the library)<p>Haskell allows this to be done in a way that makes if very hard for the user to break this abstraction (the type system will prevent you from doing this), while still allowing the full language power of Haskell to be used.')
