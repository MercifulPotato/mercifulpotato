Item(by='jleahy', descendants=None, kids=[25063389], score=None, time=1605130481, title=None, item_type='comment', url=None, parent=25062872, text='All great, then you need to port it to something other than that one specific SSD and it&#x27;s too much work.<p>We have abstraction boundaries for a reason, we give up a small % of performance and in return we can write code once (say SQLite) and use it in many scenarios. For something like SQLite it means it&#x27;s been around for a long time and had a lot of optimization work done on it (and that probably outweighs the few % gain you might get from such tight integration).<p>You&#x27;d probably get a bigger performance gain from just not using SQL (eg. a DBM).')
