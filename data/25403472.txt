Item(by='kaetemi', descendants=None, kids=[25403554, 25404057], score=None, time=1607823005, title=None, item_type='comment', url=None, parent=25402749, text='Considering the source material is a file, that needs to be loaded up by the CPU first, and the operation itself is dead simple, the real constraint is RAM bandwidth.<p>I would guess that the time of getting the data out to the GPU and back, is about the same as just calculating the difference on the CPU.<p>So, the quickest solution is probably to use the CPUs vector operations, and read ahead into the registers so you can exhaust the memory bandwidth fully.')
