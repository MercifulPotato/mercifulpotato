Item(by='fogihujy', descendants=None, kids=[25773590], score=None, time=1610611963, title=None, item_type='comment', url=None, parent=25772792, text='Pretty much anything with a serial port and a CPU should be able to get online, provided you have appropriate software.<p>Most computers built after 1980 should be able to do it with some tinkering. Pretty much anything built after 1990 should be fairly trivial to get online.<p>If you want something somewhat usable, then Lynx&#x2F;Links running on whatever a modern Linux distro can boot on should suffice.<p>I used a modem to get online with my Amiga 600 (7 MHz 68000, 2 + 4 MB RAM) in the 90&#x27;s. It was perfectly possible to use things like IRC, eMule and a WWW browser on it, but the modern web will be quite unusable with aMosaic...')
