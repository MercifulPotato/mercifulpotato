Item(by='Matthias247', descendants=None, kids=None, score=None, time=1610386295, title=None, item_type='comment', url=None, parent=25720731, text='Be aware that using mmap instead of read&#x2F;write might mess up Go’s scheduling. The runtime has special handling of other blocking system calls which tried to minimize the impact of those on other running goroutines. With directly accessing memory locations mapped via mmap you won’t get this benefit.')
