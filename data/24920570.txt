Item(by='waynecochran', descendants=None, kids=[24920999, 24920992], score=None, time=1603901856, title=None, item_type='comment', url=None, parent=24920462, text='It provides an implementation for the X Window System (aka X11) <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;X_Window_System" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;X_Window_System</a><p>It has a long a storied history. The concept of running an app remotely and use a local display server is probably the thing that sets X11 apart from the rest of the pack. This is still really useful in my humble opinion -- I can run X11 apps (e.g. xv) on a headless EC2 server running Linux in the cloud and have the UI&#x2F;display on my Mac (using Xquartz).<p>This story runs in parallel with the history of Unix itself.')
