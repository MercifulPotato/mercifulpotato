Item(by='drawfloat', descendants=None, kids=[25313428], score=None, time=1607152751, title=None, item_type='comment', url=None, parent=25311062, text='You can conditionally style elements based on an input checkbox&#x27;s `checked` state. A label element can also toggle an input checkbox&#x27;s `checked state`, but contain whatever HTML you want.<p>Put the two together and you can do interactive elements without JS:<p>```\n&lt;input id=&quot;inputId&quot; type=&quot;checkbox&quot; &#x2F;&gt;\n&lt;label for=&quot;inputId&quot;&gt;&lt;&#x2F;label&gt;\n&lt;div class=&quot;toggle-content&quot;&gt;Some DOM&lt;&#x2F;div&gt;\n```<p>```\nlabel::after{\ndisplay: block;\ncontent: &#x27;[+]&#x27;;\n} \n#input,\n.toggle-content {\ndisplay: none;\n}<p>#input:checked + label::after {\ncontent: &#x27;[-]&#x27;;\n} \n#input:checked ~ .toggle-content {\ndisplay: block;\n}\n```<p>Off the top of my head, so YMMV but principal is all there.<p>You start getting headaches if you want animate that though (keyframes that animate a max-height from 0 to 9999999px goes some of the way but feels a bit hacky&#x2F;doesn&#x27;t give you much control over consistent timing.)')
