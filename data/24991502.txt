Item(by='TheDong', descendants=None, kids=None, score=None, time=1604513582, title=None, item_type='comment', url=None, parent=24985220, text='FYI, you do lose a small amount of security using vscode remote dev. For example, if the dev server is compromised, vscode offers an api to request the client&#x27;s clipboard, which may contain sensitive information. If you&#x27;re using a password manager + browser extension, that does mitigate it quite a bit!\nIt&#x27;s also possible the remote&#x2F;local extension API has other issues since it offers a much broader attack surface than, say, just ssh + vim.<p>Clipboards are one of the several security knobs it&#x27;s hard to get just right when adding layers of security to desktop apps. Qubes does this by having separate clipboards, and requiring specific global hotkeys to move clipboard contents between specific containers [0].<p>[0]: <a href="https:&#x2F;&#x2F;www.qubes-os.org&#x2F;doc&#x2F;copy-paste&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.qubes-os.org&#x2F;doc&#x2F;copy-paste&#x2F;</a>')
