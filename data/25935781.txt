Item(by='aflag', descendants=None, kids=None, score=None, time=1611791107, title=None, item_type='comment', url=None, parent=25928154, text='Using ; will not work if you ran set -e before (eg.: if you&#x27;re using it in a script). Perhaps more importantly, cmd1; cmd2 &amp; will not work either, you&#x27;ll need a subshell. So, you need at least: ( set +e; cmd1 ; cmd2 ) &amp;. And there are probably other caveats. And we haven&#x27;t even got to the output redirection yet.')
