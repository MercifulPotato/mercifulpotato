Item(by='gnufx', descendants=None, kids=None, score=None, time=1606083330, title=None, item_type='comment', url=None, parent=25180100, text='If you haven&#x27;t read Landin&#x27;s &quot;The Next 700 Programming Languages&quot;, you should.  I don&#x27;t know if it&#x27;s still outside the ACM paywall, but you&#x27;ll find it in a web search if not. It introduced the offside rule -- not actually what Python has -- inspired by mathematical notation.  Note &quot;it can be mixed freely with [...] punctuation&quot; (e.g. Haskell).  The paper talks about &quot;physical representations&quot;, relevant to syntax highlighting mentioned below (or &quot;fontification&quot; in Emacs, reminiscent of typefaces in the publication language of ALGOL).')
