Item(by='skissane', descendants=None, kids=[25865861, 25866596], score=None, time=1611269419, title=None, item_type='comment', url=None, parent=25865249, text='&gt; Also, ideally, you&#x27;d publicly host the Dockerfile<p>Docker doesn&#x27;t make it easy to produce a reproducible config. People commonly write Dockerfiles with stuff like this in them:<p>RUN apt-get install -y my-favourite-package<p>And then which version of my-favourite-package you get depends on what is the latest one on the update servers at a time. Maybe a new version is released tomorrow with a regression bug that breaks everything.<p>There are ways to get around this problem, but the default ways of doing things in Docker doesn&#x27;t encourage reproducibility.')
