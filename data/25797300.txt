Item(by='lxe', descendants=None, kids=[25797337], score=None, time=1610749159, title=None, item_type='comment', url=None, parent=25796852, text='Lua static binary can be made to be less than ~150k, while providing very advanced high level dynamic language features you&#x27;d expect from JavaScript or PHP.<p>I used it to build a admin UI for an embedded device without an MMU.
