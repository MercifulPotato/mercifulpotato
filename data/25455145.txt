Item(by='plasticxme', descendants=None, kids=[25455529, 25455169], score=None, time=1608209911, title=None, item_type='comment', url=None, parent=25454962, text='A container is not a VM and should ONLY contain the exact files needed to run your app to run securely. Any additional packages increase the security footprint.<p>Every container is one RCE vulnerability away from being compromised and escaped. if your images are distribution based, even slim ones, you’re giving the attacker a broad set of tooling out of the box.<p>Container runtime defaults in Docker and Kubernetes are insecure and grant attackers a lot of privilege — running as UID 0, no user namespace separation, and potentially dangerous kernel capabilities added to the container’s parent process.')
