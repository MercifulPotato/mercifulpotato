Item(by='29athrowaway', descendants=None, kids=None, score=None, time=1612045050, title=None, item_type='comment', url=None, parent=25970690, text='My settings are:<p>GTK2 apps do not support HiDPI correctly. oomox (and similar) can be used to generate a scaled GTK2 theme, but it won&#x27;t solve all your problems.<p>GTK3 supports HiDPI well. If you use Gnome, Xfce and others, you can configure this via the preferences UI. Otherwise, add this to your .profile file:<p><pre><code>    export GDK_SCALE=2\n    export GDK_DPI_SCALE=0.5\n</code></pre>\nMany Java apps can be scaled by adding an argument...<p><pre><code>    java -Dsun.java2d.uiScale=2\n</code></pre>\nFor Qt you can add this to your .profile file:<p><pre><code>    export QT_AUTO_SCREEN_SCALE_FACTOR=0\n    export QT_SCALE_FACTOR=2\n\n    # For theming, requires qt5ct\n    export QT_QPA_PLATFORMTHEME=qt5ct\n</code></pre>\nFor Spotify... I use ncspot for listening, and the web version to manage my playlists (although you can also management with ncspot). ncspot is a keyboard friendly, lightweight terminal client.<p><pre><code>    https:&#x2F;&#x2F;github.com&#x2F;hrkfdn&#x2F;ncspot\n</code></pre>\nI use FreeOffice as office suite, you can configure HiDPI through the menus and works perfectly. Same with Steam.')
