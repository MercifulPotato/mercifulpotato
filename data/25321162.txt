Item(by='fiddlerwoaroof', descendants=None, kids=[25321634, 25321168], score=None, time=1607230594, title=None, item_type='comment', url=None, parent=25320588, text='I really liked Practical Common Lisp by Peter Seibel ( <a href="https:&#x2F;&#x2F;gigamonkeys.com&#x2F;book&#x2F;" rel="nofollow">https:&#x2F;&#x2F;gigamonkeys.com&#x2F;book&#x2F;</a> ) however, ignore the environment setup: the recommendations it makes are no longer maintained, these days Portacle is the best way to get started.<p>One thing I really like about this book is it assumes you know basic programming ideas, so it gets into distinctively “lispy” features relatively quickly. The chapters on exception handling and Common Lisp’s object system, in particular, really sort of blew my mind and had a fairly profound influence on the way I think about programming languages.')
