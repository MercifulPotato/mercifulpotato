Item(by='est31', descendants=None, kids=None, score=None, time=1606932633, title=None, item_type='comment', url=None, parent=25278277, text='You can do <i>cargo install --locked ripgrep_all</i> as a workaround. It uses the lockfile that&#x27;s part of the ripgrep_all package, so you miss out on some package updates, but can also get the cachedir version required.<p>There is a github issue to make this the default behaviour of cargo, but you miss out on updates which might fix security bugs so the cargo team is unwilling to change the default.<p><a href="https:&#x2F;&#x2F;github.com&#x2F;rust-lang&#x2F;cargo&#x2F;issues&#x2F;7169" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;rust-lang&#x2F;cargo&#x2F;issues&#x2F;7169</a>')
