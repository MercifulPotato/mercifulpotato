Item(by='musicale', descendants=None, kids=[25054634, 25053498], score=None, time=1605047404, title=None, item_type='comment', url=None, parent=25050050, text='Unfortunately with no native x86 support, true x86 virtualization goes away.<p>True virtualization will be for ARM (such as Linux or macOS, maybe embedded Win10 for ARM) VMs only. x86 Windows VMs will not work with native ARM virtualization, and x86 apps will run slowly if at all under emulation in ARM Win10.<p>However, Parallels (and VMware etc.) could potentially leverage Rosetta 2 to run x86 VMs in emulation on ARM, at greatly reduced speed compared to native x86 virtualization on intel Macs.<p>Boot Camp for x86 Windows is dead on these machines as well.')
