Item(by='antoinealb', descendants=None, kids=[24909962, 24925349, 24910151, 24910082, 24909099], score=None, time=1603815930, title=None, item_type='comment', url=None, parent=24904617, text='If you don&#x27;t know it, CMake can use ninja instead of make for compilation so instead of &quot;mkdir build &amp;&amp; cd build &amp;&amp; cmake .. &amp;&amp; make&quot; you can run &quot;mkdir build &amp;&amp; cd build &amp;&amp; cmake -GNinja &amp;&amp; ninja&quot;.<p>It feels faster on my side projects (haven&#x27;t benchmarked it really), but most importantly, when running a parallel build it does not interleave the output of the different steps, making it much easier to read warnings and errors.')
