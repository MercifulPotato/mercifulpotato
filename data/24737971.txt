Such a value can have two states: &#x27;None&#x27;, where it calls no value, and &#x27;Some(t)&#x27;,where it carries a value &#x27;t&#x27;.<p>For example, a division function could return (1) &#x27;None&#x27; for &#x27;div(20, 0)&#x27; because division-by-zero is undefined, and return (2) &#x27;Some 10&#x27; for &#x27;div(20, 2)&#x27; because 20&#x2F;2 is defined and is equal to &#x27;10&#x27;.')
