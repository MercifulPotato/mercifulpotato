Item(by='vbezhenar', descendants=None, kids=[24676346], score=None, time=1601779548, title=None, item_type='comment', url=None, parent=24675429, text='If your child node has unique name among its siblings and does not contain nested nodes, then it&#x27;s an attribute. Otherwise it&#x27;s an element. Seems pretty obvious to me.<p>The fundamental issue with XML is its impedance mismatch with common data structures which forces using Object to XML mappers (whether explicitly or implicitly). It&#x27;s more or less solved with XML Schemas or DTDs, but if you&#x27;re looking at just XML, you can&#x27;t tell whether some element is an array or a single node. Thus JSON is better suited for serialization.')
