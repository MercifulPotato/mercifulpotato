Item(by='amw-zero', descendants=None, kids=None, score=None, time=1610852719, title=None, item_type='comment', url=None, parent=25799719, text='My personal criticism of the web FE community is not that the tools haven’t solved problems, it’s that these problems have been solved before and that is not acknowledged.<p>For example, a single-page app is the exact same thing as a native frontend. It’s a stateful, long-running process that communicates with a backend. This is not a new paradigm, but it is treated as one in the web community.<p>Another example is declarative UI. Microsoft had XAML and data binding well before Backbone, Angular, and React.<p>I personally love where web UI is at today, but I get the sense that we think it’s full of these unique problems when it’s really not.')
