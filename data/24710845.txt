Item(by='andrewg', descendants=None, kids=None, score=None, time=1602093469, title=None, item_type='comment', url=None, parent=24709803, text='Early in my career I wrote Smalltalk in a GUI-based environment (VisualWorks&#x2F;GemStone), with the browser, debugger, transcript, etc. It was an incredible experience.<p>Later, I used Common Lisp with SLIME and had a very similar experience, at least as productive, except that I was using my preferred IDE (Emacs) and saved the code to normal files so I could use more common tools like grep, SVN&#x2F;Git, etc. I still built code into an image, which I could update live via the REPL.<p>I haven&#x27;t used GNU Smalltalk personally, but I do like the idea of a Smalltalk that&#x27;s less of an island, particularly if that makes it more palatable to a new generation of programmers.')
