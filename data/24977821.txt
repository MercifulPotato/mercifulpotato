Item(by='matsemann', descendants=None, kids=[24977963], score=None, time=1604394182, title=None, item_type='comment', url=None, parent=24959157, text='&gt; <i>Games calculate square roots and inverse square roots all the time to find the lengths of vectors</i><p>A trick here is that one often doesn&#x27;t have to do the square root. For instance, if you want something to happen when an object is 5 units away from another object, it&#x27;s normal to do<p><pre><code>    if sqrt( (x2-x1)^2 - (y2-y1)^2 ) &lt; 5 then ...\n</code></pre>\nbut instead you can do<p><pre><code>    if (x2-x1)^2 - (y2-y1)^2 &lt; 5^2 then ...\n</code></pre>\ntrading a sqrt for a squaring. And often the square of the distance can be cached or even a constant. So a lot of libraries (for instance vector library from libgdx[0]) contain a dst function, but also a dst2 function that skips the squaring to save some cycles when not needed.<p>[0]: <a href="https:&#x2F;&#x2F;libgdx.badlogicgames.com&#x2F;ci&#x2F;nightlies&#x2F;docs&#x2F;api&#x2F;com&#x2F;badlogic&#x2F;gdx&#x2F;math&#x2F;Vector2.html#dst2-com.badlogic.gdx.math.Vector2-" rel="nofollow">https:&#x2F;&#x2F;libgdx.badlogicgames.com&#x2F;ci&#x2F;nightlies&#x2F;docs&#x2F;api&#x2F;com&#x2F;b...</a>')
