Item(by='Ayesh', descendants=None, kids=None, score=None, time=1610827048, title=None, item_type='comment', url=None, parent=25804813, text='You don&#x27;t necessarily need to authenticate users on that domain with a cookie. An HMAC token would be ideal, because you don&#x27;t have to maintain state.<p>Don&#x27;t hardcore the content domain. In case the content domain gets flagged, it should be easy to change to a new domain.<p>The assets themselves (such as images, scripts, etc) can have any browser cache expiration time. HTML documents cache duration will matter, and once that has elapsed, browsers should start to use the new content domain.')
