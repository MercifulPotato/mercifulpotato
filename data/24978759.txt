Item(by='dfgdghdf', descendants=None, kids=None, score=None, time=1604404190, title=None, item_type='comment', url=None, parent=24977502, text='Writing an F# wrapper for a C# API is considerably less involved than writing an OCaml wrapper for a C API.<p>This is because F# and C# share:<p>* build tools<p>* package management<p>* garbage collector<p>* base types (string, int, etc).<p>The F# type-checker understands C# types, so you know that you are consuming the C# API correctly (at least at the type-level). F# can be be used as a new syntax for C#, if you wanted to go that way.')
