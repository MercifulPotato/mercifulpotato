When MySpace was at its peak, I used to run one of those &quot;style your profile&quot; type websites that let you generate custom CSS and all those fun things. I should try and dig it out and see if it works on SpaceHey!<p>Just a small CSS tip for your header and footer menus -- instead of using ::before&#x2F;::after pseudo elements in your anchors themselves, if you place the menu items in a list you can use them there so the vertical pipe isn&#x27;t part of the links. For example the following HTML:<p><pre><code>  &lt;ul class=&quot;menu&quot;&gt;\n    &lt;li&gt;&lt;a href=&quot;#&quot;&gt;Link 1&lt;&#x2F;a&gt;&lt;&#x2F;li&gt;\n    &lt;li&gt;&lt;a href=&quot;#&quot;&gt;Link 2&lt;&#x2F;a&gt;&lt;&#x2F;li&gt;\n  &lt;&#x2F;ul&gt;\n</code></pre>\nWith the following CSS:<p><pre><code>  .menu {\n    list-style: none;\n    padding: 0;\n    text-align: center;\n  }\n  .menu li {\n    display: inline;\n  }\n  .menu li:not(:last-child)::after {\n    color: #000;\n    content: &#x27; | &#x27;;\n  }\n</code></pre>\nLet&#x27;s you add anything to the menus (anchor tag or plain text) and they will be separated by the vertical pipe without being part of the links themselves.')
