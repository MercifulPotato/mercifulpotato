Item(by='choeger', descendants=None, kids=[25953634], score=None, time=1611866761, title=None, item_type='comment', url=None, parent=25948566, text='They didn&#x27;t say dead code analysis was decidable, but purity analysis.<p>Purity analysis is indeed trivially decidable, if you limit yourself to &quot;probably always pure&quot;, not &quot;pure when executed on a Tuesday&quot;. It boils down to &quot;is this function calling only pure functions and not using any of the impure operators?&quot;. The trouble comes with dynamic languages where you usually don&#x27;t know shit about a function&#x27;s arguments, though. In that case you need a control flow Analysis of the whole program.')
