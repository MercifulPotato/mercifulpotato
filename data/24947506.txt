Item(by='pgt', descendants=None, kids=None, score=None, time=1604097313, title=None, item_type='comment', url=None, parent=24944496, text='Here is what the SDLang example would look like in EDN (Extensible Data Notation), a subset of Clojure:<p><pre><code>    ; Maps are denoted as key-value pairs inside curly brackets:\n    {:title &quot;Hello, World&quot;}\n    \n    ; Vectors are denoted by square brackets:\n    [:bookmarks 12 15 188 1234]\n    \n    ; Or you can use a map with a keyword that maps to a vector:\n    {:bookmarks [12 15 188 1234]}\n    \n    ; Maps are collections of key-value pairs:\n    {:author  &quot;Peter Parker&quot;\n     :email   &quot;peter@example.org&quot;\n     :active? true}\n    \n    ; Data structures are heterogeneous and nest:\n    [:contents\n     [:section &quot;First section&quot;\n      [:p &quot;This is the first paragraph&quot;]\n      [:p &quot;This is the second paragraph&quot;]]]\n    ; ^ Hiccup actually renders these to HTML.\n    \n    ; Everything is an expression, so strings just work:\n    &quot;This text is the value of an anonymous node!&quot;\n    \n    ; A matrix is just a vector:\n    [1 0 0\n     0 1 0\n     0 0 1]\n    \n    ; Or you could partition it into rows:\n    [[1 0 0]\n     [0 1 0]\n     [0 0 1]]\n</code></pre>\n<a href="https:&#x2F;&#x2F;github.com&#x2F;edn-format&#x2F;edn" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;edn-format&#x2F;edn</a>')
