Item(by='pjmlp', descendants=None, kids=None, score=None, time=1605686940, title=None, item_type='comment', url=None, parent=25134004, text='As an example of how it works both ways is Eiffel.<p>While you are developing you get a JIT based enviroment with the productivity workflow one knows from languages like Java and .NET (MELT VM).<p>Then when it comes to a proper release, you use the AOT compiler which uses C or C++ based compiler as part of the build workflow (Eiffel AOT compiler generates C or C++ code as its intermediate format).<p>So you get the quick edit-compile-debug workflow, and when feeling like doing a full release build just let the C or C++ compiler do its job.<p>There are other examples, I just choose Eiffel as one of thm.')
