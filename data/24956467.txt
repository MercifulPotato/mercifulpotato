Item(by='pabs3', descendants=None, kids=None, score=None, time=1604191798, title=None, item_type='comment', url=None, parent=24951099, text='Both static linking and embedded copies have similar issues.<p>With static linking you have to track where the static library ended up (recursively) and then issue rebuilds in the correct order for all packages that contain the static library (directly or indirectly).<p>With embedded copies you have to search the source code archive for the copies and then patch and rebuild each copy.<p>In some ways static linking is more complicated to deal with, due to the case that the static library ends up in some other static library.')
