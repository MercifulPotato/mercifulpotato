Item(by='toast0', descendants=None, kids=None, score=None, time=1609703025, title=None, item_type='comment', url=None, parent=25623204, text='As implemented today, ECC is a feature of the memory controller. You need special ram, because instead of 8 parallel rams per bank, you need 9, and all the extra data lines to go to the controller.<p>Modern CPUs have integrated memory controllers, so that&#x27;s why the CPU needs to support it.<p>Correction without reporting isn&#x27;t great; anyway, you <i>need</i> a reporting mechanism for uncorrectable errors, or all you&#x27;ve done is ensure any memory errors you do experience are worse.')
