Useful if you&#x27;re using the disk a lot, for any purpose.<p>Let&#x27;s say you have 200 JS libraries that you need to install dependencies, build and test for each commit. Making the JS dependencies install into a common cache directory into &#x2F;dev&#x2F;shm will easily speed up the install phase (and make the projects share dependencies cache fairly easily), and you could also make the test data (like fixtures) go through &#x2F;dev&#x2F;shm if you need it to.')
