This system did all the compute and data management for roughly 30 users.  I&#x2F;O streaming was easy and efficient.<p>IRIX allowed for live file system backups and the drive doing compression meant all that happening with negligible user performance impact.<p>Was literally set it and forget it, aside from tape rotation into off site storage.<p>Compression would have had an impact.<p>We don&#x27;t do multi user app serving much today, so maybe a smart drive has less benefit.  But it mattered then.  2000&#x27;s era.')
