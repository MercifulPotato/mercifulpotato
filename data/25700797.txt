Item(by='shafyy', descendants=None, kids=None, score=None, time=1610204010, title=None, item_type='comment', url=None, parent=25693363, text='It is good, but not good enough for the mainstream user. With Signal, you can just install it and all your contacts who also have signal are instantly there. The UX is similar to WhatsApp.<p>With Element, you first need to create a username that in the format &lt;username&gt;:&lt;server&gt;. This is a new notion and non-tech users don&#x27;t know what this is about. I usually need to explain &quot;it&#x27;s like email, but instead of the @ there&#x27;s the :&quot;.<p>Furthermore, there is this new concept of &quot;Rooms&quot; that people are not used to.<p>Also, there are some UX flaws like that you need to include the @ when adding use (it won&#x27;t find the user if I try to add &quot;joe:matrix.org&quot;, I need to enter @joe:matrix.org). This even threw me off for a minute or so in the beginning.<p>There are some clients that look and feel more like WhatsApp (like Nio), but they are still in early development. So, I&#x27;m hopeful.<p>See the clients here: <a href="https:&#x2F;&#x2F;matrix.org&#x2F;clients&#x2F;" rel="nofollow">https:&#x2F;&#x2F;matrix.org&#x2F;clients&#x2F;</a>')
