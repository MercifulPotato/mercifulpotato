Item(by='Someone', descendants=None, kids=None, score=None, time=1611447296, title=None, item_type='comment', url=None, parent=25886880, text='Chances are that your Mac binary is a MacOS package (<a href="https:&#x2F;&#x2F;en.m.wikipedia.org&#x2F;wiki&#x2F;Package_(macOS)" rel="nofollow">https:&#x2F;&#x2F;en.m.wikipedia.org&#x2F;wiki&#x2F;Package_(macOS)</a>: <i>“In the Apple macOS operating system, a package is a file system directory that is normally displayed to the user by the Finder as if it were a single file”</i>)<p>Right-click the binary and select “Show package contents” to peek inside. Chances are you’ll find a code signature, an icon, various support files, one or more jars and an executable that launches the main jar.<p>Figuring out what the main jar is typically is easy. You may have to experiment to find out what the current directory has to be when launching it.')
