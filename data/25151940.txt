Item(by='antonf', descendants=None, kids=[25152027, 25151967, 25152571], score=None, time=1605807215, title=None, item_type='comment', url=None, parent=25151094, text='TryInto is already implemented for slices in 1.47, so `let foo = u16::from_be_bytes(some_slice.try_into().unwrap());` would work before 1.48.<p>This trait wasn&#x27;t implemented for vectors though, so you would have to write `let foo = u16::from_be_bytes(some_vec.to_slice().try_into().unwrap());`.<p>With new release you can drop `to_slice()` part. Nothing earth-shattering, but makes sense.')
