From <a href="https:&#x2F;&#x2F;docs.python.org&#x2F;3&#x2F;library&#x2F;subprocess.html#replacing-shell-pipeline" rel="nofollow">https:&#x2F;&#x2F;docs.python.org&#x2F;3&#x2F;library&#x2F;subprocess.html#replacing-...</a><p><pre><code>  output=$(dmesg | grep hda)\n</code></pre>\nbecomes<p><pre><code>  p1 = Popen([&quot;dmesg&quot;], stdout=PIPE)\n  p2 = Popen([&quot;grep&quot;, &quot;hda&quot;], stdin=p1.stdout, stdout=PIPE)\n  p1.stdout.close()  # Allow p1 to receive a SIGPIPE if p2 exits.\n  output = p2.communicate()[0]\n</code></pre>\nThis isn&#x27;t a contrived example. One reason I like using macros in Python is because it simplifies exactly this boilerplate.<p>You can write a simplifying function, but programs rarely do.')
