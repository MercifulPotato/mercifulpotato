Item(by='fnord123', descendants=None, kids=None, score=None, time=1603842010, title=None, item_type='comment', url=None, parent=24906809, text='I am intentionally conflating the C ABI with the C++ ABI because the C++ ABI is only relatively stable so if you want  to write software that doesn&#x27;t require you to rebuild everything each time you update a compiler then you push your code to use more C ABIs rather than more advanced C++isms. This is exactly what SDL has done as the headers are C compatible but it has plenty of C++ in the direct3d parts.')
