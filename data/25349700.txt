Item(by='tptacek', descendants=None, kids=[25350437, 25350338], score=None, time=1607453438, title=None, item_type='comment', url=None, parent=25349329, text='It&#x27;s generally not exploitable if the crash is triggered by a read, not a write, and it&#x27;s not clear from the advisory which it is; writes to NULL can be exploitable if NULL is mapped (which is a configuration issue more than a permissions issue; it is most frequently not) or, more commonly, if the write address includes an attacker-controlled offset (though it may be the case that we&#x27;ve stopped referring to those conditions as NULL-involved).<p>The ability to map arbitrary memory is, of course, as you point out, a game-over vulnerability by itself.')
