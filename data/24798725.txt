Item(by='gibs0ns', descendants=None, kids=[24798864], score=None, time=1602835882, title=None, item_type='comment', url=None, parent=24797643, text='It might be worth looking at ESPHome (<a href="https:&#x2F;&#x2F;esphome.io&#x2F;" rel="nofollow">https:&#x2F;&#x2F;esphome.io&#x2F;</a>), it&#x27;s very similar to Tasmota but modular and is configured via yaml files. It also supports OTA updates for firmware&#x2F;config changes (after the initial flash).\nI feel ESPHome scales better than Tasmota for these reasons - no need to visit the individual web UI&#x27;s for each Tasmota device, using ESPHome you can update your configs from a central web UI and then push the changes to your devices via wifi.<p>TLDR; Tasmota is great for bespoke devices, but IMO ESPHome is better because it&#x27;s modular and supports OTA updates.')
