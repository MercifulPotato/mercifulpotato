Item(by='fxtentacle', descendants=None, kids=None, score=None, time=1606988019, title=None, item_type='comment', url=None, parent=25286749, text='Having built an offline-capable distributed database in go, I noticed that even &quot;larger&quot; projects in go tend to be a lot less source code than building something similar in Java.<p>And maybe that&#x27;s just my personal style, but to me defer is much more important than generics.<p>I can&#x27;t even remember how often I&#x27;ve had to debug Java bugs where someone forgot to close a ResultSet for an SQL query... And then you leak memory so badly that restarting tomcat with a daily cron is still not enough :(<p>With defer, you can expect people to write the cleanup code immediately after opening something, which makes it much easier to confirm that you&#x27;re not leaking resources.')
