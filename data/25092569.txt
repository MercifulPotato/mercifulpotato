Item(by='colonwqbang', descendants=None, kids=None, score=None, time=1605365294, title=None, item_type='comment', url=None, parent=25080060, text='Concurrency is not the same as parallelism. Python has good concurrency support, I agree. Python (C Python) does not support parallelism however due to its Big Interpreter Lock which actively prevents any parallelism in Python code.<p>This was probably a conscious design decision on the part of C Python implementers and perhaps a good one. But we should not claim that Python is something which (actively and by design) it&#x27;s not.')
