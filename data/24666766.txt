Item(by='senderista', descendants=None, kids=None, score=None, time=1601671003, title=None, item_type='comment', url=None, parent=24657747, text='A good rule of thumb is &quot;if it fits in a cache line, then computation is free&quot;, where &quot;computation&quot; could mean search, hashing, compression, etc. This means, e.g., that linear search generally beats binary search if the whole array fits in a single cache line (or half a cache line if it&#x27;s unaligned).')
