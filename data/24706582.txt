Item(by='arnvald', descendants=None, kids=None, score=None, time=1602063310, title=None, item_type='comment', url=None, parent=24690947, text='An interesting thing I&#x27;ve read recently is that there can be different kinds of monolith:<p>- application monolith (what we usually talk about)<p>- database monolith (multiple apps all using one db)<p>- deployment monolith (multiple apps but need to be deployed together)<p>- organization monolith (multiple apps built by teams that have to follow the same processes and practices, or using the same technologies)<p>This made me think that migration to services has different layers and just running multiple apps might solve some problems, bet introduce others. I think learning how to slice things properly is the key here and is often only considered in terms of source code.')
