Item(by='jlebar', descendants=None, kids=[24796790, 24794602, 24794653, 24794877, 24794796, 24794914, 24794545], score=None, time=1602794967, title=None, item_type='comment', url=None, parent=24790218, text='My favorite (not actually horrible) switch statement trick:<p>Instead of<p><pre><code>  string x;\n  switch (y) {\n    case 0: x = &quot;foo&quot;; break;\n    case 1: x = &quot;bar&quot;; break;\n  }\n</code></pre>\ntry an IIFE!<p><pre><code>  string x = [&amp;] {\n    case(y) {\n      case 0: return &quot;foo&quot;;\n      case 1: return &quot;bar&quot;;\n    }\n  }();\n</code></pre>\nOnce you get used to reading it, there are a bunch of advantages.  Among them, you can&#x27;t forget a &quot;break&quot;, and you can&#x27;t forget to assign into `x`.')
