Item(by='p_l', descendants=None, kids=[24892013], score=None, time=1603674023, title=None, item_type='comment', url=None, parent=24890139, text='This is the problem with people not understanding how Secure Boot works, then reading specs that require such understanding.<p>For max-level compatibility with Windows 10 Enterprise, there&#x27;s a requirement that owner of the device can change all the keys. It doesn&#x27;t require a &quot;disable Secure Boot&quot; setting. Worst case, the device won&#x27;t permit you to keep it in what&#x27;s called &quot;Setup Mode&quot; without some inconvenience, however <i>you have to be able to enter Setup Mode</i>.<p>What&#x27;s <i>Setup Mode</i>?<p><i>Setup Mode</i> is when the PK variable (PlatformKey) is empty, effectively disabling SecureBoot. This mode exists so that you can load new PK value. According to UEFI Secure Boot spec, it&#x27;s supposed to be frozen by manufacturer. However, being able to set your own one is critical to some of the Windows Enterprise features, and as such a way to enter that mode is required by Windows certification.')
