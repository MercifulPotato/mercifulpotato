Item(by='horsawlarway', descendants=None, kids=None, score=None, time=1602872882, title=None, item_type='comment', url=None, parent=24798493, text='Duktape supports ES5, with only partial support for anything ES6+ (and not much partial support, at that).<p>That said - having used Duktape in production, it&#x27;s a very handy little library. Small, easy to drop into projects, great way to add support for custom user scripts or to run js code somewhere outside of a browser context.<p>Also - insanely easy to target still, despite the lack of ES6+, as long as you already have babel somewhere in your build pipeline.')
