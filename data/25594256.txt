Item(by='pjmlp', descendants=None, kids=[25596012], score=None, time=1609433155, title=None, item_type='comment', url=None, parent=25594010, text='AssemblyScript is one of the best options, because if the only thing you care about is some high level language that outputs WebAssembly, then you are still using Web related tools.<p>Then D is quite easy to use (LDC alongside betterC mode, runtime is still not fully ported).<p>Followed by TinyGo, Rust, C#, Nim.<p>I would leave Emscripten, C and C++, only for porting existing code into the Web, as even with the sandbox memory corruptions of data allocated in the linear memory blocks are susceptible to occur.')
