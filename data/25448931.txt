Item(by='duskwuff', descendants=None, kids=[25449069], score=None, time=1608154017, title=None, item_type='comment', url=None, parent=25448583, text='PDF is not a stack machine. This is a common misconception.<p><i>PostScript</i> was a stack machine with included operators which implemented a set of vector graphics commands. PDF has a similar rendering pipeline, but it renders a set of primitives described directly in the document -- there is no virtual machine involved.')
