Item(by='capableweb', descendants=None, kids=None, score=None, time=1604761700, title=None, item_type='comment', url=None, parent=25015430, text='Think it&#x27;s important we set the context to be around Arch Conf, where we&#x27;re talking about hosting recordings from the conference.<p>In this case: Yes, if they do run their own servers for storing the content, they are self-hosting. No, doing your own peering is not needed to be classified as self-hosting. No, you don&#x27;t need to run your own data center to be self-hosting. Unless you want to self-host the servers themselves, then yes, running your own data center is needed.<p>I think the most important point is &quot;hosted&quot; vs &quot;self-hosted&quot; here, where hosted is letting someone else run the servers where you put the content, while self-hosted is you running the servers where you put the content.')
