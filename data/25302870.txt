Item(by='rectang', descendants=None, kids=None, score=None, time=1607093765, title=None, item_type='comment', url=None, parent=25300310, text='Having written my fair share of bit-twiddling hacks in low-level code, my takeaway is that because the hack code doesn&#x27;t resemble what it actually does, it&#x27;s all too easy to create bugs.  Unit testing the living daylights out of any bit-twiddling code path is a must.<p>Also helpful: being able to represent integer types with binary.  Since I&#x27;m not a genius and I write code to be maintained by other not-geniuses, `mask = 0b_1010_1010` is clearer than `mask = 0xAA`.')
