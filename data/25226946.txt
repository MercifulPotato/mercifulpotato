Item(by='oarsinsync', descendants=None, kids=[25227652], score=None, time=1606469403, title=None, item_type='comment', url=None, parent=25226784, text='If you&#x27;re depending on HA&#x27;s built-in auth, you&#x27;re at risk of bugs in that code.<p>If you throw up HTTP basic auth on your HTTPS front end, you&#x27;ve now moved your risk away from bugs in HA&#x27;s code, to bugs in nginx&#x27;s (or whatever HTTP server you use) implementation of HTTP basic auth <i>AND</i> bugs in HA&#x27;s code (odds of this being a real event just got smaller)<p>If you throw a VPN in front of all that as well, you&#x27;ve now moved your risk away from bugs in nginx&#x27;s code, to bugs in your VPN implementation <i>AND</i> bugs in nginx&#x27;s code <i>AND</i> bugs in HA&#x27;s code (odds are now really small).<p>This is also known as defence in depth.')
