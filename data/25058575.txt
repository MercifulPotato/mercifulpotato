Item(by='andygrove', descendants=None, kids=None, score=None, time=1605104367, title=None, item_type='comment', url=None, parent=25055315, text='Apache Arrow is a specification for in-memory columnar data, IPC format + Flight protocol, with implementations in a number of languages. Some of the implementations contain code to perform computations on the in-memory data. Some of the implementations contain some form of query engine. All of these are single process &#x2F; libraries, rather than distributed systems.<p>Apache Spark is a distributed compute platform, which does have some support for Arrow for interop purposes.<p>One of the things that led me to get involved in Arrow originally was to explore the idea of building something like Apache Spark based on Arrow (and Rust) and my latest prototype of that concept is in the Ballista project [1].<p>[1] <a href="https:&#x2F;&#x2F;github.com&#x2F;ballista-compute&#x2F;ballista" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;ballista-compute&#x2F;ballista</a>')
