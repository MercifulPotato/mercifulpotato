This typically isn&#x27;t an issue since you would be unlikely to use user input in this context.<p><a href="https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-vars&#x2F;" rel="nofollow">https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-vars&#x2F;</a><p>hx-vars evaluate expressions to include in the request.  This is a bit more dangerous, because you might try to pass values through with this mechanism.  If this includes user input then you should use hx-vals instead:<p><a href="https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-vals&#x2F;" rel="nofollow">https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-vals&#x2F;</a>')
