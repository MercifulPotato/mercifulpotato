Item(by='marcan_42', descendants=None, kids=None, score=None, time=1608963238, title=None, item_type='comment', url=None, parent=25539550, text='Simple ports to small systems are actually very easy. To run Linux all that you strictly need is a timer driver and an interrupt controller driver, assuming the CPU architecture is already supported and there is a compatible MMU. On small systems, those drivers can be very simple.<p>I was bored one evening and ported Linux to run on the Wii&#x27;s IO&#x2F;security processor (not the main CPU, that&#x27;s Wii Linux). It only took a few hours to get the kernel booting to userspace.')
