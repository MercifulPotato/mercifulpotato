Item(by='masklinn', descendants=None, kids=[25151436, 25153815], score=None, time=1605804141, title=None, item_type='comment', url=None, parent=25151192, text='Yes, they&#x27;re absolutely nothing alike.<p>Shared memory means the CPU and the GPU share physical memory in different address space, so 512MB RAM with 64MB shared means the CPU gets 448MB and the GPU gets 64, and data has to be copied from one chunk to the other.<p>Unified memory means both CPU and GPU not only share physical memory but <i>share the address space</i>, all memory is accessible to both at the same time. This means that e.g. you can create commands or load data in memory on the CPU then just tell the GPU &quot;your stuff is here&quot; with no copy needed anywhere, and likewise the other way around.')
