Item(by='rndgermandude', descendants=None, kids=None, score=None, time=1604995306, title=None, item_type='comment', url=None, parent=25041276, text='C# has the using-statement + IDisposable which is similar context managers in python (except no __enter__).<p>C#8 then allowed using-statements that do not require blocks, which then are disposed when leaving the current block.<p>Pre C#8, you had to have a block, similar to how with-statement requires a block aka indentation<p><pre><code>    using (var res = OpenSomeResource()) {\n      res.doSomething();\n      &#x2F;&#x2F; res.Dispose() implicitly called\n    }\n</code></pre>\nNow you can have this (which is a bit like a go-style Defer but for blocks)<p><pre><code>    using var res = OpenSomeResource();\n    res.doSomething();\n    &#x2F;&#x2F; res.Dispose() implicitly called at end of current block\n</code></pre>\nOf course, you have to take the potentially longer lifetime into account. This way looks a lot like C++s RAII with constructors and destructors (except the destructor here is the  IDisposible.Dispose implementation).')
