Item(by='WalterBright', descendants=None, kids=None, score=None, time=1606512799, title=None, item_type='comment', url=None, parent=25227495, text='That was considered, but the problem is what does the user think when he sees `a &lt; b &lt; c` ?<p>1. is it intended to be the C way?<p>2. is it the Python way?<p>3. did the user assume it was (1) or did he assume it was (2) and guess wrong?<p>This means that such expressions are a never-ending red flag for a code review. Hence it is made an error. The user can still get the C behavior by using parentheses.<p>Note that if D was a Python-derived language, then selecting (2) semantics would be a no-brainer. But as a C-derived language, it has to be an error.')
