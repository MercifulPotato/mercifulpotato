Item(by='war1025', descendants=None, kids=None, score=None, time=1606711119, title=None, item_type='comment', url=None, parent=25250564, text='I think `pass` has an official extension to handle OTP, but here is how I do it:<p><pre><code>  #&#x2F;bin&#x2F;bash\n\n  clip $(oathtool -b --totp &quot;$(pass show otp&#x2F;$1 | head -n 1)&quot;)\n\n</code></pre>\nBasically there is a really handy utility called `oathtool` you can install that has all the one time password logic built into it.<p>Then in my `pass` store I have a `otp` folder where I put the codes for my different services.<p>So to get the OTP code for paypal for example, I do this:<p><pre><code>   pass otp paypal\n</code></pre>\nAnd then the code is copied to my clipboard and I can just paste it.')
