I did learn some new advantages of async over threading from replies, besides tired C10K. Yes, async is useful for C10K. No, I am not solving C10K problem.<p>1. If you use async in single thread mode, you can save thread synchronization.<p>2. async works better for idle connections and slow connections, even when the absolute number of connections is not large.<p>3. async task is easier to cancel than thread.<p>I still won&#x27;t use async since thread synchronization hasn&#x27;t been slow for me, thread cancellation hasn&#x27;t been problematic for me, and I use nginx to handle idle connections and slow connections, but it&#x27;s useful to know in case I need.')
