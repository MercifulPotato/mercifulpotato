Item(by='brian_cloutier', descendants=None, kids=None, score=None, time=1604021702, title=None, item_type='comment', url=None, parent=24927800, text='A small user base is absolutely a problem, if there are only so many zcash nodes then traffic analysis becomes very easy.<p>Though, if there are only a few transactions each day that&#x27;s not as large of an issue as it sounds; the blockchain still has no linkability between transactions.
