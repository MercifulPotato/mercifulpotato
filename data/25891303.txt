Item(by='xorcist', descendants=None, kids=[25891370, 25891797], score=None, time=1611489132, title=None, item_type='comment', url=None, parent=25880414, text='That answer does not seem very useful to me. The use cases are different.<p>Reasons to use dd include being able to set the block size, seek and skip to resume copies or extract parts of the data, and a whole bunch of special options for the very specific use case of large binary files or block devices.<p>Reasons to use cp include it being file based, so metadata and recursive copying is natural, efficient, and easy to use. This probably includes most copying on a daily basis. See also rsync.')
