Item(by='lqet', descendants=None, kids=None, score=None, time=1603124183, title=None, item_type='comment', url=None, parent=24826067, text='&gt; Obviously, the purpose of before&#x2F;after is to produce a visual effect; then why produce pseudo-content if CSS is oh-so-great a language for presentation?<p>Because sometimes the best way to style something is by using a character (bullet points, arrows, ...). Example: you could add a symbol to URLs by giving &lt;a&gt; a slight padding on the left and by adding an image showing an URL symbol. But by using &amp;#x1f517; in a:before, you cheaply get vector graphics, and a symbol which (in a perfect world) will always match the font.')
