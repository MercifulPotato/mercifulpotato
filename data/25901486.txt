Item(by='tsimionescu', descendants=None, kids=None, score=None, time=1611569259, title=None, item_type='comment', url=None, parent=25899362, text='Presumably, if you&#x27;re exchanging data between languages A and B, you have already figured out how to translate you want to map all of the data structures between them.<p>Then, what I would expect you do is that you have code in A that can translate Complex Data Structure -&gt; Lowest Common Denominator Data Structure -&gt; A built-in JSON serialization -&gt; JSON -&gt; B built-in JSON deserialization -&gt; Lowest Common Denominator -&gt; Complex Data Structure B.<p>Of course, the language&#x2F;library can&#x27;t help with the complex -&gt; lowest common denominator translation part, that one will have to get written by hand. But that shouldn&#x27;t mean that the lowest common denominator has to be as basic as JSON itself.')
