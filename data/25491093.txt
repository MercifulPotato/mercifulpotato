Item(by='Blikkentrekker', descendants=None, kids=[25491541], score=None, time=1608512504, title=None, item_type='comment', url=None, parent=25489700, text='It breaks everything of the sort.<p>On <i>X11</i>, a client can view the contents of any arbitrary window, simulate input, read input — this is necessary obviously for screen sharing applications; though one can argue that maybe a permission model needs to exist to allow some clients to run with lower privileges — the funny thing is that it does; it&#x27;s called te <i>XSECURITY</i> extension, it&#x27;s very easy to use, and seldom used for whatever reason though <i>SSH</i> uses it whenever it can.<p>On <i>Wayland</i>, no client can do such things, but many clients such as screen sharing tools, profiling tools, and whatever else that needs to simulate input and read surfaces obviously needs to.<p><i>Wayland</i> developers simply say “It must be built into the compositor”, but no compositor can be expected to provide all that, and no compositor does.')
