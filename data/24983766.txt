Item(by='pydry', descendants=None, kids=None, score=None, time=1604436266, title=None, item_type='comment', url=None, parent=24983664, text='I think of it as writing everything twice to appease the unit testing gods. If done indiscriminately it really is more of a ritual to ward of the bug gremlins than an effective development practice.<p>Integration tests do suffer from a whole set of problems that unit tests do not but with one fundamental difference: they&#x27;re usually problems that are soluble with better engineering techniques.<p>Integration tests that go red when something other than the code changes often provide useful feedback, so I wouldn&#x27;t necessarily rule them out.')
