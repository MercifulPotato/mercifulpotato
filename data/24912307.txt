Item(by='temporallobe', descendants=None, kids=[24915558], score=None, time=1603835189, title=None, item_type='comment', url=None, parent=24904204, text='This is similar to a UI solution a colleague and I came up with. The action the user could kick off was unstoppable and irreversible (a large batch job), and it seemed like even a confirmation prompt was too easy to simply click through. So we had the UI present a modal dialog asking the user to type in a specific word in all caps to confirm the action. Worked like a charm.')
