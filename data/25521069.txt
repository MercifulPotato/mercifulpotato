Item(by='aliceryhl', descendants=None, kids=[25521100], score=None, time=1608751286, title=None, item_type='comment', url=None, parent=25521011, text='The Rust language provides the async&#x2F;await syntax, which can turn imperative code into Future objects, but to run those Future objects, you must repeatedly call their poll method. Tokio is the piece of code that calls that method. It does so in a manner such that futures are only polled if able to continue work, and not e.g. waiting for a timer.<p>Besides that it provides lots of utilities for working with async code.')
