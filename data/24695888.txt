Item(by='unwind', descendants=None, kids=None, score=None, time=1601974691, title=None, item_type='comment', url=None, parent=24691422, text='On the topic of &quot;simple&quot; lossless compression, my favorite is probably LZJB from the ZFS file system [1].<p>I&#x27;m very far from an expert, I found that when searching a couple of years ago for something simple and usable in a semi-tight embedded setting.<p>When managing software installation&#x2F;upgrade for an embedded solution, LZJB typically shaved 30-50% off the size of the executable which was really awesome. LZJB requires a quite small amount of RAM when decompressing (around 1 KB if I remember correctly).<p>I have a repo with a BSD-licensed implementation in pure Python, which is very useful for putting in the build pipeline [2].<p>[1] <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;LZJB" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;LZJB</a><p>[2] <a href="https:&#x2F;&#x2F;github.com&#x2F;unwind&#x2F;python-lzjb" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;unwind&#x2F;python-lzjb</a>')
