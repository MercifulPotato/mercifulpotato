Item(by='amelius', descendants=None, kids=[25316442, 25316514, 25315870], score=None, time=1607185248, title=None, item_type='comment', url=None, parent=25315480, text='&gt; Why would you write your own basic data structures when the standard library provides them for you?<p>I think the problem with this view is that Rust is not being used as a true programming language (like for implementing algorithms), but more as a &quot;glue language&quot;.<p>If all the interesting stuff has to be done in libraries in &quot;unsafe&quot; mode, and all the uninteresting stuff (gluing) is done with  borrow checking, then Rust is more a glue language. Which is fine. But don&#x27;t call it a general purpose programming language.')
