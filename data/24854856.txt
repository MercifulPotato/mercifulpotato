Item(by='pjmlp', descendants=None, kids=None, score=None, time=1603340612, title=None, item_type='comment', url=None, parent=24852235, text='Calling into native is by definition the &quot;unsafe&quot; code block of managed languages, even if you are calling into Rust, unless you explicitly validated the code of the dependency being loaded.<p>There are zero guarantees that the Rust code is correct (free of logical errors), or makes use of unsafe in ways that are properly correct, and given cargo, you also need to validate every dependency that is brought into the shared library.<p>The only guarantees are that it will abort&#x2F;panic on bounds checking, numeric overflows and whatever unsafe code it has (regardless of its correctness), is explicitly marked in unsafe code blocks.')
