Item(by='Tyr42', descendants=None, kids=None, score=None, time=1602868701, title=None, item_type='comment', url=None, parent=24797687, text='Yeah, it&#x27;s called Bottom (⊥)<p><pre><code>    fun foo(): ⊥ {\n        return foo();\n    }\n\n</code></pre>\nIn Rust it&#x27;s called !, and you can write<p><pre><code>    fn will_panic() -&gt; ! {\n        panic!(&quot;uh oh&quot;);\n    }\n</code></pre>\nThis is actually useful for some embedded systems, where the main function isn&#x27;t allowed to return, and gets declared with the Nothing &#x2F; Bottom &#x2F; ! type.<p>The compiler is also smart enough to say that an infinite loop has type !, so you can have<p><pre><code>    fn main() -&gt; ! {\n        let setup = ...;\n        loop {\n            blink_led();\n            sleep(100);\n        }\n    }\n</code></pre>\nand it&#x27;ll complain if you put a break statement in the loop.')
