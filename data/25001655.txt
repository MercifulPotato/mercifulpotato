Earlier versions of Internet Explorer worked with Visual Basic Script. You enclosed Visual Basic within a script tag and a language attribute of &quot;VBScript&quot;. It did not transpile to Javascript, however.<p><a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;VBScript" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;VBScript</a><p>As I recall, you could debug VBScript by attaching a running IE process to VisualStudio. Breakpoints, instruction stepping, etc.<p>That was light years ahead of Javascript debugging. Of course, later FireBug and DevTools changed that.<p>But the original intent of the script tag was to be language agnostic.')
