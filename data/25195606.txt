If so, how much effort will it take and on what factors the effort required to port may vary substantially? (eg: Programming Language).<p>Yes.  To make Nix really work it has to be pervasive and this sometimes means porting packages again, however there&#x27;s a lot of automation around this, see[0] for ones that leverage existing build systems in other languages. Autoconf  and cmake packages are already accounted for by the default builder, and usually only the dependencies have to be specified.<p>&gt; * How is disk space usage? Is there any kind of deduplication?<p>The Nix store can be intensive on memory, as once a dependency such as glibc is updated, all the reverse dependencies (i.e. the set X such that X depends on glibc directly or transitively) have to be rebuilt as well.  There is an option to optimise the store, but I don&#x27;t know how much it helps in practice, see[1].  Garbage collection can be invoked manually or automatically to free up unused store entries.<p>&gt; * Other performance characteristics - bandwidth (delta upgrades possible?), startup time (I hope this will be good)..<p>By startup time you might be referring to NixOS.  It&#x27;s very good on my Late 2013 13-inch MacBook Pro, around 20 seconds to go from cold boot to login screen.<p>[0] <a href="https:&#x2F;&#x2F;github.com&#x2F;svanderburg&#x2F;node2nix" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;svanderburg&#x2F;node2nix</a> <a href="https:&#x2F;&#x2F;github.com&#x2F;NixOS&#x2F;cabal2nix" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;NixOS&#x2F;cabal2nix</a> <a href="https:&#x2F;&#x2F;github.com&#x2F;cargo2nix&#x2F;cargo2nix" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;cargo2nix&#x2F;cargo2nix</a> <a href="https:&#x2F;&#x2F;github.com&#x2F;nix-community&#x2F;poetry2nix" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;nix-community&#x2F;poetry2nix</a> <a href="https:&#x2F;&#x2F;github.com&#x2F;kamilchm&#x2F;go2nix" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;kamilchm&#x2F;go2nix</a> <a href="https:&#x2F;&#x2F;github.com&#x2F;nix-community&#x2F;pip2nix" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;nix-community&#x2F;pip2nix</a><p>[1] <a href="https:&#x2F;&#x2F;nixos.wiki&#x2F;wiki&#x2F;Storage_optimization" rel="nofollow">https:&#x2F;&#x2F;nixos.wiki&#x2F;wiki&#x2F;Storage_optimization</a>')
