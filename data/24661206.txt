Item(by='siraben', descendants=None, kids=[24665918, 24666795], score=None, time=1601635640, title=None, item_type='comment', url=None, parent=24657747, text='It&#x27;s not uncommon in scientific computing for the algorithm used to be automatically adjusted based on the input. An example of this is if you want to multiply very large integers. Karatsuba is first used, then Toom-Cook, then Schönhage–Strassen.<p>Karatsuba: θ(n^1.58)<p>Toom-Cook: θ(n^1.46)<p>Schönhage–Strassen: O(n log(n) log(log(n)))<p>Benchmarking is really the only thing that can reveal this kind of thing, otherwise you&#x27;ll end up with a theoretically fast but practically slow algorithm.')
