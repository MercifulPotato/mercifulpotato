If you need control, etc., characters via backslash escapes, you must use double (or semantic equivalents like %&#x2F;%Q). If you need non-eyesore strings with literal backslashes, you need single (or semantic equivalents like %q.)<p><pre><code>  irb(main):001:0&gt; puts &#x27;1\\t2&#x27;,&quot;1\\t2&quot;\n  1\\t2\n  1       2\n</code></pre>\n&gt; In Ruby, knowing that a string has interpolation is easy enough by seeing the #{} in the middle of it.<p>{} are not required for instance variables.')
