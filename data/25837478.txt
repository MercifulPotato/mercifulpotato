Item(by='gillesjacobs', descendants=None, kids=None, score=None, time=1611083701, title=None, item_type='comment', url=None, parent=25836498, text='I cannot agree with OP more. \nML model code itself is too often seen as documentation for a paper, in the sense that authors implicitly expect users to go through the pre-processing pipeline to find the actual implementation steps.<p>This is because the data handling and pre-processing nitty gritty is not actually interesting from an academic perspective.<p>I cannot count the times I went to look at a paper&#x27;s implementation source code on a benchmark dataset to find it cut-off annotated sequences to a fixed max length, essentially turning it into a different dataset making comparison to previous work invalid.<p>Good documentation costs time, time academics don&#x27;t have.')
