Item(by='ChrisMarshallNY', descendants=None, kids=None, score=None, time=1604541303, title=None, item_type='comment', url=None, parent=24994897, text='I think decompilers are so good, that any binary or pcode can be reverse-engineered by anyone that wants.<p>I generally like to open-source all my work. I&#x27;m working on a closed-source app, right now, but I think that it should be made open-source, once the embargo has passed. The backend is already open-source, as is the SDK.<p>I like to use the MIT license, which says that you can use the code how you like, but don&#x27;t come whining to me, if you pooch it.<p>But I will, sometimes use a license that says &quot;Here&#x27;s the code for you to look at. It&#x27;s not authorized for copying.&quot;<p>I think that it&#x27;s a good idea to have it available. I seriously doubt there&#x27;s anything in my stuff that is so proprietary that I&#x27;m afraid it will get ripped off. I do the same stuff everyone else does; maybe not as cleverly.<p>I just feel that it&#x27;s good to show folks what&#x27;s under the hood, if at all possible.')
