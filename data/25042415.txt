Item(by='sillysaurusx', descendants=None, kids=[25042566], score=None, time=1604976844, title=None, item_type='comment', url=None, parent=25024901, text='I’ve been studying pg’s Bel: <a href="http:&#x2F;&#x2F;paulgraham.com&#x2F;bel.html" rel="nofollow">http:&#x2F;&#x2F;paulgraham.com&#x2F;bel.html</a><p>It has a delightful idea: to ensure the current thread isn’t preempted, set a global “lock” variable to true.<p>The scheduler checks “lock”. If it’s true, the current thread is added to the front of the threads list. Otherwise it’s added to the back.<p>Then the scheduler runs the first thread in the threads list.<p>So by setting “lock” to true, you are essentially instructing the scheduler never to switch threads.')
