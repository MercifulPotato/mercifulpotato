Item(by='tylerdurden91', descendants=None, kids=None, score=None, time=1601930868, title=None, item_type='comment', url=None, parent=24691187, text='I wouldn&#x27;t put Diagram Maker in the same bucket as these tools. These tools can be used directly by an End user for drawing diagrams via drag &amp; drop or via markdown. Diagram Maker on the other hand empowers application developers to embed a tool where end users can create diagrams by drag &amp; drop within their applications.<p>Some of the reasons why an application developer might prefer using Diagram Maker over using draw.io&#x27;s embed mode in their application:\n* You want to customize the editor to match your application&#x27;s styling.\n* You want the editor to impose your application&#x27;s custom semantics, for example: your application doesn&#x27;t allow cycles.')
