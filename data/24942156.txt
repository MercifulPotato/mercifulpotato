Sum types, Option type etc<p>- Strict compiler. If it compiles, it works.<p>- Great error messages.<p>- Mature package tooling. Cargo is at least as good as npm.<p>- Mature GraphQL packages.<p>- Decent cloud support. Github actions, Container guides.<p>---<p>I&#x27;ve been running it in production for a while, usually as a remote schema for Hasura: <a href="https:&#x2F;&#x2F;github.com&#x2F;ronanyeah&#x2F;rust-hasura" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;ronanyeah&#x2F;rust-hasura</a><p>Deploy pipeline:<p>- Push to github<p>- Github action builds binary<p>- Github action pushes binary to AWS ECS in a container<p>- This triggers a rebuild of the AWS Fargate service which restarts with new Rust code')
