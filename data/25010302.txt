Item(by='mobjack', descendants=None, kids=None, score=None, time=1604690354, title=None, item_type='comment', url=None, parent=25009258, text='I prefer to say that code is like a Rube Goldberg machine.<p>While it technically does the job, it is over complicated making it hard to add new features and easy to introduce bugs.<p>If we rush to add a new feature, it will be like adding another crazy contraption making it even harder to make changes to in the future.<p>But if we take a little time to clean up the code, it will be like a well oiled machine giving us a solid foundation to build new functionality on top of.')
