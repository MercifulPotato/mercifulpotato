Item(by='thomaslkjeldsen', descendants=None, kids=None, score=None, time=1604837698, title=None, item_type='comment', url=None, parent=25024499, text='<a href="https:&#x2F;&#x2F;www.niklasroy.com&#x2F;project&#x2F;31&#x2F;pongmechanik" rel="nofollow">https:&#x2F;&#x2F;www.niklasroy.com&#x2F;project&#x2F;31&#x2F;pongmechanik</a> (2004)<p>&gt; Pongmechanik is an electromechanical version of the early digital tennis game Pong. It uses computer technology of the 1940&#x27;s to implement a computer application of the 1970&#x27;s. The control logic of the game is entirely built out of telephone relays, the same parts that the first digital computers used, which were built by pioneers like Konrad Zuse.')
