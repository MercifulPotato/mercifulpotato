Usually I keep my windows fulls screen and only have two windows side by side when copying stuff.<p>In addition to that I have a semi transparent terminal and emacs in a floating window that I can move to the scratchpad with a global hotkey. This is much alike the visor console in Quake. Terminal has tmux. Very handy for a quick ping or htop without messing up the main window layout.<p>i3 config snippet for reference:<p><pre><code>  # Visor terminal\n  exec &quot;st -c visor&quot;\n  bindsym $mod+a [class=&quot;visor&quot;] scratchpad show\n  for_window [class=&quot;visor&quot;] move scratchpad;\n  for_window [class=&quot;visor&quot;] scratchpad show;\n  for_window [class=&quot;visor&quot;] floating enable;\n  for_window [class=&quot;visor&quot;] resize set width 100 ppt;\n  for_window [class=&quot;visor&quot;] sticky enable\n  for_window [class=&quot;visor&quot;] move position center;\n  for_window [class=&quot;visor&quot;] scratchpad show;\n\n  # Visor style emacs window toggled by Meta+ &quot;less than key&quot;\n  exec emacs -name emacs-server\n  bindsym $mod+less [instance=&quot;emacs-server&quot;] scratchpad show\n  for_window [instance=&quot;emacs-server&quot;] move scratchpad\n  for_window [instance=&quot;emacs-server&quot;] floating enable\n  for_window [instance=&quot;emacs-server&quot;] move position 450px 0px\n  for_window [instance=&quot;emacs-server&quot;] sticky enable\n  for_window [instance=&quot;emacs-server&quot;] scratchpad show\n  for_window [instance=&quot;emacs-server&quot;] resize set 950 1055\n  for_window [instance=&quot;emacs-server&quot;] scratchpad show</code></pre>')
