Item(by='soheil', descendants=0, kids=None, score=3, time=1606101736, title="Show HN: The File You Didn't Know You Needed", item_type='story', url=None, parent=None, text='To prevent against accidentally deleting the content of an important folder when doing <i>rm -rf </i>* create a file named <i>-i</i>. This way <i>rm</i> command will prompt before proceeding to delete everything.<p>Here is how you create a file with that weird name:<p><pre><code>  touch -- -i\n</code></pre>\n<i>-i</i> is the name of the file when <i></i>* is expanded <i>-i</i> is passed to the <i>rm</i> command as a flag. You should probably do this in your <i>~</i> now.')
