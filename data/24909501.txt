Item(by='rgbrenner', descendants=None, kids=None, score=None, time=1603818944, title=None, item_type='comment', url=None, parent=24909127, text='Try starting with VueJS... it can be added to a page just like jquery and you get the same reactivity&#x2F;template&#x2F;etc support on the page without having to setup a compiler&#x2F;etc.<p>Then after you learn that if you want to create a full app, vue (in vue-cli) has config templates that will help you set everything up so you can just start adding the pages to the app. A full app is nice because the compiler and linting will catch a lot of js errors making your app more reliable, allowing you to write larger apps (this is the biggest reason people use these frameworks).<p>The concepts in Vue are similar enough to React, Nuxt, Next, etc that it&#x27;s pretty easy to learn the others after you have the basics down.')
