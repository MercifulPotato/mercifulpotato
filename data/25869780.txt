It is fast and open-source, with a generous set of packages and languages. I can pin it and have a stable, internally consistent GNU userland. Very good for a developer using several interpreted languages.<p>Cygwin has its quirks and flaws, like any software. But so does a <i>full distro</i> (Ubuntu, no less) running inside Windows 10.')
