Item(by='pizza234', descendants=None, kids=[25136586], score=None, time=1605705928, title=None, item_type='comment', url=None, parent=25136046, text='The `include!` is not intended to be used as a _generic_ inclusion mechanism¹:<p>&gt; Using this macro is often a bad idea, because if the file is parsed as an expression, it is going to be placed in the surrounding code unhygienically. This could result in variables or functions being different from what the file expected if there are variables or functions that have the same name in the current file.<p>¹=<a href="https:&#x2F;&#x2F;doc.rust-lang.org&#x2F;std&#x2F;macro.include.html" rel="nofollow">https:&#x2F;&#x2F;doc.rust-lang.org&#x2F;std&#x2F;macro.include.html</a>')
