Item(by='knowhy', descendants=None, kids=[25550340, 25549888, 25550301, 25549927], score=None, time=1609056489, title=None, item_type='comment', url=None, parent=25547809, text='Docker has a layer file system. Meaning if you do it right, that Chrome container will share the same 500 MB base image layer with the Gimp container, or whatever, making it less bloated than it appears when looking only at the footprint of the first image.<p>I&#x27;m not stating that I believe it is a good idea to run desktop apps in Docker containers. It is not a good idea. But it is also not true that if someone would do it, it would necessary lead to a bloated filesystem.<p>Docker will use a lot of disk space when using multiple base images.')
