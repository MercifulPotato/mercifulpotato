I was 18; lots of distractions. The situation didn&#x27;t last very long, and had hasn&#x27;t happened since.<p>Boilerplate code is conquered through, in the order of least desirable to better:<p>- manual external code generation<p>- automated external code generation that repeats itself in each build<p>- automated internal code generation: macros<p>You don&#x27;t want to be manually creating repetitive content with small variations; that&#x27;s what computers are for.<p>You can regard boilerplate as challenge: how small can I get the specification of this? And then work on the generator from that specification to the boilerplate. The boilerplate is still boring, but the generator can have fun bits in it.')
