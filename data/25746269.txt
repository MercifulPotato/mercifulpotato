Item(by='wahern', descendants=None, kids=None, score=None, time=1610456757, title=None, item_type='comment', url=None, parent=25746040, text='By default signal handlers don&#x27;t use a separate stack; they use the same stack as the process&#x2F;thread that received the signal. A process&#x2F;thread must call sigaltstack to install an alternative stack for signal handler execution.<p>So to answer the previous poster&#x27;s question: yes, AFAICT this bug could have effected userspace code unless there are some other nuances at play.')
