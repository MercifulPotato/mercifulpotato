Item(by='ywei3410', descendants=None, kids=None, score=None, time=1609984011, title=None, item_type='comment', url=None, parent=25658296, text='I&#x27;ve just set up a mail server with the same stack as mailinabox (with a few minor differences, since I didn&#x27;t care for the web-screens, self-hosted DNS and the fact that my residential ISP offers a static IP service).<p>If you do give it a try, I would personally simplify the set-up. If you&#x27;re comfortable with some downtime you can get away with running just Postfix, Dovecot (with some secondary services) and delegating to the OS for user management. All of them come with systemd services so it isn&#x27;t too bad to manage.')
