Item(by='l0b0', descendants=None, kids=None, score=None, time=1609233756, title=None, item_type='comment', url=None, parent=25549949, text='My current Bash go-to header now includes<p><pre><code>  set -o errexit -o noclobber -o nounset -o pipefail\n</code></pre>\n- Exit the script at the first non-zero exit code.<p>- Do not allow clobbering existing files (overridden on a per-line basis when I actually want to clobber).<p>- Exit the script at the first undefined and un-defaulted variable dereference.<p>- Fail if any part of a pipeline fails.<p>and<p><pre><code>  shopt -s failglob\n</code></pre>\n- Treat a glob with no matches as an error.<p>A bonus of using long names is that they are easy to search for online.')
