Item(by='ronlut', descendants=None, kids=None, score=None, time=1608911340, title=None, item_type='comment', url=None, parent=25537073, text='I had some free time so I created a benchmark of Python JSON packages.  \nVarious blog posts are discussing Python JSON serializing and deserializing performance but they are always correct only for the time they were posted.<p>I wanted to create a single, updating place for all programmers that need to decide what package to use when they start a new project and performance is a consideration.<p>Currently, JSONPerf supports Python 3 and Python 2.  \nAdditional languages will be supported in the future.<p>Another useful feature - you can benchmark using your JSON file.  \nIf most of the JSONs you deal with are in a certain structure, use that feature as it will tell you how different packages deal with your structure of JSONs.<p>Would love to hear your thoughts &#x2F; questions!')
