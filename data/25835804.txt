Instead, you can make array types which have whatever indexing behaviour you like.<p>Because of the focus on generic code, these custom array types are basically first class citizens. See for example <a href="https:&#x2F;&#x2F;github.com&#x2F;JuliaArrays&#x2F;OffsetArrays.jl" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;JuliaArrays&#x2F;OffsetArrays.jl</a><p>Almost every serious julia package is written in such a way that they can handle such arrays correctly.')
