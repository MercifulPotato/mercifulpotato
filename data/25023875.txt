Item(by='miloignis', descendants=None, kids=None, score=None, time=1604820718, title=None, item_type='comment', url=None, parent=25021897, text='Not sure if V8 does this, but a general strategy used in HotSpot is to compile assuming the good case and install a &quot;trap&quot; to deoptimize the code if the condition is violated. So you could compile the let like a const with no checks on the use of the variable, but have a trap such that if any code ever tries to assign to that variable, all code compiled under the assumption that it was constant is throw out.')
