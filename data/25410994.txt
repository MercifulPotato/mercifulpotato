Item(by='JoshTriplett', descendants=None, kids=[25412889], score=None, time=1607898168, title=None, item_type='comment', url=None, parent=25410742, text='I switch between enough different projects with different coding styles that I have shortcuts to change between them:<p><pre><code>    set expandtab\n    set shiftwidth=4\n    set softtabstop=4\n    function Spaces(...)\n        if a:0 == 1\n            let l:width = a:1\n        else\n            let l:width = 4\n        endif\n        setlocal expandtab\n        let &amp;l:shiftwidth = l:width\n        let &amp;l:softtabstop = l:width\n    endfunction\n    command! T setlocal noexpandtab shiftwidth=8 softtabstop=0\n    command! -nargs=? S call Spaces(&lt;args&gt;)\n    autocmd BufNewFile,BufRead ~&#x2F;src&#x2F;linux&#x2F;* T\n    autocmd BufNewFile,BufRead ~&#x2F;src&#x2F;git&#x2F;* T\n    autocmd FileType html S 2\n    autocmd FileType tex S 2\n</code></pre>\nThis sets the default to 4-space indents, but lets me do :T to switch to tabs (and does so automatically for Linux and Git which use that style), or :S N to switch to N-space indents (and automatically switches HTML and TeX to 2-space indents).')
