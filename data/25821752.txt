Item(by='jpsamaroo', descendants=None, kids=[25823387], score=None, time=1610978009, title=None, item_type='comment', url=None, parent=25821580, text='JavaScript&#x27;s JIT is a tracing JIT, so it can compile code in the background while the interpreter&#x2F;less optimized compiled code is actually running. In Julia, the compiler runs first, and then the compiled code is run. This will probably eventually change as Julia&#x27;s compiler improves, but regardless, it&#x27;s important to note this distinction.')
