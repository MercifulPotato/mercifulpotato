Item(by='ajuc', descendants=None, kids=None, score=None, time=1601903909, title=None, item_type='comment', url=None, parent=24686571, text='I don&#x27;t think it maps 1-1 with strong vs weak typing.<p>C (very loosely typed language) code is usually pretty close to the requirements and very strongly typed functional languages (like Haskell) are often &quot;make DSL and write the specification in it, then run it&quot;.<p>Meanwhile object oriented languages often have pretty strong typesystems and cultures of using them extensively, but they also encourage designing with margin for changes (and thus using lots of layers of abstractions instead of just implementing the specification as elegantly as possible).')
