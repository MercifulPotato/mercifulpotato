Item(by='vegetablepotpie', descendants=None, kids=[25562868, 25561909], score=None, time=1609179837, title=None, item_type='comment', url=None, parent=25561409, text='Apple added custom instructions on top of the ARM instruction set to do matrix operations.<p>Matrix operations are used <i>a lot</i> in some algorithms, such as in computer graphics and machine learning and these instructions help those operations go faster.<p>Although the ARM ISA has a multiply and accumulate instruction, it would take many more instructions on a standard ARM core to compute a matrix with only standard instructions. That enables the M1 chip to be faster in this case when developers use these instructions in their applications.')
