Item(by='majewsky', descendants=None, kids=None, score=None, time=1602110999, title=None, item_type='comment', url=None, parent=24711541, text='I do the following:<p>1. have the domain in question resolve to a server with a public IP<p>2. have that server generate the certs with any ACME client with HTTP challenge<p>3. have that server ship the certs to the actual server hosting the service via intranet<p>4. in the intranet, have the domain resolve to the actual server via &#x2F;etc&#x2F;hosts override<p>All of that is not that hard to set up even at scale with proper config management tools. Having said that, I don&#x27;t actually use it for that many services myself. The most significant one is LDAPS.')
