Item(by='sh3a4n', descendants=None, kids=None, score=None, time=1611173953, title=None, item_type='comment', url=None, parent=25851414, text='s3viewer is an open-source tool that can nicely list the content of different publicly open storage types and helps to identify leaking data. The tool allows you to view all the files in a given storage and download selected files and directories. The goal is to identify the owner of the storage as quickly as possible in order to report that data is leaking from it.<p>Supported open storage:<p>- Amazon S3 Buckets<p>- Microsoft Azure Blobs<p>- FTP servers with Anonymous access allowed<p>- HTTP Index Of &#x2F; Pages (Apache&#x2F;nginx-style directory listing)<p>Enjoy!')
