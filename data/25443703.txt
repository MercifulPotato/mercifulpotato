Item(by='jsmith45', descendants=None, kids=None, score=None, time=1608132529, title=None, item_type='comment', url=None, parent=25435338, text='That message comes from the rule where for non-core infrequently-used functionality, an app need not disclose the data collected if it is not used for advertise or tracking, it is clear and obvious what data you are collecting, and you clearly show the user id as a form field.<p>For example, if your feedback form allows users to optionally enter their email address so you can contact them to request more details if needed, you don&#x27;t need to specify that you collect email addresses if you never otherwise collect email.<p>It is not clear if this would also apply if users can optionally upload a profile photo, but profile photos are not a core part of the app (e.g. not a tinder clone). If it does apply it would let you omit the scary sounding &#x27;user photos&#x27; option from the privacy list.')
