Item(by='ufo', descendants=None, kids=None, score=None, time=1608759409, title=None, item_type='comment', url=None, parent=25505701, text='While the more obvious application of defunctionalization is to avoid stack overflow from using the implicit machine stack, another application is if you want to make your algorithm asynchronous.<p>In many languages, it&#x27;s hard to create an asynchronous recursive function that can yield from inside some deep recursion, and then resume later from where it stopped. On the other hand, if your program is rewritten to use an explicit stack plus a state machine then it&#x27;s trivial to do things like pausing the execution, or printing some debug message between each step.')
