Item(by='nobody9999', descendants=None, kids=[25581988], score=None, time=1609338144, title=None, item_type='comment', url=None, parent=25581237, text='&gt;To be clear, the Agora is not a centralized network.<p>I&#x27;d read the comment you linked and am aware of that.<p>My comment was addressed to the poster to whom I replied.<p>Agora seems interesting, although I think the requirement for <i>external</i> dependencies (a git repository) makes it less viable as a platform.<p>I get that it&#x27;s necessary to have standardized formats for data -- especially in a decentralized environment.<p>I also get that markdown&#x2F;git is a ready resource for that.<p>However, it seems to me that unless one is willing (and able) to set up <i>their own</i> git repository&#x2F;server on their own (physical, VPS or cloud) hardware, you&#x27;re getting decentralization of Agora on the back of centralized git platforms -- which kind of defeats the point of a decentralized platform.<p>Unless your <i>only</i> target set of users are developers and other technical people with the resources to host their own git instances, forcing the use of git repositories will limit both interest and adoption.<p>Removing such a requirement (or baking git into the platform) would go a long way toward making Agora more attractive, IMHO.<p>I am most certainly <i>not</i> trying to discourage anyone from using Agora, nor do I think it&#x27;s a bad idea.<p>In fact, I think it&#x27;s quite interesting.  Thank you for working on this and I wish you much success with it!')
