Item(by='flohofwoe', descendants=None, kids=None, score=None, time=1608204298, title=None, item_type='comment', url=None, parent=25451215, text='It should be possible to compile C code in a Rust project without an external C compiler, or it should be possible to import a C header into a Rust source file directly (so basically, creating separate &quot;Rust bindings&quot; for a C library shouldn&#x27;t be required).<p>See Zig as an example for excellent C interoperability.<p>As soon as a C&#x2F;C++ compiler toolchain and build tools need to be involved in a Rust project you&#x27;re basically loosing all advantages of Cargo and are back to the relative brittleness of cross-platform C&#x2F;C++ builds.')
