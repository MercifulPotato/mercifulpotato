Item(by='thamer', descendants=None, kids=None, score=None, time=1602097793, title=None, item_type='comment', url=None, parent=24708237, text='I&#x27;ve found AWS Lambda and Google Cloud Functions super useful to run automations, especially when used with the Shortcuts app on iOS.<p>For example, I have a GCF that checks my clipper card&#x27;s balance[1] when I touch it with my phone[2], and displays the amount in large font on the screen. It only took an hour or so to write with no experience with it, and costs nothing at all to host since I send only a few requests a month.<p>Being able to run custom code on demand without having to deal with hosting, config, security or any of this cruft is a game-changer.
