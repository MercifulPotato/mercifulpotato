Item(by='recursivedoubts', descendants=None, kids=[24726236], score=None, time=1602207005, title=None, item_type='comment', url=None, parent=24724548, text='You would have rows in one table, those rows would be annotated with two annotations:<p><pre><code>  hx-get=&quot;&#x2F;workflow&#x2F;&lt;workflow id&gt;&quot;\n</code></pre>\nwhere &lt;workflow id&gt; is the ID for the given workflow.  This would issue a GET request to, er, get the details for the given row, in partial html format.<p>And also<p><pre><code>  hx-target=&quot;#other-table&quot;\n</code></pre>\nThis would tell htmx to take the received content and put that into the other table on the screen, by id.  Since this target is the same for all rows in the table, it could be moved to a parent element, where it would be inherited from.<p>Finally, you could also use<p><pre><code>  hx-push-url=&quot;true&quot;\n</code></pre>\nIf you wanted to push the URL into the navigation bar, creating a history element for the navigation, and which would allow users to copy a URL to a particular spot in the two-level navigation, and enable back button support.<p><a href="https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-get&#x2F;" rel="nofollow">https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-get&#x2F;</a><p><a href="https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-target&#x2F;" rel="nofollow">https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-target&#x2F;</a><p><a href="https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-push-url&#x2F;" rel="nofollow">https:&#x2F;&#x2F;htmx.org&#x2F;attributes&#x2F;hx-push-url&#x2F;</a>')
