Item(by='Strom', descendants=None, kids=None, score=None, time=1610911705, title=None, item_type='comment', url=None, parent=25813223, text='Google provides a list of App Engine features they&#x27;ve removed. [1] Beyond that there is also a somewhat undocumented phase of <i>working-but-forgotten</i>. Classic App Engine features like the datastore, memcache, Users API, Python 2, Go 1.11 etc go under this category. These are things that still work, but get no updates. Instead you get constant e-mails and other notifications about how you should redesign your app to work with the 2nd generation App Engine system. Which means Firestore (in datastore mode) instead of datastore. Memorystore instead of memcache. Your own solution instead of the Users API etc.<p>--<p>[1] <a href="https:&#x2F;&#x2F;cloud.google.com&#x2F;appengine&#x2F;docs&#x2F;deprecations" rel="nofollow">https:&#x2F;&#x2F;cloud.google.com&#x2F;appengine&#x2F;docs&#x2F;deprecations</a>')
