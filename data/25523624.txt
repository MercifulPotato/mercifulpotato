Item(by='jandrewrogers', descendants=None, kids=None, score=None, time=1608767146, title=None, item_type='comment', url=None, parent=25522608, text='Rust is a systems programming language. Modern server software -- a primary use case for a systems language -- is heavily async by default for a long list of compelling architectural reasons. Providing first-class language tooling to support that seems eminently sensible since this is how people will want to use the language.<p>When you are writing high-performance server code, async <i>is</i> the common scenario.')
