Also rarely a problem there.<p>On Arch I run sudo pacman -Syu almost every day.<p>You could run something like this to get a warning if there is anything that requires manual intervention: <a href="https:&#x2F;&#x2F;github.com&#x2F;bradford-smith94&#x2F;informant" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;bradford-smith94&#x2F;informant</a><p>I also run my laptop with Snapper: <a href="https:&#x2F;&#x2F;wiki.archlinux.org&#x2F;index.php&#x2F;snapper" rel="nofollow">https:&#x2F;&#x2F;wiki.archlinux.org&#x2F;index.php&#x2F;snapper</a><p>I use a btrfs root volume with btrfs subvolumes instead of separate partitions. I have snapper setup with a pacman hook to take a snapshot before and after every pacman run. So worst case, if something goes horribly wrong, I can boot from an Arch install image into a previous snapshot and unbreak things.')
