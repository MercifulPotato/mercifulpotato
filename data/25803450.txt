Item(by='ufo', descendants=None, kids=None, score=None, time=1610816229, title=None, item_type='comment', url=None, parent=25789837, text='I like the approach that ShellCheck uses for its verbose error messages. Every error message is identified by an easily-googleable error ID that is not translated and there is a link to the shellcheck wiki where you can find an in-depth explanation of the error and possible ways to fix it.<p>For example, a typical run of shellcheck might look something like this:<p><pre><code>    ~&#x2F;test&gt; shellcheck blah.sh\n\n    In blah.sh line 2:\n    echo $1\n         ^-- SC2086: Double quote to prevent globbing and word splitting.\n    \n    Did you mean: \n    echo &quot;$1&quot;\n \n    For more information:\n      https:&#x2F;&#x2F;www.shellcheck.net&#x2F;wiki&#x2F;SC2086 -- Double quote to prevent globbing ...\n\n</code></pre>\n<a href="https:&#x2F;&#x2F;github.com&#x2F;koalaman&#x2F;shellcheck&#x2F;" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;koalaman&#x2F;shellcheck&#x2F;</a>')
