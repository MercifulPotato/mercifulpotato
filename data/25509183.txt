Item(by='e12e', descendants=None, kids=None, score=None, time=1608660244, title=None, item_type='comment', url=None, parent=25509050, text='&gt; Peppering is a token that is not in the database of passwords that needs to be applied for the password to be correct.<p>Well, typically a server only cares about verifying the user (still) knows a password.<p>A typical server (today) does not have a way to reconstruct the plain password, only a way to check if any given string matches.<p>A password manager, typically does have a way to supply the password.<p>Peppers and salts are typically manipulated by the server system, plain passwords are typically managed by the password manager.<p>In this case the password manager never sees the hocrux, and cannot leak it. A server will typically leak a pepper to anyone with access to ram (or access to a hw enclave, which is expected to be more difficult).')
