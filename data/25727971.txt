Item(by='vidarh', descendants=None, kids=None, score=None, time=1610367834, title=None, item_type='comment', url=None, parent=25727110, text='Which makes me think the biggest problem with the file selector as described is not that it&#x27;s flawed, but that it&#x27;s hard to replace.<p>One of the thing I loved about the Amiga was that because of how the API was structured, it was easy to replace things like this, as you could patch every API endpoint. As a result, it took very little time before more advanced replacements for the standard file requester appeared. You &quot;just&quot; had to patch (via OS-provided functions) a couple of library calls.')
