Item(by='brundolf', descendants=1, kids=[25425036], score=1, time=1607986867, title='Show HN: Bookmarklet to highlight recent comments in a Hacker News thread', item_type='story', url=None, parent=None, text='I find myself often checking into a thread to see the latest replies, which usually means visually scanning for comments that have &quot;minutes&quot; in the timestamp.<p>I decided to automate this (and a bit extra) with a bookmarklet. Create a new bookmark in your web browser and paste the following into the URL field:<p><pre><code>  javascript:(function() {\n    const cutoffStr = prompt(&quot;Highlight posts in the last:&quot;, &quot;1 hour&quot;);\n    \n    function millisForStr(s) {\n      const [num, unit] = s.split(&#x27; &#x27;);\n      const parsedNum = Number(num);\n    \n      return unit.includes(&#x27;minute&#x27;) ? parsedNum * 60 * 1000 \n        : unit.includes(&#x27;hour&#x27;) ? parsedNum * 60 * 60 * 1000 \n        : unit.includes(&#x27;day&#x27;) ? parsedNum * 24 * 60 * 60 * 1000 \n        : NaN;\n    }\n    \n    Array.from(document.querySelectorAll(&#x27;.age&#x27;))\n      .map(el =&gt; {\n        const interval = millisForStr(el.innerText);\n    \n        return [el, new Date(Date.now() - interval)];\n      })\n      .forEach(([el, date]) =&gt; {\n        const cutoff = new Date(Date.now() - millisForStr(cutoffStr));\n        if (date &gt;= cutoff) {\n          el.parentElement.parentElement.parentElement.style.background = &quot;rgba(255, 215, 0, 0.5)&quot;;\n        } else {\n          el.parentElement.parentElement.parentElement.style.background = null;\n        }\n      })\n  })();\n</code></pre>\nWhen clicked, you&#x27;ll be prompted for a string of the form &quot;1 hour&quot; or &quot;23 minutes&quot; or &quot;2 days&quot;, and after submitting you&#x27;ll see all comments on the current page that fall within that interval highlighted with a gold-colored background. Let me know what you think or if you&#x27;d like any other small features!')
