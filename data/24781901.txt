Item(by='raju', descendants=None, kids=None, score=None, time=1602711309, title=None, item_type='comment', url=None, parent=24779750, text='While this is a good idea, having a `.dockerignore` reduces how much Docker has to load into the build context. For projects with large histories, the `.git` directory itself can be rather large. Add to that directories that hold build artifacts, documentation, and you are unnecessarily increasing the time it takes to start the build process.')
