Item(by='Fwirt', descendants=None, kids=[24889436], score=None, time=1603655649, title=None, item_type='comment', url=None, parent=24886962, text='Having an EFI shell binary on a USB stick can be incredibly useful when you&#x27;re troubleshooting boot issues. I went down this rabbit hole because the Thinkpad X1 Tablet doesn&#x27;t play nice with GRUB when there&#x27;s no keyboard attached, but if you setup entries directly in EFI you can select a boot partition using the touchscreen. There used to be precompiled binaries floating around in the tianocore Github repo, but it looks like the link has gone dead.<p>See also Pete Batard&#x27;s excellent project to support all kinds of exotic filesystems in EFI.\n<a href="http:&#x2F;&#x2F;efi.akeo.ie&#x2F;" rel="nofollow">http:&#x2F;&#x2F;efi.akeo.ie&#x2F;</a>')
