Item(by='nine_k', descendants=None, kids=[25055879], score=None, time=1605065112, title=None, item_type='comment', url=None, parent=25054778, text='I suggest you try and implement some complex but well-defined algorithm, e.g. sha256, in some very, very simple language, say, brainfuck (which is very close to the original Turing machine language). Or, more realistically, in PDP11 assembly, which is beautifully transparent and uniform.<p>You might then notice that there is some kind of sweet spot in complexity: fewer features and footguns than, say, C++, but more features and abstractions than machine code.')
