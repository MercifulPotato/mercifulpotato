Item(by='foxhop', descendants=None, kids=None, score=None, time=1608041006, title=None, item_type='comment', url=None, parent=25429885, text='Before I learned vim all those years ago, it was really convenient to use Kate with fish (sftp) to &quot;mount&quot; and edit remote files.<p>I would use this to edit HTML, CSS, and JS (and later PHP) on my FreeBSD servers which hosted my websites.<p>Once I learned revision control (hg mercurial) I largely stopped modifying files directly on the remote host so I had less of a need for an sftp client built into my text editor.')
