Item(by='chriswarbo', descendants=None, kids=None, score=None, time=1603722083, title=None, item_type='comment', url=None, parent=24893122, text='Lazy IO is <i>really</i> useful for commands which stream data over stdio, e.g. this is a really useful template, where &#x27;go&#x27; is a pure function for producing a stdout string from a stdin string:<p><pre><code>    {-# LANGUAGE OverloadedStrings #-}\n    import qualified Data.ByteString.Lazy.Char8 as BS\n\n    main = BS.interact go\n\n    go :: BS.ByteString -&gt; BS.ByteString\n    go input = -- Generate output here\n</code></pre>\nIf you don&#x27;t mind Haskell&#x27;s default String implementation, then it&#x27;s just:<p><pre><code>    main = interact go\n\n    go :: String -&gt; String\n    go input = -- Generate output here</code></pre>')
