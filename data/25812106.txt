Item(by='PeterisP', descendants=None, kids=None, score=None, time=1610901287, title=None, item_type='comment', url=None, parent=25811206, text='While it&#x27;s not possible to duplicate an existing key, it is possible to create duplicate keys in the first place - you can&#x27;t get the secrets out, but you can write identical secrets to two or more keys if you want to.<p>This adds some convenience, though it also does add some risks e.g. in case of breach you can&#x27;t tell which token was misused, you can&#x27;t invalidate a lost key without invalidating others at the same time.')
