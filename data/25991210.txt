Item(by='itamarst', descendants=None, kids=[25991247, 25991600], score=None, time=1612202183, title=None, item_type='comment', url=None, parent=25990891, text='People keep misunderstanding this, so just to clarify: pip still works on Python 2! They just aren&#x27;t doing any new releases for Python 2.<p><pre><code>  $ virtualenv --python=python2.7 &#x2F;tmp&#x2F;venv\n  $ &#x2F;tmp&#x2F;venv&#x2F;bin&#x2F;pip install --upgrade pip\n  ...\n  Successfully installed pip-20.3.4\n  $ &#x2F;tmp&#x2F;venv&#x2F;bin&#x2F;pip install flask\n  ...\n  Collecting flask\n    Using cached Flask-1.1.2-py2.py3-none-any.whl (94 kB)\n  ...\n\n</code></pre>\nThe mechanism, as used by other projects as well, is that when you upload a project to PyPI you can set a minimal required version of Python. So when you upgrade, you get the latest version that supports the current version of Python.<p>Sometimes maintainers screw this up, but so long as the maintainers do this everything Just Works: people on old Python get old packages, people on new Python get new packages.')
