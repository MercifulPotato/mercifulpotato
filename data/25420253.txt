Item(by='PaulHoule', descendants=None, kids=[25420716], score=None, time=1607966761, title=None, item_type='comment', url=None, parent=25419190, text='Modern documents are composite documents.<p>A Microsoft Word DOCX document is really a ZIP file and if you unzip it you will find XML files,  PNG files,  all sorts of things you can read with ordinary tools.<p>Similarly a PDF file is a graph of more-or-less JSON objects which can be nested inside &quot;filters&quot; which support compression,  encryption,  etc.<p>The organization of numerous small subfiles makes it easy to pick one piece out to work with in RAM which would be important on a small target.')
