Item(by='llimos', descendants=None, kids=None, score=None, time=1611235622, title=None, item_type='comment', url=None, parent=25854411, text='I once did a migration of an auth system by having the new system attempt an auth against the old system if there was no password hash saved in the new one. If it succeeded, it hashed the password and saved it. Seamless for users.<p>Obviously the old system isn&#x27;t going to stick around forever, but you can get a good chunk of the passwords migrated this way.')
