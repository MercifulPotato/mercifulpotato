Item(by='tga', descendants=None, kids=None, score=None, time=1607761415, title=None, item_type='comment', url=None, parent=25396447, text='When using Redux, components are supposed to select small parts of the state tree, and then they only rerender when those change. It is absolutely not the case that all components render on each unrelated update of the global state, that would be crazy.<p><a href="https:&#x2F;&#x2F;react-redux.js.org&#x2F;next&#x2F;api&#x2F;hooks#useselector" rel="nofollow">https:&#x2F;&#x2F;react-redux.js.org&#x2F;next&#x2F;api&#x2F;hooks#useselector</a><p>It is true that <i>all of the selector functions</i> need to run on each change, but those are supposed to be really small pure functions and their performance should be negligible, compared with everything else that normally happens in an app.')
