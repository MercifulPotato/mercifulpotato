Item(by='MarkSweep', descendants=None, kids=None, score=None, time=1605068012, title=None, item_type='comment', url=None, parent=25049655, text='There is sort of an event loop: the .NET Threadpool. It is used by async by default. It will by default quickly create an number of threads equal to the number of CPU cores, but then slowly increase it beyond that. If you are blocking in a threadpool thread, the work item queue will back up.
