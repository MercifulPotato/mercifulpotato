Item(by='roryokane', descendants=None, kids=[24894195], score=None, time=1603645126, title=None, item_type='comment', url=None, parent=24882480, text='If you want to do single-file conversions with Pandoc without having to install it, try <a href="http:&#x2F;&#x2F;markup.rocks&#x2F;" rel="nofollow">http:&#x2F;&#x2F;markup.rocks&#x2F;</a>. It’s a compilation of Pandoc into 2.2MB of JavaScript so you can convert documents (and preview their HTML conversion) in your browser as you type. Its source code: <a href="https:&#x2F;&#x2F;github.com&#x2F;osener&#x2F;markup.rocks" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;osener&#x2F;markup.rocks</a>.<p>I most often use <a href="http:&#x2F;&#x2F;markup.rocks&#x2F;" rel="nofollow">http:&#x2F;&#x2F;markup.rocks&#x2F;</a> for converting HTML to Markdown and for testing that my reStructuredText syntax is correct when contributing to docs.<p>Pandoc also has a demo web page for trying it out (<a href="https:&#x2F;&#x2F;pandoc.org&#x2F;try&#x2F;" rel="nofollow">https:&#x2F;&#x2F;pandoc.org&#x2F;try&#x2F;</a>). The demo supports all of Pandoc’s formats and doesn’t require a large JS download, but it silently truncates inputs to 3,000 characters.')
