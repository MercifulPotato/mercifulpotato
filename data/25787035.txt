Item(by='SoSoRoCoCo', descendants=None, kids=[25788663, 25787806], score=None, time=1610685169, title=None, item_type='comment', url=None, parent=25786470, text='&gt; Everything is a Command Line<p>This is a really controversial pattern for GUIs. In one camp, the GUI is really a skin over the CLI that acts like a virtual user, translating GUI inputs into underlying CLI. In the other camp, the GUI is all there is (e.g., Windows) and there is no underlying OS that can be accessed via CLI: in fact, the CLI is a &quot;fake GUI&quot; (win32 apps written without a Window). I can&#x27;t say which is better, but it is fascinating to see that this was an &quot;original pattern&quot;.')
