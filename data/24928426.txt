Item(by='jononor', descendants=None, kids=None, score=None, time=1603960775, title=None, item_type='comment', url=None, parent=24927414, text='If you want to install such a package into its own prefix (not directly to &#x2F;) the convention is typically to use &#x2F;opt&#x2F;$SOFTWARENAME. So one would have bin,lib etc under that directory. If the software is not designed to be relocatable like this, one may have to set up PATH, LD_LIBRARY_PATH, etc so that it can find its executables, libraries etc')
