Item(by='lilyball', descendants=None, kids=[25076226], score=None, time=1605217457, title=None, item_type='comment', url=None, parent=25074615, text='Huh I had no idea $1&#x2F;$2&#x2F;etc worked as-is in an alias. The advice I learned years ago for dealing with any alias that needs to do something custom with parameters is to write it like<p><pre><code>  git config alias.foo &#x27;!f() { actual command goes here }; f&#x27;\n</code></pre>\nas that will pass all the args to the shell function. But if git is already setting it up so the args work then suffixing the alias with &quot;;:&quot; seems simpler.')
