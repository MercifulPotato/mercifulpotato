Item(by='apple_innocent', descendants=None, kids=[25829024], score=None, time=1611014136, title=None, item_type='comment', url=None, parent=25828079, text='&quot;Where is the encryption being added?&quot;<p>It is being added by the proxy server listening on the loopback which connects to the remote website.<p>Browser connects to forward proxy on port 80, forward proxy (compiled with SSL library) connects to target IP on port 443.<p>This is how one can, e.g, use clients that are not SSL-enabled to access websites, etc. that require SSL.<p>For example, if forward proxy is listening on 127.0.0.1:80, we can make an encrypted connection to example.com using original *hobbit netcat which does not support SSL.<p><pre><code>   echo -e &#x27;GET &#x2F; HTTP&#x2F;1.1\\r\\nHost: example.com\\r\\nConnection: close\\r\\n\\r\\n&quot; |nc -vvn 127.0.0.1 80\n</code></pre>\nIt is probably more popular to use stunnel for this purpose instead of a forward proxy.')
