Item(by='jstrong', descendants=None, kids=None, score=None, time=1609448671, title=None, item_type='comment', url=None, parent=25592679, text='I&#x27;ve experienced excellent results building distributed systems where the nodes talk to each over via raw udp. often, in the context of message queues, there is much talk about delivery guarantees, but in many contexts the same goals can be accomplished by designing the system so it can handle lost messages. for example, rather than a node sending a single message notifying that it has started work on something, it sends a message once a second, saying, &#x27;I am currently working on <i>x</i>&#x27;. is that reinventing the wheel? to me it just seems much simpler.')
