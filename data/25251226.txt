Item(by='ghayes', descendants=None, kids=[25251506], score=None, time=1606717688, title=None, item_type='comment', url=None, parent=25251154, text='Not necessarily. You could use homomorphic encryption and, for example, encode the file with a hash of the file as the key material for encryption. All such files will encode identically for each user, but it will be opaque to the server what the contents are unless the server already has the plain-text of the original file. This was used by other cloud storage companies (that are now defunct).')
