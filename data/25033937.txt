Item(by='wccrawford', descendants=None, kids=None, score=None, time=1604924132, title=None, item_type='comment', url=None, parent=25033870, text='In both cases, the server is the thing that&#x27;s actually in control.  Your browser doesn&#x27;t know if code is malicious or not and has no choice but to run it as the server says to.<p>The server, on the other hand, does know if some text is potentially malicious and can take steps to prevent it from being malicious.<p>So the server is responsible in both cases because it has all the knowledge necessary.<p>If the browser tried to prevent code execution then no code would execute.  It can&#x27;t know the original origin of the text.')
