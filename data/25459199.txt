Item(by='aidos', descendants=None, kids=None, score=None, time=1608232756, title=None, item_type='comment', url=None, parent=25458943, text='I find it easiest to think in terms of diffs as patches. It’s just a bunch of search and update commands. Sure, there’s some hairiness around files moving or whatever, but for me, a cherry pick is just getting a patch file and applying it in the appropriate place. It doesn’t care about the file it’s being applied to, it just needs to find similar content in the file so it can apply itself.')
