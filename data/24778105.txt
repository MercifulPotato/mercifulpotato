Item(by='rkangel', descendants=None, kids=None, score=None, time=1602691114, title=None, item_type='comment', url=None, parent=24777991, text='What it does is show up something that is true anyway for unit tests: automated tests are software and they should be designed like it.<p>It might affect multiple tests, but all you should have to change is one or two places in some shared code pulled in by those tests. Too many tests have too much repeated code, and so become expensive to maintain, exactly like normal code. Refactor, think about shared functions etc. exactly as you would do with your production code.')
