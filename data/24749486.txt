Item(by='nybble41', descendants=None, kids=None, score=None, time=1602453868, title=None, item_type='comment', url=None, parent=24749018, text='It&#x27;s possible to create a Wayland compositor that works with the proprietary nVidia drivers, but it requires using nVidia-specific interfaces because nVidia refuses to support the same interfaces for non-GLX, non-X11 hardware acceleration provided by every other Linux graphics driver.<p>It&#x27;s hardly surprising that a lot of Wayland compositor developers would rather not put in a ton of extra effort to add a special case for one particular set of proprietary drivers, which they would then need to maintain and support separately from the common code path.')
