Item(by='zeroimpl', descendants=None, kids=None, score=None, time=1605664312, title=None, item_type='comment', url=None, parent=25131782, text='They could have added namespaces to branch names. Right now branch names are just a free-for-all. Instead, if they insisted that branch names are always prefixed with &quot;username&#x2F;&quot;, then you wouldn&#x27;t need forks and anybody could freely submit code to any project. PRs would become MRs.<p>I believe this is how most Git repos are managed within organizations. I&#x27;m not sure why public repos <i>need</i> to require users to fork things to propose changes.<p>And isn&#x27;t this basically how Github works internally anyways?
