Item(by='richbradshaw', descendants=None, kids=[25979198], score=None, time=1612104144, title=None, item_type='comment', url=None, parent=25979123, text='It&#x27;s common (for novices&#x2F;experts who are in a rush ) to accidentally commit AWS keys for dev environments, or other API keys (e.g. Mailchimp) or other secrets (2FA, SAML certs etc).<p>We all know how to do it right, but particularly early on in a project it&#x27;s easy to test something with a hardcoded API key, then forget and commit it.<p>Picking a framework that makes this a less obvious choice (e.g  using a .env file that&#x27;s already in gitignore by default) helps a lot here.')
