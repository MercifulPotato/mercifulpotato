Item(by='zomglings', descendants=None, kids=None, score=None, time=1612189746, title=None, item_type='comment', url=None, parent=25986927, text='Preemptible instances should not have any associated auto-scaling. It is common for people to run pre-emptible instances as part of a &quot;Managed Instance Group&quot;, with which you can associate some kind of autoscaling policy.<p>You can run preemptible instances without any kind of management or with only very simple management and you should be okay. I recommend a simple managed group with a policy of, &quot;I want maximum one of these instances running at all times and, if it gets preempted, then please start a new one.&quot;<p>If you&#x27;re using Google Cloud, don&#x27;t deny yourself the cost savings of preemptible instances. When I was using the same infrastructure, preemptions were very rare!')
