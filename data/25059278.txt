Item(by='morelisp', descendants=None, kids=None, score=None, time=1605109554, title=None, item_type='comment', url=None, parent=25059143, text='You could argue that a REST DTO for reads is a query string and a REST DTO for writes is a JSON body. That seems comparable to the GraphQL case to me.<p>In the end it&#x27;s about whether the term tells you something about the architecture, and in neither of those cases would I gain useful knowledge of the system from someone saying &quot;it&#x27;s CQRS.&quot;<p>A better litmus test is probably something to do with enumerating possible commands. So a GraphQL API <i>with a specific set of mutations</i> could be a CQRS design. But if your mutations are just CRUD spelled differently, it&#x27;s probably not.')
