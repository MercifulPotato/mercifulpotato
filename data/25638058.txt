Item(by='kccqzy', descendants=None, kids=None, score=None, time=1609794451, title=None, item_type='comment', url=None, parent=25637026, text='I did use dithering for unimportant background images of a website. Use just 256 colors, and both PNG and GIF will use a color palette instead of describing each pixel separately. Really helps with the file size. Afterwards muck with the various lossless compression parameters in PNG with optipng to shave off a few more percent.')
