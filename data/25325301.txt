Item(by='alexvoda', descendants=None, kids=[25330277], score=None, time=1607278326, title=None, item_type='comment', url=None, parent=25322288, text='This article only mentions in passing the best approach so far: Uniform Type Identifiers (1). When it comes to file types and associations Mac OS X hit a homerun. This mechanism way more powerful than MIME types because:<p>UTIs use a reverse-DNS naming structure.\nUTIs support multiple inheritance (which is a good thing in this context)\nUsing UTIs allow per file exceptions.<p>It really is a shame this was not implemented by any other OS than Mac OS X. It is one of the brilliant technical features it has. Together with bundles (2), they make for very powerful and flexible filing.<p>(1) <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Uniform_Type_Identifier" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Uniform_Type_Identifier</a><p>(2) <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Bundle_(macOS)" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Bundle_(macOS)</a>')
