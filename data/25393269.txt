Item(by='dathinab', descendants=None, kids=[25394622], score=None, time=1607730233, title=None, item_type='comment', url=None, parent=25389261, text='Nesting them can lead to very not-so-easy to understand logical expressions, furthermore if the parts between ?: is long enough it can also noticeable reduce readability.<p>Instead of `&lt;cond&gt; ? &lt;left&gt; : &lt;right&gt;` I prefer `if &lt;cond&gt; { &lt;left&gt; } else { &lt;right&gt; }` the additional brackets noticeable improve readability and you can extend it to support `if &lt;cond&gt; { &lt;a&gt; } else if &lt;cond2&gt; { &lt;b&gt; } else { &lt;c&gt;` instead of `&lt;cond&gt; ? &lt;a&gt; : &lt;cond2&gt; ? &lt;b&gt; : &lt;c&gt;`.\n(Oh and that last example might be wrong needing brackets depending on operator precedence...)<p>Through if you don&#x27;t nest it it doesn&#x27;t matter (oh and because it&#x27;s a expression evaluation `else` is not optional but required as you need a value the expression resolves to).')
