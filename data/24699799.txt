Item(by='tom_mellior', descendants=None, kids=None, score=None, time=1602002804, title=None, item_type='comment', url=None, parent=24698432, text='You can do that in Frama-C as well. But now this new precondition must be propagated somewhere. For example, if you have some function (syntax from memory as well):<p><pre><code>    &#x2F;*@ ensures \\result &gt; a; *&#x2F;\n    int increment(int a) {\n        return a + 1;\n    }\n</code></pre>\nyou would change it to:<p><pre><code>    &#x2F;*@ requires a &lt; INT_MAX;\n        ensures \\result &gt; a; *&#x2F;\n    int increment(int a) {\n        return a + 1;\n    }\n</code></pre>\nBut if this function has N call sites, you now have N problems: At every call site you must prove that the precondition holds.')
