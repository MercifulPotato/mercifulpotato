Item(by='throwaway894345', descendants=None, kids=None, score=None, time=1611993944, title=None, item_type='comment', url=None, parent=25964677, text='Your conclusion might be right, but your reasoning is certainly wrong. Calling native functions in Python is often quite expensive because you need to marshal between PyObjects and the native types (probably allocating memory as well). This doesn’t “feel” so slow in Python because, well, everything in Python is slow. But you really start to notice it when you’re optimizing.')
