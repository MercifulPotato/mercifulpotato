Fzf even offers keybindings for your shell. For example, it binds Alt+C to fuzzy-finding a directory, and Enter cds to it ( <a href="https:&#x2F;&#x2F;github.com&#x2F;junegunn&#x2F;fzf#key-bindings-for-command-line" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;junegunn&#x2F;fzf#key-bindings-for-command-lin...</a> ).<p>Fzf is great for other things too; here is a fish function to bing Alt+G to fuzzy-pick a Git branch and jump to it:<p><pre><code>   function fish_user_key_bindings\n     bind \\eg &#x27;test -d .git; or git rev-parse --git-dir &gt; &#x2F;dev&#x2F;null 2&gt;&amp;1; and git checkout (string trim -- (git branch | fzf)); and commandline -f repaint&#x27;\n     bind \\eG &#x27;test -d .git; or git rev-parse --git-dir &gt; &#x2F;dev&#x2F;null 2&gt;&amp;1; and git checkout (string trim -- (git branch --all | fzf)); and commandline -f repaint&#x27;\n   end</code></pre>')
