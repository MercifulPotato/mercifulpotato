Item(by='chrisdalke', descendants=None, kids=[25454490], score=None, time=1608182869, title=None, item_type='comment', url=None, parent=25452146, text='One other common use for useRef that isn&#x27;t mentioned here is to bridge your React component with a JS library outside the React ecosystem.<p>For example, if you wanted to instantiate a charting library like uPlot (<a href="https:&#x2F;&#x2F;github.com&#x2F;leeoniya&#x2F;uPlot" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;leeoniya&#x2F;uPlot</a>) you might instantiate a uPlot object storing state and methods for the chart. You can perform the chart initialization in a useEffect hook, and use a ref to keep a reference to the instance you&#x27;ve created.')
