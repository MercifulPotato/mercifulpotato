Item(by='jiehong', descendants=None, kids=[25254434], score=None, time=1606747567, title=None, item_type='comment', url=None, parent=25253471, text='Context is quite important, I agree.<p>However, when possible, giving _actionable_ info can really help, especially if the user can see that error.<p>For example, you can try to explain why this file is expected, and give a link to the spec&#x2F;doc.\nExtremely helpful when called at 4am in production.<p>Alternatively, I&#x27;ve seen some experiments with extreme reproducibility: capture the full context (network requests, filesystem, db state, etc.), and spin up an instance with that state. This way devs can deterministically reproduce the issue, and fix it. But, it supposes that the application is written to support this fairly well (by either being very functional, or by having probes in all non-pure functions).')
