Item(by='kbrosnan', descendants=None, kids=None, score=None, time=1609403911, title=None, item_type='comment', url=None, parent=25590459, text='Getting major UI added to a browser is not Igalia&#x2F;Tidelift&#x2F;etc. forte. They provide development for engine level features like supporting a new CSS element or ECMAscript function. SSB needs OS integration on Windows, MacOS and Linux. It needs UI elements such as toolbar buttons and menu items. Getting that sorted is a non-trivial task. Then there is the ongoing maintenance. This is not something where a few unit tests are going to keep the feature functional as new code is developed.')
