Files in a ZipFile can be read in a streaming manner. There is no need to store all the data in memory.<p><pre><code>    import io, csv, zipfile\n\n    max_lines = 10\n    with zipfile.ZipFile(&quot;data.zip&quot;) as z:\n        for info in z.infolist():\n            with z.open(info.filename) as f:\n                reader = csv.reader(io.TextIOWrapper(f))\n                for i_line, line in enumerate(reader):\n                    if i_line &gt;= max_lines: break\n                    print(line)</code></pre>')
