Item(by='steveklabnik', descendants=None, kids=None, score=None, time=1604930955, title=None, item_type='comment', url=None, parent=25033787, text='To be a little more explicit about it:<p>* The index of packages on crates.io is stored in a git repository<p>* Cargo knows how to clone that down without you having git installed<p>* The contents of packages are not acquired by doing a git clone; source code is stored in an S3 bucket, and Cargo downloads it directly.<p>* You <i>can</i> have your project depend on a package that lives in a git repository, but you cannot upload a crate to crates.io that depends on a git repository. This means that this feature is mostly useful for end projects and private libraries, not public dependencies.')
