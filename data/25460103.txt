Item(by='ehutch79', descendants=None, kids=[25460328], score=None, time=1608237550, title=None, item_type='comment', url=None, parent=25459793, text='I would honestly question if you actually need JWT anyways.<p>jwt is useful for independently verified auth tokens. i.e. service a auths the user, generates a token that says &#x27;\nthis is definitely my user with this data&#x27;. service b can&#x27;t auth the user because it has not access to the data, but can trust the jwt from service a.<p>every time i&#x27;ve seen someone with a jwt issue in django, they&#x27;ve just been wrapping the session id from django in a jwt, and then unwrapping it and verifying it in the db anyways.')
