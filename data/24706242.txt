Item(by='vardump', descendants=None, kids=None, score=None, time=1602058239, title=None, item_type='comment', url=None, parent=24693665, text='SetEvent&#x2F;WaitForSingleObject is a thread yield from SetEvent thread to the waiting thread. Scheduler does that when SetEvent is called. So microsecond level waiting time is expected.<p>You don&#x27;t need to wait for timer tick.<p>IOW, parent was talking about timer wait, not event wait. Timer events are only processed at timer interrupt ticks.')
