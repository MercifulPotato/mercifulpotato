Item(by='pwg', descendants=None, kids=[25311183], score=None, time=1607132485, title=None, item_type='comment', url=None, parent=25304260, text='I did not see a personal pet-peeve of mine:<p>If a user of your CLI tool has <i>explicitly requested</i> the built-in help text via --help (or an equivalent switch that requests help) then that help text shall be output on stdout.<p>It always peeves me to do:<p><pre><code>    command --help | less\n</code></pre>\nonly to find that the explicit request for help (the --help switch) has output the help text on stderr, and I then have to redo the invocation:<p><pre><code>    command --help 2&gt;&amp;1 | less \n</code></pre>\nto cause the help text to actually be piped into less.')
