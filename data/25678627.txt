Item(by='fouric', descendants=None, kids=[25678964], score=None, time=1610058954, title=None, item_type='comment', url=None, parent=25675474, text='Removing the dimensions from the image tags is absolutely the wrong thing to do.<p>If you do this, then a browser&#x2F;user&#x2F;client that wants that information simply doesn&#x27;t have it - you can&#x27;t create bits from thin air.<p>If you leave the metadata in, then any browser that wants it can use it, and any brower that doesn&#x27;t want it can ignore it.<p>I can code up a Greasemonkey script in a few minutes that will strip the width and height attributes from img tags, but nobody on earth can write a script that will magically detect what size the author wants them to be if they don&#x27;t send that data to you.')
