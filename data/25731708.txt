Item(by='cerplum', descendants=None, kids=None, score=None, time=1610384194, title=None, item_type='comment', url=None, parent=25723398, text='One potential shortcut is if you install their CubeMX and take a look in it&#x27;s data files there are XML files for each distinct peripheral. So there&#x27;ll be something like RTC_v1.0.xml and RTC_v2.0.xml. Someone uploaded an old copy of them [1] if you just want a quick look.<p>Then there&#x27;s also device &amp; device family xml files that link to the various peripherals they have [2].<p>There&#x27;s probably no guarantees but in my experience if two devices link to the same version xml file their peripheral registers at least will be identical.
