Item(by='dangus', descendants=None, kids=None, score=None, time=1605964621, title=None, item_type='comment', url=None, parent=25167033, text='Nobody’s stopping you from using the web version of Slack or Spotify.<p>That said I can think of some pretty good reasons.<p>- Users don’t understand it as well or are less comfortable with it.<p>- Users’ general preference for logical separation<p>- Mobile safari doesn’t support browser notifications.<p>- You won’t get browser notifications if you close the tab by accident<p>- You lose OS integration, like replying via OS notifications in Slack or using OS Music controls with Spotify<p>On top of all that a browser tab uses plenty of RAM as well so I don’t really see the upside of using the browser for these apps, either.')
