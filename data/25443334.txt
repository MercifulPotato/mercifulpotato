Item(by='ChrisMarshallNY', descendants=None, kids=[25443399, 25443805, 25443358], score=None, time=1608130958, title=None, item_type='comment', url=None, parent=25442734, text='This type of thing can happen easily with Git. We just submit a directory, and every damn file in that directory gets submitted.<p>What I do, is have a file that aggregates my sensitive stuff (like server secrets and whatnot), and call that file something like &quot;DoNotCheckThisIntoSourceControl.swift&quot;. I then add a git ignore line, on that name.<p>I&#x27;ll also sometimes store it outside of the repo root (I use Xcode, so I can drag files in from anywhere).')
