Item(by='hunter2_', descendants=None, kids=None, score=None, time=1605536731, title=None, item_type='comment', url=None, parent=25111609, text='No, when a script tells a browsers to make a cross-origin request such as a GET or POST, the browser first makes a &quot;pre-flight&quot; request (without the payload) using the OPTIONS method to see what CORS-related response headers come back. If headers are returned that allow for it to proceed, the browser then makes whatever request the script asked for.<p>The network tab of developer tools should reveal all of this.')
