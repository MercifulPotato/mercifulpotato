Item(by='worble', descendants=None, kids=None, score=None, time=1603317145, title=None, item_type='comment', url=None, parent=24850565, text='I think he means when using git from the command line. Trying to push to your repo will trigger a github sign in process, which will then save the credentials to Windows Credential Manager. Git will then ALWAYS use those from that point on credentials unless you go into the manager and delete them, which is a massive pain if you use multiple github accounts.')
