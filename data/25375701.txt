Item(by='mbreese', descendants=None, kids=[25375941], score=None, time=1607619551, title=None, item_type='comment', url=None, parent=25375453, text='I’m a big fan of FreeBSD (in fact, I was planning to install CentOS on a new storage server, but I’ll probably go back to FreeBSD instead). But, I imagine anyone wanting to run Docker on FreeBSD would want to be able to run Linux-based containers. In which case, you’d end up with a Mac like situation — you could have command line Docker communicating with a Docker daemon running in a Linux VM.<p>If that’s what you’d need, then why not just run a Linux distro to start with?')
