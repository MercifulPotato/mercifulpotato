Item(by='gvb', descendants=None, kids=[25375423], score=None, time=1607617554, title=None, item_type='comment', url=None, parent=25375184, text='Remove snaps:<p><pre><code>  sudo apt-get remove snapd\n  sudo apt-get remove lxd-agent-loader # 20.04\n</code></pre>\nThe key to keeping them uninstalled is to pin the packages so they don&#x27;t get &quot;upgraded,&quot; which results in them being reinstalled.<p><pre><code>  sudo apt-mark hold snapd\n  sudo apt-mark hold lxd-agent-loader # 20.04\n</code></pre>\nIf you want to go the next step in cleanup:<p><pre><code>  sudo umount &#x2F;var&#x2F;lib&#x2F;snapd&#x2F;snaps&#x2F;*.snap\n  sudo rm -rvf &#x2F;var&#x2F;lib&#x2F;snapd&#x2F;*\n  sudo rm -vf &#x2F;etc&#x2F;systemd&#x2F;system&#x2F;snap-*.mount\n  sudo rm -vf &#x2F;etc&#x2F;systemd&#x2F;system&#x2F;snap-*.service\n  sudo rm -vf &#x2F;etc&#x2F;systemd&#x2F;system&#x2F;multi-user.target.wants&#x2F;snap-*.mount\n  sudo rm -vrf &#x2F;snap&#x2F;*</code></pre>')
