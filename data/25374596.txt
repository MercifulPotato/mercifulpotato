Item(by='remram', descendants=None, kids=[25375951], score=None, time=1607613700, title=None, item_type='comment', url=None, parent=25374490, text='AGPL considers network use to be <i>distribution</i>, so if you have modified the AGPL-licensed software you have to make the source code available to users of that software. That does not mean you have to make the source of downstream systems available.<p>In fact I am not sure you have to make the source of your modified AGPL software available to end users if you are using it internally (&quot;users of that server&quot; from the license would be you), but IANAL.')
