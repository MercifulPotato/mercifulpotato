Item(by='Const-me', descendants=None, kids=[24754294, 24753278], score=None, time=1602500111, title=None, item_type='comment', url=None, parent=24751875, text='I find it strange Rust marketing people are insisting the language is safe yet it can&#x27;t dynamically allocate a byte of memory without some unsafe code.<p>In languages like C# or Java the whole standard library is written in C# or Java.
