Anyway, for out parameters, the common convention is:<p>Caller provides one object for output:<p><pre><code>   void callee(Obj *obj);\n</code></pre>\nCaller provides an array for output:<p><pre><code>    void callee(Obj *obj, size_t obj_size);\n</code></pre>\nCaller provides an pointer, to which callee allocates a new object:<p><pre><code>    void callee(Obj **obj)</code></pre>')
