Item(by='stewartesmith', descendants=None, kids=None, score=None, time=1606442202, title=None, item_type='comment', url=None, parent=25224859, text='What this practically means is that a plain &quot;make -j&quot; will spawn all the processes that could run concurrently. So if you&#x27;re building 1,000 files, it&#x27;ll spawn 1,000 processes. This can either be fine, or completely terrible, depending on the number of CPUs you have an the amount of memory each process uses.')
