Item(by='fayten', descendants=None, kids=[25448537, 25448172, 25447981], score=None, time=1608145310, title=None, item_type='comment', url=None, parent=25446170, text='I&#x27;m a big fan of using plain SWT[1] without JFace.  In fact GTK4 support is already being work on!<p>I tend to see a lot of developers think of Eclipse Platform UI&#x2F;JFace[2] when they see SWT.  Eclipse Platform UI is Eclipse&#x27;s abstraction over SWT that allows them to make consistent components across platforms.  SWT is lighter weight and it&#x27;s just platform native components. Also since you are dealing with native components you get easy access to platform accessibility.<p>Eclipse has some great documentation for the platform as well as loads of snippets[3] and examples[4] to pick apart.<p>There is also an incredibly useful charting library[5] that is used heavily by OpenChrom(Uses Eclipse platform)[6].<p>I have only tried it with some small samples, but SWT does work with GraalVM native-image[7]. This should allow you to develop a nice little statically link executable instead of deploying a jar if needed.<p>[1]<a href="https:&#x2F;&#x2F;www.eclipse.org&#x2F;swt&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.eclipse.org&#x2F;swt&#x2F;</a><p><a href="https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;eclipse.platform.swt&#x2F;tree&#x2F;master&#x2F;bundles&#x2F;org.eclipse.swt" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;eclipse.platform.swt&#x2F;tree&#x2F;master&#x2F;...</a><p>[2]<a href="https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;eclipse.platform.ui" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;eclipse.platform.ui</a><p>[3]<a href="https:&#x2F;&#x2F;www.eclipse.org&#x2F;swt&#x2F;snippets&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.eclipse.org&#x2F;swt&#x2F;snippets&#x2F;</a><p>[4]<a href="https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;eclipse.platform.swt&#x2F;tree&#x2F;master&#x2F;examples" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;eclipse.platform.swt&#x2F;tree&#x2F;master&#x2F;...</a><p>[5]<a href="https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;swtchart" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;eclipse&#x2F;swtchart</a><p>[6]<a href="https:&#x2F;&#x2F;lablicate.com&#x2F;platform&#x2F;openchrom" rel="nofollow">https:&#x2F;&#x2F;lablicate.com&#x2F;platform&#x2F;openchrom</a><p><a href="https:&#x2F;&#x2F;lablicate.com&#x2F;static&#x2F;media&#x2F;openchrom-GC-MS.722d5aa3.png" rel="nofollow">https:&#x2F;&#x2F;lablicate.com&#x2F;static&#x2F;media&#x2F;openchrom-GC-MS.722d5aa3....</a><p>[7]<a href="https:&#x2F;&#x2F;github.com&#x2F;mbarbeaux&#x2F;graalvm-swt-native-image" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;mbarbeaux&#x2F;graalvm-swt-native-image</a>')
