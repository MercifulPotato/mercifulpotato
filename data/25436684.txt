Item(by='noisenotsignal', descendants=None, kids=None, score=None, time=1608073975, title=None, item_type='comment', url=None, parent=25430119, text='It&#x27;s not really &quot;run anywhere&quot;, but you can write apps for a trusted execution environment like Intel SGX enclaves; not even the OS can look at what&#x27;s running. Enclave code is cryptographically signed so that you can both validate the identity of the signer as well as the code contents. In the latter, you&#x27;d have to compare the MRENCLAVE value to a published value, which you could reproduce by building from source if it&#x27;s open.<p>Microsoft calls this &quot;confidential computing&quot; and has some related Azure products, including providing VMs standalone and in Kubernetes.')
