Item(by='a1369209993', descendants=None, kids=None, score=None, time=1602618976, title=None, item_type='comment', url=None, parent=24767795, text='&gt; allow the creation of a new FAT-32 image file <i>of a specified length</i> given a target directory [emphasis added]<p>I think that&#x27;s your problem: file systems are supposed to go onto a preexisting block device, which you&#x27;ll need to create seperately. For example, on my system:<p><pre><code>  dd if=&#x2F;dev&#x2F;zero of=&#x2F;tmp&#x2F;foo.dos bs=512 count=128K\n  mkdosfs -F 32 &#x2F;tmp&#x2F;foo.dos\n</code></pre>\nOnce you&#x27;ve created the filesystem, fill it with:<p><pre><code>  mkdir &#x2F;tmp&#x2F;foo&#x2F;\n  sudo mount -t msdos -o loop &#x2F;tmp&#x2F;foo.dos &#x2F;tmp&#x2F;foo&#x2F;\n  # dump stuff into &#x2F;tmp&#x2F;foo&#x2F;\n  sudo umount &#x2F;tmp&#x2F;foo.dos # or &#x2F;tmp&#x2F;foo&#x2F;\n  # filesystem is in &#x2F;tmp&#x2F;foo.dos</code></pre>')
