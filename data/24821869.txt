Item(by='mfateev', descendants=None, kids=[24822014], score=None, time=1603062608, title=None, item_type='comment', url=None, parent=24821819, text='The short answer is that Temporal is a fork of Cadence by the original founders of the project. The Temporal fork also has a VC backed company behind it. The Temporal is still an open-source project under MIT License. We believe the success of the open source project is essential for the success of the company, so we are putting significant resources into its development.<p>We spent almost year working on various improvements before declaring the first production release. The most important technical difference is that Temporal uses gRPC&#x2F;Protobuf when Cadence is TChannel&#x2F;Thrift. One implication is that TChannel did not support any security and Temporal supports mTLS. We are currently working on a comprehensive security story.<p>Here is a stackoverflow post that has more details about this: <a href="https:&#x2F;&#x2F;stackoverflow.com&#x2F;questions&#x2F;61157400&#x2F;temporal-workflow-vs-cadence-workflow" rel="nofollow">https:&#x2F;&#x2F;stackoverflow.com&#x2F;questions&#x2F;61157400&#x2F;temporal-workfl...</a>.')
