Item(by='yantrams', descendants=None, kids=[24960482], score=None, time=1604239853, title=None, item_type='comment', url=None, parent=24950291, text='A popular mnemonic for remembering the formulae of the basic units of Sanskrit metre is probably the first ever recorded example of De Bruijn Sequence. The phrase &quot;Ya-Maa-Taa-Raa-Ja-Bhaa-Na-Sa-La-Gam&quot; encodes all 8 possible combinations of syllables of length 3.<p>The basic unit of Sanskrit metre is called Gana and is composed of 3 syllables. Since a syllable can be either short (I) or long (U), we have 2^3 = 8 Ganas in total corresponding to UUU, UUI, UIU etc. To know the composition of a specific Gana say for example Ja Gana, one has to find the index of &quot;Ja&quot; in the aforementioned phrase and extract the next two syllables as well so Ja Gana equals Ja-Bhaa-Na =&gt; IUI (because Ja is short, Bhaa is long and Na is short again).<p>Code in Python if my explanation wasn&#x27;t clear<p><pre><code>  mnemonic = &quot;Ya-Maa-Taa-Raa-Ja-Bhaa-Na-Sa-La-Gaam&quot;\n\n  def symbol(syllable):\n      if syllable.count(&quot;a&quot;) == 1:\n        return &quot;I&quot;\n   else:\n        return &quot;U&quot;\n\n  def ganaSymbol(ganaName):\n      mnerray = mnemonic.split(&quot;-&quot;)\n      ganaIndex = mnerray.index(ganaName)\n      return &quot;&quot;.join([symbol(syllable) for syllable in mnerray[ganaIndex : ganaIndex + 3]])\n</code></pre>\nMetres are then defined in terms of Ganas and the number of lines in a stanza, for example Utpala Mala metre can be defined simply as Na-Ja-Bhaa-Ja-Ja-Ja-Raa, 4 which means a stanza in that meter should comprise of 4 lines, with each line having 21 syllables corresponding to Na-Ja-Bhaa-Ja-Ja-Ja-Raa sequence<p><pre><code>  utpalaMala = &quot;Na-Ja-Bhaa-Ja-Ja-Ja-Raa&quot;\n\n  def printMetre(metreGanas, numLines):\n    print(&quot;\\n&quot;.join([&quot;&quot;.join([ganaSymbol(ganaName) for ganaName in metreGanas.split(&quot;-&quot;)])]*numLines))\n\n</code></pre>\nEdits: Formatting')
