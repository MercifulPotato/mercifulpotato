Item(by='AnthonyMouse', descendants=None, kids=None, score=None, time=1608444262, title=None, item_type='comment', url=None, parent=25484054, text='Most of what you&#x27;re listing are Layer 7 services. The time cost there is in the configuration. You can put Active Directory in the cloud, but it&#x27;s still going to be Active Directory, i.e. a massively complicated proprietary framework that touches every Windows system in your network like an octopus.<p>And some of those things actually make sense. You can&#x27;t really locally host a CDN, can you? If you need a big amount of compute for an hour and then never again, it doesn&#x27;t make much sense to buy hardware for that.<p>But the point isn&#x27;t that it never makes sense to put anything in the cloud at all. It&#x27;s that companies regularly overuse it as some kind of buzzword panacea when there are only a specific set of things that it&#x27;s actually good for.')
