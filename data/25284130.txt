Item(by='t0astbread', descendants=None, kids=None, score=None, time=1606958750, title=None, item_type='comment', url=None, parent=25279909, text='Slightly off-topic but this is an important PSA.<p>Firefox keyword search has one little known killer feature: You can combine it with data URIs and JavaScript to run small &quot;command line snippets&quot; stored in your bookmarks from your browser bar.<p>To get started, create a keyword search from any form (like the search bar on duckduckgo.com) and edit the URL of the entry in the bookmark manager to point to<p><pre><code>  data:text&#x2F;html,&lt;script&gt;alert(&quot;%s&quot;)&lt;&#x2F;script&gt;\n</code></pre>\ninstead.<p>What you can do with this is (fortunately) limited by cross-origin restrictions but there are some useful applications. For example, I use this snippet<p><pre><code>  data:text&#x2F;html,&lt;script&gt;i=&quot;%s&quot;;firstSep=i.indexOf(&quot; &quot;);if(firstSep==-1)firstSep=i.length;subreddit=i.substr(0,firstSep);searchTerm=i.substr(firstSep+1);location=`https:&#x2F;&#x2F;old.reddit.com&#x2F;r&#x2F;${encodeURIComponent(subreddit)}${(searchTerm!=null&amp;&amp;searchTerm.replace(&#x2F;\\s&#x2F;,&quot;&quot;).length&gt;0)?`&#x2F;search?q=${encodeURIComponent(searchTerm)}&amp;restrict_sr=1`:&quot;&quot;}`&lt;&#x2F;script&gt;\n</code></pre>\nas a nice shortcut to Reddit (&quot;&lt;keyword&gt; &lt;subreddit&gt;&quot; to jump to a Subreddit, &quot;&lt;keyword&gt; &lt;subreddit&gt; &lt;search string&gt;&quot; to search within a Subreddit).<p>You can also insert content directly into the document which opens the possibility for instant marquee<p><pre><code>  data:text&#x2F;html,&lt;marquee&gt;%s&lt;&#x2F;marquee&gt;</code></pre>')
