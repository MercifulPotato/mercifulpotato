Item(by='eigenspace', descendants=None, kids=[24703884], score=None, time=1602023088, title=None, item_type='comment', url=None, parent=24702543, text='Just to echo Tkf here, I think it&#x27;d be great to have something like ThreadsX.map in base someday, but when the multi-threading features in Julia first came out, it was unclear what the best interface to provide would be.<p>Since Julia is committed to semantic versioning, this is a kinda scary prospect because it means that any high level, exported interface we decide on, we&#x27;ll be stuck with until at <i>least</i> 2.0 and probably forever.<p>So with things like this, it&#x27;s important to be conservative about the interfaces we expose and instead, we&#x27;ve have people explore different approaches out in the package ecosystem. One day, if someone can make a strong enough argument, I&#x27;m sure we&#x27;ll see one of these package solutions end up in Base.')
