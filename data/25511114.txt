Item(by='mwcampbell', descendants=None, kids=None, score=None, time=1608670486, title=None, item_type='comment', url=None, parent=25511062, text='Windows 10 on ARM is real Windows, with Win32, MFC, and all. You can build and distribute native applications using any of those APIs. Some ARM-based Windows PCs come configured with Windows 10 S, which only accepts apps from the Microsoft Store, but those can be upgraded to Windows 10 Pro, which has full support for side-loading just like x86&#x2F;x64.<p>Disclosure: I used to work on the Windows accessibility team at Microsoft.')
