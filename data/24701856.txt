Item(by='slaymaker1907', descendants=None, kids=None, score=None, time=1602015258, title=None, item_type='comment', url=None, parent=24698497, text='I think in general R7RS-large is supposed to be what mainline implementations implement with R7RS-small mainly being for implementations used for embedding scheme as part of a larger application (and even then you might still use large) as well as for experimental&#x2F;hobbyist implementations.<p>Particularly if you are just using scheme as a scripting language which mostly calls out to C, it doesn&#x27;t necessarily make a bunch of sense to include complex features like reflection or weak boxes. Such an implementation might not even have a GC in the first place and just rely on the scheme process ending to release memory back to the OS.')
