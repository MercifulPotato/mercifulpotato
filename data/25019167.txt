In the US, it just stayed big and grew.<p>The thing about doing something is you can always claim what you did was necessary. It&#x27;s asymmetric. Do nothing, and if the problem doesn&#x27;t go away you&#x27;re an idiot. Do something, and if the same thing happens you can claim it would have been worse if you hadn&#x27;t. There&#x27;s always some example somewhere you can abuse to say whatever you want.<p>Whether someone did badly depends on expectations about how it would have gone if they had done otherwise. Like in sports, you judged less harshly if you lose to the favourite. If you let the &quot;otherwise&quot; happen, people complain that you let it happen.<p>&quot;Something must be done. This is something. Therefore it must be done.&quot;<p>Note I&#x27;m saying nothing at all about what ought to be done about the virus, just how to handle the PR.')
