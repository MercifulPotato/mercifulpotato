Item(by='lokedhs', descendants=None, kids=[25313649, 25313656], score=None, time=1607161578, title=None, item_type='comment', url=None, parent=25311114, text='No, please don&#x27;t use escape codes in your output. Use the library that is designed for this purpose: terminfo.<p>Explicit escape codes is problematic when using any terminal that isnt&#x27; fully compatible with the xterm control codes, and doesn&#x27;t allow me to turn those codes off by setting TERM to dumb.<p>Far too many times have I redirected output from a program to a file only be be bombarded with escape codes, breaking grep and other tools that process the output.<p>Not to mention the fact that using terminfo is much easier than manually outputting the control codes.')
