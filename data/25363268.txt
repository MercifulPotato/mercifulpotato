Item(by='enricozb', descendants=None, kids=[25363769], score=None, time=1607542170, title=None, item_type='comment', url=None, parent=25357138, text='I personally don&#x27;t like `curl | bash` because I don&#x27;t know _how_ something will install:\n 1) What are all of the directories that something will insert itself into?\n 2) What of my files (.bashrc, etc.) will it modify?\n 3) If it modifies those things, will it tell me?<p>The `curl | bash` install pattern means that it can do _anything_.<p>Using a package manager I know that the install will be &quot;typical&quot;, and easy to uninstall (that&#x27;s the case with most of the package managers that I use anyways). Each package manager has a different pattern, sure, but at least it will be predictable.')
