For the following reasons:<p>1. It was very fast. Code written in it was comparable to the C code I was writing at the same time.<p>2. Very good C interop. Writing code to make use of C libraries was a near painless experience.<p>3. Very good collection of eggs (packages, think Rust&#x27;s crates). I rarely needed to make use of (2).')
