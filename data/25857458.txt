Item(by='angus_gh', descendants=None, kids=None, score=None, time=1611224979, title=None, item_type='comment', url=None, parent=25856644, text='The reason you get warned for doing<p>`logger.debug(&quot;Got %s results&quot;, num)`<p>instead of<p>`logger.debug(&quot;Got {} results&quot;.format(num))`<p>is that in the first case, internally it checks if the logger is disabled and if so skips formatting the string, whereas in the second case you&#x27;ve already formatted it before passing it in, so that work can&#x27;t be avoided. This isn&#x27;t the case for print statements because they don&#x27;t execute conditionally, so you don&#x27;t ever skip doing the format.')
