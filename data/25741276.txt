Item(by='ncmncm', descendants=None, kids=None, score=None, time=1610416556, title=None, item_type='comment', url=None, parent=25732902, text='What you say about performance on NV 3090 probably applies better to much older GPUs.<p>NV has put a great deal of recent work into getting their system to execute regular multi-threaded code with the same sort of speedup as they have had with SIMD workloads. It might be better to think of today&#x27;s NV GPU as a super-threadripper <i>when</i> the data to be processed is already in GPU memory. The coder&#x27;s job becomes making sure enough threads are ready to run to use up all the compute units at all times.')
