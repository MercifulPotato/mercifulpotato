Item(by='jsjohnst', descendants=None, kids=None, score=None, time=1608171757, title=None, item_type='comment', url=None, parent=25447529, text='Can you run a container of Windows on an x86 machine? The answer is no, and for the same reason it won’t work on ARM. A “container” is not a virtual machine, you can only  run the same Linux executables you would on a normal Linux system.<p>That said, as another person commented, you can run Windows for ARM in a VM on an Apple M1.')
