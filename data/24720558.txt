Item(by='nendroid', descendants=None, kids=None, score=None, time=1602173816, title=None, item_type='comment', url=None, parent=24719877, text='&gt;At the same time, if you have 10 micro services which are accessed by 1 frontend, it may be a little bit messy to keep all that code in the same place.<p>Why is this messy? Using folders to separate your code has no intrinsic organizational difference than using an entire repo. There&#x27;s no issue in throwing every app in your company under one repo and just using a folder to organize your stuff.<p>Separating code into multiple repos makes one repo less aware of changes in another repo. It actually makes things harder and worse.<p>This is the same issue with monoliths and microserves. You don&#x27;t need to separate your code into several computers just for organizational issues. You can use folders to organize things.')
