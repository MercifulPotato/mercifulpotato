Item(by='kevindong', descendants=None, kids=[25005311], score=None, time=1604626199, title=None, item_type='comment', url=None, parent=25001789, text='For my personal home server, I bought a Synology NAS which provides me with file sharing (via Samba), a built in VPN server, automatic RAID, easy dynamic DNS, Plex server, and support for Docker.<p>Who needs Kubernetes (however stripped) down, when you&#x27;re running a single-machine &quot;cluster&quot;? I write all of the services I want running as Docker containers and use a single docker-compose.yml to bring them up&#x2F;down as needed. Specify `restart: unless-stopped` for each container and bam it&#x27;s self healing too.')
