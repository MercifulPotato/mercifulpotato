Item(by='user5994461', descendants=None, kids=[24999027], score=None, time=1604589572, title=None, item_type='comment', url=None, parent=24998387, text='You can log all HTTP requests and it is standard practice to do so at the load balancer level (haproxy and co have very configurable logging).<p>However you MUST NOT logs cookies or headers or content, because these can contain private information.<p>Last but not least, developers should never pass tokens&#x2F;passwords in the request path because it will be systematically logged and leaked &#x2F;api&#x2F;token&#x2F;abcdec. That should go into a header or the body.')
