Item(by='dantillberg', descendants=None, kids=None, score=None, time=1603656823, title=None, item_type='comment', url=None, parent=24887820, text='When writing quick-and-dirty Go programs, I define a function called &quot;bailif&quot; (pun intended) that turns non-nil errors into panics. i.e. `func bailif() { if err != nil { panic(err) } }`. Each error can be &quot;handled&quot; at first by just `bailif(err)`.')
