Item(by='simonh', descendants=None, kids=None, score=None, time=1606390057, title=None, item_type='comment', url=None, parent=25218644, text='There is already a thriving ecosystem of very powerful development apps on iOS.<p>Pythonista is a full copy of Python 2.7 and 3.x, with an IDE, links to the GUI APIs and even a basic visual GUI designer for the iPad and iPhone. You can sync your scripts source code with Macs and other iOS devices through iCloud.<p><a href="http:&#x2F;&#x2F;omz-software.com&#x2F;pythonista&#x2F;" rel="nofollow">http:&#x2F;&#x2F;omz-software.com&#x2F;pythonista&#x2F;</a><p>Pyto is another Python app for the iPad, it&#x27;s not as mature but is 100% open source on Github.<p><a href="https:&#x2F;&#x2F;apps.apple.com&#x2F;us&#x2F;app&#x2F;pyto-python-3&#x2F;id1436650069" rel="nofollow">https:&#x2F;&#x2F;apps.apple.com&#x2F;us&#x2F;app&#x2F;pyto-python-3&#x2F;id1436650069</a><p>Codea is an IDE for Lua with extensive integration with the iPad native APIs and fantastic graphics capabilities.<p><a href="https:&#x2F;&#x2F;codea.io&#x2F;" rel="nofollow">https:&#x2F;&#x2F;codea.io&#x2F;</a><p>There are several others for different languages, but I think these are the most mature. Some of these have even been used to publish apps on the App Store, by dropping the code into a Python or Lua app container project in XCode.  You can also integrate them with Git from iOS using an App called Working Copy.')
