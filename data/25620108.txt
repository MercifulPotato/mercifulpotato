Item(by='carlmr', descendants=None, kids=None, score=None, time=1609657117, title=None, item_type='comment', url=None, parent=25619672, text='If you have a lot of string handling the strictness and ease of safe zero copy in Rust would make it unbeatable for performance and safety.<p>Due to lifetimes it&#x27;s easy to keep string slices as references into the original memory and if you do need to modify some strings, you can use CoW.<p>The way Rust deals with encoding also makes it much safer to use Rust. Since you need to be very explicit about which encoding you convert to what and if you want to allow lossy conversions.<p>I don&#x27;t think there is much of an equivalent in ADA for this. ADA has other benefits, like the delta (fixed point) types which help a lot in embedded projects.')
