Item(by='xg15', descendants=None, kids=[25476908, 25476929], score=None, time=1608379123, title=None, item_type='comment', url=None, parent=25476855, text='I think WASM has the added benefit of being platform-independent - which is generally a good thing but can even be a security feature: With native code, even if run without privileges, you need to expose a lot of details about your hardware that malicious users could use to find vulnerabilities.<p>With WASM, the host can keep the actual hardware architecture and even the actual machine code that is generated a secret.')
