Item(by='bbotond', descendants=None, kids=[24720230], score=None, time=1602170006, title=None, item_type='comment', url=None, parent=24719525, text='I don&#x27;t have a lot of experience with monorepos but I&#x27;ve found them very useful for:<p>- Simple projects with a server and an SPA component - frontend and backend code for the same feature is on the same feature branch, can be tested and reviewed together.<p>- Projects with a couple of microservices that share some common libraries - these libs can be directly referenced by the microservices instead of being published on an internal package server. This of course has its drawbacks too but overall, the benefits have outweighed them.')
