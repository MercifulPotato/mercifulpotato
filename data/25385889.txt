Item(by='Macha', descendants=None, kids=None, score=None, time=1607695664, title=None, item_type='comment', url=None, parent=25383662, text='The one slight gotcha is if you go for the page object pattern the intuition is that each page object holds a reference to the cypress element that formed it so you can do new LoginForm(cy.get(&#x27;#loginForm&#x27;)) or similar which is a mistake because of the way cypress chaining works - it will result in only being able to use a single method once on the page object. Instead you need to query fresh again each time.<p>Which is fine when the page object is a login form, but a bit more comfortable complicated when it&#x27;s notification 5 or something.')
