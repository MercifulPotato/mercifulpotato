Item(by='andris9', descendants=None, kids=None, score=None, time=1607615278, title=None, item_type='comment', url=None, parent=25374166, text='XOAUTH2 is supported but all the heavy lifting about refreshing access tokens etc must be done on your application side. Whenever IMAP API requires an OAuth2 token it makes a HTTP request to an URL you have defined and whatever is returned from that URL is the used to authenticate against the IMAP server. Here&#x27;s an example of such an authentication server: <a href="https:&#x2F;&#x2F;github.com&#x2F;andris9&#x2F;imapapi&#x2F;blob&#x2F;master&#x2F;examples&#x2F;auth-server.js" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;andris9&#x2F;imapapi&#x2F;blob&#x2F;master&#x2F;examples&#x2F;auth...</a> Basically you would have to run your own and configure IMAP API to requests tokens from that server.')
