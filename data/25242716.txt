Item(by='to11mtm', descendants=None, kids=None, score=None, time=1606620408, title=None, item_type='comment', url=None, parent=25233455, text='So, I&#x27;ve thought about this a lot, especially since there are sometimes some challenges with things such as datetimes in SQLite if you&#x27;re not very careful and specific (thinking about if you actually want to use in query for more than basic comparisons).<p>However, on the OTHER hand... if we think about SQLite as a general purpose data storage tool, it&#x27;s actually a really handy feature, letting the in datatypes almost every case <i>be more of an additional type of label</i>.<p>For example, if I required a high-precision timestamp, I could declare the column type something like &#x27;UTCEpochNanos&#x27;, providing a reminder to myself and other consumers of the data how it should be interpreted.')
