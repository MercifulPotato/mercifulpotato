Item(by='bnegreve', descendants=None, kids=None, score=None, time=1606287685, title=None, item_type='comment', url=None, parent=25204620, text='&gt; <i>Programs often reserve a lot more memory than they actually use (zero hit in performance) so memory stats are misleading, and the OS is really good at swapping memory not touched in a while to the SSD without you noticing.</i><p>The stats are absolutely reliable because no physical memory page is allocated until it is actually used to store something. So allocating a large chunk of unused memory wouldn&#x27;t show in the (physical) memory usage stat.')
