Item(by='wjakob', descendants=None, kids=[24837340, 24838292], score=None, time=1603201166, title=None, item_type='comment', url=None, parent=24836643, text='Caution: if you rely on pybind11 or a project using pybind11 (many projects do, like NumPy&#x2F;SciPy&#x2F;Tensorflow&#x2F;PyTorch..), hold off on upgrading to Python 3.9.0 for now.<p>A change in Python 3.9.0 introduces undefined behavior in combination with pybind11 (rarely occurring crashes, but could be arbitrarily bad). We will work around it in an upcoming version of pybind11, and Python will separately also fix this problem in 3.9.1 slated for release in December.<p>Details available here: <a href="https:&#x2F;&#x2F;pybind11.readthedocs.io&#x2F;en&#x2F;latest" rel="nofollow">https:&#x2F;&#x2F;pybind11.readthedocs.io&#x2F;en&#x2F;latest</a> and <a href="https:&#x2F;&#x2F;github.com&#x2F;python&#x2F;cpython&#x2F;pull&#x2F;22670" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;python&#x2F;cpython&#x2F;pull&#x2F;22670</a>.')
