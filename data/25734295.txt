Item(by='jarkhen', descendants=None, kids=[25747617], score=None, time=1610391630, title=None, item_type='comment', url=None, parent=25733555, text='That protects against an attacker reading network traffic. It does <i>not</i> protect against an attacker that has the hashed password from the DB.<p>The only thing you need in order to authenticate at any given time is the hash of (hashed password + nonce). The latter you get <i>for free</i>, at any time, from the server, so you only need to know the hashed password -- not the password itself. Since the hashed password is directly stored in the DB, if you get your hands on that you can authenticate.')
