I feel like the setups I have now save a good chunk of time while working, but I&#x27;m always looking for new tools to incorporate. \nMy list:<p>- Vim<p><pre><code>   - The plugin for Intellij primarily\n - Notion.so (https:&#x2F;&#x2F;www.notion.so&#x2F;)\n   - for general note-taking, planning, documenting\n - Karabiner Elements (https:&#x2F;&#x2F;karabiner-elements.pqrs.org&#x2F;)\n   - Rebind capslock to work as a modifier\n     - caps + hjkl = left, down, up, right\n     -  + 1 = escape\n     -  + 2 = Alfred search\n - Alfred Powerpack (https:&#x2F;&#x2F;www.alfredapp.com&#x2F;)\n   - Clipboard manager + snippets\n   - Search\n     - bookmarks\n     - prefixes for searching on Sumologic, Jira, Confluence, etc.\n - LazyGit (https:&#x2F;&#x2F;github.com&#x2F;jesseduffield&#x2F;lazygit)\n   - great for quick commits, branch switching, creating new branches and opening PRs\n   - not the best for reviewing actual changes though\n - Chrome extensions\n   - Quick tabs (https:&#x2F;&#x2F;chrome.google.com&#x2F;webstore&#x2F;detail&#x2F;quick-tabs&#x2F;jnjfeinjfmenlddahdjdmgpbokiacbbb?hl=en)\n     - Not something I use much, but makes switching back to the previous tab pretty easy\n   - Vimium (https:&#x2F;&#x2F;chrome.google.com&#x2F;webstore&#x2F;detail&#x2F;vimium&#x2F;dbepggeogbaibhgnhhndojpepiihcmeb?hl=en)\n     - Found this while looking for an extension that would offer VIM keybindings for any given text input. Vimium doesn&#x27;t do that but does offer a number of helpful bindings for navigating\n     - I don&#x27;t actually use it much other than to use &quot;gg&quot; and &quot;G&quot; to scroll to the top or bottom of pages. Marking is also pretty helpful\n - Iterm2 (https:&#x2F;&#x2F;iterm2.com&#x2F;)\n - Oh My Zsh (https:&#x2F;&#x2F;ohmyz.sh&#x2F;)\n  - I&#x27;ve used this pretty much the entire time I&#x27;ve used zsh so I honestly can&#x27;t remember which features it adds\n - Hotkey to edit current terminal command in Vim (https:&#x2F;&#x2F;unix.stackexchange.com&#x2F;questions&#x2F;6620&#x2F;how-to-edit-command-line-in-full-screen-editor-in-zsh)\n  - Makes it waay easier to fix a typo or update something in a long command</code></pre>')
