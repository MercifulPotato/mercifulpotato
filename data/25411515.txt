Item(by='simonw', descendants=None, kids=[25411571], score=None, time=1607901833, title=None, item_type='comment', url=None, parent=25411231, text='If you want to try it out on a Mac laptop you can do the following:<p><pre><code>    brew install datasette # if you don&#x27;t have it already\n    wget https:&#x2F;&#x2F;gitlab.com&#x2F;shanghai-ccp-member-db&#x2F;shanghai-ccp-member-db&#x2F;-&#x2F;raw&#x2F;master&#x2F;shanghai-ccp-member.db\n    datasette shanghai-ccp-member.db --setting sql_time_limit_ms 10000 --setting facet_time_limit_ms 10000\n</code></pre>\nI recommend bumping up those default time limits because the database has almost 2,000,000 rows in and faceting will time out if you don&#x27;t increase them.<p>You can easily facet by sex, ethnicity, hometown and organization using the cog icon on each column.
