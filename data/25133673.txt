Item(by='MrRadar', descendants=None, kids=None, score=None, time=1605679447, title=None, item_type='comment', url=None, parent=25130139, text='It&#x27;s probably down to browser&#x2F;OS support and the increasing importance of user-generated content on the web. Windows only got Unicode support by default on the consumer side with Windows XP which launched in 2001. Mac OS X was also released in 2001 and had Unicode support from the start (unlike Classic Mac OS which had it bolted on with poor app support). Similarly, according to this old blog post I dug up[1] Internet Explorer only got good Unicode support with 5.5 in 2000, and Netscape never had good Unicode support. Firefox, which had Unicode support from the start, was released around 2002. By 2006, which is clearly an inflection point on that graph, browsers and OSes with poor Unicode support were highly obsolete and global social media platforms like Myspace, Facebook, and Twitter, which needed Unicode to enable sharing of user posts between users writing in different languages, were on the ascendance.<p>[1] <a href="https:&#x2F;&#x2F;www.worldtimzone.com&#x2F;2002&#x2F;09&#x2F;24&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.worldtimzone.com&#x2F;2002&#x2F;09&#x2F;24&#x2F;</a>')
