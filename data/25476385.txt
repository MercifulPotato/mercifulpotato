Item(by='daitangio', descendants=None, kids=None, score=None, time=1608371536, title=None, item_type='comment', url=None, parent=25469708, text='I use a similar approach: first code, then take a refactoring session and look if some repetitive code can be &quot;condensed&quot; in a common method.<p>By the way the copy-pasta code is very resilient: a bug introduced in one place cannot spread, because the other code is duplicated and old.<p>Instead if you have too much condensed code, you end up with complex method which are difficult to maintain, because you have a huge &quot;coverage land of code&quot;, and you get scared when you need to change it.')
