Item(by='skohan', descendants=None, kids=None, score=None, time=1606932948, title=None, item_type='comment', url=None, parent=25278994, text='I would say even in a shifting landscape of different tools&#x2F;frameworks, the differences are mostly aesthetic, and the fundamentals mostly carry over.<p>For instance, at various stages in my career, I&#x27;ve done professional work in a lot of different languages and ecosystems: c&#x2F;c++, objC, Java, Node.js, Swift, React, Rust - and each one of these has it&#x27;s unique features, strengths, and weaknesses, but I think in general the more different tools you use, the more you see how much they have in common.<p>For instance, when I pick up a new framework or language, I&#x27;m looking for the same basic things: like how does the build system work, and how do I make an http request, how do I optimize for performance, and then I&#x27;m generally plugging the new tool into the same design patterns I&#x27;ve been using for years.')
