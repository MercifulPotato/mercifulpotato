Item(by='Osiris', descendants=None, kids=None, score=None, time=1608312799, title=None, item_type='comment', url=None, parent=25459506, text='The way I like to work is first to write out all the code I need to make something work correctly, then I go back over the code to see if there&#x27;s anything that could be simplified or split into separate functions, etc.<p>I really like to see DRY code, but if you have to make a helper function that takes a bunch of parameters with a bunch of conditionals to something slightly different, you might be better off just sticking the specific logic you need in each place.<p>The worst case of copy-pasta I saw in a codebase I came into was a function that was 1000 lines long, duplicated 3 times with &lt; 10 lines of it different for each copy. That&#x27;s a classic case for DRY to be applied.')
