The internet killed that kind of simplicity. You just can&#x27;t have the kind of minimalistic OS that you could back then. It wouldn&#x27;t be able to handle the security and performance challenges of the modern web.<p>Of course, if you just want a simpler GUI for Linux, then that is indeed possible. But that&#x27;s an uphill battle for another reason... the underlying libraries that you&#x27;d build you hypothetical GUI on are all built with the assumption that the user wants a rich <i>command-line</i> and <i>config file</i> interface to everything. Also you&#x27;ll have to give up modularity. If you build a GUI frontend for ALSA, you&#x27;ll also need to make it work with PulseAudio and Jack.')
