Item(by='mhh__', descendants=None, kids=None, score=None, time=1609707933, title=None, item_type='comment', url=None, parent=25624837, text='Firstly, (for the uninitiated) Bluspec is both a Haskell DSL (Bluespec Classic) and a Verilog-like language (Bluespec SystemVerilog)<p>It compiles to Verilog, but the stack is much more integrated than other similar compile-to-verilog HDLs - the simulator is similar to verilator and much easier to get started with.<p>I&#x27;m kind of beginning to feel that Haskell isn&#x27;t a good medium for HDL code - Verilog already encourages unreadable names like &quot;mem_chk_sig_state&quot; and Haskell code is almost unstructured to my eye (I like functional programming but it seems hard to keep it readable because of the style it imposes - the flow is there but the names are usually way too short for my taste)')
