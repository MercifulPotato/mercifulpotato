Item(by='sradman', descendants=None, kids=[25135998, 25112121], score=None, time=1605526608, title=None, item_type='comment', url=None, parent=25110025, text='Similar to SQL, GraphQL allows dynamic manipulation of a result set. Projection (filtering columns), selection (filtering rows), and sort order can all be specified on the client. Traditionally, REST interfaces have a fixed response representation for each endpoint.<p>For a dating site, a user can have a custom configuration specifying which profile attributes to include or exclude which helps optimize response size. A user can also generate advanced queries based on a faceted search interface.')
