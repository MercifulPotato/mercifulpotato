Item(by='DHowett', descendants=None, kids=[25867734, 25873512], score=None, time=1611274879, title=None, item_type='comment', url=None, parent=25863477, text='&quot;still requires a special WinRT call in order to enable them&quot;<p>Which WinRT call would that be?<p>Hi, owner of the Windows Console here.<p>Enabling VT control sequences is a matter of setting the ENABLE_VIRTUAL_TERMINAL_PROCESSING mode on the output handle, and it&#x27;s available through the same interface as every console mode flag that came before it. SetConsoleMode has a long history--dating back to the nineties--that this just builds on.<p>It&#x27;s behind a flag so that applications developed before ca. 2015 that like to emit control characters to the screen don&#x27;t melt away into gibberish.')
