Item(by='knlje', descendants=None, kids=[25025454], score=None, time=1604838025, title=None, item_type='comment', url=None, parent=25025269, text='I have used NixOS only for a year so I&#x27;m no expert but I try to answer.<p>The config file has this line:<p><pre><code>  imports = [ .&#x2F;hardware-configuration.nix ];\n</code></pre>\nThis file hardware-configuration.nix is typically generated semi-automatically during the installation. It has stuff like<p><pre><code>  fileSystems.&quot;&#x2F;&quot; =\n  { device = &quot;&#x2F;dev&#x2F;disk&#x2F;by-uuid&#x2F;14b5b22f-6cd2-4da5-a9a5-6ee74b76509d&quot;;\n    fsType = &quot;ext4&quot;;\n  };\n\n  fileSystems.&quot;&#x2F;boot&quot; =\n  { device = &quot;&#x2F;dev&#x2F;disk&#x2F;by-uuid&#x2F;7D69-81E5&quot;;\n    fsType = &quot;vfat&quot;;\n  };\n</code></pre>\nAFAIK you can include these details in configuration.nix if you happen to know them. But most of the time these are so hardware-specific details that you don&#x27;t want to write them manually. Does it make sense?')
