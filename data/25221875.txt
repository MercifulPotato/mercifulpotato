This is not the case for Firefox extension development.  It uses plain ZIP files.<p>You do not need this or any command-line tool to create Firefox extensions.  You can create and publish an extension entirely with Notepad, Firefox, and the built in ZIP functionality of Windows Explorer if you want.')
