Item(by='AnIdiotOnTheNet', descendants=None, kids=None, score=None, time=1609686925, title=None, item_type='comment', url=None, parent=25620841, text='Anonymous struct&#x2F;tuple&#x2F;array is a convenience feature that can only be used for values.<p>Consider the above code without it:<p><pre><code>  std.debug.print(&quot;hello world!\\n&quot;, struct{}{});\n</code></pre>\nNow add a value:<p><pre><code>  std.debug.print(&quot;{}!\\n&quot;, struct{string: []const u8}{ .string = &quot;hello world&quot;});\n</code></pre>\nQuite unwieldy. Zig used to have variadic functions and it was difficult to work with, complicated metaprogramming, and made how things work less obvious during reading. It was also pretty much only ever used for the &quot;print&quot; function.')
