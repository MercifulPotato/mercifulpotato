Item(by='acemarke', descendants=None, kids=None, score=None, time=1603259356, title=None, item_type='comment', url=None, parent=24845048, text='React and web components are different tools trying to solve different problems.<p>Web components are primarily about trying to add new virtual HTML tags to &quot;extend the platform&quot;.<p>React and other frameworks are about trying to build interactive applications that require larger-scale UI management, efficiently, on top of the DOM, by defining pieces of that UI as a tree of reusable components (and using techniques that web components don&#x27;t have available to them).<p>If I want to add a color picker to an otherwise static HTML page, I might use a color picker web component.<p>If I want to build a meaningful-sized app, I&#x27;d reach for React.')
