Item(by='kristjansson', descendants=None, kids=[25653128], score=None, time=1609888013, title=None, item_type='comment', url=None, parent=25650298, text='At the risk of contributing my own probably-subtly-broken advice to the collection already present in this thread, I would note that  there is a relatively sane system python3 on Catalina&#x2F;Big Sur, and most projects are building and distributing compatible wheels [1] (binary packages). So something like:<p><pre><code>    &#x2F;usr&#x2F;bin&#x2F;python3 -m venv .&#x2F;venv # create a virtual environment in .&#x2F;venv\n    .&#x2F;venv&#x2F;bin&#x2F;pip install wheel # optional\n    .&#x2F;venv&#x2F;bin&#x2F;pip install dep1 dep2 dep3 ...\n    .&#x2F;venv&#x2F;bin&#x2F;python ... # up and running\n</code></pre>\nis enough to get started in most cases.  Of course, all bets are off for GPU or Apple Silicon support...<p>[1]: <a href="https:&#x2F;&#x2F;pythonwheels.com&#x2F;" rel="nofollow">https:&#x2F;&#x2F;pythonwheels.com&#x2F;</a>')
