Item(by='int_19h', descendants=None, kids=None, score=None, time=1603284720, title=None, item_type='comment', url=None, parent=24845673, text='Not quite - CPython docs specifically say that some things are implementation-specific, and that other implementations are free to do them differently. For example, there&#x27;s no guarantee of refcounting, and portable Python code has to assume that implicit cleanup is non-deterministic (which is observable with __del__).<p>The problem, rather, is that coders tend to ignore this, and write code that only works on CPython.')
