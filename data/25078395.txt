So you either have normal Tabs or Tabs with Fav Icons, not both. ( This is new in Safari 14 )<p>FYI, there is a way to get around this annoying new setting.<p>1. Enable Preferences -&gt; Tabs -&gt; Show website icons.<p>2. Quit Safari.<p>3. Enable Safari Debug menu as described in <a href="https:&#x2F;&#x2F;yuriy.eu&#x2F;design&#x2F;2020&#x2F;09&#x2F;21&#x2F;safari-tab-preview.html" rel="nofollow">https:&#x2F;&#x2F;yuriy.eu&#x2F;design&#x2F;2020&#x2F;09&#x2F;21&#x2F;safari-tab-preview.html</a><p>4. Open Safari and disable Debug -&gt; Tab Features -&gt; Narrow Tabs.')
