Item(by='tikhonj', descendants=None, kids=[25098972, 25098917], score=None, time=1605422166, title=None, item_type='comment', url=None, parent=25098802, text='I&#x27;m not sure about the specifics, but providing something a lot more &quot;batteries-included&quot; seems to be the idea behind Emacs distributions like Spacemacs and Doom Emacs.<p>One difficulty is that language-specific tools (language servers, linters... etc) are standalone pieces of software with their own dependencies, requirements... etc. It&#x27;s hard to bundle together a fully &quot;batteries-included&quot; Emacs without figuring out a way to manage native software and services.<p>I think this is one place where Nix can be a real advantage: even if you don&#x27;t care about Nix&#x27;s reproducibility, it&#x27;s ability to manage Emacs, Emacs packages <i>and</i> language-specific tools is pretty unparalleled. If I had some more time on my hands, I would try putting together a Nix-based Emacs &quot;distribution&quot; with some easy way to configure which languages you want to work with. Coupled with lsp-mode and language-servers, this has some real potential!')
