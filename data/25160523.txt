Item(by='gh02t', descendants=None, kids=[25162242, 25160858], score=None, time=1605881207, title=None, item_type='comment', url=None, parent=25159799, text='ZFS can use that many drives, but it&#x27;s not a distributed filesystem so it&#x27;s intended to use within a single server. For 100 disks (or some arbitrarily large number) I&#x27;d assume you want something distributed, in which case ZFS could serve as the underlying FS on each node with e.g. Gluster or Ceph. ZFS as the backing store for Gluster is fairly common, for example.')
