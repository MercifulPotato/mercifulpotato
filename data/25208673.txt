Item(by='DavidSJ', descendants=None, kids=[25208775, 25208695], score=None, time=1606308692, title=None, item_type='comment', url=None, parent=25208574, text='My longstanding explanation for this has been as follows:<p>* Improvements to open-source projects come primarily from developers scratching a personal itch.<p>* User interfaces are first and foremost a way for the application to communicate to the user what is possible and how to achieve it.<p>* If there&#x27;s a problem with a user interface, it therefore most adversely impacts a user who does not yet know what is possible or how to achieve it.<p>* To fix a problem with the user interface, you must learn enough about the application such that you now know what is possible and how to achieve it. Hence the problem no longer affects you as much and is no longer a personal itch. So the problem doesn&#x27;t get fixed.')
