Item(by='alblue', descendants=None, kids=None, score=None, time=1607289471, title=None, item_type='comment', url=None, parent=25324930, text='The bcc-tools ship a Clang&#x2F;LLVM compiler to compile and build the BPF programs, which takes additional resources and adds a requirement for kernel headers to be installed on the machine.<p>The libbpf-tools uses a compiler at build time (not run time) to generate a bytecode (portable) program that need no additional dependencies on the server other than the libbpf, and so requires less setup and resources at runtime to install the filter.')
