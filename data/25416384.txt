Item(by='chrismorgan', descendants=None, kids=None, score=None, time=1607947875, title=None, item_type='comment', url=None, parent=25411610, text='You can set a few variables to change where these files go:<p>• &#x27;directory&#x27; controls where swap files go, toggled by &#x27;swapfile&#x27; (default on), filenames like […].swp; ~&#x2F;.cache&#x2F;vim&#x2F;swap&#x2F;&#x2F; is a good choice.<p>• &#x27;undodir&#x27; controls where undo files go, toggled by &#x27;undofile&#x27; (default off), filenames […].un~; ~&#x2F;.cache&#x2F;vim&#x2F;undo&#x2F;&#x2F; is a good choice.<p>• &#x27;backupdir&#x27; controls where backup files go, toggled by &#x27;backup&#x27; (default off), filename […]~; ~&#x2F;.cache&#x2F;vim&#x2F;backup&#x2F;&#x2F; is an interesting choice.<p>Arch Linux actually ships a runtime file that sets these paths, creating the directories if they don’t exist: <a href="https:&#x2F;&#x2F;github.com&#x2F;archlinux&#x2F;svntogit-packages&#x2F;blob&#x2F;packages&#x2F;vim&#x2F;trunk&#x2F;archlinux.vim" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;archlinux&#x2F;svntogit-packages&#x2F;blob&#x2F;packages...</a>.<p>(Note that the double trailing slashes are significant and necessary if you’re using a central location for these files. Refer to the help for explanation.)')
