Item(by='29athrowaway', descendants=None, kids=[25006656], score=None, time=1604608287, title=None, item_type='comment', url=None, parent=24998260, text='My IntelliJ tips:<p>- exclude folders from indexing<p>- disable unused plugins<p>- install a theme like Nord<p>- setup debugging<p>- make sure you can run unit tests from the IDE<p>- add JSdoc tags so that IntelliJ can detect type mismatches.<p>- setup scopes in your project view to show only the files that you are working on.<p>And these are my personal preferences:<p>- Disable smooth scrolling<p>- Disable animations<p>- Disable code folding<p>- Hide status bar, navigation bar, toolbar. If you need something use the actions menu... Ctrl (or Cmd) Shift A.<p>My font settings: Monoid size 14, line spacing 1.5, greyscale antialiasing.<p>And check out distraction free mode + full screen')
