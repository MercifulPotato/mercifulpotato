Item(by='lunixbochs', descendants=None, kids=None, score=None, time=1610330814, title=None, item_type='comment', url=None, parent=25713915, text='As this doesn&#x27;t use `__slots__`, every empty Array() will be 176 bytes vs the 56 bytes of [], and incur a dict allocation per array.<p>This is due to classes without `__slots__` gaining a `__dict__` attribute for dynamic attribute assignment.<p>Currently:<p><pre><code>    &gt;&gt;&gt; sys.getsizeof([])\n    56\n\n    &gt;&gt;&gt; a = Array()\n    &gt;&gt;&gt; sys.getsizeof(a)\n    72\n    &gt;&gt;&gt; sys.getsizeof(a.__dict__)\n    104\n</code></pre>\nwith `__slots__ = []` in the Array class definition:<p><pre><code>    &gt;&gt;&gt; a = Array()\n    &gt;&gt;&gt; sys.getsizeof(a)\n    56\n    &gt;&gt;&gt; sys.getsizeof(a.__dict__)\n    AttributeError: &#x27;Array&#x27; object has no attribute &#x27;__dict__&#x27;</code></pre>')
