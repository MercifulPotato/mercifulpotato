Item(by='GlitchMr', descendants=None, kids=None, score=None, time=1609757259, title=None, item_type='comment', url=None, parent=25624615, text='Few issues with this statement:<p>- That bit is not stored in RAM but rather somewhere within CPU which can have different bit flip frequency characteristics.<p>- Even assuming this bit can be flipped, most computers wouldn&#x27;t have ECC RAM inserted because well... the CPU doesn&#x27;t support it.<p>- And even assuming you have ECC RAM inserted, this bit flipping while your computer is running will likely lead to CPU hang considering the CPU didn&#x27;t fill in error correction bits and will lead CPU to think that pretty much every single byte is corrupt.<p>- And all in all, even if you did manage to turn on ECC bit, it would only work until the computer gets turned off.')
