Item(by='jere', descendants=None, kids=None, score=None, time=1606603595, title=None, item_type='comment', url=None, parent=25241011, text='Just want to echo all this praise for Godot. I developed games in JS for a while, then switched to Unity for platform&#x2F;performance reasons. I really did not get on with Unity and struggled through the tutorials (I was so frustrated I wrote a tutorial about JS game dev in response [1]). Unity felt bloated, slow, making simple things unintuitive, and with a mess of an API.<p>Godot on the other hand is straightforward, fast, and lightweight in a way I didn&#x27;t think was possible. Like I&#x27;m absolutely floored the executable is ~50MB and the low memory usage made me question intially if I had even launched it properly. Regardless of the pros&#x2F;cons, I&#x27;m just <i>enjoying</i> gamedev again.<p>Coming from JS, the optional type system doesn&#x27;t bother me none. My only annoyances are a few missing language features like increment operators or type coercion (even in Java I don&#x27;t need to cast a number to a string to print it).<p>[1] <a href="https:&#x2F;&#x2F;nluqo.github.io&#x2F;broughlike-tutorial&#x2F;" rel="nofollow">https:&#x2F;&#x2F;nluqo.github.io&#x2F;broughlike-tutorial&#x2F;</a>')
