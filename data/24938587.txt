Item(by='jlokier', descendants=None, kids=[24947404], score=None, time=1604025759, title=None, item_type='comment', url=None, parent=24934716, text='Of course there is a way to duplicate a line in Emacs by default.  Like in vi, you compose simple commands.<p>Let&#x27;s see:<p><pre><code>  ; Duplicates the current line\n  S-C-backspace C-[y y]\n\n  ; Duplicates the current line\n  C-[a k k y y]\n\n  ; Duplicates the current line\n  C-[a SPC n w y y]\n\n  ; Duplicates the current line\n  C-[a SPC n] ESC w C-y\n</code></pre>\nRepeat duplicating until it looks like enough:<p><pre><code>  ; Duplicate again until happy\n  C-[y y y...]\n\n  ; Duplicate again until happy\n  C-x z z z...\n</code></pre>\nDuplicate 3 lines instead of 1:<p><pre><code>  ; Duplicate this line and the next two\n  C-[a u] 3 C-[k y y]\n</code></pre>\nDuplicate 10 times:<p><pre><code>  ; Duplicate the current line 10 times\n  C-[a k k] C-x ( C-y C-x ) C-u 10 C-x e\n\n  ; Duplicate the current line 10 times\n  C-[a SPC n w] C-x ( C-y C-x ) C-u 10 C-x e\n\n  ; Duplicate the current line 10 times\n  S-C-backspace C-x ( C-y C-x ) C-u 10 C-x e\n</code></pre>\nRemove duplicate lines instead:<p><pre><code>  ; Select region, then\n  M-x delete-duplicate-lines</code></pre>')
