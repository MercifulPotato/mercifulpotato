Item(by='samatman', descendants=None, kids=None, score=None, time=1607291099, title=None, item_type='comment', url=None, parent=25326470, text='The motive behind ORM is in the name: it is to Map Objects to Relations.<p>The promise is that a Java (then Ruby) developer, could simply design the objects needed for the program, and the fields which need to be persistent could be automatically mapped to the database using ORM.<p>The reality is quite different of course, there&#x27;s a reason ORM is so widely derided. But ORM is more about skipping the bookkeeping involved in setting up persistence for application code, rather than testability or opacity of SQL.')
