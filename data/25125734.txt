In the past, one had to read the documentation to figure out how to use a tool.  So, for instance, running git --help or git branch --help would give the information needed.<p>Now having a what&#x27;s considered an intuitive interface is expected, which is why many people have problems using tools where the documentation is there, but people don&#x27;t want to take the time to read it.')
