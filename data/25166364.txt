Item(by='t-writescode', descendants=None, kids=[25166569], score=None, time=1605914654, title=None, item_type='comment', url=None, parent=25166328, text='I&#x27;ve still run into issues where an external service took approx 1 second per call. We had 20 threads on each docker container running Python. We had a lot of web requests.<p>It regularly took down the multithreaded python instance.<p>The solution we went with was to put all the different web requests into 2 flask servers and then take the output of all those different web requests and make one big request tot the django instance that had all the business logic.<p>If memory serves, we only had 2 flask instances handling all the traffic that the old array of django servers.<p>Async pays dividends.')
