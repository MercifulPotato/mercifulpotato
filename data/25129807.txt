Item(by='takeda', descendants=None, kids=None, score=None, time=1605647049, title=None, item_type='comment', url=None, parent=25129037, text='You absolutely can, at least in Firefox you can right click on search field, select &quot;Add a Keyword for this search...&quot; Then save it as bookmark and enter the keyword (you don&#x27;t have to use !, but it is an option if you chose so).<p>You can also create such bookmark manually and use %s in the url as a placeholder where search query should be placed.<p>The manual configuration can be useful when there&#x27;s no direct search field. For example freshports.org allows querying freebsd.org. I can add a bookmark with search keyword &quot;fp&quot; to point to <a href="https:&#x2F;&#x2F;freshports.org&#x2F;%S" rel="nofollow">https:&#x2F;&#x2F;freshports.org&#x2F;%S</a><p>After that I can type in address bar: fp lang&#x2F;python39 to land on <a href="https:&#x2F;&#x2F;freshports.org&#x2F;lang&#x2F;python39" rel="nofollow">https:&#x2F;&#x2F;freshports.org&#x2F;lang&#x2F;python39</a> (the capital %S doesn&#x27;t escape special characters like &#x2F;)')
