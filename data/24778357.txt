Item(by='mfontani', descendants=None, kids=[24778758], score=None, time=1602692417, title=None, item_type='comment', url=None, parent=24777783, text='One could do worse than using (more) stable distribution for their base images.<p>20.10 is not a LTS, so it&#x27;s more likely to get semi-spurious updates than a LTS version like 20.04 or, say, debian stable.<p>For most of my non-alpine-based images, I use debian:buster-slim as base, as it&#x27;s got a fairly stable base and gets quite routinely updated:<p><pre><code>    $ docker run --rm -it debian:buster-slim bash -c &quot;apt update &gt;dev&#x2F;null 2&gt;&amp;1 &amp;&amp; apt upgrade&quot;\n    Reading package lists... Done\n    Building dependency tree       \n    Reading state information... Done\n    Calculating upgrade... Done\n    0 upgraded, 0 newly installed, 0 to remove and 0 not upgraded.</code></pre>')
