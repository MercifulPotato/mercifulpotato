Item(by='UK-Al05', descendants=None, kids=None, score=None, time=1607161332, title=None, item_type='comment', url=None, parent=25310362, text='The mock should display the same behaviour as the real thing.<p>So I don&#x27;t think sharing one is bad.<p>The ideal fake is one that shares the same behaviour as the real thing. And you have tests that run across both to validate this.<p>Quite frankly mocks become the bane of large code bases.<p>I don&#x27;t know how many times I&#x27;ve seen tests that are basically useless because the mock behaves nothing like how it behaves in the real world. Like throwing an exception for not found rather than null.<p>Keeping those contracts in sync are much easier with fakes.')
