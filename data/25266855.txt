That&#x27;s really what Java&#x27;s checked exceptions are: one of the only places in Java where you can specify an alternative &#x27;return&#x27; type. But it isn&#x27;t quite complete enough to use in Java; little quirks here and there make it difficult to use widely, especially since Java 8 and lambdas arrived.<p>You can still use Java&#x27;s type system to get something approximating Rust&#x27;s approach. One upshot of Result-based error handling is that you can often replicate it in any language by shaving off one or two of the biggest benefits to suit that language&#x27;s constraints.')
