Item(by='adamkl', descendants=None, kids=[25860370], score=None, time=1611244643, title=None, item_type='comment', url=None, parent=25859929, text='To make it workable I had to just give up on the local host file integration.<p>I basically used the container like a VM. Configured it with all the tools I normally use (e.g. OhMyZsh, etc) and had it constantly running in the background. I would use VS Code as a front end and work directly inside the container (cloning repos and pushing commits).<p>It had its quirks but the main benefit was that my local machine was no longer a snowflake. I could easily move to any machine, pull my &quot;development&quot; image, spin up the container and everything was exactly as I liked it.')
