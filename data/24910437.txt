Item(by='bhattisatish', descendants=None, kids=None, score=None, time=1603824069, title=None, item_type='comment', url=None, parent=24907582, text='Well, this could be because you are running out of memory, which results in the yarn build process getting killed. The simplest solution is to enable &quot;swap&quot; in your instance by using the following commands:<p><pre><code>  dd if=&#x2F;dev&#x2F;zero of=&#x2F;swapfile bs=1M count=1024\n  sudo chown root:root &#x2F;swapfile \n  chmod 600 &#x2F;swapfile\n  mkswap &#x2F;swapfile\n  swapon &#x2F;swapfile\n  add &#x2F;etc&#x2F;fstab\n  &#x2F;swapfile               swap                    swap    \n  defaults        0 0 \n\n  #echo 0 &gt; &#x2F;proc&#x2F;sys&#x2F;vm&#x2F;swappiness\n  sudo sysctl vm.swappiness=10\n  add to &#x2F;etc&#x2F;sysctl.conf \n  vm.swappiness=10\n</code></pre>\nAll I am doing is creating a swap file of 1GB and enabling it with a not so aggressive swappinness on the kernel.')
