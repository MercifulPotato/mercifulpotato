Item(by='blabitty', descendants=None, kids=None, score=None, time=1610322797, title=None, item_type='comment', url=None, parent=25714822, text='For simple document or file type storage for an application I think solutions like S3 or azure blob storage are really great. In my real world experience we replaced a highly finicky way too heavyweight COTS CMS and all the associated database and backend SAN complexity with S3 and it ended up being way cheaper, easier and more reliable. To me object storage is one place where the cloud hype does deliver. Have had mixed results with the other services, even with things like RDS you have to do a lot of tuning and backup &#x2F; DR work yourself.')
