Item(by='jononor', descendants=None, kids=None, score=None, time=1604696406, title=None, item_type='comment', url=None, parent=25010921, text='A major reason for not shipping GTK3 port, which was largely completed a long time ago, is that GIMP 2.x has a stable plugins API and GTK2+ is exposed in said API. At the same time there has been other major revamps of the API with changing image processing layer to use GEGL. It was decided to release both breaking changes as GIMP 3.x, instead of breaking first the one, then the other.')
