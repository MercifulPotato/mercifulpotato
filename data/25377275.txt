Item(by='soraminazuki', descendants=None, kids=None, score=None, time=1607627807, title=None, item_type='comment', url=None, parent=25372162, text='Did you ever read a single line of alien source code? It extracts the metadata and files from a package, and re-archives it. That’s about it. It doesn’t handle package version differences, path differences, distro-specific rules, nothing. If your package has dependencies, forget about it because alien won’t retain dependency information it all. In other words, it only “works” for the most simple of cases.<p>I suggest you read the source code too, because otherwise you can cause real damage if you use it without understanding how it works.<p><a href="https:&#x2F;&#x2F;sources.debian.org&#x2F;src&#x2F;alien&#x2F;8.95&#x2F;alien.pl&#x2F;" rel="nofollow">https:&#x2F;&#x2F;sources.debian.org&#x2F;src&#x2F;alien&#x2F;8.95&#x2F;alien.pl&#x2F;</a>')
