Item(by='brabel', descendants=None, kids=None, score=None, time=1607343111, title=None, item_type='comment', url=None, parent=25330850, text='I&#x27;ve played a little bit with Unison and it&#x27;s definitely very interesting and a little bit of a new paradigm (some people compare their &quot;images&quot; to Smalltalk images, but I think they differ enough to be considered as distinct paradigms)... but they&#x27;re still working on very basic things, like how to enable people to do code reviews when the committed code is an AST, not just text... and how to actually distribute such software (I asked in their very friendly chat but apparently you can&#x27;t run the code outside the interpreter for now, which I think is written in Haskell)... also, the only help you can get writing code is some syntax highlighting, even though the ucm CLI can display function docs and look up functions by type, for example, similar to Haskell&#x27;s Hoogle (but in the CLI!!). ... so just be aware it&#x27;s very early days for Unison (and they do make that clear by &quot;forcing&quot; you to join the #alphatesting Slack chat to install it, which is a great idea IMO as it sets expectations early).')
