Item(by='Macha', descendants=None, kids=None, score=None, time=1606824436, title=None, item_type='comment', url=None, parent=25250622, text='So you can self host for free. Most of the older premium features are included in the open source server - it does have a license check, which loads a license file you&#x27;re provide by Bitwarden if you purchase one. These older premium features and the license check are themselves AGPL licensed last time I looked. Alternatively, you can use bitwarden_rs, a re-implementation of the server under GPLv3. Many of the premium features (such as yubikey support) are implemented in bitwarden_rs out of the box so you don&#x27;t need a license for those.<p>Some of the newer premium features, such as SSO integration are under the Bitwarden license, which is more of a visible source license.')
