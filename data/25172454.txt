Item(by='theevilsharpie', descendants=None, kids=[25176548], score=None, time=1605989797, title=None, item_type='comment', url=None, parent=25169813, text='Without hardware-level ECC support, how would a memory self-check (that isn&#x27;t along the lines of a diagnostic like Memtest86) work?<p>You would have to have some way of detecting that memory is corrupted, which non-ECC platforms do by writing a known pattern to memory and accessing it to verify that the pattern matches. You obviously can&#x27;t do that during runtime because that memory is being used by the kernel and applications.')
