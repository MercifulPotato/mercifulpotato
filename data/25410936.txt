If you call &quot;vim&quot; it&#x27;ll start in nocompatibe mode by default.<p>The nocompatible mode means adding some features to vim rendering it &quot;incompatible&quot; with the original vi. For example in the original vi you can only undo the last action, with compatible mode on vim will mimic this behavior.')
