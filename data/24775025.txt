Item(by='axilmar', descendants=None, kids=[24780177], score=None, time=1602670614, title=None, item_type='comment', url=None, parent=24762867, text='The newline character plays a very important role in formatting source code: long declarations&#x2F;statements are usually broken up into several lines of code using hidden newlines. These newlines do not alter the meaning of code.<p>The same hidden newlines though in a string will alter the string: they will make the string multiline.<p>So, in order to be more consistent with how code is formatted, I&#x27;d prefer hidden newlines in strings to not modify the strings.')
