Item(by='mumblemumble', descendants=None, kids=None, score=None, time=1607114365, title=None, item_type='comment', url=None, parent=25306071, text='I would love to see a different metric be introduced: Code <i>elusion</i>. All it is is the inverse of code coverage.<p>It would do a better job with the mental game. You may not be able to conclude that code is adequately tested just by knowing that the tests exercise it. But you can be pretty confident that code is not being tested if it eludes all of the tests.')
