Item(by='simiones', descendants=None, kids=None, score=None, time=1610549947, title=None, item_type='comment', url=None, parent=25759157, text='The only difference between Go and Java 1.4 in terms of collections is that Go has a generic map type, which Java didn&#x27;t. Java 1.4 still had generic arrays, just like Go. In fact, it was a little easier to program with generic arrays in Java vs Go (but also less safe) because in Java a subtype[] can be passed to a function which takes a supertype[] (arrays are covariant).')
