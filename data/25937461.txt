Item(by='giantrobot', descendants=None, kids=None, score=None, time=1611801673, title=None, item_type='comment', url=None, parent=25932633, text='Flexbox is by no means necessary to make a responsive web page. Media queries have been supported by mobile browsers for over a decade as has the viewport meta tag. Between the two it <i>should</i> be straightforward to make most page designs responsive.<p>Old browsers (old IE especially) that don&#x27;t handle those features aren&#x27;t a problem because they&#x27;ll just fall through to the screen style. Trying to use JavaScript to help old browsers is a <i>worse</i> solution as you&#x27;re more likely to run into JavaScript incompatibility problems than CSS.')
