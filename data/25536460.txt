Item(by='rwmj', descendants=None, kids=None, score=None, time=1608904338, title=None, item_type='comment', url=None, parent=25536310, text='I&#x27;ve been using git submodules for years but there are some real problems with them:<p>* Changing from a subdirectory to a submodule breaks lots of things like git reset and git bisect.<p>* Having to remember to git module init, update etc.  I always have to look up the commands and never remember what the difference is.<p>* I don&#x27;t care that there are unlisted files in a submodule, either don&#x27;t bug me about this in status, or integrate commands in such a way that they work transparently across the main module and submodule.<p>* Related to the previous: Coordinating a single logical change across submodules involves several manual steps and has plenty of scope to go wrong.')
