Item(by='calibas', descendants=None, kids=None, score=None, time=1611454267, title=None, item_type='comment', url=None, parent=25887930, text='The problem is finding people who actually know how to use it.  It&#x27;s great, but it takes years to learn properly. Before that it can be frustrating as hell.<p>It seems easy enough at first, like you can just use whatever contributed modules you need, but that only works on simple sites (like the same ones Wordpress is great for). To use Drupal well, you have to actually know the APIs and write your own custom modules, otherwise the site becomes a buggy resource hog.<p>I like Drupal, I have multiple 8&#x2F;9 sites I still run and I&#x27;ve used it since 6, but there&#x27;s some pretty good reasons not to use it.')
