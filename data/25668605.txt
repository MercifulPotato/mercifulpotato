Authors and users of tidyverse value other things like consistency and new features over API stability and backward compatility. The base-R ecosystem is actually very stable and so the original package manager is very simple.<p>With R spreading out from the academic environment and with many new authors breaking their packages&#x27; APIs we observe new attempts to solve the issues with dependencies (such as renv or <a href="https:&#x2F;&#x2F;rsuite.io" rel="nofollow">https:&#x2F;&#x2F;rsuite.io</a>)')
