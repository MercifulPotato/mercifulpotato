Every single gain is somebody&#x27;s loss. Either you rip the other guy&#x27;s heart out and eat it, or he rips your heart out and eats yours. There is no other choice.<p>Choosing Java over C++ is choosing to be that second guy.<p>Sure, Java coders are cheaper and easier to hire, and they produce <i>lots</i> more code. But if lots of code is a liability, lots of  Java code is a bigger liability.<p>A trading house with a lot of Java code is a red warning.')
