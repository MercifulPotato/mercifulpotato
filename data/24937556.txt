Item(by='herpderperator', descendants=None, kids=[24943522], score=None, time=1604015837, title=None, item_type='comment', url=None, parent=24937364, text='RSpec is definitely better, but I&#x27;ve mimicked it this way with \npytest because I didn&#x27;t like each Given&#x2F;When decorator in pytest-bdd requiring a separate method to setup. It was way too verbose. I want to be able to quickly nest them, like in RSpec.<p>So I do:<p><pre><code>  TestSomeMethod:\n      TestWhenDateProvided:\n           def test_it_does_not_use_system_date(fixture, ...):\n           assert ...\n\n      TestWhenDateNotProvided:\n           def test_it_uses_system_date(fixture, ...):\n           assert ...</code></pre>')
