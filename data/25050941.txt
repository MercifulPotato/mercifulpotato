Item(by='rkagerer', descendants=None, kids=[25051098, 25051961], score=None, time=1605038177, title=None, item_type='comment', url=None, parent=25047021, text='For anyone wondering, the .NET 2.0 dependency gives this really widespread compatibility on Windows.<p>It will run on Server 2003 R2 through Windows 7 without the need to install any framework beyond what comes out of the box in the OS.  i.e. Just copy the EXE and run it.<p>And 2.0 is the last version of the framework still installable on Windows 98.<p>Unfortunately Windows 8 and 10 would need you to install the 3.5 framework (which supports running 2.0 apps... yeah, I know, confusing).<p>I used to write all my little command line utilities in this version (with a theoretical exception if there was a really pressing need for new language features, but that didn&#x27;t tend to occur for those kind of utilities).<p>Of course not as lightweight as a Russinovich or Sofer style single-EXE done in C or whatever, but if you&#x27;re using NET Framework it&#x27;s not a bad choice of version.')
