Item(by='gnusty_gnurc', descendants=None, kids=[25061191], score=None, time=1605106672, title=None, item_type='comment', url=None, parent=25057730, text='Unit tests are fine for testing side effects if you&#x27;ve written the code properly with proper interfaces, etc.<p>You need to test that you&#x27;re using the contracts you&#x27;ve made, and integration tests verify that everything works as expected when both sides are complying with the contract.<p>I&#x27;m saying that if I&#x27;ve abstracted out the filesystem into an interface the distinction of asserting against whether a method was called on that interface and asserting a side-effect is somewhat trivial. I&#x27;ve made it so that side-effects must be handled in integration tests.')
