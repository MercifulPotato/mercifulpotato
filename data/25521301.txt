Item(by='skulk', descendants=None, kids=[25526461], score=None, time=1608752472, title=None, item_type='comment', url=None, parent=25505492, text='&gt; 13 Taming circular type dependencies<p>I really despise the &#x27;common&#x27; module approach because it scatters code that should be in the same file.<p>One workaround I use to avoid this is to break the cycle of type dependencies by replacing one of the dependencies with a generic parameter. This works great if the type that has a  generic parameter only deals with opaque objects of that type.<p>Once concepts are fully supported (they may already be now), this pattern will be able to describe more complex relationships.')
