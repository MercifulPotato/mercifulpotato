Item(by='arendtio', descendants=None, kids=None, score=None, time=1605432290, title=None, item_type='comment', url=None, parent=25099458, text='Actually, the commands you listed are more or less the basics, but the magic comes when you learn about the structure of the commands.<p>For example, `2dw` is a typical example. So you start with the number of times you want to do something, then you add a verb like `d` for delete and you end with a noun like `w` for word. As you probably have guessed, it deletes the next two words.<p>Since working with lines is quite common, many commands are executed for a line when you repeat the verb (easy to type). So `2dd` deletes the next two lines.<p>When you use this language you start using vim. There is still more to explore (e.g. macros), but this is where vim is different.')
