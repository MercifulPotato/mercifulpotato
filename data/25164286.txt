Item(by='yegle', descendants=None, kids=[25164436], score=None, time=1605902356, title=None, item_type='comment', url=None, parent=25163685, text='People don&#x27;t consider ChromeOS as a Linux desktop but I consider it a great one.<p>With the addition of Crostini, you get a full Debian based Linux running near native speed inside a KVM vm, and you can communicate between the ChromeOS and the VM seamlessly.<p>You can run Linux GUI programs and it will work (albeit the UI is a bit off, font etc), so vscode&#x2F;IntelliJ works.
