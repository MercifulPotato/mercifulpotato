Item(by='Technically', descendants=None, kids=None, score=None, time=1607988974, title=None, item_type='comment', url=None, parent=25417383, text='&gt; One of the key interest of C compared to more recent languages is that everything is explicit. With a finger you can follow the code as it runs and know exactly what is going on and when exactly.<p>I don&#x27;t agree! There&#x27;s a lot of behavior that&#x27;s implicit and you essentially need to internalize the C standard&#x2F;compiler behavior to follow. Weak typing, for instance; defaults for memory access&#x2F;fencing, handling faults, runtime semantics with respect to initialization of the process, cleanup via atexit, and signal handling.<p>Memorable, sure, but hardly <i>explicit</i>.')
