Item(by='badsectoracula', descendants=None, kids=[25425012, 25424314], score=None, time=1607983268, title=None, item_type='comment', url=None, parent=25422427, text='&gt; What happens when &#x27;a&#x27; is allocated to a CPU register?<p>When the &amp; operator is used on &#x27;a&#x27; it should mark it as unsafe to place the &#x27;a&#x27; on a CPU register - CPU allocation is an optimization and optimizations should <i>never</i> affect how the program behaves (except making it run faster, of course) and as such they should only be applied when the compiler can be sure that they&#x27;re safe to do so.<p>(and yes, the same applies on using &amp; on something like an array or struct element - the use of &amp; should taint any variable)')
