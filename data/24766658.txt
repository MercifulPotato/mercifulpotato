Item(by='chrismorgan', descendants=None, kids=[24766915, 24771699, 24767117], score=None, time=1602603813, title=None, item_type='comment', url=None, parent=24764605, text='Point of document implementation: please don’t use &lt;script type=&quot;math&#x2F;tex&quot;&gt;…&lt;&#x2F;script&gt; for equations, or else the equations are simply <i>missing</i> in environments where MathJax doesn’t run (e.g. JS disabled, text mode browser, or JS fails to load—all up, it’s more common than you realise). Use a form of markup that will fail <i>visible</i> rather than fail <i>invisible</i>. TeX mathematics notation is far better than a void.<p>You <i>could</i> make this abuse of &lt;script&gt; a bit more tolerable with something like the CSS `script[type^=&quot;math&#x2F;tex&quot;]:not([id]) { display: inline }`, but that will still be hidden to text-mode browsers or browsers where the CSS fails to load. It’s much better to just not use &lt;script&gt; in this way.')
