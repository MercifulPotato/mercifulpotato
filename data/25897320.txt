Item(by='tsimionescu', descendants=None, kids=[25899362], score=None, time=1611531054, title=None, item_type='comment', url=None, parent=25870793, text='I think the article is missing the immense value of simply having a good default serialization.<p>For most data models expressed in most languages, you can define a single canonical JSON representation that can be easily provided by a library.<p>Then, there come the corner cases of course. And you do need to add a lot of code to handle them. But good defaults + some customizability is a very good strategy for many problems.<p>It is true that you can encounter problems where your data model is so far away from JSON that it&#x27;s a better idea to just transform it entirely by hand. But the very reason JSON caught on is that these cases are rare in the industry.')
