Item(by='jitl', descendants=None, kids=[25005799], score=None, time=1604638823, title=None, item_type='comment', url=None, parent=25004913, text='SQLite database files themselves are binary, which typically isn&#x27;t diff-able with standard tools, so (eg) git wouldn&#x27;t show you anything useful unless you configure a special tool for diffing sqlite files.<p>And in fact, there is a sqlitediff.exe tool for diffing sqlite databases: <a href="https:&#x2F;&#x2F;sqlite.org&#x2F;sqldiff.html" rel="nofollow">https:&#x2F;&#x2F;sqlite.org&#x2F;sqldiff.html</a><p>But, it would still be a bummer to store them in git because git won&#x27;t efficiently track changes between sqlite databases, it&#x27;ll just add the whole binary blob on every commit. oof.')
