Item(by='ptx', descendants=None, kids=[25086736], score=None, time=1605260947, title=None, item_type='comment', url=None, parent=25079824, text='You don&#x27;t have to activate the environment. I never do; it&#x27;s a strictly optional convenience (it you think it&#x27;s convenient).<p>Instead, simply run the interpreter installed in the environment when you run your app, e.g. &quot;.&#x2F;my_env&#x2F;bin&#x2F;python my_app.py&quot;, and things will just work. No activation required, no special mode, nothing to forget.<p>The part about requirements.txt and installing packages could also be simplified if you did it the other way around: install first and create the requirements file from that:<p><pre><code>  $ python3 -m venv my_env\n  $ my_env&#x2F;bin&#x2F;pip install some-dependency\n  $ my_env&#x2F;bin&#x2F;pip freeze &gt;requirements.txt\n  $ my_env&#x2F;bin&#x2F;python3 my_app.py\n</code></pre>\nThere you go. Setup, install and run in four steps and zero modes.')
