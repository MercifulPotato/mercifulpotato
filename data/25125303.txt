Item(by='siraben', descendants=None, kids=[25127878, 25139073], score=None, time=1605628952, title=None, item_type='comment', url=None, parent=25124211, text='Having used the reMarkable 1 tablet for over a year now, the hackability is so refreshing to have in a consumer device. The tablet runs Linux and you can SSH into with root access.<p>The hacking community is quite active, one frequently asked question is how to get and run custom software on the device.<p>There&#x27;s the Entware[0] package repository for running ARMv7 software on the tablet. Recently a community maintained package repository, toltec[1] for tablet-specific software was created and is actively maintained.<p>For Nix users, I&#x27;ve been helping build a Nix-based cross-compile toolchain[2], so you could easily cross-compile a good amount of the 60K packages on Nixpkgs for the tablet.  Upstreaming the cross toolchain to Nixpkgs is in the works.<p>To see what the community is up to, see the IRC channel[3], Reddit[4] and Discord server[5].<p>[0] <a href="https:&#x2F;&#x2F;github.com&#x2F;evidlo&#x2F;remarkable_entware" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;evidlo&#x2F;remarkable_entware</a><p>[1] <a href="https:&#x2F;&#x2F;github.com&#x2F;toltec-dev&#x2F;toltec" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;toltec-dev&#x2F;toltec</a><p>[2] <a href="https:&#x2F;&#x2F;github.com&#x2F;siraben&#x2F;nix-remarkable" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;siraben&#x2F;nix-remarkable</a><p>[3] #reMarkable on Freenode<p>[4] <a href="https:&#x2F;&#x2F;reddit.com&#x2F;r&#x2F;RemarkableTablet&#x2F;" rel="nofollow">https:&#x2F;&#x2F;reddit.com&#x2F;r&#x2F;RemarkableTablet&#x2F;</a><p>[5] <a href="https:&#x2F;&#x2F;discord.gg&#x2F;JSSGnFY" rel="nofollow">https:&#x2F;&#x2F;discord.gg&#x2F;JSSGnFY</a>')
