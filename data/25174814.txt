Item(by='geofft', descendants=None, kids=None, score=None, time=1606017410, title=None, item_type='comment', url=None, parent=25174672, text='X11 (aka X, because older versions haven&#x27;t been relevant in decades) is a protocol&#x2F;architecture for GUIs. The X.Org Foundation is a group that maintains the Xorg server (the engine that actually draws graphics somewhere) as well as several X protocol client libraries (used by applications), other server implementations, and associated tools. libX11 is the primary client library.<p><a href="https:&#x2F;&#x2F;magcius.github.io&#x2F;xplain&#x2F;article&#x2F;x-basics.html" rel="nofollow">https:&#x2F;&#x2F;magcius.github.io&#x2F;xplain&#x2F;article&#x2F;x-basics.html</a> is probably a good place to start, as is the official documentation (indexed at <a href="https:&#x2F;&#x2F;www.x.org&#x2F;releases&#x2F;current&#x2F;doc&#x2F;index.html" rel="nofollow">https:&#x2F;&#x2F;www.x.org&#x2F;releases&#x2F;current&#x2F;doc&#x2F;index.html</a> , but the stuff at the top may not be what you want - you probably want to start with the protocol docs <a href="https:&#x2F;&#x2F;www.x.org&#x2F;releases&#x2F;current&#x2F;doc&#x2F;xproto&#x2F;x11protocol.html" rel="nofollow">https:&#x2F;&#x2F;www.x.org&#x2F;releases&#x2F;current&#x2F;doc&#x2F;xproto&#x2F;x11protocol.ht...</a> or perhaps the libX11 docs <a href="https:&#x2F;&#x2F;www.x.org&#x2F;releases&#x2F;current&#x2F;doc&#x2F;libX11&#x2F;libX11&#x2F;libX11.html" rel="nofollow">https:&#x2F;&#x2F;www.x.org&#x2F;releases&#x2F;current&#x2F;doc&#x2F;libX11&#x2F;libX11&#x2F;libX11....</a> ).')
