Item(by='panarky', descendants=None, kids=None, score=None, time=1602947065, title=None, item_type='comment', url=None, parent=24809794, text='SQL is great for relational tables with a two-dimensional structure of rows and fixed columns.<p>It&#x27;s not so great for schemaless object databases or navigating nested arrays and hashmaps.<p>Some relational databases can store JSON objects, and incorporate SQL extensions to navigate those objects.<p>But in general, SQL doesn&#x27;t seem to be a great tool for data that isn&#x27;t a collection of two-dimensional data structures.')
