Item(by='fanf2', descendants=None, kids=None, score=None, time=1604544355, title=None, item_type='comment', url=None, parent=24990997, text='The keyboard matrix driver is on the main board, and there’s no access to the USB2 bus that connects the matrix driver to the USB controller. So you would have to run the whole Raspberry Pi in USB “gadget mode”, so it pretends to be a device rather than a host. The USB-C power port can also support USB-2 OTG in gadget mode (but it has no high bandwidth features).')
