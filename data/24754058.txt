Item(by='dathinab', descendants=None, kids=None, score=None, time=1602508640, title=None, item_type='comment', url=None, parent=24746716, text='The problem is not the CDN (or arbitrary shared source domain) being able to track you but the sites which use the CDN.<p>Furthermore a CDN can&#x27;t track you as simple as you might think, it often would require thinks which need explicitly opt-in agreements <i>on a per website basis</i> to be legal.<p>Furthermore due to technical limitations you can only get that permission from the user after the CDN was already used.<p>CDNs can still track aggregated information to some degree but they can&#x27;t legally act like a tracker cookie.')
