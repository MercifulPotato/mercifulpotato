Item(by='jetrink', descendants=None, kids=None, score=None, time=1605623594, title=None, item_type='comment', url=None, parent=25123067, text='I have a separate playbook called &#x27;provision&#x27; that, among other things, creates an unprivileged user, gives it sudo permissions, disables root login and password authentication in sshd_config and then restarts ssh. It&#x27;s the only playbook that uses root. All subsequent connections are made using the unprivileged user and the become directive for privilege escalation. It&#x27;s a bit awkward, because the playbook can only be run once, but it works in practice.')
