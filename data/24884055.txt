Item(by='gizmo686', descendants=None, kids=None, score=None, time=1603603376, title=None, item_type='comment', url=None, parent=24884026, text='Yes.<p>You can get it with: git fetch &lt;remote&gt; refs&#x2F;pull&#x2F;&lt;pr&gt;&#x2F;head<p>My git config has the alias:<p><pre><code>  pr = !f() { git fetch $1 refs&#x2F;pull&#x2F;$2&#x2F;head:pr&#x2F;$1&#x2F;$2; } ; f\n</code></pre>\nwhich will create a local branch corresponding to the provide PR. This is useful for evaluating large PRs that would be difficult to fully evaluate with just the online UI.')
