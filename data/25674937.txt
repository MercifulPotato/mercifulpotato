C++ since it was added.<p>As far as type systems, there&#x27;s really two different types of &quot;types&quot;: individual types objects that can have generic functions attached to them, etc.  This is not as well known, and there are actually several object systems for typing:<p><a href="http:&#x2F;&#x2F;adv-r.had.co.nz&#x2F;OO-essentials.html" rel="nofollow">http:&#x2F;&#x2F;adv-r.had.co.nz&#x2F;OO-essentials.html</a><p>But these sort of objects are not quite as commonly created by programmers, because the second type of &quot;types&quot; are much more useful: data frames, which is kind of a vectorization of structs. This is what would be used in data oriented design, which is apparently much more common in modern game design.')
