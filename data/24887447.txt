This means that it will return Cloudflare (i.e. ASN of CLOUDFLARENET), but probably the same Cloudflare as if you are using Cloudflare&#x27;s public DNS servers.<p>Tip:  You can check a specific DNS server with dig and curl:<p><pre><code>  UUID=$(uuidgen)\n  dig ${UUID}.which.nameserve.rs @1.1.1.1\n  curl --silent https:&#x2F;&#x2F;which.nameserve.rs&#x2F;debug.txt | grep ${UUID}\n</code></pre>\nand then do a ASN lookup on the IP address<p>Note: the debug page is unofficial, and may change, so don&#x27;t bake this into anything.')
