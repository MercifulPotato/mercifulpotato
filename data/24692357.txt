Item(by='sergeykish', descendants=None, kids=None, score=None, time=1601934135, title=None, item_type='comment', url=None, parent=24689006, text='The more I use linux (12 years), the more I believe that common users should not use &quot;stable distribution&quot;. Upgrade is risky, software is always outdated. It solves nothing and provides unjustified feeling of stability.<p>LTS makes sense as deployment platform. Push it to server and deploy own application. Make a demo like subject NVIDIA Jetson Nano SDK and forget about it.<p>There are better ways to help underpowered machines - use light desktop environment, block ads in browser, add RAM if possible. Kernel and base requirements has not changed much in a decade. About 120 MB with XOrg.')
