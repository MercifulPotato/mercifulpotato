Item(by='mschuster91', descendants=None, kids=[25061490], score=None, time=1605120064, title=None, item_type='comment', url=None, parent=25061188, text='As long as you&#x27;re not doing push stuff:<p>1) set up a series of N docker registry mirrors in pull-through mode (<a href="https:&#x2F;&#x2F;docs.docker.com&#x2F;registry&#x2F;recipes&#x2F;mirror&#x2F;" rel="nofollow">https:&#x2F;&#x2F;docs.docker.com&#x2F;registry&#x2F;recipes&#x2F;mirror&#x2F;</a>, it&#x27;s as simple as &quot;docker run --rm --name registry -d -p 5000:5000 -e REGISTRY_STORAGE_DELETE_ENABLED=true -e REGISTRY_PROXY_REMOTEURL=<a href="https:&#x2F;&#x2F;registry-1.docker.io" rel="nofollow">https:&#x2F;&#x2F;registry-1.docker.io</a> -v &#x2F;mnt&#x2F;persistentdata&#x2F;registry:&#x2F;var&#x2F;lib&#x2F;registry registry&quot;)<p>2) expose them on the same domain name (multiple A records, loadbalancers, whatever you want)<p>3) set them as mirrors in each machine&#x27;s docker daemon<p>In case one of your mirrors go down, take them out of the DNS&#x2F;LB rotation. That&#x27;s it.')
