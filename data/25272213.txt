Everything should copy that.<p>All ELF systems support the $ORIGIN RPATH macro. It lets you build and package an application exactly like, if not better than, you would a macOS bundle. Specifically, it let&#x27;s you specify at compile time a library load path relative to the invoked binary. You&#x27;d specify it as $ORIGIN&#x2F;..&#x2F;lib for the canonical bin&#x2F;, lib&#x2F;, etc layout.')
