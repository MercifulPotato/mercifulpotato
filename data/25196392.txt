Item(by='joegibbs', descendants=None, kids=None, score=None, time=1606205903, title=None, item_type='comment', url=None, parent=25195695, text='It does it by this CSS:<p><pre><code>    main &gt; * {\n        display: none;\n    }\n\n    main &gt; *:last-child {\n        display: block;\n    }\n\n    main &gt; *:target {\n        display: block;\n    }\n\n    main &gt; *:target ~ * {\n        display: none;\n    }\n</code></pre>\nWhich hides all children of the &lt;main&gt; element except the last element, which is the default page. Then when an element is targeted by its ID it&#x27;s displayed, otherwise it&#x27;s hidden.<p>It saves that into a CSS file and injects that into the &lt;head&gt;.')
