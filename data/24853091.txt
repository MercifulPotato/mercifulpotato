Item(by='jlokier', descendants=None, kids=None, score=None, time=1603320469, title=None, item_type='comment', url=None, parent=24852675, text='That sounds like it could be that you&#x27;re running out of memory.<p>Open the Activity Monitor, switch to the Memory tab, and watch the Memory Pressure graph as well as Memory Used.  If memory pressure isn&#x27;t green, or memory used is close to Physical Memory, you&#x27;re using too much.  In that state, the whole MacOS GUI can become rather unresponsive, fitting your description.')
