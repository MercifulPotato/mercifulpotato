Item(by='marcosdumay', descendants=None, kids=None, score=None, time=1606503131, title=None, item_type='comment', url=None, parent=25231220, text='There is an interesting property of the speed of causality, that it is fixed independently of the observer (as in, any observer will measure the same speed).<p>Our mechanics allow only one such speed, and if you try to use one that doesn&#x27;t have this property for causality, you&#x27;ll get all kinds of paradoxes.<p>But it&#x27;s probably something that can be solved. All said, the reason we assume causality moves at that speed is because it agrees with the experiments.')
