Item(by='sarnu', descendants=None, kids=None, score=None, time=1606200772, title=None, item_type='comment', url=None, parent=25195482, text='I use it for developing inside docker containers.\nWe have a dockerized Apache Airflow setup for which I  am adding plugins.\nRemote VSCode allows to work inside one of the containers spun up by docker-compose, so I do not have to simulate the environment the final system is running in.  \nThink of it as python venv on steroids.<p>Another use case is remote ssh where I can work on a headless system with a graphical editor.\nI use it e.g. to work in a minimal Linux VM from my Windows desktop.<p>The remote docker plugin is a little fiddly to setup, but once running works pretty good.')
