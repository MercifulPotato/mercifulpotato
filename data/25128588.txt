Item(by='joshspankit', descendants=None, kids=[25130997], score=None, time=1605641893, title=None, item_type='comment', url=None, parent=25127304, text='I <i>suspect</i> (based on similar issues years ago) that you could get it working on MacOS in either one of two ways:<p>1. Set a password for the setting “VNC viewers may control screen with password:” in the screen sharing settings. Most of the issue with standard VNC clients is that MacOS uses MacOS-specific login security, but setting a password opens it up to VNC-standard security<p>2. Install an alternate VNC server on the Mac. This bypasses all the MacOS security nonsense.')
