Item(by='Rygian', descendants=None, kids=[24847851], score=None, time=1603280474, title=None, item_type='comment', url=None, parent=24846679, text='Every Western-style input method should come with a Compose function enabled, and a simple tutorial on how to use it.<p>For example, on Windows I use Wincompose [1], that lets me type things like &quot;⋄*g&quot; for Greek gamma: &#x27;γ&#x27;, or &quot;⋄ee&quot; for &#x27;ə&#x27;, or even &quot;⋄⋄plane&quot; for a &#x27;&#x27; (and of course the list is customizable). The &quot;⋄&quot; symbol represents the Compose key, which I chose to map to the Caps Lock key on my keyboard.<p>[1] <a href="http:&#x2F;&#x2F;wincompose.info&#x2F;" rel="nofollow">http:&#x2F;&#x2F;wincompose.info&#x2F;</a><p>Edit: HN removed the airplane icon.')
