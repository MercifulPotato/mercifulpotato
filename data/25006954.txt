Item(by='dsr_', descendants=None, kids=[25007942], score=None, time=1604663331, title=None, item_type='comment', url=None, parent=25004585, text='Wrapping a new abstraction layer around a single server does not help, it is an expense you do not need. &quot;Recreating these manifests elsewhere&quot; will not work, because there is no elsewhere.<p>You cannot add complexity to a system to make it simpler.<p>You cannot abstract away the configuration of a system when there is only one system: you must actually do the configuration.<p>There is no point in having a high-level representation of all the things on the system: you have the actual things on the system, and if you do not know how to configure them, you should not be running them.')
