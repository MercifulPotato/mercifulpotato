Item(by='alex-treebeard', descendants=None, kids=None, score=None, time=1609756061, title=None, item_type='comment', url=None, parent=25629566, text='Thought I&#x27;d drop my xmas project in here: A visual disk usage analyser for docker images.<p>I have regularly found Disk Inventory X to be illuminating when fighting congestion on my macbook, but when it came to de-congesting Python docker images there was no similar tool.<p>Unlike Disk Inventory X and other similar tools, Whaler works from the command-line both on local dirs and images. It is built using modern front end tech: D3, Canvas, React, making it portable to all platforms including headless environments.<p>I have tested the tool with directories containing up to 500K nodes which should be fine for most Python Docker images. Beyond that it may chug&#x2F;crash.<p>A live demo of my CI environment is <a href="https:&#x2F;&#x2F;treebeardtech.github.io&#x2F;whaler&#x2F;" rel="nofollow">https:&#x2F;&#x2F;treebeardtech.github.io&#x2F;whaler&#x2F;</a>')
