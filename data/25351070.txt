Function(T,T) f, T? x0)\n         =&gt; this.moveNext()\n            ? (this.reduce(f, (x0 == null || this.current == null) ? null : f(x0,current)))\n            : x0;\n    }\n</code></pre>\nNot sure if this is the best notation but then I&#x27;ve never written Dart before.<p>Of course this is all a bit easier to read if we use some syntax specially built for Monads such as Haskell&#x27;s do syntax:<p><pre><code>    reduce f (head:tail) x0 = do { x &lt;- x0 \n                                 ; y &lt;- head\n                                 ; reduce f tail (f x y) }\n    reduce f [] x0 = x0\n</code></pre>\nwhich has the advantage that it works for all Monads.')
