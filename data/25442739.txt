Item(by='PragmaticPulp', descendants=None, kids=[25443083, 25443023], score=None, time=1608128378, title=None, item_type='comment', url=None, parent=25441464, text='When secure boot started to go mainstream there was a lot of FUD that it was really just a mechanism to lock out alternative OSes.<p>The reality is that hardware manufactures do not care what OS you run, as long as it doesn’t result in costly hardware returns or endless support calls. This is why hardware typically has options to disable OS-specific. secure boot, but it requires some technical steps that average users won’t accidentally stumble upon.<p>The M1 macs were never locked down, either. Apple has said that Microsoft can make Windows builds for them if they’d like. Linux on M1 macs is only a matter of time.<p>Secure boot is a net win for everyone. Having an option to disable it lets the 0.01% of people who want to run Linux use the hardware too.')
