Bare metal so far has been a &quot;we built it because we could&quot; kind of feature, so please be warned it&#x27;s not as polished. Here&#x27;s a quick tutorial.<p>Basically, ape&#x2F;ape.S is a traditional BIOS bootloader. It also embeds a GRUB stub, which basically just switches back to the real mode BIOS bootloader entrypoint. So it should work with or without GRUB. The boot sector loads the rest of your executable of disk and prints &quot;actually portable executable&quot; to your display. It then configures standard i&#x2F;o to use serial uart, sets up the page tables, enters long mode, and calls _start(). That&#x27;s all I&#x27;ve got working so far. I would love to have e1000 and virtio though!<p>To try it out, here&#x27;s how you build the included PC BIOS system emulator and boot an x86-64 ape executable that prints a spinning deathstar:<p><pre><code>    m=tiny; make -j12 MODE=$m o&#x2F;$m&#x2F;tool&#x2F;build&#x2F;blinkenlights.com o&#x2F;$m&#x2F;tool&#x2F;viz&#x2F;deathstar.com &amp;&amp;\n      o&#x2F;$m&#x2F;tool&#x2F;build&#x2F;blinkenlights.com -rt o&#x2F;$m&#x2F;tool&#x2F;viz&#x2F;deathstar.com\n</code></pre>\nVideo screencast: <a href="https:&#x2F;&#x2F;justine.lol&#x2F;cosmopolitan&#x2F;deathstar.html" rel="nofollow">https:&#x2F;&#x2F;justine.lol&#x2F;cosmopolitan&#x2F;deathstar.html</a><p>See also: <a href="https:&#x2F;&#x2F;justine.lol&#x2F;blinkenlights&#x2F;manual.html" rel="nofollow">https:&#x2F;&#x2F;justine.lol&#x2F;blinkenlights&#x2F;manual.html</a><p>Enjoy!')
