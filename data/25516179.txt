So if you prefer this to CSS, go ahead.<p>Most of the angst around &lt;table&gt; was before aria support, so using &lt;table&gt; purely for layout impacted accessibility, which meant it was not viable for most real-world scenarios.<p>That said, if you need stuff like colspans for a layout, you are probably better of using grid.')
