Item(by='chrismorgan', descendants=None, kids=None, score=None, time=1601802749, title=None, item_type='comment', url=None, parent=24677275, text='Not all styli have erasers, but a very substantial fraction do. Detecting it is a basic operation when you’re using the Pointer Events API (event.button == 5 and event.buttons &amp;== 32). I use not supporting this as a <i>proxy</i> for other basic errors of implementation, such as using mouse events instead of pointer events. This is why I say it’s a very bad <i>sign</i>. It’s a simple thing that I can point at and say, “this thing is half-baked; they have very probably done a bad job”, without needing to go through a larger battery of comparative and subjective tests.')
