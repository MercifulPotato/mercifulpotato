Item(by='skissane', descendants=None, kids=None, score=None, time=1611630202, title=None, item_type='comment', url=None, parent=25911334, text='&gt; An example off the top of my head -- a CNC mill at my previous job, purchased new perhaps ten years ago, runs DOS 6.22 with Windows CE on top<p>I thought Windows CE was a standalone operating system like current Windows versions are, not a layer on top of DOS like Windows 3.x and earlier were.<p>But, Googling, I see Windows CE, on x86, did use DOS as a bootloader, just like how Windows 9x did. DOS boots, then runs LOADCEPC.EXE to start Windows CE. (By contrast, CE on other CPU architectures didn&#x27;t do this, it just booted Windows CE directly.)')
