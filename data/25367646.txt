Item(by='kibwen', descendants=None, kids=[25368955, 25368244], score=None, time=1607559654, title=None, item_type='comment', url=None, parent=25367042, text='I think the belief that Go does no dynamic linking whatsoever is outdated; I seem to recall that they recanted this approach on Mac and Windows since these platforms only offer a stable system interface via dynamically-linked libraries, and trying to re-implement all of it ended up being a lot of work only to end up with broken code after system updates. On Linux I think Go still tries to link mostly statically, but if you use parts of the stdlib (e.g. `net`) you&#x27;ll still end up dynamically linking to libc unless you instruct it otherwise. If anyone more knowledgeable could speak up I&#x27;d welcome it.')
