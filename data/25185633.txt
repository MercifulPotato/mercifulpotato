Item(by='bonzini', descendants=None, kids=None, score=None, time=1606130844, title=None, item_type='comment', url=None, parent=25185548, text='There&#x27;s a way for odd divisors, since they have a multiplicative inverse mod 2^n.<p>For example, the inverse of 3 modulo 2^32 is 0xAAAAAAAB. \nIf you multiply an unsigned number x by 0xAAAAAAAB, the result is less than or equal to 0xFFFFFFFF&#x2F;3 if and only x is divisible by 3.<p>Of course this is only possible for non-bignum computations.<p>But there are other tricks for bignums, for example the number of even set bits in a multiple of three is equal to the number of odd set bits, because 4*n+k = n+k (mod 3) and the property is true for 0 and 3 but not 1 and 2.')
