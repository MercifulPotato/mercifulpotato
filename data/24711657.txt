Item(by='phaer', descendants=None, kids=[24711716], score=None, time=1602098440, title=None, item_type='comment', url=None, parent=24711194, text='Emacs can use GPG to encrypt files at rest pretty transparently. Just save a files with the extension `.org.gpg` and it should get encrypted automatically with Spacemacs (I personally use <a href="https:&#x2F;&#x2F;github.com&#x2F;hlissner&#x2F;doom-emacs&#x2F;" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;hlissner&#x2F;doom-emacs&#x2F;</a>).<p>Note that adding an encrypted file to your agenda, .e.g.\n`(setq org-agenda-files &#x27;(&quot;~&#x2F;org&#x2F;secret-diary.org.gpg&quot;))` will let Emacs decrypt that file upon calling &quot;org-agenda&quot;. Something similar should be possible for search if it doesn&#x27;t work out of the box')
