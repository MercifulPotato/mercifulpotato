Item(by='otabdeveloper4', descendants=None, kids=None, score=None, time=1611150425, title=None, item_type='comment', url=None, parent=25845241, text='&gt; It&#x27;s designed to force people to handle the damn error as near to the call as possible.<p>This is always the wrong way to handle errors.<p>If a function returns an &#x27;error&#x27; that needs be handled at the call site, then it isn&#x27;t an error, it&#x27;s a variant return type.<p>Errors are things that can&#x27;t be recovered from but must be handled to release resources.<p>You want this to happen in some central place, not scattered ad-hoc in every place where you use resources; releasing them by hand is worse than manual memory management.')
