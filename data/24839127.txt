Item(by='skohan', descendants=None, kids=None, score=None, time=1603210585, title=None, item_type='comment', url=None, parent=24836678, text='It&#x27;s good for isolation: i.e. in general if you copy a directory containing an NPM project to another location, in general it will just work, and deleting something elsewhere on the file system should never affect the project.  In general I would argue this is a more important property for dependency managers to have rather than using the least amount of space possible.<p>If you <i>really</i> want to avoid it you can install everything as a global dependency, but it&#x27;s not best practice.')
