Item(by='dahfizz', descendants=None, kids=[25306646, 25310450, 25307886], score=None, time=1607110272, title=None, item_type='comment', url=None, parent=25306071, text='There is a big difference here between strongly typed, compiled languages and weakly typed interpreted languages.<p>High code coverage is much more important on e.g. Javascript or Python because running the code is the only way you know that it &quot;compiles&quot; and you didn&#x27;t do something dumb like mistake the type of arguments, access fields that don&#x27;t exist, etc.<p>In a compiled codebase, I think functional &#x2F; end-to-end tests are more important. I don&#x27;t care as much if every single function is exercised by unit tests if I know that the software works as intended for all supported use cases.')
