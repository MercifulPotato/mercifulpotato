Item(by='ikreymer', descendants=None, kids=None, score=None, time=1608844333, title=None, item_type='comment', url=None, parent=25531469, text='I&#x27;m excited to share the latest update from Webrecorder project, the release of a new OldWeb.today (<a href="https:&#x2F;&#x2F;oldweb.today" rel="nofollow">https:&#x2F;&#x2F;oldweb.today</a>), now running emulated browsers using JS&#x2F;WASM entirely in your browser!<p>This is an update to the previous version (discussed 5 years ago at: <a href="https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=10653033" rel="nofollow">https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=10653033</a>)<p>This builds on much previous work, including two excellent emulators, v86 (previously discussed at: <a href="https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=11155203" rel="nofollow">https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=11155203</a>) and Basilisk in JS (mentioned here: <a href="https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=20632843" rel="nofollow">https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=20632843</a>) running in the browser. These were modified to support a browser-based network stack, an Emscripten build of picotcp created by the bwFla Emulation-as-a-Service team (<a href="https:&#x2F;&#x2F;gitlab.com&#x2F;emulation-as-a-service&#x2F;picotcp.js" rel="nofollow">https:&#x2F;&#x2F;gitlab.com&#x2F;emulation-as-a-service&#x2F;picotcp.js</a>)<p>The IE and Netscape browsers support older versions of Flash and Java. For good measure, there is also an option to run just the Ruffle emulator (discussed at: <a href="https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=25242115" rel="nofollow">https:&#x2F;&#x2F;news.ycombinator.com&#x2F;item?id=25242115</a>) for Flash-only emulation in your current browser.<p>The system can browse live web pages as well as archived pages from Internet Archive&#x27;s Wayback Machine.\nOldWeb.today can be deployed as a static site and connected to other archives as well. (Only a server-side CORS proxy is necessary, for connecting to external websites or archives)<p>More details on how it works at:\n<a href="https:&#x2F;&#x2F;github.com&#x2F;oldweb-today&#x2F;oldweb-today" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;oldweb-today&#x2F;oldweb-today</a><p>Blog post:\n<a href="https:&#x2F;&#x2F;webrecorder.net&#x2F;2020&#x2F;12&#x2F;23&#x2F;new-oldweb-today.html" rel="nofollow">https:&#x2F;&#x2F;webrecorder.net&#x2F;2020&#x2F;12&#x2F;23&#x2F;new-oldweb-today.html</a>')
