Item(by='dgb23', descendants=None, kids=None, score=None, time=1605779438, title=None, item_type='comment', url=None, parent=25146987, text='It systematically abstracts CSS to an atomic subset.<p>You either take the default configuration with a few tweaks, for example if you are a programmer first.<p>Or you define a more complete, unique system upfront, for example if you implement a bespoke interface from a UI designer.<p>If you see emerging patterns that ask for composition, then you can easily compose utilities and give your components semantic class names.<p>It is a blessing for productivity, performance and maintainability, because it decouples CSS concretions by introducing a light, composable abstraction layer between raw CSS and component&#x2F;semantic CSS.<p>Think of it as a lightweight, well defined query builder on top of SQL that you use for safe parameter binding and composability. (The analogy leaks a bit but you get the drift)')
