Item(by='mratsim', descendants=None, kids=[25359799], score=None, time=1607520872, title=None, item_type='comment', url=None, parent=25352253, text='Swift leaves it in the hand of programmers and says &quot;use weak references&quot;.<p>Otherwise traditional GC use a copying mechanism, what doesn&#x27;t end up copied is dead.<p>The old reference counting GC in Nim had that extra copying pass when a type could have cycles.<p>That said, it&#x27;s very easy to tell if a type can have cycles or not at compile-time, you just need to check if it refers to itself i.e. does the Node type has a Node field (or a field that can recursively contain a Node field).')
