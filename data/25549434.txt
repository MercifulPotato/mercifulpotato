Item(by='ogre_codes', descendants=None, kids=None, score=None, time=1609053135, title=None, item_type='comment', url=None, parent=25547845, text='A MacOS App is just what you need to run the app on MacOS. A Docker Image is essentially a mini operating system in a can. Most Docker images contain shells, the entire Python install, an init sequence... piles and piles of redundant stuff.<p>If you are running a Docker image on MacOS or Windows, you first have to start docker which is itself a Linux Virtual machine.<p>Docker is a great dev tool, but if you don&#x27;t need it, it&#x27;s a ton of bloat.')
