They have some good starter docs here:<p><a href="https:&#x2F;&#x2F;docs.mapbox.com&#x2F;mapbox-gl-js&#x2F;example&#x2F;simple-map&#x2F;" rel="nofollow">https:&#x2F;&#x2F;docs.mapbox.com&#x2F;mapbox-gl-js&#x2F;example&#x2F;simple-map&#x2F;</a><p>You can learn a lot by signing up for an API key from Mapbox and tweaking their example code. They even provide online editors so you don&#x27;t even need to work with an HTML file locally (though I&#x27;d still recommend that for permanence). No React, no Webpack needed!')
