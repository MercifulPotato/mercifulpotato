Item(by='nickjj', descendants=None, kids=[25535809, 25535815], score=None, time=1608893577, title=None, item_type='comment', url=None, parent=25534258, text='I&#x27;m surprised Dexpot[0] isn&#x27;t listed. It&#x27;s a really great virtual desktop management tool and has a number of things included that&#x27;s not included with the Windows 10 default virtual desktop feature.<p>Then there&#x27;s VcXrv[1] to share your clipboard between WSL and Windows, it also lets you run Linux GUI tools in WSL. Essential IMO, if only for clipboard sharing.<p>Sizer[2] is another great tool for quickly resizing and positioning windows programmatically. Amazingly useful for spinning up specific sets of apps quickly.<p>IrfanView[3] is IMO one of the best quick image viewers &#x2F; editors. It also has handy dev features (batch processing, quickly getting a hex value from an image, etc.).<p>I keep a list of dev related tools I use on Windows too at <a href="https:&#x2F;&#x2F;nickjanetakis.com&#x2F;blog&#x2F;the-tools-i-use" rel="nofollow">https:&#x2F;&#x2F;nickjanetakis.com&#x2F;blog&#x2F;the-tools-i-use</a>.<p>[0]: <a href="https:&#x2F;&#x2F;dexpot.de&#x2F;?lang=en" rel="nofollow">https:&#x2F;&#x2F;dexpot.de&#x2F;?lang=en</a><p>[1]: <a href="https:&#x2F;&#x2F;sourceforge.net&#x2F;projects&#x2F;vcxsrv&#x2F;" rel="nofollow">https:&#x2F;&#x2F;sourceforge.net&#x2F;projects&#x2F;vcxsrv&#x2F;</a><p>[2]: <a href="http:&#x2F;&#x2F;www.brianapps.net&#x2F;sizer4&#x2F;" rel="nofollow">http:&#x2F;&#x2F;www.brianapps.net&#x2F;sizer4&#x2F;</a><p>[3]: <a href="https:&#x2F;&#x2F;www.irfanview.com&#x2F;" rel="nofollow">https:&#x2F;&#x2F;www.irfanview.com&#x2F;</a>')
