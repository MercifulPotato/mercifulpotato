Item(by='foldr', descendants=None, kids=None, score=None, time=1606243545, title=None, item_type='comment', url=None, parent=25201091, text='It&#x27;s easy to call C++ code from Rust but it&#x27;s often not trivial to wrap a C++ library with a safe Rust interface. C++ APIs often have implicit ownership structures that are difficult to encode using Rust&#x27;s lifetimes.<p>To be fair, that doesn&#x27;t put Rust in a worse position than any other language binding a C++ API. It&#x27;s just that Rustaceans tend to have higher expectations in terms of safety.')
