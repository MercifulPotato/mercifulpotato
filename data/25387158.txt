Item(by='gempain', descendants=None, kids=[25397859, 25392215], score=None, time=1607702330, title=None, item_type='comment', url=None, parent=25387157, text='Hi there !<p>We used to host our sites on Netlify, but our eyes fill with glitter when we hear open source and self-hosted. So, we built Meli, which essentially is a Netlify alternative that lets you deploy static sites and frontend applications with ease, featuring per-branch deployments, web&#x2F;slack&#x2F;mattermost&#x2F;email hooks, an API, and a way to manage organizations, teams and sites easily.<p>We built Meli on top of Caddy, a very powerful HTTP server (<a href="https:&#x2F;&#x2F;github.com&#x2F;caddyserver&#x2F;caddy" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;caddyserver&#x2F;caddy</a>). We&#x27;ve used Typescript, Node (backend), React (frontend) and MongoDB for the database.<p>It&#x27;s a beta, but you can install super easily with Docker Compose: <a href="https:&#x2F;&#x2F;docs.meli.sh&#x2F;get-started&#x2F;installation" rel="nofollow">https:&#x2F;&#x2F;docs.meli.sh&#x2F;get-started&#x2F;installation</a><p>Check us out at <a href="https:&#x2F;&#x2F;github.com&#x2F;getmeli&#x2F;meli" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;getmeli&#x2F;meli</a> !<p>Looking forward for your feedback :)')
