The 21x21 QR has an error correction of &quot;L&quot;. That means up to 7% of the data area can be obscured. Realistically, as the code is a square, you could only lose data from the right hand side while maintaining integrity.<p>A QR with &quot;H&quot; error correction can lose about 30% - but the code takes up a lot more pixels.')
