Item(by='l0b0', descendants=None, kids=None, score=None, time=1609968710, title=None, item_type='comment', url=None, parent=25643940, text='Some acceptance tests for a software engineer:<p>- Should know many tools to the level where they know their strengths and weaknesses. This allows them to select a useful collection of them for any new piece of work.<p>- Should produce incremental project changes which all either improve the value or reduce the cost.<p>- Should <i>balance</i> all the improvements to produce useful, working software with a reasonable cost. For example, producing super-secure software with no useful features, featureful software with a bajillion bugs, or bug-free software with a gigantic cost are all bad.')
