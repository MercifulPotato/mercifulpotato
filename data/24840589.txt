Item(by='Arathorn', descendants=None, kids=None, score=None, time=1603217561, title=None, item_type='comment', url=None, parent=24839895, text='The problem is that you wouldn&#x27;t have to &#x27;hack the server&#x27;, at least for Matrix, given anyone can run a server - so all it would take is for one server in the room to be vulnerable to a social or technical attack (e.g. a nosey sysadmin).<p>If an escrow public key has been mixed into the e2ee encryption, then all it takes is for the private key to be leaked (e.g. for a price on a dark market) and the nosey sysadmin can go and break the encryption of all its users.<p>The same goes for a centralised service too (c.f. the twitter hack) - while it might be less likely given the smaller attack envelope, it&#x27;s a much bigger prize.')
