Item(by='jniedrauer', descendants=None, kids=None, score=None, time=1604431126, title=None, item_type='comment', url=None, parent=24982587, text='There is some value in unit tests for plumbing. Something I see a lot is a set of values being transformed across several domains, losing data at each step. A very simple test that inputs values on one end of the pipe, and checks the values on the other end of the pipe, makes it very easy to be sure that you didn&#x27;t drop something.')
