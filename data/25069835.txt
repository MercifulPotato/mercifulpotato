Item(by='pavlo', descendants=None, kids=[25069896, 25070342, 25071467], score=None, time=1605189341, title=None, item_type='comment', url=None, parent=25069350, text='This is a great looking tool.<p>What I have been doing by hand for some time is putting code for different customers in different directories and having a conditional in `~&#x2F;.gitconfig` to determine what config applies there:<p><pre><code>    [includeIf &quot;gitdir:~&#x2F;projects-private&#x2F;**&quot;]\n      path = .&#x2F;.gitconfig-private\n\n    [includeIf &quot;gitdir:~&#x2F;projects-client&#x2F;**&quot;]\n      path = .&#x2F;.gitconfig-work\n</code></pre>\nThen in .gitconfig-private or .gitconfig-work I have all the usual gitconfig settings that apply, for example the [user] section...<p>Switching to the right directory thus automatically changes the settings.')
