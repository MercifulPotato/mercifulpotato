Item(by='pkolaczk', descendants=None, kids=None, score=None, time=1611734726, title=None, item_type='comment', url=None, parent=25925448, text='If you are forced to write pure C (or a very low level subset of C++) , then that is often because:<p>- you need every tiny bit of performance,<p>- you optimize for low executable size or low memory footprint (e.g ATTiny13)<p>- you have an exotic architecture that offers only a C compiler or ancient C++ compiler from 90s,<p>- you need to use C for other reasons like safety certification,  e.g. MISRA C.<p>In all of these cases Python wouldn&#x27;t work at all.<p>And in the others your can use modern C++ which can be just as productive as Python (in my case it is more productive, but I have probably spent more time with it than with Python, so I&#x27;m biased).')
