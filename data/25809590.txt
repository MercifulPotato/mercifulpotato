Item(by='juniperplant', descendants=None, kids=None, score=None, time=1610878614, title=None, item_type='comment', url=None, parent=25808827, text='SSB = Single Site Browser<p>PWA = Progressive Web App<p>SSB refers to the ability to run a website inside its own window, like a native application. Also, it is expected to integrate into the desktop environment, with its own icon&#x2F;shortcut.<p>PWA is more an umbrella term for websites using various technologies to close the gap between web and native applications (e.g. the ability to run offline). I would say it includes SSB (at least on desktop), among other things.')
