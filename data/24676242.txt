Item(by='karmakaze', descendants=None, kids=[24676628], score=None, time=1601780777, title=None, item_type='comment', url=None, parent=24674490, text='I see two problems with this for my use.<p>&quot;The format holds dictionaries (ordered collections of name&#x2F;value pairs), lists (ordered collections of values) and strings (text) organized hierarchically to any depth.&quot;<p>1. Ordered dictionaries are not conveniently supported in all languages I tend to use.<p>2. The only element type is string which means that parsing of common types has to be separately and differently by each implementation that uses such a file with potential for unspecified differences.')
