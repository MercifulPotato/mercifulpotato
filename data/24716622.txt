Item(by='tgv', descendants=2, kids=[24719409], score=10, time=1602140546, title='Docker on macOS much more efficient now', item_type='story', url=None, parent=None, text='I hadn&#x27;t been using docker on my mac for a few weeks, and when I fired up the old containers this morning after updating Docker, I found that it no longer had one CPU going at 100% while my processes were idling as it did before, and some very high CPU usage during more demanding tasks. It now seems to behave like it does on Linux.<p>I had given up on docker on my mac, and instead ran it inside an Ubuntu VM on my mac, but this is much better. If you had given up on it too, take another look.')
