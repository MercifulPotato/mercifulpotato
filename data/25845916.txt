Item(by='justsomeuser', descendants=None, kids=[25846315], score=None, time=1611147868, title=None, item_type='comment', url=None, parent=25844731, text='CDNs allow serving files closest to the user cutting out 100s of ms of latency.<p>But they are limited to read only files.<p>Cloud flare workers allow running simple programs at the CDN node closest to the user.<p>SQLite is embeddable C code that is like a normal SQL server just without the network parts - just the sync functions embedded into your process.<p>This wraps SQLite with the networking and CDN abilities of cloud flare, allowing you to http-get a subset of data dynamically using SQL (that would not be possible with the simple “download 100% of file X”).')
