Item(by='alphachloride', descendants=None, kids=None, score=None, time=1602566846, title=None, item_type='comment', url=None, parent=24752348, text='PyOxidizer generates standalone binaries for python.<p>It requires the rust (and optionally C) toolchain to be installed first.<p>I think requiring another language development environment set up to package code is a big turn-off. A friendlier approach is either to have a package in the same language so it interpolates with the ecosystem, or have a standalone binary which doesn&#x27;t need dependencies.<p>At this point, I look at it as a python distribution tool for rust developers.')
