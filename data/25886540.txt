Item(by='int_19h', descendants=None, kids=None, score=None, time=1611439281, title=None, item_type='comment', url=None, parent=25886064, text='JNI is a bit different - you have to write code in C or C++ that wraps native APIs you want to call, in a certain very specific way that then makes it possible to call it via methods declared as &quot;native&quot; in Java.<p>P&#x2F;Invoke does not require such a layer - you simply declare enums, structs, and functions from the native API directly in your C# code, and then you can call them.<p>For those familiar with Python, the Java way is more like writing a Python extension module in C, and the C# way is more like using ctypes (but way faster).<p>Java has some third party libraries that are basically P&#x2F;Invoke equivalent; JNA is probably the most used one.')
