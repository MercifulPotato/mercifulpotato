Item(by='a1369209993', descendants=None, kids=[25044028, 25044073], score=None, time=1604946909, title=None, item_type='comment', url=None, parent=25036708, text='Actually they do perform (drastically cutailed) computation; for example INI performs computations of the form:<p><pre><code>  [foo]    # section = &quot;foo&quot;\n  bar=baz  # table[section][&quot;bar&quot;] = &quot;baz&quot;\n</code></pre>\nSimilarly, wc is a programming language with ~4 operations:<p><pre><code>  &#x27;a&#x27; # inc byte counter, add word flag to word counter, clear word flag\n  # any other non-space&#x2F;newline is the same as &#x27;a&#x27;\n  &#x27; &#x27; # inc byte counter, mark word flag\n  \\n  # inc byte counter, mark word flag, inc line counter\n  EOF # (implied) print counters\n</code></pre>\nThat set of operations is of extremely limited usefulness, but that&#x27;s more or less true of any non-turing-complete language (and many that are turing-complete, even).')
