Shell scripts handle the &#x27;run this build&#x2F;test task&#x27; but not &#x27;pull these artifacts from another set of tasks, run this task to produce a bunch more artifacts, cache them for later.&#x27;<p>In these scenarios the lines between build system, package manager, and task runner all become pretty blurred.
