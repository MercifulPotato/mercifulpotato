Item(by='open-source-ux', descendants=None, kids=[24894165], score=None, time=1603702708, title=None, item_type='comment', url=None, parent=24893336, text='Affinity Designer is cross-platform (Mac and Windows) and is cheaper than Adobe (and not based on an online subscription) but it&#x27;s not focused wholly on UI design. It doesn&#x27;t match XD feature-by-feature but it can be used for UI app design. It has no prototyping features and is unlikely to have such a feature in the future.<p>It does have features like symbols that are essential for any UI design app (a symbol is a visual component that can be repeated throughout a design; editing one instance updates all instances).<p>However, the Affinity Designer audience skews more towards illustration and my impression is it isn&#x27;t used as much for UI design even though it has very capable tools to accomplish this. This means fewer tutorials and a small community using it solely for UI design.<p>Here&#x27;s an example tutorial on creating a mobile app UI using Affinity Design that shows what can be done in the app:<p><a href="https:&#x2F;&#x2F;www.youtube.com&#x2F;watch?v=WgaRAkohwMU" rel="nofollow">https:&#x2F;&#x2F;www.youtube.com&#x2F;watch?v=WgaRAkohwMU</a>')
