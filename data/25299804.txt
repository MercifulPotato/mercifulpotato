Item(by='BlueTemplar', descendants=None, kids=None, score=None, time=1607065735, title=None, item_type='comment', url=None, parent=25294704, text='One of the huge questions is : who takes responsibility for the issues with the black box learning systems ?<p>In France, since you have to be able to explain in plain words the algorithm that was used to help to make an administrative decision, it indirectly makes these black box learning systems illegal.<p>(And yet they probably already have been used by the police?)<p>Now, the naive response is that the engineer that implemented a system should take responsibility for any harm done by the system - and I guess that if these engineers could lose their license to use a computer and face jail time, they might also start to put pressure upstream on researchers to provide better systems ?')
