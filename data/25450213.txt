Item(by='thescriptkiddie', descendants=None, kids=None, score=None, time=1608161361, title=None, item_type='comment', url=None, parent=25449948, text='Qemu is only able to achieve native performance when running in conjunction with a hypervisor like KVM. Hypervisors don&#x27;t do binary translation, so the guest architecture needs to match the host architecture. Running x86_64 qemu under rosetta would likely be much slower than running aarch64 qemu, because it would be running an emulator inside of an emulator.')
