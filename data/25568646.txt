Item(by='elihu', descendants=None, kids=[25571317], score=None, time=1609239459, title=None, item_type='comment', url=None, parent=25566445, text='If the OS can run in a virtualized environment, you can kind of cheat by letting some other OS like Linux handle the hardware compatibility.  It&#x27;s not as aesthetically satisfying as running on bare metal, but it can work.<p>If some company wanted to create an Apple-like desktop experience with purpose-built hardware but with an open OS, this could be a good route: implement all the drivers needed to run well on bare metal on the company&#x27;s own hardware products, but anyone can run it in a hypervisor on any computer.')
