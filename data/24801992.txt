And you should be able to run D3.js in it.<p>Some browser compatibility layer will be required though.<p>The plan for Sciter.JS is to provide minimal DOM implementation to keep it lightweight.<p>For example there is no document.getElementsByTagName method but there is document.querySelectorAll.<p>So if that exact method is needed anyone can add:<p><pre><code>    Document.prototype.getElementsByTagName = function(tag) {\n      return  this.querySelectorAll(tag);\n    }\n</code></pre>\nThat&#x27;s the idea of Sciter.JS: engine with core functionality that can be configured to run browser and electron applications, just a matter of adequate compatibility layer.')
