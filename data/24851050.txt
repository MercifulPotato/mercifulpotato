From [1]:<p>&gt; If you commercially distribute binaries not accompanied with source code, the GPL says you must provide a written offer to distribute the source code later.<p>So if you don’t send the source along with the binaries then you need to make a written offer to your users. However, [2] states that:<p>&gt; If you make object code available on a network server, you have to provide the Corresponding Source on a network server as well.<p>So you can’t publish the binaries online but only distribute the source via physical media, for example.<p>[1] <a href="https:&#x2F;&#x2F;www.gnu.org&#x2F;licenses&#x2F;gpl-faq.html#WhatDoesWrittenOfferValid" rel="nofollow">https:&#x2F;&#x2F;www.gnu.org&#x2F;licenses&#x2F;gpl-faq.html#WhatDoesWrittenOff...</a><p>[2] <a href="https:&#x2F;&#x2F;www.gnu.org&#x2F;licenses&#x2F;gpl-faq.html#AnonFTPAndSendSources" rel="nofollow">https:&#x2F;&#x2F;www.gnu.org&#x2F;licenses&#x2F;gpl-faq.html#AnonFTPAndSendSour...</a>')
