Item(by='jcynix', descendants=None, kids=[25955357, 25949456, 25949644, 25948772], score=None, time=1611857282, title=None, item_type='comment', url=None, parent=25944866, text='Modern shells are powerful enough to help you remember, if you learn to configure them appropriately. My histories are always saved because each shell instance gets its own HISTFILE, like so:<p><pre><code>  export HISTFILE=$HOME&#x2F;.history&#x2F;${TTY##\\*&#x2F;}.$SHLVL\n</code></pre>\nAs I use different terminal windows for different tasks, this keeps history files rather concise thematically.<p>And I let the shell add timestamps too, so I can grep for entries produced during a certain time span:<p>zsh:<p><pre><code>  setopt EXTENDEDHISTORY # add timestamps\n</code></pre>\nbash:<p><pre><code>  HISTTIMEFORMAT=&quot;%F %T &quot;\n</code></pre>\nI write perl or shell script files, of course, if it&#x27;s more than some a handful of lines.')
