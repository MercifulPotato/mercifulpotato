If you are using wg-quick, the the script automatically adds interface level routes to the routing table that would override any manual static routes. You could add a Table=off flag to the wg interface config, and then you won&#x27;t have the automatically generated route table entries based on the AllowedIPs flags, and your manual routes will be respected.')
