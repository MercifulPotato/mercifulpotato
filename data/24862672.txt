Item(by='toast0', descendants=None, kids=[24864264], score=None, time=1603396725, title=None, item_type='comment', url=None, parent=24861110, text='Of course you&#x27;ll need to write a OS specific driver for critical hardware like interrupt controllers and system timers, but drivers for other things like network cards and disk controllers can be reused if you adopt the other OS&#x27;s interfaces. Things like ndiswrapper exist to use windows drivers on Linux or FreeBSD; Linux, FreeBSD, and NetBSD also have a large pool of drivers, which can be used depending on which system&#x27;s driver model you like best.')
