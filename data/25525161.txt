Plugins like <a href="https:&#x2F;&#x2F;datasette.io&#x2F;plugins&#x2F;datasette-upload-csvs" rel="nofollow">https:&#x2F;&#x2F;datasette.io&#x2F;plugins&#x2F;datasette-upload-csvs</a> can be used to let people add new tables to those existing databases, but there isn&#x27;t a mechanism to add a whole new database file without restarting the server.<p>In the next release I hope to be able to support pointing Datasette at a directory and having any new SQLite files that are added to that directory automatically show up in the interface without having to restart the server.<p>You can follow this issue for updates: <a href="https:&#x2F;&#x2F;github.com&#x2F;simonw&#x2F;datasette&#x2F;issues&#x2F;417" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;simonw&#x2F;datasette&#x2F;issues&#x2F;417</a>')
