Item(by='gitgud', descendants=None, kids=None, score=None, time=1603783841, title=None, item_type='comment', url=None, parent=24898016, text='A great web-scraping architecture is the pipeline model similar to 3D rendering pipelines. <i>|Stage 1|</i>: Render and HTML, <i>|Stage 2|</i> Save HTML to disk, <i>|Stage 3|</i>: Parse and translate HTML to whatever output you need; JSON, CSV etc...<p>It&#x27;s great if each of these processes can be invoked separately, so that after the HTML is saved, you don&#x27;t need to redownload it, unless the source has changed.<p>By dividing scraping into; rendering, caching and parsing you save your self a lot of web requests. This also helps prevent the website from triggering IP-blocking, DDOS protection and Rate-limiting.')
