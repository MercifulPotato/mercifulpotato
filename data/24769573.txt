Item(by='TheRealPomax', descendants=None, kids=None, score=None, time=1602618657, title=None, item_type='comment', url=None, parent=24767536, text='It doesn&#x27;t, it uses correct contextual capitalisation. &quot;De&quot; is the definite article in Dutch, and in a full name the correct capitalisation is written using lowercase (&quot;Hans de Zwart&quot;).<p>However, when used in isolation it needs to be capitalised so as to differentiate it from &quot;an incorrectly capitalised noun&#x2F;adjective following the definite article&quot;.<p>If English had surnames that started with &quot;the ...&quot;, the same rule would be applied, where &quot;Jason the Bouvier&quot; would be the correct full name spelling, but in isolated context, such as &quot;According to The Bouvier, this policy has [...]&quot;, capitalisation should be used to make it clear that this is expressly not the definite article.')
