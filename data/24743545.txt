Item(by='susam', descendants=None, kids=None, score=None, time=1602387422, title=None, item_type='comment', url=None, parent=24742518, text='I considered switching from MathJax to KaTeX for my MathJax-based projects such as <a href="https:&#x2F;&#x2F;github.com&#x2F;susam&#x2F;texme" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;susam&#x2F;texme</a> and <a href="http:&#x2F;&#x2F;mathb.in&#x2F;" rel="nofollow">http:&#x2F;&#x2F;mathb.in&#x2F;</a> but I had to decide against doing so because MathJax supports commands like \\begin{align}, \\begin{equation}, \\label, \\eqref, \\notag, etc. that KaTeX does not, so switching to KaTeX would break many existing documents written using TeXMe or MathBin. I am going to stick with MathJax for now.<p>Commands supported in MathJax: <a href="http:&#x2F;&#x2F;docs.mathjax.org&#x2F;en&#x2F;latest&#x2F;input&#x2F;tex&#x2F;macros&#x2F;" rel="nofollow">http:&#x2F;&#x2F;docs.mathjax.org&#x2F;en&#x2F;latest&#x2F;input&#x2F;tex&#x2F;macros&#x2F;</a><p>Commands supported in KaTeX: <a href="https:&#x2F;&#x2F;katex.org&#x2F;docs&#x2F;supported.html" rel="nofollow">https:&#x2F;&#x2F;katex.org&#x2F;docs&#x2F;supported.html</a>')
