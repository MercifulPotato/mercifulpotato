Item(by='markdown', descendants=None, kids=[25896445, 25896374, 25896326, 25896377, 25898284, 25899884, 25900702], score=None, time=1611523818, title=None, item_type='comment', url=None, parent=25896185, text='The problem with inlining all CSS is serving all the global styles all over again with every page load. You&#x27;re gaining a great first impression at the cost of a poorer experience for every subsequent page load.<p>Have you considered inlining CSS in the head (as you&#x27;ve done), but then serving it <i>again</i> with a linked css file just before &lt;&#x2F;body&gt;.<p>Then, with subsequent page loads (of the current or other pages), you don&#x27;t have to inline any CSS anymore.<p>Of course this requires that you serve two versions of all your pages, one for if that page is a first hit, and another to be served to users who already have your CSS cached.')
