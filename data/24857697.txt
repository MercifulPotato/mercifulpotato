Item(by='peterwwillis', descendants=None, kids=None, score=None, time=1603372860, title=None, item_type='comment', url=None, parent=24857342, text='This is why we need standard protocols and data formats for these common systems. If there were just a &quot;secrets protocol&quot; or &quot;secrets data format&quot;, any program could just implement it for input and output, along with a standard interface to perform the actions (a single URL for example). It used to be commonplace to just write an RFC for what you were doing and then other people would use the RFC. Which wasn&#x27;t perfect, but nothing ever is...<p>Instead, the standard solution today is custom integration, which leads to a lot of reinventing the wheel and incompatibility with extremely similar products.')
