Item(by='pdonis', descendants=None, kids=None, score=None, time=1607733886, title=None, item_type='comment', url=None, parent=25389867, text='<i>&gt;  All __slots__ does is mean that myObj doesn&#x27;t need its own dictionary, it still has to go to __class__ for a dictionary hit</i><p>In view of a follow-up comment about slots, this should be clarified: an class with __slots__ does <i>not</i> need to do a dictionary lookup for slot attributes; those are looked up by index into an array, just as local variables in a function are. The dictionary lookup is only done for attributes that aren&#x27;t listed in __slots__.')
