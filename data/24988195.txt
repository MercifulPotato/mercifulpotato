You are looking indeed for the thrush operator. I think it should be built into Scala&#x27;s standard library, but until that happens, you can use the mouse library or build it yourself. Here is an example:<p><pre><code>    &#x2F;&#x2F; Need to define this once somewhere in your project\n    implicit class TrushExtension[A](anything: A) {\n      def |&gt;[B](function: A =&gt; B) = function(anything)\n    }\n    \n    \n    &#x2F;&#x2F; Your application code\n    def fun(arr: Iterable[Int]) = arr.map(_.toDouble &#x2F; Math.PI)\n    \n    val arr = 0 to 100\n    val newArr = arr.map(_*2) |&gt; fun\n    \n    newArr.foreach(println)  &#x2F;&#x2F; prints 0, 0.63662, 1.27324, ...\n</code></pre>\nExecute or change the code here: <a href="https:&#x2F;&#x2F;scalafiddle.io&#x2F;sf&#x2F;WAKhZtJ&#x2F;0" rel="nofollow">https:&#x2F;&#x2F;scalafiddle.io&#x2F;sf&#x2F;WAKhZtJ&#x2F;0</a><p>This is maybe not exactly as convenient, but it comes pretty close.<p>&quot;someA.someB.someC&quot; becomes &quot;someA |&gt; someB |&gt; someC&quot;.')
