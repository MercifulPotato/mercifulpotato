Item(by='02020202', descendants=None, kids=None, score=None, time=1611060478, title=None, item_type='comment', url=None, parent=25820057, text='if there is mutex involved it will always be blocking in some way. the best way i think would be to package the client&#x27;s request with response channel. loop through incoming requests and pass them to workers(goroutines) which will send the response to individual client and close said response channel. then it is merely a decision on whether to go with N workers that will process the request channel or if each request will result in its own worker via goroutine.')
