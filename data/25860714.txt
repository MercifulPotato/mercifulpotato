Item(by='NohatCoder', descendants=None, kids=None, score=None, time=1611246858, title=None, item_type='comment', url=None, parent=25859338, text='For added easiness, outputting high-level language code instead of machine code is a &quot;cheat&quot; that is often good enough, and you can typically borrow a lot of the high level features with relatively little effort. Say you want to run a domain specific language in your program? Include V8, and write a compiler that turns the DSL into JavaScript. You get much better performance than a simple interpreter, sandboxing, and somewhat useful error messages.')
