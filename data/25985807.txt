Item(by='johnisgood', descendants=None, kids=[25986352], score=None, time=1612156985, title=None, item_type='comment', url=None, parent=25983630, text='This supports macOS only.<p>&gt; Requirements: at the moment, msync supports only macOS. This is because it uses macOS&#x27;s built-in afinfo tool to determine music files&#x27; bitrates.<p>You can simply use <i>file</i>[1] to get the bitrate on Linux if you have the metadata (if you lack the metadata, read below). I bet even macOS supports it, it is available on Unix, Unix-like, Plan 9, and IBM i. That, or you can use the <i>MediaInfo</i>[2] program or library, although I am not sure if it works with metadata or not.<p>Using <i>file</i>:<p><pre><code>   file foo.mp3 | sed &#x27;s&#x2F;.*, \\(.*\\)kbps.*&#x2F;\\1&#x2F;&#x27;| tr -d &quot; &quot;\n</code></pre>\nThe result is in <i>kbps</i>.<p>[1] <a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;File_(command)" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;File_(command)</a><p>[2] <a href="https:&#x2F;&#x2F;mediaarea.net&#x2F;en&#x2F;MediaInfo" rel="nofollow">https:&#x2F;&#x2F;mediaarea.net&#x2F;en&#x2F;MediaInfo</a>, <a href="https:&#x2F;&#x2F;mediaarea.net&#x2F;en&#x2F;MediaInfo&#x2F;Download" rel="nofollow">https:&#x2F;&#x2F;mediaarea.net&#x2F;en&#x2F;MediaInfo&#x2F;Download</a><p>You can download the CLI, GUI, or the library of <i>MediaInfo</i>. You want the CLI or the library.<p>---<p>The above solutions may work only if you have the metadata  (ID3, for example) set, although I am not sure about <i>MediaInfo</i>.<p>Another solution would be using <i>ffprobe</i>[3] (requires <i>ffmpeg</i>), which for example returns this for a FLAC file:<p><pre><code>  Metadata:\n    Comment         : Processed by SoX\n  Duration: 00:00:26.20, start: 0.000000, bitrate: 527 kb&#x2F;s\n    Stream #0:0: Audio: flac, 48000 Hz, stereo, s16\n</code></pre>\nIt should not be based on any metadata.<p>[3] <a href="https:&#x2F;&#x2F;ffmpeg.org&#x2F;ffprobe.html" rel="nofollow">https:&#x2F;&#x2F;ffmpeg.org&#x2F;ffprobe.html</a>')
