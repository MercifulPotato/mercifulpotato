Item(by='mjw1007', descendants=None, kids=None, score=None, time=1601739282, title=None, item_type='comment', url=None, parent=24672408, text='In addition to the issues other people have mentioned, Python benefits from a large standard library in a way that Rust doesn&#x27;t, because Python scripts don&#x27;t require a build step.<p>So it&#x27;s easier to use Python than Rust for a little bit of &quot;glue&quot; in a place where maintaining a pile of dependencies would be inconvenient.<p>If you can just stick a single .py file in a git repo somewhere and arrange for it to be run, that&#x27;s a big gain compared to having to maintain a list of dependencies and arrange for them to be installed and so on. So it makes a big difference that you have access to regular expressions and tar files and whatever in that situation.')
