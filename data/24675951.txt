Item(by='alphachloride', descendants=None, kids=None, score=None, time=1601774519, title=None, item_type='comment', url=None, parent=24671680, text='Hugo is a single executable - it doesn&#x27;t require any knowledge of Go itself. Granted, you have to be familiar with the templating syntax to add build-time logic to the website, but I believe that syntax is common across many other static site generators as well.<p>Yes, if you want to add logic in addition to the templating primitives provided by default, then being able to write code into the source pages will be of great help. However I think that when such a level of dynamism is required, then one can start debating the utility of a static generator in the first place.')
