Item(by='andylynch', descendants=None, kids=None, score=None, time=1612124998, title=None, item_type='comment', url=None, parent=25979444, text='It&#x27;s worth pointing out that with Oracle&#x27;s new Java licensing model whether something is free on charge depends on a number of things including how Oracle thinks it&#x27;s being deployed.<p>The Oracle JDK is indeed free of charge for developing aplications, <i>but</i> running that app using Oracle Java on a server as opposed to a desktop needs a paid license subscription &amp; generates exposure to Oracle license audits.<p>Many, many firms are now using OpenJDK and the like, and have policies against even downloading Oracle&#x27;s version.')
