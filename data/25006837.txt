Item(by='theshrike79', descendants=None, kids=[25007203, 25007218], score=None, time=1604661817, title=None, item_type='comment', url=None, parent=25006277, text='From the FAQ, before someone asks the obvious:<p>Why shouldn&#x27;t I just use jq?<p>jq is awesome, and a lot more powerful than gron, but with that power comes complexity. gron aims to make it easier to use the tools you already know, like grep and sed.<p>gron&#x27;s primary purpose is to make it easy to find the path to a value in a deeply nested JSON blob when you don&#x27;t already know the structure; much of jq&#x27;s power is unlocked only once you know that structure.')
