But that goes all the way down. If you don&#x27;t trust docker and the OS, run them in a VM. If you don&#x27;t trust the VM, run it on a spare computer. If you don&#x27;t trust that computer&#x27;s hardware, then use minecraft to make a PC with your own instruction set :) Same with networking. Use HTTPS. No? Also use a VPN. No? Make your own VPN. No?
