Item(by='Cyph0n', descendants=None, kids=None, score=None, time=1602343791, title=None, item_type='comment', url=None, parent=24739266, text='Not including headers in header files can be confusing.<p>Say you maintain a library (lib1.h) that exports an API that takes in an argument defined in lib2.h.<p>If you do not include lib2.h in lib1.h, then the users of your API need to include lib2.h <i>before</i> lib1.h in every source file and header. With a larger number of dependencies, this can become unwieldy for users.<p>In addition, future maintainers of your library may also find it confusing why things break when includes are reordered.')
