Item(by='jimbokun', descendants=None, kids=None, score=None, time=1603722113, title=None, item_type='comment', url=None, parent=24893821, text='I think a good candidate for macros is anything you would do with Annotations in Java or similar languages.  Macros are a much cleaner and easier to reason about way of doing code generation than whatever it is Java annotation processors do to generate code.<p>And in the same way you should think twice about introducing a new code of annotation to your program, you should think twice about adding a macro to a Lisp program.<p>But boy, when you really need it, it can really be a life saver.')
