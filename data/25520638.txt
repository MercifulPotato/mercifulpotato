Item(by='strenholme', descendants=None, kids=None, score=None, time=1608748976, title=None, item_type='comment', url=None, parent=25517322, text='Actually, the original HTML was <i>not</i> designed to have the end user agent determine how a page looks, and was designed to specify presentation.  &lt;FONT&gt; was not part of the original HTML specification.  &lt;TABLE&gt; was never meant to lay out a page.<p>The WYSIWYG-like tags were added in the late 1990s during the dot-com era because companies wanted to be able to make web pages “glossy brochures” and were doing things like making web pages huge imagemaps until Netscape added HTML tags specifying presentation.')
