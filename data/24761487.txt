Item(by='schoen', descendants=None, kids=None, score=None, time=1602553706, title=None, item_type='comment', url=None, parent=24760129, text='We&#x27;re interested in routing it on the public Internet, although I also know that some people are using it internally and that one of the two older public proposals about this suggested that it might be better as new private address space.<p>One way that I&#x27;ve been thinking about the reachability issue is by analogy to<p><a href="https:&#x2F;&#x2F;blog.cloudflare.com&#x2F;fixing-reachability-to-1-1-1-1-globally&#x2F;" rel="nofollow">https:&#x2F;&#x2F;blog.cloudflare.com&#x2F;fixing-reachability-to-1-1-1-1-g...</a><p>Many organizations used 1.1.1.1 as a test address (in addition to other uses of 1&#x2F;8), so once Cloudflare started working to get a public service up there, it had to perform various reachability tests and also set up a way for people to report problems. This is apparently an ongoing process to this day, but Cloudflare&#x27;s hard work helped make reachability that was initially patchy get asymptotically better. I think we can much do the same with other previously unassigned address ranges.')
