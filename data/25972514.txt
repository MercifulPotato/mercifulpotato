Item(by='xolve', descendants=None, kids=None, score=None, time=1612037463, title=None, item_type='comment', url=None, parent=25970690, text='I am using Retina display on which everything looks tiny. I struggled with setting up QT scaling, GTK3.0 scaling, icon sizes, font DPI and resolutions. The problem is there is no single setting which all UI libraries follow and this makes your life hell.<p>Even worse is when you connect an external monitor which is not HiDPI and now everything is bigger than usual.<p>Instead scale the whole display and this works perfectly. I have explained it here: <a href="https:&#x2F;&#x2F;np.reddit.com&#x2F;r&#x2F;linuxquestions&#x2F;comments&#x2F;4c2o4p&#x2F;what_does_panning_and_fb_in_xrandr_stand_for&#x2F;" rel="nofollow">https:&#x2F;&#x2F;np.reddit.com&#x2F;r&#x2F;linuxquestions&#x2F;comments&#x2F;4c2o4p&#x2F;what_...</a> (Except one hiccup, but its way better than tuning inidividual knobs)<p>I have added the command to &#x2F;etc&#x2F;X11&#x2F;xinit&#x2F;xinitrc so it scales automatically when I login:<p>xrandr --output eDP1 --scale 0.60x0.60')
