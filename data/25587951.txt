Item(by='nimih', descendants=None, kids=[25589988], score=None, time=1609373458, title=None, item_type='comment', url=None, parent=25585707, text='The general idea is that the discipline is automatically enforced via the type system, e.g. you have a library which provides a type FragileAndDangerousResource, but the only way to interact with such a thing is via (linear) functions along the lines of<p><pre><code>    withFADR :: (FragileAndDangerousResource #-&gt; a) -&gt; a\n    \n    addToFADR :: Int -&gt; FragileAndDangerousResource #-&gt; FragileAndDangerousResource\n</code></pre>\nIn other words, linear types are purportedly useful <i>precisely because</i> they make it reasonable to ask for that extreme discipline on the programmer&#x27;s part, because the compiler is right there to tell them when they screw it up--ideally with a helpful error message.')
