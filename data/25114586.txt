Item(by='DonaldPShimoda', descendants=None, kids=None, score=None, time=1605548490, title=None, item_type='comment', url=None, parent=25114457, text='It&#x27;s so frustrating that people immediately jump for &quot;forced&#x2F;planned obsolescence&quot; whenever something like this happens.<p>Forced obsolescence is the <i>deliberate</i> undermining of a device to ensure that it cannot be used in the future. That would be if Apple <i>intentionally</i> included code in Big Sur to cause older MacBooks to mysteriously brick.<p>There is a big difference between intentionally writing code like this versus not performing adequate Q&amp;A to ensure that such bug doesn&#x27;t exist, and only the former of these qualifies as planned obsolescence.')
