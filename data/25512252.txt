Item(by='wwweston', descendants=None, kids=None, score=None, time=1608676937, title=None, item_type='comment', url=None, parent=25509577, text='The interesting thing is that if you don&#x27;t think of the browser as just another runtime, nothing more than The VM That Lived (where applets and flash died), but actually think of your applications as Web Applications, then you get the ideas behind this faster.<p>JSON is just a media type that a resource can be rendered as. HTML is another media type for the same resource. Which is better? Neither, necessarily, it depends on the client application. But if you are primarily using JSON to drive updates to custom client code to push to HTML, well, that should give you something to think about.')
