Item(by='yongjik', descendants=None, kids=[25198652], score=None, time=1606201686, title=None, item_type='comment', url=None, parent=25193003, text='An interesting side effect is that naive use of std::vector::reserve can turn an O(N) algorithm into O(N^2).  If you just do push_back() N times, it&#x27;s guaranteed O(N).  If you have a function that inserts 10 elements, and prefix it with reserve(v.size() + 10), then it will only reserve ten more slots - if you keep calling it, the result is O(N^2).')
