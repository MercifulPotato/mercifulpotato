Item(by='toast0', descendants=None, kids=None, score=None, time=1611695522, title=None, item_type='comment', url=None, parent=25920817, text='Layer 4 load balancing can be a huge reduction in work for the load balancer; especially if it&#x27;s in a Direct Server Return configuration, where the load balancer only sees incoming packets, and response packets go directly from the server.<p>The downside is you lose any ability to balance based on details of the application protocol, it requires some specific network setup, and it&#x27;s hard to find a DSR load balancer in managed hosting or cloud. I&#x27;m not sure if there&#x27;s off the shelf software to manage DSR either (the basic pieces are there in most firewalls, but management isn&#x27;t)')
