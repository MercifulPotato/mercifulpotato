Item(by='aduros', descendants=None, kids=None, score=None, time=1611601785, title=None, item_type='comment', url=None, parent=25905447, text='pjs is for filtering and transformation like awk&#x2F;sed&#x2F;grep, but it supports many different text-based formats and its programs are written in plain JavaScript.<p>Here&#x27;s an example that converts a file to uppercase:<p><pre><code>    cat file.txt | pjs &#x27;_.toUpperCase()&#x27;\n</code></pre>\nIt can also scrape websites using CSS selectors:<p><pre><code>    curl https:&#x2F;&#x2F;aduros.com | pjs --html &#x27;h1,h2&#x27; &#x27;_.text&#x27;\n</code></pre>\nAnd JSON using jq filters:<p><pre><code>    cat database.json | pjs --json &#x27;.users[]&#x27; &#x27;_.age &gt;= 21&#x27;\n</code></pre>\nLots more examples on the README&#x2F;man page. It&#x27;s still quite new and may have bugs, if you find one let me know!')
