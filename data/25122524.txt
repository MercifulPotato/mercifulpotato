Item(by='wscott', descendants=None, kids=None, score=None, time=1605612764, title=None, item_type='comment', url=None, parent=25121416, text='Part of the problem with git is that too much of the metadata around a repository (config, heads, remotes, branches, tracking etc) is stored in separate little files somewhere in &#x27;.git&#x27;  This critical state of the repository should itself be version controlled so changes can be examined and rolled back in a clean consistent manor.  Git has a reflog so experts can go dig around for the stuff you lost. Imagine if git were more like &#x27;nix&#x27;. (that last statement might not help my argument for some of you ;-)<p>Read the various stores of git blunders in this thread with the idea in our head about how it would be different with this change.')
