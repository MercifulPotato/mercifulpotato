Item(by='throwaway09223', descendants=None, kids=None, score=None, time=1610389791, title=None, item_type='comment', url=None, parent=25730145, text='Not transmitting the password is good practice for the same reason it&#x27;s good practice to not store passwords cleartext: systems can be compromised.<p>I trust we all agree that storing cleartext passwords in a database and doing a simple string compare is a problem so I won&#x27;t rehash that bit.<p>If a login server is compromised then attackers can harvest cleartext passwords. It&#x27;s the same class of problem with a  reduced attack surface.<p>There is no good reason to transmit a persistent authentication secret as part of authentication. Just don&#x27;t do it.')
