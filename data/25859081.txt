Item(by='analog31', descendants=None, kids=None, score=None, time=1611239064, title=None, item_type='comment', url=None, parent=25858214, text='I would actually venture so far as to say that the purpose is to <i>not</i> run an operating system. Programming i&#x2F;o intensive stuff that has to work in real time, on a computer with an OS can be hard, especially for a beginner. The OS &quot;gets in the way&quot; if you will.<p>A useful approach for applications that need an OS is to run the OS on a regular computer, and connect it to a microcontroller that handles your custom i&#x2F;o tasks. Connect them via USB or wireless.')
