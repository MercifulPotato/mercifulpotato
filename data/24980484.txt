Item(by='imtringued', descendants=None, kids=[24980828], score=None, time=1604417322, title=None, item_type='comment', url=None, parent=24979552, text='It&#x27;s pretty much common knowledge that if you want linear performance scaling on a multiprocessor then each workload has to be effectively independent from the others. Synchronization and communication is expensive and only takes away from your per core performance but it never makes a given core faster. So yes, sharding is a critical part of this &quot;thread per core&quot; architecture.')
