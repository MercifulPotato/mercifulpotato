Item(by='creata', descendants=None, kids=None, score=None, time=1602169848, title=None, item_type='comment', url=None, parent=24719269, text='The main way to make the compiler more forgiving is to use Rc&lt;RefCell&lt;T&gt;&gt; (and Arc&lt;Mutex&lt;T&gt;&gt; when you need thread-safety) to move borrow checking from compile time to run time. Please avoid doing that, though --- it&#x27;s ugly, it can be slow, and the borrow checker is usually yelling at you for a good reason.<p>There are also some immature crates for tracing garbage collection in Rust, like rust-gc, which typically has higher throughput than reference counting.')
