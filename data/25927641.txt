Item(by='unsigner', descendants=None, kids=None, score=None, time=1611752264, title=None, item_type='comment', url=None, parent=25926090, text='Zstd is very different - it includes an entropy coder.\nLZ4 only finds repeated matches, but then doesn&#x27;t encode them very efficiently.<p>To put it simplistically, if you have a file which is a (good) random mix of an equal number A and B characters, LZ4 won&#x27;t be able to compress it significantly, while Zstd will compress it 8:1 converging to an encoding where a &#x27;1&#x27; bit is A, and a &#x27;0&#x27; bit is B.')
