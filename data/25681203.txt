Item(by='Wowfunhappy', descendants=None, kids=None, score=None, time=1610076699, title=None, item_type='comment', url=None, parent=25680814, text='I was running into similar problems running OS X 10.9. What I eventually did was set up a proxy server which MiTMs all https traffic and re-encrypts it. This <i>totally</i> worked, suddenly reviving all sorts of apps that I&#x27;d thought were broken forever.<p>Since other people were also having this problem, I ultimately put together a nice little installer. It&#x27;s very Mac-focused, but if you have access to a Mac, you could extract the pkg to see how Squid (the proxy I use) is configured: <a href="https:&#x2F;&#x2F;jonathanalland.com&#x2F;downloads&#x2F;wowfunhappy-https-proxy.dmg" rel="nofollow">https:&#x2F;&#x2F;jonathanalland.com&#x2F;downloads&#x2F;wowfunhappy-https-proxy...</a><p>It&#x27;s probably not worth the effort to set up something like this for your parents, but I do think these sorts of proxies are a good (if imperfect) solution for old systems. You could plausibly set them up at the router level too.')
