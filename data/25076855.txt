Item(by='ben509', descendants=None, kids=None, score=None, time=1605224381, title=None, item_type='comment', url=None, parent=25076397, text='You could absolutely use a simple certificate revocation list instead of OCSP. I don&#x27;t know how large that would be, though. It could run into problems if there was a heartbleed like issue that required revoking many certs.<p>All the extra connections are enough of an issue that there&#x27;s OCSP stapling, where a web server attaches a copy of the OCSP check to the response.<p>Seems like it&#x27;d be possible to inject a file into Cool.app&#x2F;Contents&#x2F;ocsp.staple in a downloaded .dmg.<p>That could be considered valid for a few days so that, for the common case of &quot;download app and try it out&quot;, there&#x27;s no need to phone home.')
