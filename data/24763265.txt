Reasons for that would be customization, integration, and data handling issues. For me the go-to system would be Gecode, but there are a lot of very interesting and varied solvers available.<p>Worth noting is that MiniZinc is starting to get more and more support for integration into large systems, for example with the python interface (<a href="https:&#x2F;&#x2F;pypi.org&#x2F;project&#x2F;minizinc&#x2F;" rel="nofollow">https:&#x2F;&#x2F;pypi.org&#x2F;project&#x2F;minizinc&#x2F;</a>).')
