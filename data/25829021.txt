Item(by='TheDong', descendants=None, kids=[25829125, 25829173, 25830393, 25829074, 25830140], score=None, time=1611020784, title=None, item_type='comment', url=None, parent=25809640, text='Maybe I&#x27;m misunderstanding the PsyStar case, but this seems likely to be illegal.<p>Psystar, among other things, released &quot;Rebel EFI&quot; (<a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Psystar_Corporation#Rebel_EFI" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Psystar_Corporation#Rebel_EFI</a>), which was a way to circumvent OSX&#x27;s checks that you were running on mac hardware.<p>Judge Alsup ruled that Rebel EFI was a circumvention technology, and thus not legal under the DMCAs anti-circumvention provisions.<p>The snap is not running macOS on mac hardware. Even if you run it on a mac laptop, it&#x27;s using virtualized qemu hardware. It includes a magic hardware string to make macOS run (<a href="https:&#x2F;&#x2F;github.com&#x2F;popey&#x2F;sosumi-snap&#x2F;blob&#x2F;def1652e916f&#x2F;snap&#x2F;local&#x2F;bin&#x2F;launch#L3" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;popey&#x2F;sosumi-snap&#x2F;blob&#x2F;def1652e916f&#x2F;snap&#x2F;...</a>). That magic string is a circumvention measure which dodges a check macOS makes on boot to see if it&#x27;s on legitimate hardware, and so I think this project is performing circumvention, similar to Rebel EFI.')
