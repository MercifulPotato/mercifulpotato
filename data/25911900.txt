Item(by='souprock', descendants=None, kids=[25912951], score=None, time=1611629941, title=None, item_type='comment', url=None, parent=25910496, text='My memory of GNU autotools and friends, from when I shipped one release of a project with it:<p>* the size of my project doubled<p>* naturally, autotools would check for FORTRAN despite my project being in C<p>* if users built the software as root, it would trash critical system files<p>* instead of installing ELF library files in &#x2F;lib, it installed non-functional shell scripts in &#x2F;usr&#x2F;lib<p>* cross compiling broke<p>The solution was plain old make. You can do a lot with make. It performs really well. I even did a &quot;make install&quot; that would depend on the final locations for everything. It was all configurable. It worked great.')
