Item(by='theevilsharpie', descendants=None, kids=None, score=None, time=1609697146, title=None, item_type='comment', url=None, parent=25623001, text='On Windows, to check if ECC is working, run the command &#x27;wmic memphysical get memoryerrorcorrection&#x27;:<p><pre><code>    PC C:\\&gt; wmic memphysical get memoryerrorcorrection\n    MemoryErrorCorrection\n    6\n</code></pre>\nSuperUser has a convenient decoder[1], but modern systems will report &quot;6&quot; here if ECC is working.<p>When Windows detects a memory error, it will record it in the system event log, under the WHEA source.
