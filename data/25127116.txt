Item(by='usefulcat', descendants=None, kids=None, score=None, time=1605636027, title=None, item_type='comment', url=None, parent=25126982, text='Pretty sure this is already a thing for NUMA systems, e.g. an Intel system with a pair of N-core processors. Each processor gets &quot;its own&quot; half of memory. Memory which belongs to the other processor is slower to access.<p>See also Linux cpusets (cset command), which can be used to control which NUMA nodes a process has access to.')
