Item(by='flubert', descendants=None, kids=None, score=None, time=1609189236, title=None, item_type='comment', url=None, parent=25559503, text='How about C?<p><a href="https:&#x2F;&#x2F;github.com&#x2F;vsedach&#x2F;Vacietis" rel="nofollow">https:&#x2F;&#x2F;github.com&#x2F;vsedach&#x2F;Vacietis</a><p>================<p>Vacietis is a C compiler for Common Lisp systems.<p>Vacietis works by loading C code into a Common Lisp runtime as though\nit were Lisp code, where it can then be compiled or evaled. The loaded\nC code has the same function calling convention as regular CL code and\nuses the same numerical representations. C memory is backed by regular\nCommon Lisp arrays.<p>Vacietis comes with a libc implemented in portable Common Lisp.')
