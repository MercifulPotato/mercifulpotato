Item(by='cxr', descendants=None, kids=None, score=None, time=1607110793, title=None, item_type='comment', url=None, parent=25304746, text='There&#x27;s nothing inherent to Web apps that&#x27;s forcing the hand of developers to send user data to a server outside the user&#x27;s control (i.e. opting not to give the choice of sending it to a server the user does control). Keeping user data on servers that the Web app developer controls is a decision that developers make because control over user data is usually an adversarial conflict, and when given to power to do developers go for the choice that works out to their advantage.')
