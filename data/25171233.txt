Item(by='Conan_Kudo', descendants=None, kids=None, score=None, time=1605980904, title=None, item_type='comment', url=None, parent=25170949, text='The current RPM repository metadata format has existed since Fedora Core 2 in 2004. The build system infrastructure changed to Koji in 2007[1]. The development model has changed exactly <i>once</i> when Fedora switched from CVS to Git in 2009[2].<p>There have been no significant changes to Fedora packaging model until three years ago, when Modularity was introduced[3] and Pagure was deployed to ease contributions and support building modules[4]. And the modularity concept is primarily used for alternate software streams in Fedora, so the vast majority of Fedora packages don&#x27;t use this feature.<p>[1]: <a href="https:&#x2F;&#x2F;fedoraproject.org&#x2F;wiki&#x2F;Releases&#x2F;FeatureNewBuildSystem" rel="nofollow">https:&#x2F;&#x2F;fedoraproject.org&#x2F;wiki&#x2F;Releases&#x2F;FeatureNewBuildSyste...</a><p>[2]: <a href="https:&#x2F;&#x2F;fedoraproject.org&#x2F;wiki&#x2F;Dist_Git_Proposal" rel="nofollow">https:&#x2F;&#x2F;fedoraproject.org&#x2F;wiki&#x2F;Dist_Git_Proposal</a><p>[3]: <a href="https:&#x2F;&#x2F;docs.pagure.org&#x2F;modularity&#x2F;" rel="nofollow">https:&#x2F;&#x2F;docs.pagure.org&#x2F;modularity&#x2F;</a><p>[4]: <a href="https:&#x2F;&#x2F;fedoraproject.org&#x2F;wiki&#x2F;Changes&#x2F;ArbitraryBranching" rel="nofollow">https:&#x2F;&#x2F;fedoraproject.org&#x2F;wiki&#x2F;Changes&#x2F;ArbitraryBranching</a>')
