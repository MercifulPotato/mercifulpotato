Item(by='rualca', descendants=None, kids=[25232851], score=None, time=1606459955, title=None, item_type='comment', url=None, parent=25224966, text='I would add that cmake is a makefile generator.<p>cmake&#x27;s value is in the way it provides a high-level abstraction over all supported platflom&#x27;s capabilities, and that it works by generating and running build scripts from lower level systems such as standard make.<p>For example, Cmake allows us to set the version of c++ a project uses with a single build target property, which is then used to pick which compiler&#x27;s flags it uses with a given compiler. It then generates the makefile with all the relevant flags set, and runs that with ease.<p>This works regardless of platform or compiler.')
