Item(by='simonbw', descendants=None, kids=None, score=None, time=1611956813, title=None, item_type='comment', url=None, parent=25962557, text='I think the QA process isn&#x27;t usually as pipelineable as you&#x27;re suggesting. On a website it&#x27;s often easy to QA changes that happen on page A at the same time you QA changes on page B. But in a game, especially one like factorio, it&#x27;s probably a lot harder to find features that can be tested and deployed simultaneously with other features.<p>It would be easy to test changes to the design of Level 3 at the same time as changes to the design of Level 5, but it would be hard to test changes to the way inserters work at the same time as testing changes to the way conveyor belts work.')
