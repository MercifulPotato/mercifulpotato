Item(by='ncmncm', descendants=None, kids=None, score=None, time=1606267832, title=None, item_type='comment', url=None, parent=25203690, text='Nobody seems to mention this when comparing, but code to handle the case of getting a bad error code is almost always very poorly exercised: therefore, buggy.<p>That is in contrast to the code run during an exception, the destructors, which run all the damn time.<p>If you do make the effort to test your error-handling code, that is a significant expense on top of the reduced performance. Maintaining those tests as the code changes from under them is another expense.')
