Item(by='geophile', descendants=None, kids=[24758850, 24760311, 24758536], score=None, time=1602528888, title=None, item_type='comment', url=None, parent=24744653, text='Another pythonic shell: <a href="https:&#x2F;&#x2F;marceltheshell.org" rel="nofollow">https:&#x2F;&#x2F;marceltheshell.org</a>. Unlike Xonsh, marcel provides a strict separation between shell syntax and Python syntax. Specifically, some shell commands take Python functions as arguments. E.g., in the current directory, find all the .c and .h files, and count lines:<p><pre><code>    ls -fr \\\n    | select (f: f.suffix in (&#x27;.c&#x27;, &#x27;.h&#x27;)) \\\n    | map (f: f.read().count(&#x27;\\n&#x27;)) \\\n    | red +\n</code></pre>\n- List only files (-f) recursively (-r).<p>- Select those whose suffix is .c or .h.<p>- Map qualifying files to the count of \\n in the file.<p>- Reduce those counts to a total using +.<p>The parentheses delimit Python functions (marcel permits omission of &quot;lambda&quot;).')
