Item(by='kazinator', descendants=None, kids=None, score=None, time=1606507846, title=None, item_type='comment', url=None, parent=25231370, text='IEEE 754 specifies some not-a-number values X that are not equal to themselves; i.e. such that X != X.<p>The bit pattern is actually not equal to itself: if we compared X bitwise, like with memcmp in C, it would be equal: memcmp(&amp;X, &amp;X, sizeof X) == 0.<p>That obnoxiously violates the philosophical Law of Identity, as we would like to see it applied in programming languages.<p><a href="https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Law_of_identity" rel="nofollow">https:&#x2F;&#x2F;en.wikipedia.org&#x2F;wiki&#x2F;Law_of_identity</a>')
