Item(by='kazinator', descendants=None, kids=None, score=None, time=1602791054, title=None, item_type='comment', url=None, parent=24792493, text='In Common Lisp, methods are the bits of code which <i>specialize</i> a <i>generic function</i>.<p><i>defgeneric</i> most certainly defines a function. The symbol becomes <i>fbound</i> and can be used like function: (mapcar #&#x27;mygeneric ....) and so on.<p><i>defmethod</i> defines specializations for it. If you write a <i>defmethod</i> without a matching <i>defgeneric</i>, then the generic function gets implicitly defined.')
