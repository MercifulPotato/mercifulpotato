They focus on automating tasks either for linx developers or normal users. Here are some of the highlights:<p>* Suggestions-enabled search using googler on a variety of websites. For example:<p><pre><code>    Google\n    Amazon\n    Youtube\n    Github\n    Ebay\n    Imdb\n    Urban dictionary: Word&#x2F;Slang definitions lookup\n    Python, OpenCV, Dlib, C++ documentation lookup\n</code></pre>\n* image_search: Search, preview and directly copy-to-clipboard images from the web<p>* Words - Get the definition, synonyms and antonyms for a word<p>* Killproc - Kill process(es); allows killing multiple processes with the same name or based on a glob<p>* Timezones - Fuzzy-search timezones<p>* Colors - Lookup the RGB triplet or the hex code for the given color<p>* Jira - Issue Tracking<p>* Saxophone - Listen to internet Radio Streams<p>* Zoopla - Search Property to Buy, Rent, House Prices<p>* Xkcd - Fetch xkcd comics like a boss<p>* Taskwarrior - Interact with Taskwarrior<p>* Remmina - Search and start remmina connections easily<p>* Google Maps - Fetch instructions from&#x2F;to a specific place<p>* Google Translate<p>* TL;DR pages lookup - Get 2FA codes using totp-cli<p>* Lookup HTTP URL status codes<p>* Lookup errno status codes<p>* Show Internal, External IPs, Default Gateways')
